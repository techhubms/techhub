# Foundry IQ: The Future of Retrieval-Augmented Generation with Azure AI Search and Knowledge Retrieval

Agents need context to deliver relevant and actionable responses. This session, delivered by Microsoft Events at Ignite 2025, introduces Foundry IQ, which serves as a knowledge layer for AI agents, facilitating optimal connection of enterprise data to agent workflows.

## Core Topics

- **Multi-source RAG Orchestration**: Learn how Foundry IQ enables orchestration across diverse data sources for robust retrieval-augmented generation.
- **Retrieval Steering**: Techniques to guide and refine retrieval processes, particularly with integration into GitHub documentation and targeted issue searches.
- **Dynamic Security Controls**: Explore methods for securing knowledge bases, allowing for both flexibility and compliance in enterprise scenarios.
- **Agentic RAG**: How agents leverage Foundry IQ to connect and contextualize enterprise knowledge for improved application intelligence.
- **Active Indexing and Remote Sources**: Insight into accessing and indexing remote knowledge bases, utilizing Azure AI Search for seamless data retrieval.
- **Enhanced Data Indexing with Azure Content Understanding**: Leverage Azure's content analysis features to improve data quality and retrieval precision.
- **Evaluation Methodology**: Practical approaches to assessing improved context, security, and effectiveness of AI-driven applications.

## Session Breakdown

- **Foundry IQ Technician Agent Demonstration**: Practical example showing the use of Foundry IQ with Azure AI Search.
- **Enterprise Integration**: How Foundry IQ links enterprise software issues to contextual resources such as GitHub documentation.
- **Security & Scaling**: Growing needs for dynamic controls as agent adoption increases.
