# Data Protection in the Age of the Adversary: Accelerating Microsoft Purview Adoption

**Speakers:**

- Caleb McDowell
- Quentin Rhoads-Herrera

## Overview

Cyber adversaries are constantly evolving, leveraging vast amounts of data and advanced AI techniques to launch increasingly sophisticated attacks. This Microsoft Ignite 2025 session spotlights real-world threat stories and explores cutting-edge defensive measures, focusing on the strategic adoption of Microsoft Purview for sensitive data protection.

## Session Highlights

- **The Shifting Threat Landscape:**
  - Exploration of how adversarial tactics have changed with advancements in AI.
  - Specific example: misuse of Apple’s AI protocol to bypass security tools.

- **Defensive AI and Security Copilot:**
  - Discussion of applying AI technologies for cyber defense and the pivotal role of Security Copilot in data discovery and response automation.
  - Emphasizing a secure AI implementation framework.

- **Data Governance as a Security Imperative:**
  - Rise of data governance as a core component of modern security strategies.
  - Practical steps: tracking and tagging documents for enhanced visibility and turning on auditing features to enforce data governance and collaboration.

- **Four-Phase Adoption Approach for Microsoft Purview:**
  1. **Assessment:** Identify sensitive data and assess risks.
  2. **Planning:** Map out governance and compliance needs.
  3. **Implementation:** Roll out Microsoft Purview capabilities for classification, protection, and monitoring.
  4. **Continuous Improvement:** Regularly update processes to keep pace with evolving threats and regulatory demands.

## Recommendations

- Stay informed on the latest tools and threat vectors.
- Foster a culture of continuous learning in security teams.
- Leverage integrated Microsoft solutions—such as Security Copilot and Purview—to create resilient, compliant, and secure data environments.
