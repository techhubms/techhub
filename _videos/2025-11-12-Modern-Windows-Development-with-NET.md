# Modern Windows Development with .NET

Developing for Windows has advanced significantly with the introduction of .NET 10 and new frameworks. This session explores how WinUI 3, Windows App SDK, and .NET 10 empower developers to build Windows desktop applications with modern features and native performance.

## Key Topics Covered

- **WinUI 3** for modern, native Windows UI
- **Windows App SDK** for accessing new Windows platform capabilities
- **.NET 10** as a unified, intelligent development platform
- Integrating **AI** features into desktop apps
- Delivering **native, performant Windows experiences**
- Approaches to **modernizing existing applications** or starting new ones
- **.NET Aspire** for cloud-native orchestration
- **ASP.NET Core 10** for secure, full-stack development
- Cross-platform UI with **.NET MAUI 10**
- **C# 14** and **F# 10** for expressive, productive development
- Improvements and new features in **Visual Studio 2026**
