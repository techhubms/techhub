# Azure Migrate: AI Insights for Strategic Cloud Transformations

## Session Overview

This advanced Microsoft Ignite 2025 session provides actionable guidance for IT teams looking to migrate and modernize their IT estates using Azure Migrate and its newest AI capabilities.

### Key Topics

- **Workload Assessment & Inventory**: Learn how to comprehensively assess your current environment, inventory all workload types (Windows Server, SQL Server, .NET, Linux, PostgreSQL, Java), and map interdependencies.
- **AI-Assisted Planning**: Discover how Azure Migrate’s AI enhancements—such as automated business case generation—provide cost savings analysis, modernization benefits, and ROI comparisons to build a strong migration case.
- **Azure Copilot Migration Agent Preview**: Get early insights into the new migration agent that uses AI for decision making and automation of migration waves.
- **Migration Roadmap**: Build actionable wave planning strategies using Azure Migrate tools, enabling coordinated and automated migration wave setups, including landing zone configuration recommendations.

## Session Chapters

- *Prioritizing Migration Workloads*: Practical advice for identifying and sequencing workloads for migration.
- *Modernization Successes*: Real-world examples of successful modernization leveraging agent-based automation.
- *Business Case & ROI Analysis*: Tools for generating and presenting ROI-driven business cases using the AI features of Azure Migrate.
- *Comparative Insights & TCO Analysis*: Using results and insights from Azure Migrate to compare modernization approaches and analyze Total Cost of Ownership (TCO).
- *Wave Planning & Automation*: Automated migration wave planning and landing zone configuration using agent expertise.
