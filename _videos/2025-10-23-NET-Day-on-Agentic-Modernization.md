# .NET Day on Agentic Modernization

.NET Day on Agentic Modernization showcases strategies and tools for accelerating the modernization of legacy applications, shipping more reliably, and building advanced agentic apps using .NET and Microsoft cloud technologies.

## Event Highlights

- **GitHub Copilot App Modernization:**
  - Learn how GitHub Copilot integration in Visual Studio streamlines .NET upgrades and Azure migration, accelerating the modernization process for legacy apps.
  - Explore automated code suggestions and refactoring support for upgrading projects.
- **Rehosting Hard-to-Change Apps:**
  - See techniques for rehosting legacy or complex applications on Azure App Service Managed Instance, simplifying migration and operations.
- **Boosting Reliability with Azure SRE Agent:**
  - Understand how Azure SRE Agent enhances reliability, automating error detection and remediation in cloud-hosted .NET workloads.
- **Building Agentic Experiences:**
  - Discover the Microsoft Agent Framework and Azure AI Foundry for creating agentic solutions within .NET, enabling intelligent app experiences.
- **GitHub Copilot for Azure (via Azure MCP):**
  - See how GitHub Copilot for Azure leverages Azure MCP to automate repetitive Azure development tasks, further boosting developer productivity and consistency.

## Key Technologies and Tools

- .NET Framework and Modernization Strategies
- GitHub Copilot and Visual Studio Integration
- Azure App Service Managed Instance
- Azure SRE Agent
- Microsoft Agent Framework
- Azure AI Foundry
- Azure MCP (Managed Control Plane)
