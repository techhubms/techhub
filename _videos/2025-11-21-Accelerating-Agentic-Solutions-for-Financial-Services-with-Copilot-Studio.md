# Accelerating Agentic Solutions for Financial Services with Copilot Studio

## Session Overview

This Microsoft Ignite breakout (BRK366) explores how financial services organizations are leveraging Copilot Studio to develop intelligent agents that address complex workplace and customer challenges.

**Key Topics Covered:**

- The role of contextual intelligence and 'Work IQ' in agent design
- Prominent AI use cases in banking, capital markets, and insurance
- Best practices in model selection, sub-agent usage, and optimization through evaluation loops
- Techniques for continuous optimization of agent performance
- Modular problem solving and the significance of sub-agents in scalable AI development

## Case Study: Edward Jones

- Introduction by Adrian Crockett, Jared Lambert, Hemanth Sundararaj, and Adam Wheat
- Edward Jones firm-wide rollout of tailored AI assistants for employee workflows
- Integration of Edward Jones data with Morningstar via the Model Context Protocol (MCP)
- Live demonstration: Generating diversification alert emails using connected AI agents

## Technical Best Practices

- How model selection affects agent outcomes in financial domains
- Deploying sub-agents for modular task execution
- Using evaluation loops to continuously enhance solution quality and performance

## Solution Impact

Participants can learn:

- How to build agentic solutions using Copilot Studio for financial use cases
- Integration strategies for connecting proprietary financial data into AI workflows
- Approaches for automating personalized communications and decision alerts with AI assistants
