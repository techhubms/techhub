# Managing and Auditing GitHub Copilot Agents: Insights and Governance Tools

GitHub introduces a suite of evolving tools to help administrators manage and audit GitHub Copilot agents. As Copilot capabilities expand, so has the need for robust governance and oversight mechanisms.

## Key Features Covered

- **Copilot Insights Dashboard:**
  - Provides an overview of Copilot usage within your organization, helping administrators understand how developers interact with Copilot agents.

- **Delegating Control:**
  - Enhanced controls for delegating administrative rights, ensuring the correct personnel can manage Copilot settings and resources.

- **Copilot Control Panel:**
  - Central location to customize agent access, manage user permissions, and review audit logs, streamlining the governance process.

- **Managing Custom Agents:**
  - Options for managing custom Copilot agents, allowing organizations to tailor AI assistance to their specific requirements.

- **Auditing Copilot Agent Activity:**
  - Comprehensive auditing features, including actionable logs of Copilot activity, support compliance, transparency, and traceability.

- **MCP Access Management:**
  - Tools to manage and control access to Managed Control Plane (MCP) elements related to Copilot agents.

## Administrative Benefits

- Improved visibility into Copilot adoption and utilization
- Streamlined delegation and access management
- Detailed audit logging for compliance and incident review
- Control over custom agent deployment and policy enforcement
