# Building Multi-Agent Systems with MCP and Copilot Studio

## Session Overview

This Microsoft Ignite 2025 session dives into advanced, scalable multi-agent functionality in Copilot Studio. Speakers Matthew Bolanos, Zankar Desai, Kendra Springer, and Ziv Ventura walk through key architectural concepts and demonstrate how to use Microsoft Copilot Studio's extensibility.

### Key Topics

- Overview of available MCP (Multi-agent Control Plane) servers
- Extending Copilot Studio with agents from Microsoft Foundry, Fabric Data Agents, M365 Agents SDK, and A2A
- Addressing configuration challenges for non-technical users
- How Copilot Studio agents flag and assign problematic results
- Adding subagents and external agents from outside Copilot Studio
- Demonstration of a Policy Expert Agent and responses to policy queries
- Protocols for enterprise integration, including agent communication and interoperability

### Demonstration Highlights

- Adding and managing subagents within Copilot Studio
- Integrating external and policy expert agents for advanced enterprise solutions
- Using Fabric Data Agents to pull enterprise-ready data into agent workflows
- Quick demonstration of MCP server configuration and troubleshooting

### Developer & Enterprise Insights

- Scaling low-code AI agent solutions
- Integrating with the M365 Agents SDK for custom agent development
- Best practices for protocol-based integration across Microsoft platforms
