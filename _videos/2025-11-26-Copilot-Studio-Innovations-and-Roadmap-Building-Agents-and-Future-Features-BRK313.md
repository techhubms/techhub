# Copilot Studio Innovations and Roadmap: Building Agents and Future Features (BRK313)

## Overview

This session—presented by Omar Aftab, Erez Altus, and Zac Cadi at Microsoft Ignite 2025—focuses on Copilot Studio’s evolution, highlighting recent enhancements, practical agent-building capabilities, and upcoming roadmap items for the platform.

## Key Topics

- **Two Copilot Studio Versions**: Introduction to the embedded builder for quick agent creation and the enterprise-grade suite for advanced scenarios.
- **Agent Development**: Walkthrough of agent creation tools, including question set generation, scoring methods, and evaluation processes tailored for enterprise needs.
- **Information Worker Features**: Integration with Microsoft 365 Copilot Agent Builder extends agent-building capabilities for business users, facilitating streamlined automation and improved user experiences.
- **Governance and Management**: Deep dive into platform governance, management capabilities, and cost control options for deploying and operating agents within organizations.
- **Testing and Debugging**: Robust new tools for developing, testing, and debugging agents ensure reliability and scalability.
- **Generative Orchestration**: Introduction of generative orchestration features that enhance knowledge capabilities and enable seamless interactions between agents and backend systems.
- **Agent Identity**: Demonstration of agent identity management and how agents function within the Copilot Studio ecosystem.

## Customer Impact

Real-world success stories illustrate how Copilot Studio accelerates automation initiatives, increases operational efficiency, and elevates user experiences in business contexts.
