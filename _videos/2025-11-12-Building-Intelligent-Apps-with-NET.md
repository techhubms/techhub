# Building Intelligent Apps with .NET

Presented by dotnet, this session focuses on empowering developers to build smarter applications using the latest AI capabilities within the .NET ecosystem.

## Key Topics Covered

- **Unified AI Abstractions**: Learn how Microsoft.Extensions.AI provides a consistent way to integrate various AI features into .NET applications.
- **Large Language Model Integration**: Practical approaches to embedding LLMs for tasks like summarization and classification.
- **Semantic Search**: Implement semantic search to enhance information retrieval in your apps.
- **Intelligent Features**: Add summarization, classification, and AI-powered assistants directly to .NET solutions.
- **Cloud and Local Model Support**: Patterns for using both cloud-hosted and on-premises AI models.
