# AI-Powered Email Sorting with n8n and OpenAI

**Author:** Alireza Chegini | AI Skills for Your Career

This tutorial demonstrates how to automate email classification and labeling using the n8n workflow engine and OpenAI's text classification.

## What You'll Learn

- Connecting n8n with Gmail for email retrieval
- Designing a classification prompt for OpenAI
- Building a workflow for automated email categorization
- Creating custom label logic for emails
- Updating Gmail with classified label results
- Customizing and testing the workflow for accuracy

## Workflow Steps

1. **Inbox Chaos Introduction:** Overview of email overload and practical goals for sorting.
2. **Project Setup:** Configuration of n8n and Gmail connection.
3. **Prompt Engineering:** Building AI prompts for OpenAI to classify incoming messages.
4. **Automation Flow:** Implementing logic in n8n to process, classify, and label emails dynamically.
5. **Custom Label Logic:** Setting up custom folders/categories to organize emails.
6. **Automated Gmail Update:** Real-time labeling and organization within Gmail.
7. **Testing and Customization:** Running full workflow trials and refining category logic as needed.

## Tools Used

- **n8n:** No-code automation platform
- **Gmail Integration:** For email management
- **OpenAI Text Classification:** AI-powered categorization
- **Custom Labels:** Personalization for folder assignment
