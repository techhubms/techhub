# Transforming GitHub Copilot Metrics into Business Value

**Speakers:**

- Sharanya Doddapaneni, VP, Engineering, GitHub
- Shruti Corbett, Senior Director of Engineering, GitHub

## Overview

This session addresses the question: “Now that your GitHub Copilot deployment is live, how do you measure its real impact?” It is aimed at engineering leaders interested in using data to turn AI adoption into a business-critical component of their organization.

## Key Topics

- **Refreshed Copilot Metrics Dashboard:**
  - Walkthrough of the redesigned dashboard offering comprehensive, actionable usage insights.
- **Understanding Adoption Patterns:**
  - How to interpret Copilot usage data to identify high-performing teams and areas for improvement.
- **Driving Broader Adoption:**
  - Practical approaches for increasing Copilot utilization across engineering teams.
- **Demonstrating ROI:**
  - Translating usage metrics into compelling narratives to secure ongoing AI investment.
- **Business Outcomes:**
  - Strategies tied to three outcomes: measuring effective use, expanding adoption, and proving business value.
- **Live Demo:**
  - Hands-on demonstration of the metrics dashboard.
- **Feedback Loop:**
  - Using observed data for continuous improvement: “Observe, learn, and act.”
- **Future Roadmap:**
  - Looking ahead to additional impact and SDLC metrics.
