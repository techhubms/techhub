# What's New in NuGet: Security, AI, and Modern Package Management for .NET

NuGet, the backbone of .NET package management, is undergoing major improvements to help developers stay secure and productive.

## Modern AI-Powered Productivity with MCP

- **Model Context Protocol (MCP) Server**: Enables integration of AI assistants for streamlined package management workflows.
- **NuGet.org MCP Registry**: NuGet.org becomes the central MCP registry for .NET, making it easier to discover and utilize AI-driven packages.
- **AI-Driven Package Management**: Automate package updates, vulnerability checks, and integration through AI-powered tools.

## Security Innovations

- **NuGet Audit**: Enhanced capabilities for detecting and mitigating vulnerable dependencies in your projects.
- **Trusted Publishing**: Methods for ensuring secure and compliant package distribution, protecting consumers from supply chain risks.

## Ecosystem and Developer Experience

- **Modern Package Discovery & Sponsorship**: Improved tools for finding relevant packages and supporting package maintainers.
- **Streamlined Workflows**: Updates designed to help teams maintain secure, reliable dependencies while moving quickly.
