# Modernize on-premises VMware environments with Azure VMware Solution

## Overview

This Microsoft Ignite 2025 breakout session explores strategies and tools available for organizations seeking to migrate, scale, and modernize VMware environments using Azure VMware Solution (AVS).

### Key Topics Covered

- **AVS Customer Adoption & Managed Service Model:** Insights into real-world customer scenarios and adoption trends, including how managed services simplify operations.
- **Choosing the Right Azure Path:** How to evaluate migration vs modernization, and when AVS is the best fit.
- **Accelerating Cloud Strategy:** Approaches for migrating large legacy VMware workloads with minimal disruption, focusing on scalability and operational efficiency.
- **Integration with Azure Storage:** Enhancing VMware environments with Azure NetApp Files and Elastic SAN to improve performance and flexibility.
- **VCF Plus Subscription Compatibility:** Understanding the bring-your-own-license (BYOL) model for subscription management.
- **Leveraging Azure and AI Services:** Ways to enrich migrated applications using Azureâ€™s AI capabilities, such as Azure OpenAI Service.
- **Azure Arc Integration:** Extending management with Azure Arc for vSphere, enabling unified management and governance across hybrid environments.
- **Demo:** Live demonstration of a banking chatbot built with Azure OpenAI Service and managed identities, showcasing AI integration and practical cloud implementation.
- **Migration Planning:** Step-by-step guidance for planning migrations and establishing a cycle of continuous modernization.
