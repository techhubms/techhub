# Gemini 3 Flash Public Preview Released for GitHub Copilot Users

Gemini 3 Flash, designed for scenarios where performance matters, is being introduced as a new AI model option in public preview for GitHub Copilot users.

## Availability

- **Supported plans:** Copilot Pro, Pro+, Business, and Enterprise
- **Supported environments:**
  - Visual Studio Code (all modes: chat, ask, edit, agent)
  - Copilot Chat on github.com
  - GitHub Copilot Mobile for iOS and Android

The rollout is staggered, so some users may not see Gemini 3 Flash immediately. Availability will expand over time.

## How to Enable Gemini 3 Flash

- **Enterprise and Business:** Organization administrators must enable Gemini 3 Flash in Copilot settings. Once activated, team members will see Gemini 3 Flash as an option in their model picker.
- **Pro and Pro+:** Individual users can enable the new model by selecting it in the Copilot model picker and proceeding with the one-time confirmation prompt.
- **Bring Your Own Key (BYOK):** In Visual Studio Code, choose **Manage Models** in the picker, select Gemini 3 Flash, and enter your own API key when prompted.
