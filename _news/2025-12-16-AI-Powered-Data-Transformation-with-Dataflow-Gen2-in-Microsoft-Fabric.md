# AI-Powered Data Transformation with Dataflow Gen2 in Microsoft Fabric

## Introduction

Efficiency and adaptability are more important than ever in today's business world. Dataflow Gen2, a new feature in Microsoft Fabric, combines artificial intelligence (AI) with natural language prompts to streamline data workflows and unlock insights without complex coding.

![Screenshot of AI Prompt entry in the Add column tab of the Power Query editor ribbon](//dataplatformblogwebfd-d3h9cbawf0h8ecgf.b01.azurefd.net/wp-content/uploads/2025/12/image-19.png)

## Key Features of Dataflow Gen2

- **AI-Powered Transformations:** Perform tasks like summarization, classification, translation, and sentiment analysis directly within dataflows, using AI rather than manual scripts.
- **Prompt-Driven Automation:** Natural language prompts enable users to operate on data tables, specifying context and desired outcomes using an intuitive interface.
- **Scalable Personalization:** Outputs are context-aware, adapting to the underlying data and workflows.
- **Seamless Fabric Integration:** Works with Microsoft Fabric's data engineering, data factory, and analytics workloads. This unified experience reduces friction and helps teams stay productive.
- **Preview Availability:** The AI Prompt feature will be available in the Add column tab for Dataflow Gen2, rolling out globally by December 18, 2025.

![Screenshot of AI Prompt dialog in Dataflow Gen2 (CI/CD)](//dataplatformblogwebfd-d3h9cbawf0h8ecgf.b01.azurefd.net/wp-content/uploads/2025/12/image-20-1024x697.png)

## Working with AI Prompts in Dataflow Gen2

- **Access:** Find the AI Prompt option in the Add column tab of the ribbon in Dataflow Gen2.
- **Usage:** Enter natural language prompts and select the columns to use as context. This lets you quickly transform data, automate repetitive tasks, and experiment with workflows.
- **Documentation:** More details and guidance are available at the official documentation: [Fabric AI Prompt in Dataflow Gen2 (Preview) – Microsoft Fabric | Microsoft Learn](https://learn.microsoft.com/fabric/data-factory/dataflow-gen2-ai-functions)

## Summary

With Dataflow Gen2 and its AI Prompt feature, Microsoft Fabric is making it possible for teams to implement powerful data transformations and analytics—without writing code. Automation, personalization, and seamless integration support productivity and innovation within enterprise data workflows.

## Availability

- **Preview rollout:** Begins December 15, 2025
- **Global availability:** By December 18, 2025

For hands-on details and screenshots, visit the Microsoft Fabric documentation linked above.
