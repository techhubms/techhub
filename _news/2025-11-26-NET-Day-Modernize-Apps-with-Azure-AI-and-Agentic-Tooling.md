# .NET Day on Agentic Modernization: What to Expect

**Event Date:** December 9, 2025, 9AM-1PM Pacific.

Join a free, live virtual event designed to help developers modernize their .NET applications with the latest Microsoft tools and agentic patterns. The agenda includes eight technical sessions led by Microsoft experts, focusing on AI integration, cloud native migration, secure architectures, and developer productivity.

## Key Sessions

- **Choose Your Modernization Adventure with GitHub Copilot**: See how GitHub Copilot and Visual Studio accelerate modernization, upgrade legacy code, and facilitate secure, cloud-ready migrations into Azure.

- **Agentic DevOps: Enhancing .NET Web Apps with Azure MCP**: Discover AI-powered DevOps tooling for .NET apps, including streamlined code, storage, and IaC workflows.

- **Performance Guardrails with Azure SRE Agent**: Learn how the Azure SRE Agent monitors .NET app performance, auto-detects regressions, and enables safe rollbacks to maintain reliability.

- **No-Code Modernization for ASP.NET**: Explore how Azure App Service Managed Instance provides frictionless ASP.NET modernization, enhanced performance, and integration with modern agentic workflows.

- **Building Agentic Solutions in .NET**: Integrate the Agent Framework in .NET apps for multi-agent coordination and advanced orchestration support with practical starting guidance.

- **Bulletproof Agents with Durable Task Extension**: See how Microsoftâ€™s Durable Task Extension brings reliable, distributed, and debuggable AI agents to Azure, enabling robust agentic workflows.

- **Secure AI Agents on Azure SQL and SQL Server**: Learn secure deployment strategies, enforcing least-privilege access and schema-aware protection for AI agents interacting with data platforms.

- **Secure and Smart AI Agents with Azure Redis**: Discover how Azure Redis supports secure data access and agentic app workflows, leveraging Redis-backed tools and modern security patterns.

## Why Attend?

- Get hands-on insights to upgrade and modernize existing .NET applications
- Understand how to leverage cloud, AI, and agentic frameworks for real-world solutions
- Direct access to Microsoft engineers and opportunity for live Q&A
- Discover approaches to integrate security, monitoring, and robust agentic patterns in app architecture
