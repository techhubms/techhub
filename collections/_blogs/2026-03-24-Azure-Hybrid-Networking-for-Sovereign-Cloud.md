# Azure Hybrid Networking for Sovereign Cloud

Thomas Maurer shares a video conversation with **Cynthia Treger (EMEA GBB for Networking at Microsoft)** focused on **Azure Hybrid Networking for Sovereign Cloud scenarios**.

## What the conversation covers

- **Microsoft Global Network** as the foundation for hybrid and sovereign cloud architectures
- **Azure connectivity options** and when to use each:
  - **ExpressRoute**
  - **Site-to-Site VPN / internet-based scenarios**
- **Connectivity design considerations**:
  - Regional connectivity
  - Inter-regional connectivity
  - Trade-offs when choosing an approach

## Azure Local and Azure Arc in sovereign/hybrid scenarios

The discussion also covers how Azure networking capabilities can provide **secure connectivity** for:

- **Azure Local**
- **Azure Arc–enabled services**

…supporting consistent operations across **cloud, on-premises, edge, and regulated environments**.
