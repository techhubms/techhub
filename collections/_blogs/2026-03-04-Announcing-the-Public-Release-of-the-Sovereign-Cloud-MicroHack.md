# Announcing the Public Release of the Sovereign Cloud MicroHack

By Thomas Maurer

## Overview

The Sovereign Cloud MicroHack is now publicly available, offering engineers and architects a focused, hands-on experience to master the core concepts and technical controls behind Microsoft's Sovereign Cloud offerings. The MicroHack is designed for practical application—guiding participants through common challenges and scenarios in both public and private sovereign cloud environments.

### Key Details

- **Format:** Hands-on workshop (<6 hours)
- **Availability:** Public (self-paced) and instructor-led sessions
- **Target Audience:** Engineers, architects, and technical decision-makers interested in Azure cloud sovereignty, data protection, and hybrid models
- **Official MicroHack URL:** [https://aka.ms/SovereignCloudMicroHack](https://aka.ms/SovereignCloudMicroHack)

## What You'll Learn

Participants tackle six carefully constructed technical challenges:

1. **Enforcing Sovereign Controls in Azure**
   - Use Azure Policy, RBAC, and native platform controls for governance.
2. **Encryption at Rest with Customer Managed Keys**
   - Learn to use Azure Key Vault for key management and securing stored data.
3. **Encryption in Transit**
   - Enforce TLS and secure data as it moves between services.
4. **Encryption in Use (VM-based)**
   - Leverage Azure Confidential Compute to protect data while it's being processed.
5. **Encryption in Use (Container/Application-based)**
   - Apply confidential computing techniques to containerized and application workloads on Azure.
6. **Operating Sovereign Workloads in Hybrid Environments**
   - Integrate Azure Local and Azure Arc to manage workloads across public and private clouds.

## Technical Resources Included

- Step-by-step lab guides for each challenge
- Architecture diagrams
- Deployment scripts and automation
- Prerequisite checklists

All resources and lab content are maintained in the public [Microsoft/MicroHacks GitHub repository](https://aka.ms/SovereignCloudMicroHack) for easy access and contribution.

## Upcoming MicroHack Events

Eight public MicroHacks are already scheduled across Europe—Belgium, Denmark, Finland, Norway, Czech Republic, Romania, Switzerland, and Hungary. Check with your local Microsoft representative for opportunities to participate.
