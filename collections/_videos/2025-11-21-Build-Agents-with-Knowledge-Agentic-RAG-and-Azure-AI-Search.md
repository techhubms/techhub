# Build Agents with Knowledge, Agentic RAG, and Azure AI Search

**Speakers:** Pamela Fox, Matthew Gotteiner
**Event:** Microsoft Ignite 2025 (Session BRK193)

Learn how to create advanced AI agents with cutting-edge features in Azure AI Search. This session is designed for developers and technical practitioners who want to:

- Connect agentic retrieval engines to new knowledge sources, including SharePoint, web content, and Azure Blob Storage
- Leverage hybrid search methods and reciprocal rank fusion for optimal relevance
- Apply the Semantic Ranker in AI Search to improve search result quality
- Understand and implement indexed vs remote knowledge source integration
- Optimize query planning and knowledge source selection
- Use iterative retrieval strategies for complex, multi-query scenarios
- See code-focused walkthroughs and demos of setting up and refining RAG solutions

### Session Highlights

- **Connecting to Diverse Knowledge Sources:**
  - How to link your agent to SharePoint, web content via Bing, and Azure Blob
- **Hybrid Search and Reciprocal Rank Fusion:**
  - Techniques to blend multiple search signals for more robust retrieval
- **Enhanced Retrieval Flow:**
  - End-to-end walkthrough of hybrid search pipeline and semantic ranking
- **Query Planning and Answer Generation:**
  - Strategies to select and sequence knowledge sources for best results
  - Multi-stage retrieval to refine queries and answers
- **Code Samples and Step-by-Step Demos:**
  - Practical coding examples and workflow guides
