# Comprehensive Data Security and Governance in AI Workloads with Microsoft Purview

## Session Overview

This session, presented by Microsoft Events at Ignite 2025, focuses on advanced strategies for securing and governing organizational data in environments increasingly driven by AI technologies. The talk addresses how Microsoft Purview provides a unified solution for data protection, governance, and compliance across several Microsoft platforms.

### Key Topics & Solutions

- **Security and Governance Focus**: Building trust by securing data and ensuring robust governance practices throughout the organization.
- **AI Threats and Urgency**: Real-world threat examples and the unique challenges of securing AI agents are presented.
- **Dynamic Data & Audit Logs**: Introduction to evolving audit logs as AI agents become faster and more autonomous, emphasizing the need for agile security monitoring.
- **Purview + AI Foundry Integration**: Purview tracks sensitive data usage in AI models and agents via deep integration within the AI Foundry platform.
- **Purview Protections for Copilot Studio**: The session introduces new methods for applying data protections across Copilot Studio, enabling secure AI application development.
- **Compliance Manager**: Launch of Compliance Manager for evaluating AI applications against regulatory and business requirements.
- **Fabric and Power BI Integration**: Purview's extended role in protecting sensitive data processed in Microsoft Fabric and Power BI analytics workloads.
- **Security Dashboard for AI**: A unified dashboard is showcased, providing consolidated AI risk views, actionable insights, and monitoring capabilities.
- **Unified Microsoft Security Ecosystem**: Emphasis on the transition from fragmented tools to a cohesive security platform spanning identity, data, cloud, and AI operations.

## Actionable Insights

- Adopt Microsoft Purview for comprehensive data security and governance across cloud and AI services.
- Leverage integrations between Purview, Azure AI Foundry, Copilot Studio, Microsoft Fabric, and Power BI to monitor and manage sensitive data.
- Utilize Compliance Manager to ensure AI app development aligns with regulatory obligations.
- Employ the Security Dashboard for an enterprise-wide overview of AI risk and compliance status.
