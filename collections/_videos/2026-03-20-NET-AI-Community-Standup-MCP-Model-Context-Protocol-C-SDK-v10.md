# .NET AI Community Standup: MCP (Model Context Protocol) C# SDK v1.0

The official MCP (Model Context Protocol) C# SDK v1.0 is available. This session focuses on what MCP is, why a standardized protocol matters, and how to use the C# SDK from .NET to connect apps with AI models.

## What this session covers

- What **MCP (Model Context Protocol)** is and the problems it aims to solve
- Why using a **standardized protocol** can help when connecting apps to AI models
- How the **MCP C# SDK v1.0** enables AI-powered scenarios in **.NET**
- **Real-world use cases**, plus a walkthrough of **code samples** and integration patterns

## Who it’s for

- .NET developers getting started with AI integrations
- Teams already building AI features who want a more interoperable, protocol-based approach
