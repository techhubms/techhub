# On .NET Live: Rx.NET v7 and Futures

**Panelists:** Ian Griffiths, Katie Savage, Frank Boucher, Cam Soper

Join the .NET team and guest experts as they walk through the latest features of Rx.NET (Reactive Extensions for .NET) version 7. The session covers:

- Overview of Rx.NET v7 updates and enhancements
- Demonstrations of new APIs and reactive programming patterns
- How Rx.NET empowers developers to work with asynchronous data streams more effectively
- Discussion of the project's direction and future plans for reactive extensions in the .NET landscape
- Best practices for integrating Rx.NET into modern .NET applications

This episode provides .NET developers with actionable insights into using reactive patterns, upcoming library improvements, and where the Rx.NET ecosystem is heading next.

**Key topics:**

- Introduction to Rx.NET and reactive programming concepts
- Deep dive into v7 features and breaking changes
- Q&A with the .NET team
