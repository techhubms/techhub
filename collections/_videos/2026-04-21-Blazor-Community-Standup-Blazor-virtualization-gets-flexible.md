# Blazor Community Standup: Blazor virtualization gets flexible

The **dotnet** team hosts a Blazor Community Standup previewing upcoming **virtualization improvements in .NET 11 for Blazor**, focusing on making the **`Virtualize`** component work better in more real-world UI scenarios.

## What’s being covered

- **Variable-height item support** for the **Blazor `Virtualize` component**
  - This targets scenarios where list items don’t have a fixed, uniform height.
- **New APIs for controlling scroll position**
  - Programmatic control over scrolling behavior, aligned with virtualization scenarios.
