# Advanced Agent Development with Copilot Studio

**Speaker:** Scott Durow
**Event:** Microsoft Ignite 2025 (Session BRK1742)

## Overview

This session dives into advanced techniques for building intelligent, autonomous agents using Microsoft Copilot Studio. It's designed for both makers and developers interested in moving beyond foundational skills to orchestrate AI-driven agents capable of streamlining business processes and enhancing operational efficiency.

## Key Topics Covered

- **Creating and Configuring Agents:** Step-by-step guidance on setting up agents within Copilot Studio and configuring them for specific scenarios.
- **Managed and Anthropic Models:** Exploration of different AI models, focusing on reasoning capabilities and decision logic.
- **Agent Settings and Knowledge Grounding:** Adjusting agent parameters and grounding them with business-specific knowledge for contextual intelligence.
- **Automated Triggers:** Practical demonstration of setting up automated triggers and verifying email-driven actions for business workflows.
- **Explaining AI-driven Actions:** Deep dive into how agents use triggers autonomously to drive business impact.
- **Multi-Agent Orchestration:** Introducing strategies for coordinating multiple agents, separation of concerns, and designing scalable automation solutions.
- **Product Inventory Scenario:** Example walkthrough of orchestrating a product inventory adjustment using agent collaboration.
- **Integrating Computer Use for Agents (CUA):** Hands-on demonstration of leveraging UI automation to extend agent capability and interact with external systems.

## Value for Attendees

By the end of this session, you’ll gain:

- Advanced knowledge of Copilot Studio’s agent orchestration features
- Techniques for grounding AI agents with business knowledge
- Best practices for combining triggers, actions, and multi-agent setups
- Insights into integrating automation tools for broader business transformation
