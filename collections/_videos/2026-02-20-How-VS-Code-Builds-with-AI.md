# How VS Code Builds with AI

## Overview

The Visual Studio Code team shares their experience integrating AI agents into their daily development workflow.

- **AI in Development:** Real-world examples of AI agents assisting in building and improving VS Code.
- **Daily Integration:** Methods for incorporating AI into engineering cycles.
- **Lessons Learned:** Challenges, growing pains, and the collaborative dynamic between engineers and AI agents.
- **Impact of Feedback:** How the team's own usage and iterative feedback loop inform and inspire new features in the VS Code editor.
- **Continuous Improvement:** The role of telemetry and agent interaction metrics in product evolution.
