# Using Goose and MCPUI with the GitHub MCP server to create and manage GitHub issues

GitHub’s GitHub Universe 2025 demo shows how to automate GitHub issue management using an AI agent connected through the GitHub MCP server.

## What the demo covers

- Using the **Goose AI agent** with the **GitHub MCP server**
- **Automatically creating and assigning** a new **GitHub issue** directly from the **terminal**
- Using **MCPUI** so the agent can respond with **interactive visual interfaces** (example shown: a **team project calendar**)
