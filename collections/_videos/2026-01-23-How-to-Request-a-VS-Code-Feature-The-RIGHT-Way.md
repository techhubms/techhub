# How to Request a VS Code Feature (The RIGHT Way)

**Presented by Reynald Adolphe (@ReynaldAdolphe)**

Are you missing a feature in Visual Studio Code that could boost your productivity or simplify your workflow? This concise tutorial shows you how to properly submit a feature request right within VS Code—ensuring your suggestion is clearly seen and increases its chances of review by the development team.

## Key Steps Covered

- **Demo:** Step-by-step walkthrough on initiating and submitting a feature request
- **Best Practices:** Tips on providing concise, actionable details to make your request more effective
- **Where Requests Go:** Explanation of how VS Code team reviews GitHub issues and requests

## How To Submit a Feature Request from VS Code

1. Open the command palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
2. Type “Report Issue” and select the associated option
3. Choose the “Feature Request” template
4. Provide clear, detailed information about your suggestion (what you want and why it matters)
5. Submit the request, which will be filed in the VS Code GitHub repository
