# Innovation Session: Build & Manage AI Apps with Your Agent Factory

**Speakers:** Yina Arenas, Sarah Bird, Amanda Silver, Marco Casalaina  
**Event:** Microsoft Ignite 2025

## Overview

Join engineering leaders as they introduce Microsoft Foundry’s approach to modular AI development. This session covers how to build, orchestrate, and manage intelligent agents using new workflow features, with a special focus on unifying models, data, and tools under strong enterprise security and observability.

## Key Topics

- **Modular AI with Microsoft Foundry:** Approaches for assembling and orchestrating AI agents within scalable, secure workflows.
- **Enterprise-Grade Security & Observability:** How Foundry delivers robust security guardrails and advanced monitoring for agent operations.
- **Model and Tool Unification:** Strategies for leveraging diverse models and integrating different data sources for flexible solutions.
- **Scalability and Agent Management:** Techniques for managing hundreds of agents at scale using the Foundry Hub.
- **Demonstrations:**
    - Workflow and agent integration using LangGraph.
    - Inventory analysis agent set up with a human-in-the-loop workflow.
    - Showcasing Lufthansa’s use of the Microsoft Agent Framework for scalable operations.
- **Developer Enablement:**
    - How GitHub Copilot is integrated for agent evaluations.
    - CI/CD integration to streamline agent testing and deployment.

## Detailed Agenda Highlights

- The importance of trust, security, and runtime guardrails in AI applications.
- Implementing defense measures against attacks within agent workflows.
- Model diversity and the flexibility of the Foundry platform.
- Real-world success stories demonstrating AI agent factory deployment.
- Best practices for evaluation using developer tools.
