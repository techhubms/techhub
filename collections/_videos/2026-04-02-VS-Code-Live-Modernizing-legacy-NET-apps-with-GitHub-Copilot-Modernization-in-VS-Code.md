# VS Code Live: Modernizing legacy .NET applications with GitHub Copilot Modernization in VS Code

Visual Studio Code hosts a live session where **Joel Norman** and **Jorge Balderas** “let Copilot loose” to try **modernizing a .NET application** using **GitHub Copilot Modernization** inside **Visual Studio Code**.

## What this session is about

- Using **GitHub Copilot Modernization** workflows in **VS Code** to assist with **modernizing legacy .NET applications**.
- Following the **Modernize .NET** guidance and tooling provided by Microsoft.
