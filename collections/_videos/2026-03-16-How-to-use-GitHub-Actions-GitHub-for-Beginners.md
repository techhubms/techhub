# How to use GitHub Actions | GitHub for Beginners

GitHub’s **GitHub for Beginners** episode covers the fundamentals of **GitHub Actions** and the core idea of **CI/CD**. It includes a practical walkthrough where you build a workflow that **automatically labels new issues** in a repository.

## What this video covers

- What **GitHub Actions** is
- Core **CI/CD** concepts in the context of GitHub
- Building your first **workflow**
- Creating an **Action**
- How to **test** your workflow
