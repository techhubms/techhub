# Powering Financial Workflows with AI: Microsoft and LSEG Partnership

**Speakers:** Matthew Kerner, Nej D'Jelal, Max Grigoryev  
**Event:** Microsoft Ignite 2025 | Session BRK367

---

## Strategic LSEG–Microsoft Partnership

- Description of transformation goals for financial workflow automation and interoperability.
- Focus on user-centered design and the enabling of semantic search for better data access and decision-making.

## Technical Approach

- Details the integration strategy: build vs. partner considerations and leveraging open standards for data sharing across organizations using Open Directory.

## AI and Agent-Based Automation

- In-depth demonstration of preparing and deploying a financial analytics agent in Copilot Studio.
- Practical walkthrough of no-code agent creation using natural language interfaces in Copilot Studio.
- Use of LSEG data with M365 Copilot, enabling dynamic insights directly in productivity apps like Word, Excel, PowerPoint, and Teams.

## Applications in Financial Services

- Enabling post-trade analysis and intercompany collaboration using AI-driven tools.
- Automated preparation of investment review documents leveraging agent-based orchestration.
- A new Researcher Agent demonstrates integrating external financial data with M365 for accelerated document and research workflows.

## Session Highlights

- Summary covers AI integration, scalable agent creation, and improvements to technical collaboration between LSEG and Microsoft.
- Showcases the transformation of financial workflows to be more productive and data-driven using interoperable Microsoft technology.
