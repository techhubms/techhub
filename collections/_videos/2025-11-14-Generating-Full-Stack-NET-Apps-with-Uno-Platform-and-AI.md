# Generating Full-Stack .NET Apps with Uno Platform and AI

The Uno Platform empowers developers to create pixel-perfect mobile, web, desktop, and embedded applications using a single .NET codebase. Building on its established productivity features like Hot Design—a next-generation visual designer—Uno Platform now incorporates AI to accelerate full-stack app generation.

## Key Highlights

- **Cross-Platform .NET Development:**
  - Target iOS, Android, WebAssembly, Windows, macOS, and embedded platforms.
  - Share logic and UI across all form factors.
  
- **Hot Design:**
  - Enables rapid visual iteration during development.
  - Accelerates prototyping and reduces feedback cycles.

- **AI Integration:**
  - Use AI tools to scaffold and generate full-stack applications.
  - Boost productivity from design to deployment.

- **.NET Ecosystem Updates:**
  - **.NET 10:** Enhanced intelligence and unified experience.
  - **.NET Aspire 13:** Simplified orchestration for scalable, cloud-native apps.
  - **ASP.NET Core 10:** Modern, secure full-stack web development.
  - **.NET MAUI 10:** Sophisticated UI for truly cross-platform experiences.
  - **C# 14 & F# 10:** Advancements in expressiveness, safety, and productivity.
  - **Visual Studio 2026:** Faster, smarter IDE designed for the future of .NET.
