# Open Source Friday with Unity-MCP

GitHub interviews Andy Tsen about **Unity MCP (Model Context Protocol for Unity)** and how it makes “AI inside Unity” more practical.

Repo: https://github.com/CoplayDev/unity-mcp

## What this video covers

- **Why MCP matters for Unity + AI**
  - How MCP can help **tools and agents talk to Unity in a structured way**.

- **What “context” means in a game engine**
  - Discussion of how context applies specifically to Unity workflows (as opposed to generic chat prompts).

- **Getting started / experimenting**
  - High-level guidance aimed at developers who want to start trying **AI-assisted workflows** with Unity using Unity MCP.
