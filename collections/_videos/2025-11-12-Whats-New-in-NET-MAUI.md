# What's New in .NET MAUI

Explore the latest features in .NET MAUI for .NET 10, which continues to solidify its place as the premier framework for building native applications across platforms such as iOS, Android, macOS, and Windows. This session details:

## Key Enhancements

- **Performance Optimizations**: Improved efficiency to ensure fast and responsive apps.
- **New Controls and Layouts**: Additional UI elements and layouts to support flexible, modern designs.
- **Tooling Improvements**: Enhanced developer experience through Visual Studio integration, supporting rapid development and debugging.
- **Expanded Platform Support**: Broader reach across mobile and desktop environments with a single codebase.

## Real-World Impact

Learn from examples of companies shipping production applications with .NET MAUI, demonstrating its maturity and reliability.

## Unified Development with .NET

- **.NET 10**: Delivers a more intelligent and unified platform for developers.
- **.NET Aspire 13**: Simplifies orchestration for cloud-native applications.
- **ASP.NET Core 10**: Advances modern and secure full-stack web development.
- **C# 14 & F# 10**: Offers greater productivity and expressive power for developers.
- **Visual Studio 2026**: Provides a faster and smarter IDE tailored for the future of .NET.
