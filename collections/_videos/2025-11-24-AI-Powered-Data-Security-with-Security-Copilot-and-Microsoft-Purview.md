# AI-Powered Data Security with Security Copilot and Microsoft Purview

**Speakers:** Tanay Baldua, Paige Johnson, Arun Kumar

## Overview

This advanced Microsoft Ignite 2025 breakout session explores how Security Copilot and new AI agents in Microsoft Purview revolutionize data security operations. It addresses common security team challenges such as alert fatigue and limited staffing, and demonstrates AI-driven automation in handling sensitive data protection at speed and scale.

## Key Topics

- **Security Team Challenges**: Managing overwhelming alerts and staffing limitations
- **Introduction to Security Copilot**: How AI agents in Microsoft Purview automate key security workflows
- **Live Demonstration**: Arun Kumar presents a practical scenario of data leak detection and response
- **Team Scenario**: Security professionals Lauren and Adi address alerts related to potential leaks from 'Project Nebula'
- **AI-Powered Automation**:
  - Alert triage and prioritization
  - Workload reduction using AI automation
  - Cross-functional management of data security alerts
- **Risk Types Explained**:
  - Sensitivity risk
  - Exfiltration risk
  - Policy risk
- **Policy Review**:
  - Comparison between custom project security policies and default policies
  - How policy configuration impacts data leak prevention
- **Signal to Outcome Workflow**:
  - Review of the six steps in a typical security incident workflow, with focus on automation and efficiency
- **Automation Benefits**:
  - Faster risk mitigation
  - Reduced manual intervention
  - Improved alert management across security operations
