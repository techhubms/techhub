# Building an Intelligent Vendor Onboarding Agent with Copilot Studio

## Overview
This video walks through building an enterprise vendor onboarding agent using **Copilot Studio** and the **Power Platform** to reduce manual effort while keeping **governance and control**.

## What the agent does
- Automates **vendor onboarding** end-to-end
- Performs **document validation**
- Produces **risk scoring**
- Routes **approvals** through a controlled workflow

## Architecture and components used
- **Copilot Studio**: used to design and orchestrate the agent experience
- **AI Builder**: used for AI capabilities that support tasks like document-related processing
- **Microsoft Dataverse**: used as the data layer for vendor onboarding information
- **Power Automate**: used to implement workflow automation for approvals and operational steps

## Key implementation themes
- How the above components integrate to form a **scalable, enterprise-ready** agent
- Maintaining **governance** while automating sensitive onboarding processes
