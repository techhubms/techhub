# Securing Data Across Microsoft Environments with Microsoft Purview

**Speakers:** Nathalia Borges, Maithili Dandige, Aashish Ramdas, Sushma Rao

## Introduction

This Microsoft Ignite session (BRK252) dives deep into Microsoft Purview's capabilities for securing data across a range of Microsoft environments, including Microsoft 365, Azure, Windows, and Microsoft Fabric.

## Topics Covered

- **Overview of Microsoft Purview**: Centralized platform for enterprise data security and compliance.
- **Data Security Posture Management (DSPM)**: Introduction to DSPM concepts and how agent-driven security enhances risk detection and automated remediation.
- **Remediation for M365 Copilot**: Steps organizations should take to prepare for new AI-driven integrations, including advanced data protection.
- **Reporting & Insights**: Overview of built-in reporting features, specifically for auto labeling performance, helping admins assess and improve compliance automation.
- **Demonstrations**:
  - Working of Recall and DLP together to prevent sensitive data leakage
  - Prompt DLP announcement and live setup walkthrough
  - Application of DLP protections in the OneLake Catalog
  - Real-world example using the Credit Card Usage Semantic Model with enforced DLP policies

## Key Features and Announcements

- **Prompt DLP**: New feature to bolster protection against sensitive data leakage in AI-related scenarios.
- **Agent-driven Security**: Deploying agents throughout your Microsoft data estate for more granular visibility and control.
- **OneLake Catalog Protections**: How Purview delivers policy enforcement on data stored in Fabric’s OneLake.
- **AI Readiness**: Ensuring your data security is robust enough to handle integrations with AI-powered solutions like M365 Copilot.

## Takeaways

- Microsoft Purview delivers integrated tools for uncovering data security risks across hybrid enterprise environments.
- New and enhanced DLP, auto labeling, and Prompt DLP help protect sensitive information proactively.
- Seamless coordination between Azure, Microsoft 365, Windows, and Fabric data platforms allows unified security policies and centralized oversight.
