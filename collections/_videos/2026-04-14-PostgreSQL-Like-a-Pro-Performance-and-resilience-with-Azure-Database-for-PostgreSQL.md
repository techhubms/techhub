# PostgreSQL Like a Pro: Performance and resilience with PostgreSQL on Azure

Gauri Kasar (Microsoft Developer) walks through how **Azure Database for PostgreSQL** can be deployed and operated for **performance under pressure**—with a focus on resilience features, read scaling, and recovery.

## What the video covers

- How PostgreSQL on Azure is positioned for **performance and resilience**
- A demo of common **deployment and operations** tasks
- Scaling patterns (including **horizontal scale** via elastic clusters)
- How Azure supports **failover, recovery, and scaling** to keep apps online
- Notes on **AMD-powered SKUs** for price/performance considerations
