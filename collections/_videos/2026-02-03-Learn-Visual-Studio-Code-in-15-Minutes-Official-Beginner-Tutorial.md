# Learn Visual Studio Code in 15 Minutes: Official Beginner Tutorial

**Featuring:** Reynald Adolphe (@ReynaldAdolphe)

Get started quickly with Visual Studio Code in this official tutorial. The video is structured in clearly defined chapters for easy reference.

## Contents

- **UI Exploration:** Learn about the layout and interface of VS Code.
- **Creating Files:** Step-by-step guidelines on creating and managing new files.
- **Using Chat:** Discover chat-based features within the editor to enhance workflows.
- **Source Control:** Integrate version control directly in your workspace for streamlined collaboration.
- **Install MCP Server:** Walkthrough of installing the MCP server to enable specific features.
- **Issue Creation:** Learn to create and manage code issues efficiently.
- **Session Management:** Review current and previous work sessions, helping you track progress.

**Highlighted Features**

- [IntelliSense](https://aka.ms/intellisense): Boost your coding with intelligent code completions.
- [Inline Suggestions](https://aka.ms/ai-ps): Leverage AI-powered suggestions while typing.
