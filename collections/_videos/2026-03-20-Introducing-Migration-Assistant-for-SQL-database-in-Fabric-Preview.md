# Introducing Migration Assistant for SQL database in Fabric (Preview)

Migration Assistant for SQL Database in Fabric simplifies moving from SQL Server to Microsoft Fabric by providing an end-to-end, guided migration experience inside the Fabric portal.

## What it does

- Provides a guided, end-to-end migration flow from **SQL Server** to **SQL database in Fabric**.
- Helps you **assess schema compatibility**.
- Helps you **proactively resolve issues** with **Copilot assistance**.
- Moves data using **built-in Fabric copy workflows**.

## Why it matters

- Faster time to value
- Fewer migration surprises
- Smoother path to modernizing SQL workloads in Fabric

## Availability

- This is a **Preview** experience.
- Rolling out soon and expected to be available over the next few weeks.
