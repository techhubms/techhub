# ASP.NET Core Server & APIs Roadmap Discussion for .NET 11

## Overview

This community standup video, hosted by the .NET team and featuring Daniel Roth, Mike Kistler, and Jose Perez Rodriguez, centers on the upcoming roadmap for ASP.NET Core as .NET 11 approaches. The discussion focuses on backend server functionality and API advancements, with an emphasis on soliciting community feedback to help shape future development.

## Key Topics Covered

- Planned enhancements to server and API infrastructure in ASP.NET Core for .NET 11
- Review of current pain points and community-submitted feedback
- API design improvements and performance initiatives
- Discussion on priorities for API developers and backend maintainers
- Links to related resources and feedback channels: [ASP.NET Standup Links](https://www.theurlist.com/aspnet-standup-20260106)
