# Azure Weekly Update: November 7, 2025

*Author: John Savill's Technical Training*

Stay ahead with this week's Azure updates focused on new infrastructure, data, and developer tooling enhancements. The video covers:

## Key Highlights

- **HBv5 Series VMs**: Introduction of new hardware options for compute-heavy workloads
- **ExpressRoute Enhancements**:
  - Resiliency validation for improved reliability
  - Deeper insights and end-to-end connectivity monitoring
- **Azure NetApp Files (ANF)**:
  - Object REST API access for storage management
- **Storage Updates**:
  - Planned failover scenarios
  - Object replication metrics for tracking data redundancy
  - Ultra disks get new flexible provisioning
- **Database Improvements**:
  - **DocumentDB Kubernetes Operator**: Deploy and manage DocumentDB via Kubernetes
  - **Cosmos DB**: Query advisor and geospatial enhancements (e.g., distance ordering)
  - **MySQL**: Flexible dedicated ALB
  - **SQL Database**: Hyperscale multi geo-replica and easy restarts from portal
  - **PostgreSQL**: Flex Prem SSDv2 adds read replica capabilities
- **Developer & DevOps Features**:
  - **SSMS (SQL Server Management Studio)**: Integration with GitHub Copilot for improved developer productivity
  - Infrastructure automation topics, including Kubernetes and REST APIs
  - DevOps monitoring tools and infrastructure-as-code enhancements
- **Other Notable Mentions**: Azure MCP Server updates, ongoing learning and certification resources
