# GPT-5.4 Now Available in VS Code with GitHub Copilot

OpenAI's GPT-5.4 has just launched in Visual Studio Code via GitHub Copilot, bringing significant improvements in logical reasoning and handling of complex, multi-step, tool-dependent workflows.

- **Enhanced Logical Reasoning**: GPT-5.4 improves at interpreting developer intent and executing complicated tasks.
- **Multi-Step Task Execution**: The new model can break down large problems and coordinate tool usage within VS Code, benefiting real-world engineering scenarios.
- **Developer Integration**: This update is available now in GitHub Copilot for VS Code, as confirmed by the [GitHub changelog](https://github.blog/changelog/2026-03-05-gpt-5-4-is-generally-available-in-github-copilot/).
