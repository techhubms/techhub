# The GitHub Copilot SDK is here! - Rubber Duck Thursdays

This session introduces the GitHub Copilot SDK and its capabilities for developers. The presentation covers:

- What the GitHub Copilot SDK is and its main purpose

- Example use-cases where the SDK enhances productivity
