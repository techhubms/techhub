# AI Foundry for .NET Developers

Azure AI Foundry provides an end-to-end platform for .NET developers to build, test, and deploy AI-powered applications with confidence. This session walks through the process of leveraging AI Foundry capabilities in contemporary .NET development workflows.

## Key Takeaways

- **AI Model Integration**: Learn how to embed and work with AI models using Azure AI Foundry tools.
- **Prompt Management and Orchestration**: Manage prompts and orchestrations to optimize your AI model interactions.
- **Evaluation and Testing**: Implement systematic evaluation strategies and automated testing for AI systems.
- **Deployment to Production**: Discover streamlined deployment tools for transitioning from prototyping to reliable, production-ready AI applications.
- **.NET Ecosystem Integration**: See how AI Foundry connects with .NET 10, .NET Aspire, ASP.NET Core 10, .NET MAUI 10, C# 14, F# 10, and Visual Studio 2026.
