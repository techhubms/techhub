# Innovation Session: Microsoft Fabric and Azure Databases – The Data Estate for AI

**Speakers:** Arun Ulag, Michel Porter  
**Event:** Microsoft Ignite 2025 (Session BRK1702)

## Overview

This session explores Microsoft’s latest innovations in data and AI, unveiling how Microsoft Fabric, Power BI, Azure SQL, and Azure PostgreSQL are delivering a unified, intelligent foundation for organizations aiming to differentiate with AI.

### Key Announcements & Themes

- **AI Transformation & High-Quality Data**  
  The role of high-quality, unified data is emphasized as a core element for successful AI-driven initiatives.
- **Microsoft Fabric Growth**  
  Milestones highlighted include adoption by 28,000 customers—80% of the Fortune 500.
- **Open Standards and Ecosystem Expansion**  
  Expansion of the Fabric ecosystem through open standards and new ISV (Independent Software Vendor) partnerships.
- **DocumentDB Open Sourcing**  
  DocumentDB is now open sourced, featuring AI-readiness and robust security features for broader community contributions.
- **VS Code Extension for PostgreSQL**  
  The general availability of a new Visual Studio Code extension boosts developer productivity when working with PostgreSQL on Azure.
- **Three-Step Framework for AI-Ready Data**  
  Introduction of a new framework using Microsoft Fabric to prepare data for AI, from ingestion and unification in OneLake to semantic knowledge curation.
- **Fabric IQ Launch**  
  Announcing Fabric IQ, which brings semantic models to both operational and AI scenarios.
- **Customer Story: EY**  
  Showcasing EY’s successful digital transformation leveraging Microsoft Fabric’s unified analytics platform.

## Demo Highlights

- Live demonstrations of integrating Fabric, Power BI, Azure SQL, and PostgreSQL for real-time analytics and transactional workloads
- Practical guidance for organizations aiming to ready their data for AI

## Roadmap and Future Directions

- Insights into Microsoft’s roadmap for analytics, with a focus on operational efficiency, scalability, and new AI-driven features across the platform
