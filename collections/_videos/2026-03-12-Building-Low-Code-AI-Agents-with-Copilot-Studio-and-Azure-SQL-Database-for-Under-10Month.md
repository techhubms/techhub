# Building Low-Code AI Agents with Copilot Studio and Azure SQL Database for Under $10/Month

**Presented by:** Bob Ward, Principal Architect, Azure Data team

## Overview

This session walks you through creating a custom AI application that understands your data using Copilot Studio and Azure SQL Database. Designed for real-life use cases such as helping custom shoppers, the approach is low-code and cost-conscious—enabling you to deploy scalable AI agents for under $10 per month.

### Key Topics Covered

- **Introduction:** What you will learn and the scenario overview (AdventureWorks sample)
- **Prerequisites:** Getting started with Azure SQL Database (free options highlighted)
- **Building with Copilot Studio:** Step-by-step guidance using Copilot Studio's low-code environment to create AI agents
- **Walkthrough:** Navigating the Azure portal, integrating with data, and testing the AI agent
- **Deployment & Budgeting:** Final results discussion and a breakdown of the total cost—demonstrating truly affordable AI integration

## Steps Demonstrated

1. **Blueprint/Overview:** Understanding the architecture of the solution
2. **AdventureWorks Scenario:** Framing the business problem with practical data
3. **Setting Up Azure SQL:** Using the free tiers for cost-efficient data storage
4. **Creating AI Agents:** Leveraging Copilot Studio for no-code/low-code agent development
5. **Testing:** How to validate and refine the AI agent in real time
6. **Scaling Up:** Adapting the approach for different use cases
7. **Budget Analysis:** Clear explanation of keeping the solution under $10/month

## Why Choose This Approach?

- Enables rapid prototyping and deployment of data-driven AI agents
- Suitable for non-developers or teams with limited coding experience
- Demonstrates real cost optimization using Azure's free or low-cost tiers
- Practical roadmap to scale solutions across various business scenarios
