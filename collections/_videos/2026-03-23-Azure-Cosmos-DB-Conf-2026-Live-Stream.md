# Azure Cosmos DB Conf 2026 (Live Stream)

Azure Cosmos DB Conf 2026 is a free, global, developer-first conference focused on building high-scale, distributed, and AI-powered applications with **Azure Cosmos DB**.

- **Date:** April 28, 2026
- **Time:** 9:00 AM – 2:00 PM Pacific
- **Format:** Live on YouTube (5-hour live show)

## What the event covers

The agenda is described as practical and demo-driven, with sessions led by Microsoft engineers and community experts. Topics include:

- Real-world architecture patterns for distributed applications
- Performance tuning guidance
- Production lessons learned from real workloads
- Coverage of **Azure Cosmos DB**, **Azure DocumentDB**, and **open source DocumentDB**
