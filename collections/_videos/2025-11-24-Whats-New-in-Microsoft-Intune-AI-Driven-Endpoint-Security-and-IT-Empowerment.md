# What's New in Microsoft Intune: AI-Driven Endpoint Security and IT Empowerment

**Speakers:** Eugenie Burrage, Jason Roszak
**Event:** Microsoft Ignite 2025, Session BRK341

## Overview

This session dives into major updates across Microsoft Intune, focusing on:

- Deeper data exploration with Copilot
- Simplified workflows and cloud-native deployments
- Scaling with new agent-based capabilities

The session emphasizes creating resilient, secure, and productive IT environments to counter evolving threats, manage app updates, and ensure device compliance.

## Key Announcements and Features

### 1. Intune Recovery for Device Restoration

A new feature that streamlines device recovery operations for organizations, minimizing downtime and supporting robust device management.

### 2. Enhanced Application Inventory & Reporting

Significant overhaul of app inventory and reporting provides IT admins with deeper visibility across all managed endpoints. This enables more informed decision-making for compliance and updates.

### 3. AI Capabilities in Intune

#### a. Assistive and Agent Experiences

- Transitioning from traditional assistive experiences to more autonomous, agentic AI models
- Goal: Reduce manual admin effort and anticipate IT needs

#### b. Copilot Chat & Explorer

- Integrate natural language processing for device management, policy queries, and trouble-shooting
- Enable administrators to interact with Intune using conversational prompts

### 4. Endpoint Privilege Management (EPM)

A new dashboard centralizes the management of endpoint privilege, helping IT teams assign, audit, and control elevated privilege tasks securely.

### 5. Defender and Intune Integration

Leverages automation to remediate vulnerabilities, reducing attack surfaces with minimal human intervention. Integrates threat signals from Defender directly into Intune for seamless security operations.

## Chapters Highlights

- **Intune Recovery Announcement**: Streamlines device restoration
- **App Inventory Overhaul**: Modernizes reporting and visibility
- **AI Experiences**: Copilot Chat/Explorer for management in natural language
- **Privilege Management Dashboard**: Centralizes and controls endpoint privileges
- **Automation via Defender**: Automatically remediates vulnerabilities
