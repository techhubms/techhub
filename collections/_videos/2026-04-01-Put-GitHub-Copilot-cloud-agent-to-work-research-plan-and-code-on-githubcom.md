GitHub shares a short update on the GitHub Copilot cloud agent on github.com, highlighting that it can now research, plan, and make code changes without requiring you to open a pull request first, and letting you choose when a PR gets created.<!--excerpt_end-->

## Summary

GitHub Copilot cloud agent on **github.com** has been updated to be more flexible in how it helps you move work forward.

## What changed

You can now use the Copilot cloud agent to:

- **Research**
- **Plan**
- **Make code changes**

…**without needing to open a pull request first**.

Instead, Copilot gives you more flexibility by letting **you decide when** a pull request should be created for a code change.
