# NAT Gateway v2 Overview

## Introduction

This video presents an in-depth look at the new capabilities introduced in Azure NAT Gateway v2. The session is intended for professionals working with Microsoft Azure networking who need to understand recent service enhancements and best practices for deployment.

## Topics Covered

- **Why We Need NAT:** An explanation of the necessity of NAT in cloud architectures (timestamp: 00:12)
- **Azure NAT Gateway Architecture:** Overview of how NAT Gateway operates in Azure (07:21)
- **Creation and Subnet Linking:** Step-by-step details on deploying NAT Gateway and linking it to Azure subnets (08:38)
- **What's New in V2:** Key updates, including:
    - **Zone-Redundant Deployments:** Increases availability across Azure regions (12:08)
    - **IPv6 Support:** Expanded for modern networking needs (16:12)
    - **Performance Enhancements:** Improved data throughput and reliability (18:16)
    - **Flow Log Support:** Enables more detailed network monitoring (18:40)

- **Summary and Closing Thoughts:** Recap of the main points and guidance on further learning (20:00)
