# Simplifying Azure Local Deployments with Lenovo ThinkAgile MX and LOCA

**Speaker:** Adam Mandelbloom
**Event:** Microsoft Ignite 2025

## Overview

Lenovo ThinkAgile MX, integrated with Lenovo Open Cloud Automation (LOCA), is designed to simplify and accelerate Azure Local deployments. This session demonstrates:

- How IT teams and field technicians can deploy full Azure clusters with minimal manual effort using LOCA.
- Preparation of deployment templates including network and Azure subscription details.
- Secure technician access via password-protected files—eliminating the need for direct infrastructure access.
- Options for connection configuration and restoring factory settings for consistency.

## Key Features Highlighted

- **Automation:** Deploy clusters in a few clicks using LOCA.
- **Edge Focus:** Solution tailored for edge environments where rapid, remote setup is critical.
- **Reliability:** Emphasis on fully tested updates and non-disruptive cluster maintenance.
- **Lenovo/Microsoft Partnership:** Collaboration ensures up-to-date, predictable operations and streamlined maintenance.
- **Migration Tools:** Support provided for Azure Local migrations and future initiatives.

## Session Breakdown

- **Introduction and Overview:** Lenovo’s Adam Mandelbloom sets the scene for automating Azure Local deployments.
- **Demonstration:** Step-by-step live demo of cluster setup using LOCA, highlighting streamlined workflow for both administrators and technicians.
- **Administrator Workflow:** Deployments start with preparing templates containing all needed network and subscription information.
- **Technician Workflow:** Secure process to access deployment files, select options, and initiate setup without full infrastructure access.
- **Update and Maintenance:** Description of reliable, non-disruptive update mechanisms, ensuring high availability.
- **Partnership Initiatives:** Ongoing improvements through the Microsoft and Lenovo relationship, as well as migration tools for future Azure Local enhancements.
