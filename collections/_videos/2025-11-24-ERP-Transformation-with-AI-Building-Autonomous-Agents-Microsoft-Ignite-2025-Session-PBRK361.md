# ERP Transformation with AI: Building Autonomous Agents

**Speakers:** Jim Bretschneider, Jared Hall, Niels Jensen, Sameer Verma, Ben Vollmer
**Session:** Microsoft Ignite 2025, PBRK361

## Overview

Microsoft is moving beyond copilots to autonomous agents for ERP, aiming for businesses to automate daily operations with agentic frameworks. The session explains Microsoft's agent framework, adaptive ERP experiences, and technical mechanisms to build and deploy agents.

## Key Topics Covered

- **Agentic ERP:** Shifting from copilots to agents that autonomously run business processes.
- **Technical Deep Dive:** Microsoft’s agent framework, market releases, and the MCP server in ERP environments.
- **Integration:** How autonomous agents tie into Dynamics 365 for enhanced process automation.
- **LLM Capabilities:** Using large language models to process business data, generate communication, and enable automated reasoning.
- **Agent Ecosystem:** Enabling partners and customers to build tailored agents for vertical or industry needs.
- **Industry Examples:** RSM’s industry-specific agents and real-world customer engagement.
- **Partner Opportunities:** Programs, investments, and practical ways to accelerate AI adoption and specialization.

## Actionable Insights

- Companies can leverage Microsoft’s pre-built agents or develop their own using the agent framework.
- LLMs can translate unstructured business inputs into actionable, structured information for ERP systems.
- The MCP agent server architecture supports custom vertical and cross-industry solutions.
- Microsoft Ignite offers extensive resources for partners to specialize and innovate in ERP automation.

## Next Steps

- Explore more sessions and resources at [Microsoft Ignite](https://ignite.microsoft.com).
- Review programs in the Microsoft AI Cloud Partner Program to accelerate your agentic ERP initiatives.
- Build expertise in deploying and scaling autonomous agents for business process transformation.

## Conclusion

The session concludes with strategies for building AI practices, achieving industry specialization, and driving customer success through advanced business automation.
