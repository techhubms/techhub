# AI Dev Days: Building AI Applications with Azure and GitHub

Presented by Microsoft Developer, this video session kicks off AI Dev Days by diving into core topics for building modern AI applications.

## Event Focus

- **Azure & GitHub Integration**: Explore how Azure and GitHub enable streamlined and scalable AI development workflows.
- **Agentic DevOps**: Learn how Agentic DevOps approaches bring automation and intelligence to developer practices.
- **Azure SRE Agent**: Introduction to Microsoft's advanced Site Reliability Engineering agent for Azure environments.
- **Microsoft Foundry & MCP**: Get hands-on with the latest tools, including Microsoft Foundry and Model Customization Platform (MCP), showcasing real-world implementations.
- **Model Innovations**: Review current model capabilities (including GPT, GPT-5) and AI features.

## Agenda Highlights

- **Hands-On Lab**: Apply new product features directly in AI application projects
- **Industry Updates**: Key takeaways from Microsoft Ignite and GitHub Universe 2025
- **Practical Demos**: See AI solutions in action and get tips for immediate implementation

## Technologies Covered

- Azure AI and Agentic workflows
- GitHub for code management and collaboration
- Python for AI development
- Latest generation models (e.g., GPT-5)

## Who Should Watch?

- Software developers, engineers, and technical leads building AI-powered apps
- DevOps practitioners looking to integrate AI into automation pipelines
- Anyone interested in using Microsoft cloud and open source tools for machine learning and AI

## Key Takeaways

- Guidance for building robust AI applications leveraging Microsoft cloud platforms
- Real-world examples of Agentic DevOps and SRE agent deployments
- Actionable insights from recent industry events
