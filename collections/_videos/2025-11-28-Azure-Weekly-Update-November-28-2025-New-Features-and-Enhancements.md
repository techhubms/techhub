# Azure Weekly Update: November 28, 2025

Join John Savill's Technical Training for a rundown of significant updates and features in Microsoft Azure this week. This comprehensive video covers:

## Key Updates

- **Azure Functions Flex Custom Handler**: Introduction to flexible custom handlers for Azure Functions, enabling tailored execution environments for serverless apps.
- **Azure Bastion Portal RDP Entra ID Integrated**: Secure RDP access via Azure Bastion, now supporting identity authentication with Microsoft Entra ID.
- **Network Watcher Managed Identity Support**: Enhanced monitoring capabilities with Network Watcher now integrating managed identities.
- **Azure File Sync**: New regional support rolled out for New Zealand North, streamlining hybrid file share deployments.
- **Azure SQL Database Regex in T-SQL**: Regular expression support for more advanced QUERY logic in Azure SQL databases using T-SQL.
- **Azure MCP Server MySQL Support**: Additional flexibility as MCP Server now accommodates MySQL deployments.
- **AML Low-Priority VM Retirement**: Retiring low-priority VMs in Azure Machine Learning workloads; guidance for migration and workload planning.
- **Azure Load Testing in Italy North**: Public preview or regional launch of Azure Load Testing, available in Italy North for scalable performance benchmarking.
- **API Management and APIC MCP Support**: Updates to the API Management platform (APIM) and APIC with new MCP integrations for modern API infrastructure.
- **Claude Opus 4.5 in Foundry**: The Claude Opus 4.5 model is now accessible via Azure Foundry, broadening AI and ML toolsets.
