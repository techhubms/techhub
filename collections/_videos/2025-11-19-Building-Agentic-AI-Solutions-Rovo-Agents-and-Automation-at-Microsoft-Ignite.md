# Building Agentic AI Solutions: Rovo Agents and Automation at Microsoft Ignite

**Speaker:** Karen Sartor  
**Event:** Microsoft Ignite 2025, Session ODSP1486  
**Level:** Intermediate (200)

---

## Overview

This session introduces practical strategies for building and deploying Rovo agents using agentic automation principles. Both beginners and experts will gain best practices for defining agent use cases, creating action-focused workflows, and solving team problems—without coding.

### Key Topics

- **Concept of Mini Agents:** Using specialized mini agents that collaborate in real-world scenarios.
- **Identifying Workflow Opportunities:** Steps for spotting agent automation possibilities in daily team routines.
- **Templates for Automation:** Automating common tasks such as release notes and community blog generation.
- **Creating Agents from Scratch:** Best practices for agent setup and configuration, including prompt design.
- **Prompt Engineering:** Why specific and detailed prompts are vital for effective agent performance.

## Session Chapters

- **Agenda:** How to identify, build, test, and improve AI agents using prompts and orchestration (0:00)
- **Scenarios:** Mini agents working together (4:28)
- **Workflow Identification:** Finding automation opportunities (5:04)
- **Templates for Automation:** Release notes and blog workflows (7:48)
- **Agent Creation:** Step-by-step configuration (8:30)
- **Prompt Crafting:** Maximizing agent effectiveness (11:35)
- **Conclusion:** Final strategies and presenter remarks (14:20)
