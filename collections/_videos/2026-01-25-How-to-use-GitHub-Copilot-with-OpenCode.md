# How to use GitHub Copilot with OpenCode

GitHub demonstrates how developers can use their existing Copilot subscription (Pro, Business, or Enterprise) with OpenCode—a new open-source agent designed for terminal and desktop environments.

## What You'll Learn

- Which Copilot subscription types are supported (Pro, Business, Enterprise)
- Step-by-step process to connect your Copilot credentials to OpenCode
- How to start leveraging Copilot’s AI coding capabilities inside terminal and desktop workflows

## Key Points

- **OpenCode** is an open-source agent that extends Copilot’s usability beyond traditional environments.
- You can use GitHub Copilot with OpenCode if you have an eligible subscription plan.

## How to Connect Your Credentials

1. Open OpenCode on your terminal or desktop.
2. Select the GitHub Copilot integration option.
3. Log in with your GitHub credentials associated with your Copilot subscription.
4. Authorize OpenCode to access Copilot on your behalf.
5. Confirm the setup and start using Copilot features within OpenCode.

For detailed instructions and troubleshooting, visit the GitHub Docs or OpenCode official documentation.
