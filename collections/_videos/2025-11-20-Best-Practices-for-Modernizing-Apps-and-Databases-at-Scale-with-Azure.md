# Best Practices for Modernizing Apps and Databases at Scale with Azure

## Overview

This Microsoft Ignite 2025 session demonstrates how leading enterprises are transforming legacy applications and databases into agile, cloud- and AI-ready solutions. The speakers discuss best practices, customer success stories, and the newest Azure innovations to support scalable modernization.

## Key Topics

- Migrating legacy systems to Azure cloud
- Integrating AI with applications and data stores
- Real-world examples of enterprise digital transformation
- Best practices for app and database modernization
- Accessibility and multilingual support in digital sessions
- Unified SecOps modernization with platforms like GitHub
- Collaboration and cross-team strategies
