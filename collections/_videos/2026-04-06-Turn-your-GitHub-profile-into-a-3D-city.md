# Turn your GitHub profile into a 3D city

GitHub highlights **GitCity**, an open-source project that visualizes your GitHub profile as a **3D pixel art city**.

## How the visualization maps to your GitHub activity

- **Contributions** determine **building height**
- **Repository count** determines the **city width**
- **Stars** “light up” the **windows**

## Tech stack

- **Next.js 15**
- **Three.js**

## Exploration feature

- Try **flight mode** to move around and explore your generated city.
