# Building Connected Managed Agentic Apps with Microsoft Foundry (Ignite BRK113)

## Session Overview

Agentic applications are changing how organizations drive innovation with AI. However, deploying production-ready agents demands more than advanced models—it requires a unified, AI-native application platform. This Ignite 2025 session highlights how Microsoft Foundry enables:

- **Creation, deployment, and scaling of AI agents** that leverage real-time business context.
- **Integration of custom tools** and workflows (with a focus on Python-based code creation).
- **Comprehensive governance** with central policy and visibility management via AI Gateway.

**Speakers:** Robert Finlayson, Shawn Henry, Mike Hulme, Balan Subramanian

## Key Chapters and Demonstrations

- **0:00:** Challenges in deploying AI agents and governance complexities
- **4:21:** Introduction to Microsoft Foundry for AI agent management
- **5:32:** New capabilities for the Microsoft Foundry platform
- **13:44:** Turnkey platform for building apps and agents
- **25:55:** Sean demonstrates building agents with Python code
- **28:37:** Creating a multi-agent coupon generation workflow
- **32:43:** Connecting AI workflows to business process automation
- **40:01:** AI Gateway integration for centralizing policy and visibility
- **42:45:** Summary—end-to-end integration and governance of enterprise AI agents

## Practical Highlights

- **Code-Based Agent Creation:** Step-by-step demonstration using Python in Foundry to create and manage agents.
- **Business Process Automation:** Integration of workflows to automate enterprise processes.
- **AI Governance:** Using AI Gateway for unified management, policy enforcement, and visibility.
- **Real-World Impact:** Case study from KPMG on deploying Microsoft Foundry to accelerate business outcomes with enterprise AI.
