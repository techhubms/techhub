# From Code to Real Results: AI Impact with Foundry Models at Microsoft Ignite

This session, BRK191 from Microsoft Ignite 2025, features technical leaders from Adobe, eClinicalWorks, and others, sharing how they leverage Microsoft Foundry models to accelerate AI innovation on Azure.

## Session Highlights

- **AI-driven Customer Experience Solutions**: Overview of Sierra’s use of AI to enhance customer interaction.
- **Model Breadth and Partnerships**: Foundry’s diverse model portfolio and strategic collaborations.
- **Customer Service Demo**: Real-world application of AI in customer service scenarios.
- **Omnichannel Experience**: The Conversation Relay framework enabling seamless interactions across channels.
- **Costar Group Case Study**: Leveraging Azure AI for advanced home search solutions.
- **Document Processing Architecture**: Deep dive into document retrieval and processing systems using Azure services.
- **Accelerating Review with AIR**: Integrating AIR to streamline document review workflows.
- **Enterprise Security & Reliability**: Details on security and reliability features essential for deployment at scale.
- **Customer Testimonials**: Feedback highlighting Azure’s trust, transparency, and accuracy.

## Technical Takeaways

- Practical architecture diagrams illustrating document processing and AI agent integration
- Strategies for rapid AI deployment and iterative model improvement
- Security, compliance, and reliability considerations for enterprise AI on Azure
