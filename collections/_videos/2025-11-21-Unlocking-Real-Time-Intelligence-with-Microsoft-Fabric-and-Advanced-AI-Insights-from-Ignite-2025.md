# Unlocking Real-Time Intelligence in the Era of AI

### Presented at Microsoft Ignite 2025 by Yitzhak Kesselman, Fred Bellio, Tessa Kloster, Bharathi Rajan, Ashokkan Somuveerappan, and Will Thompson

## Overview

Real-Time Intelligence is redefining how organizations access and leverage data insights for immediate, informed decision-making. This session examines the evolution, implementation, and transformative impact of Real-Time Intelligence using Microsoft's data platform technologies.

---

## Key Topics Covered

- **Industry Use Cases**
  - Adoption in hospitals (patient care, operations)
  - Hospitality analytics (hotels, traveler behaviors)

- **Operational Impact**
  - Identifying bottlenecks like luggage delays
  - Improving customer experiences through real-time analytics

- **Addressing Cost and Value Misconceptions**
  - Dispelling myths around real-time intelligence investment

- **Integration with Microsoft Fabric**
  - Unifying data sources for actionable intelligence
  - Scalable graph analytics and advanced data processing

- **Microsoft Sentinel Integration**
  - Enhancing security monitoring with real-time data insights
  - Large-scale event detection

- **Copilot Data Exploration**
  - Harnessing natural language for rapid data exploration
  - Demo: AI-driven insights from operational datasets

- **Machine Learning and Anomaly Detection**
  - Automatically surfacing unknown data anomalies
  - ML-powered detectors for continuous improvement

- **Preview Feature: Operations Agent**
  - Live feedback loop with customer-driven enhancements
