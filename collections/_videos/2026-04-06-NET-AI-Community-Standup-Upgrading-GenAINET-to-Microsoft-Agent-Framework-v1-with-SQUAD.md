# .NET AI Community Standup: Upgrading GenAINET to Microsoft Agent Framework v1 with SQUAD

In this session, the **dotnet** team upgrades real samples from **Generative AI .NET for Beginners (GenAINET)** to **v1 of the Microsoft Agent Framework (MAF)**.

The workflow uses **GitHub Copilot CLI** together with **SQUAD** to coordinate **AI agents directly in the repository** and accelerate the upgrade process.

As an extra step, the session extends the samples with new capabilities, including scenarios that leverage **Microsoft Foundry Hosted Agents**, to demonstrate moving from **local experiments** to **scalable, production-ready AI systems** in **.NET**.
