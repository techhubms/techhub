# .NET AI Community Standup: Building AI Apps with the New .NET AI Stack

Join Bruno Capuano and Jeremy Likness for an open, unscripted session diving into what it takes to build AI applications with .NET today. This community standup skips the slides and scripts, opting instead for real conversation, demonstrations, and C# coding examples.

## What You'll Learn

- **Building agents in .NET:** Insights into implementing AI-powered agents and how the DevUI approach changes traditional application design.
- **Moving beyond chatbots:** Exploration of modern AI application patterns, how to transcend simple chatbot experiences, and build more sophisticated intelligent solutions.
- **New .NET + AI building blocks:** An overview of new libraries, toolkits, and practical utilities that make AI integration in .NET more accessible and robust.
- **Lessons learned:** Candid discussion of what works, what doesn't, and tips you can apply directly to your own projects.

## Session Highlights

- Informal, energetic, and interactive Q&A
- Live C# code demos (no slides)
- Open sharing of practical AI development lessons
