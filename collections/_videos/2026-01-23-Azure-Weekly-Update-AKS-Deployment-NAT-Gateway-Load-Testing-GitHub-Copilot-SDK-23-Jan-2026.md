# Azure Update - 23rd January 2026

**Presented by John Savill's Technical Training**

## This Week's Azure Announcements

- **AKS Deployment Safeguards**: Enhanced mechanisms for safer Kubernetes deployments on Azure Kubernetes Service (AKS).
- **StandardV2 NAT Gateway**: New networking option for Azure workloads, providing optimized outbound connectivity.
- **User Delegated SAS for More Services**: Expanded support for using Azure Storage's User Delegated Shared Access Signatures, increasing secure access options for services.
- **Azure File Share (AFS) in Israel Central**: Azure File Share is now available in the Israel Central region, enabling data residency compliance and local storage options.
- **Azure NetApp Files (ANF) App Volume Group for Oracle Data Protection**: Improved data protection strategies for Oracle workloads using ANF's app volume groups.
- **Azure Load Testing New Region**: Azure Load Testing service now available in additional regions, increasing test coverage flexibility for global deployments.
- **App Testing Reporting Enhancements**: New reporting capabilities streamline analysis and automation of app test results.
- **GitHub Copilot SDK**: The latest development kit supporting integration of GitHub Copilot code suggestions and AI assistance directly into developer workflows.
