# VS Code 1.107 Release Highlights

In this video, Reynald Adolphe presents several important updates introduced in the Visual Studio Code 1.107 release. Here are the main improvements discussed:

- **Inline Chat Editing**: Enhanced editing capabilities that allow more efficient use of inline chat features directly in the code editor.
- **Smarter Rename Suggestions**: Improved rename suggestions provide more accurate and context-aware options for refactoring code.
- **Persistent Local Agents**: Local agents now remain active even when the editor window is closed, improving background processes and automation.
- **Additional Improvements**: The release contains many other small but useful optimizations across the editor experience. For a comprehensive list of all changes, users are encouraged to review the official release notes.
