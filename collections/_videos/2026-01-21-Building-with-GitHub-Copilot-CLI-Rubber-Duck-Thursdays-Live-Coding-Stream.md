# Rubber Duck Thursdays: Building with GitHub Copilot CLI

In this episode of Rubber Duck Thursdays, GitHub hosts an informal and lighthearted live coding stream focused on experimenting with the GitHub Copilot CLI. This session is an opportunity for developers to see practical demonstrations of how the Copilot CLI can be leveraged from the command line and integrated into day-to-day development workflows.

## What You'll See

- Live coding with GitHub Copilot CLI
- Building and experimenting with command-line agents
- Workflow integration tips
- Informal Q&A and developer insights

## Key Takeaways

- **Getting Started:** How to launch and utilize GitHub Copilot commands directly from your terminal
- **Experimentation:** Learn by watching the team try out different use cases live
- **Integration:** See best practices for integrating Copilot CLI into your own coding and automation workflows

## About the Hosts

Presented by GitHub as part of their Rubber Duck Thursdays series, the session is designed for developers of all levels interested in seeing Copilot CLI in action and brainstorming new ways to improve daily development tasks.
