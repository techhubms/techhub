# Power Next-Generation AI Workloads with NVIDIA Blackwell on Azure

Presented by Microsoft Events at Microsoft Ignite 2025, this session explains how next-generation AI workloads are powered by NVIDIA Blackwell architecture integrated with Microsoft Azure cloud services.

## Key Topics

- **Agentic and Physical AI:** Introduction to new AI paradigms for autonomous reasoning and robotics.
- **NVIDIA Blackwell Architecture:** Breakthroughs in accelerated computing for complex AI tasks including multimodal inference and simulation.
- **Azure Collaboration:** Secure and flexible deployment leveraging Azure's capabilities for scalable AI solutions.
- **Multi-Dimensional AI Design:** Introduction of new framework and acronyms for designing advanced AI systems.
- **Extreme Co-Design & Mixture of Experts (MoE):** Techniques to optimize AI model communication and performance.
- **NVLink Acceleration:** Performance optimization through advanced GPU interconnects and Pareto curve analysis.
- **Confidential Computing:** Secure AI execution using Azure and NVIDIA's H100 GPUs for privacy and compliance.
- **RTX Pro 6000 Blackwell Server Edition:** Dedicated hardware for visualization and AI-focused workloads.
- **Omniverse Platform:** Use cases in manufacturing and warehouse optimization via simulation-driven AI agents.
