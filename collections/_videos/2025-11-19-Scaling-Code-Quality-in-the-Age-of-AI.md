# Scaling Code Quality in the Age of AI

**Speakers:** Caro Galvin (Senior Product Manager, GitHub), Marcelo Oliveira (VP, Product Management, GitHub)

## Overview

GitHub presents an exclusive look at how AI agents—especially GitHub Copilot—are changing application creation by integrating context around codebase, security, and coding standards. The talk explores innovations that enhance code quality and security across the entire developer workflow—from initial code writing to review and code lifecycle management.

## Key Topics Covered

- **Code Quality Standards**: Introduction of GitHub Code Quality and approaches to defining and enforcing coding standards for teams.
- **AI in Code Review**: Live demonstration of how AI agents support code review processes, automatically identifying and suggesting improvements aligned with quality requirements.
- **Fixing Legacy Code with AI**: Practical walkthrough of using AI tooling to modernize legacy codebases and apply updated security and quality checks.
- **Organization Dashboards**: Preview of new dashboards that provide high-level views of code quality and security across projects, helping organizations track improvements and enforce standards.
- **Advanced Security Integration**: How GitHub Advanced Security now works seamlessly with Copilot and other developer tools to proactively protect and guide coding practices.
- **Developer Experience Enhancements**: Insights into speeding up the entire workflow for teams and individual developers—moving from code creation to review and lifecycle management, aided by AI-powered automation.

## Demo Highlights

- Pull request automation for code quality enforcement
- AI-powered suggestions for security flaws and legacy code refactoring
- Dashboard analytics for organization-wide code and security metrics

## Takeaways

- AI agents in developer workflows can provide deep context, helping teams maintain high security and quality without slowing down productivity.
- GitHub Copilot and Advanced Security set new standards for safe, robust software development, leveraging automation and continuous improvement.
- Developers can expect evolving innovations making it easier to collaborate, ship quality code, and track security posture at scale.
