# What’s New in Azure Local: Portfolio Enhancements & Edge AI Innovations

**Speakers:** Meena Gowdar, Mohamed Khalid, Dean Paron  
**Event:** Microsoft Ignite 2025 (Session BRK147)

## Overview

This session dives into the latest developments in Azure Local, a solution enabling organizations to leverage Azure's capabilities within their own environments. The session is aimed at professionals seeking advanced (Level 300) insights into private and sovereign cloud adoption for regulated industries.

## Key Topics Covered

### Customer Use Cases & Industry Examples

- Real-world examples of how organizations use Azure Local to modernize operations and comply with industry regulations.

### Layers of Sovereignty & Private Cloud Role

- New sovereignty options supporting data residency and regulatory compliance.
- Azure Local as a private/sovereign cloud platform, enabling customers to maintain control over data and workloads.

### Microsoft Private Sovereign Cloud for Regulated Industries

- Introduction to Microsoft’s private sovereign cloud offering for sectors like healthcare, finance, and government.

### AI Innovations: NVIDIA RTX GPU & Edge AI Acceleration

- Updates on edge computing capabilities using NVIDIA RTX GPU support.
- Enhanced edge AI acceleration, enabling organizations to process AI workloads closer to where data is created.

### General Availability & Feature Enhancements

- General availability announcements for Azure Local.
- New features in migration, network security, and identity management.

### Flexibility Updates

- Additional updates that provide greater deployment flexibility for hybrid and edge scenarios.

### Industry Highlight: Healthcare

- Case study featuring GSK’s use of Azure Local for healthcare edge deployment, demonstrating the role of Azure Local in critical industries.

- Azure Local now provides expanded data sovereignty, edge AI capability, and reinforced security and identity tools for regulated and innovative environments.
- Portfolio enhancements are designed to address regulatory requirements, drive innovation, and offer flexible deployment wherever data is stored.
