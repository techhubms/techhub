# ASP.NET Community Standup: Build Agentic UI with AG-UI and Blazor

**Featuring:** Daniel Roth, Javier Calvarro Nelson, Tyler Slaton (CopilotKit)

The ASP.NET Community Standup explores how to design and develop agentic user interfaces (UIs) with AG-UI and Blazor, unlocking new ways for .NET developers to build intelligent and interactive web applications.

## What You'll Learn

- Principles of agentic UI: What are agentic UIs and why do they matter?
- How to use AG-UI with Blazor for building interactive, intelligent components
- Live demonstrations showing code samples and core concepts
- Insights from the developer community, including practical Q&A and tips

## Key Technologies Discussed

- **Blazor**: A powerful framework for building interactive web UIs with .NET and C#
- **AG-UI**: Toolkit for creating agent-based interfaces
- **CopilotKit**: Insights from Tyler Slaton on leveraging AI and agent capabilities in Blazor apps

## Demo Highlights

- Step-by-step walkthrough of integrating AG-UI into a Blazor application
- Implementing agentic behaviors to enhance user experience
- Best practices and common pitfalls when working with agent-based patterns in .NET

After watching, you'll understand how to start building intelligent, agent-driven web apps using the latest .NET tools. The session bridges practical development tips with emerging UI design concepts in the Microsoft web ecosystem.
