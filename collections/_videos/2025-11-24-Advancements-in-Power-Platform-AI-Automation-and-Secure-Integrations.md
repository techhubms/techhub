# Advancements in Power Platform: AI, Automation, and Secure Integrations

**Speakers:** Marc Schweigert, Tiffany Treacy  
**Event:** Microsoft Ignite 2025 (Session BRK324)  
**Type:** Advanced Breakout

## Session Overview

This session demonstrates recent advancements across Power Platform applications with an emphasis on leveraging AI and automation to create smarter, more secure enterprise solutions:

- **Power Apps:** Responsive Canvas App Authoring for adaptable UIs; Copilot integrations to simplify app development; new options for incorporating server-side logic and third-party systems.
- **Power Automate:** RPA (Robotic Process Automation) improvements; new automation features for more efficient workflow management.
- **Power Pages:** Demonstration of building secure sites from AI-driven prototypes; introduction of server-side logic; improved support for secure and flexible integrations.

## Key AI and Copilot Innovations

- Deep integration of AI, including Copilot Studio, enabling agent-first experiences across Power Platform products.
- Use of AI for data exploration, natural language queries, and in-app agents to deliver business insights (demonstrated through inventory data scenarios).
- Step-by-step demonstration of incorporating Copilot-powered features within Power Apps and researching with AI agents.

## Governance & Security Enhancements

- Introduction of more secure connectors and new governance capabilities to better manage low-code solutions at scale.
- Focus on observability to improve monitoring and compliance across Power Platform deployments.

## Sample Demos

- Responsive Canvas App with AI-powered authoring.
- Creating secure Power Pages sites using AI-generated prototypes.
- Using in-app agents for real-time data exploration and visualization.
- Demonstrating automation advancements via Power Automate workflows.
