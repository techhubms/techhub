# Top 5 New VS Code Agent Features to Improve Your Workflow

In this video, Olivia Guzzardo McVicker from the Visual Studio Code team showcases recent advancements that make agents in VS Code more practical and adaptable for developers. The presented features aim to let agents fit your development style, rather than forcing you to adapt.

## Featured Chapters & Features

1. **Agent Skills on Demand** (00:26)
   - Dynamically request specialized skills from agents as needed.
2. **Message Steering** (00:59)
   - Influence and direct agent responses with clearer communication and guidance.
3. **Integrated Browser** (01:30)
   - Browse documentation and resources directly from the editor to support the development process.
4. **Fork a Conversation** (01:52)
   - Split discussions with agents, maintaining context while exploring different solution paths.
5. **Hooks** (02:12)
   - Automate and trigger custom behaviors in response to specific agent activities.
