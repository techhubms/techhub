# Security Copilot: Automating and Accelerating Defense with Agentic Workflows

**Speakers:** Lizzie Heinze, Sarat Subramaniam  
**Event:** Microsoft Ignite 2025 | Session BRK238  
**Level:** Advanced (300)

## Session Overview

This session presents an in-depth look at Microsoft Security Copilot agents and how these AI-driven tools can automate critical security operations, accelerate response to threats, and facilitate smarter defenses across complex environments.

### Key Topics Covered

- **Automating Security Operations:**
  - How Security Copilot agents help streamline and automate security workflows
  - Custom and partner-built agents for tailored security solutions

- **Phishing Triage Agent Demo:**  
  - Live demonstration of a Security Copilot agent designed for phishing investigation and mitigation
  - Deep dive into agent workflow, decision factors, and impact

- **Measuring Agent Effectiveness:**
  - Overview of agent impact metrics for assessing value delivered
  - Techniques for building trust in agent decisions and training

- **Building Trust & Transparency:**
  - How Security Copilot enables transparency in agent actions
  - Interactive queries to explain individual agent decisions

- **Extending to Device & Identity Security:**
  - Expanding AI-driven automation to protect identities and devices
  - Transition from identity management to device-level security orchestration

- **Partner and Custom Agent Marketplace:**
  - Showcasing the security store marketplace for partner-generated and custom security agents
  - Deployment and orchestration of agents across multiple domains

- **Job Agent Orchestration:**
  - Managing agent workflows spanning phishing, identity, and device security domains
  - Best practices for operationalizing agentic security at scale
