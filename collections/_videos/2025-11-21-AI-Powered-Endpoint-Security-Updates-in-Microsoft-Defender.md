# AI-Powered Endpoint Security Updates in Microsoft Defender

## Session Overview

This Microsoft Ignite 2025 breakout session reviews how Microsoft Defender leverages AI to redefine endpoint security, spanning device onboarding to autonomous, proactive protection. Experts Dana Bargury Shina and Kim Kischel demonstrate how organizations can benefit from enhanced, real-time threat defense capabilities.

### Key Session Chapters

- **Threat Landscape Insights & Customer Listening**
- **Complexity of Protecting Diverse Devices**
- **Announcement: Expanded Support for Windows 7 & Server 2008 R2**
- **Mission Focus: Efficient & Customizable Endpoint Protection**
- **Customer Success Stories & Defender Effectiveness**
- **Proactive Defense: Just-in-Time Enforcement & Real-Time Protection**
- **Live Demo: Attack Detection & Predictive Shielding**
- **Threat Analytics Profile: Incident Contextualization**
- **Final Thoughts: Evolving Defender for Modern Threats**

## Technical Highlights

- **AI-powered endpoint protection:** Autonomous defense against diverse threats using real-time analytics and predictive shielding.
- **Legacy OS support:** Inclusion of Windows 7 and Server 2008 R2 in updated security mechanisms.
- **Proactive enforcement:** Just-in-time protection and enforcement against emerging attack vectors.
- **Threat analytics:** Dashboard profiles providing contextual threat insights during security incidents.
- **Security Demos:** Real-world scenarios showing effectiveness of Defender’s new features.
