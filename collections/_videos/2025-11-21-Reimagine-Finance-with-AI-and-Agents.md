# Reimagine Finance with AI and Agents | BRK354

**Speakers**: Sigita Cepaitiene, Matt Jutila, April Olson  
**Event**: Microsoft Ignite 2025  
**Session Type**: Breakout (Advanced, Level 300)

## Session Overview

This session demonstrates how leading finance teams can adopt AI-first strategies to automate financial processes, reduce manual work, and improve oversight by integrating Microsoft technologies like Azure AI Foundry and Copilot Studio.

### Key Topics Covered

- **Introduction to Frontier Firms and AI-First Finance Strategies**
  - Changing role of AI in advancing finance operations

- **Microsoft Tools Supporting Finance Transformation**
  - Overview of Azure AI Foundry and Copilot Studio as key technologies
  - Real-life finance transformation examples using Microsoft's tech stack

- **Finance-Specific Copilot Features**
  - Integrating Copilot into tools such as Outlook and Excel
  - Automation of ERP data processing, transaction reconciliation, and troubleshooting
  - Examples of variance analysis and voucher mismatch automation

- **Human Oversight and Auditability**
  - Maintaining financial control while leveraging automation
  - Emphasis on audit readiness, compliance, and transparency

- **Copilot Studio for Custom Financial Agents**
  - Building custom AI-powered agents tailored for finance with low-code/no-code approaches
  - Simplification of agent creation without deep coding expertise
