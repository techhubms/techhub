# Safe and Scalable DevOps with GitHub Copilot AI Agents

**Speakers:** Luke Hoban, Chris Patterson  
**Event:** Microsoft Ignite 2025 Session BRK107

AI agents like GitHub Copilot are redefining how developers build software. This advanced session provides actionable guidance for scaling Copilot agents in enterprise environments:

## Session Overview

- **Copilot Adoption:** Performance impact and organizational strategies at GitHub
- **Interactive Collaboration:** Using AI agents for collaborative, multi-task development workflows
- **Agentic DevOps:** Introduction to Copilot coding agents for parallel task execution
- **Custom Copilot Agents:** How to create targeted AI agents and extend Copilot across GitHub, CLI, and VS Code
- **Automation Demos:** Demonstrations of Copilot custom agents automating development tasks, including management via the AI Controls interface
- **Governance & Security:** Leveraging GitHub’s built-in governance, oversight, and security tools while scaling agent deployment
- **Q&A Highlights:** Practical advice on running agentic tasks within VS Code and cloud or hybrid environments

## Key Best Practices

- Align agent adoption with developer workflows for maximum efficiency
- Enable oversight with GitHub’s security and governance capabilities
- Facilitate scalable customization for organization-wide Copilot deployment
- Automate routine development and DevOps tasks using Copilot custom agents
