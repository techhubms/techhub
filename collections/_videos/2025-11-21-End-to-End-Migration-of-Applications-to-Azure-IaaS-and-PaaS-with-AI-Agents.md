# End-to-End Migration of Applications to Azure IaaS and PaaS with AI Agents

Presented at Microsoft Ignite 2025, this session demonstrates how Azure’s new AI-powered migration agents simplify the migration of applications, databases, and dependencies from VMware environments into Azure’s IaaS and PaaS platforms.

**Key Topics Covered:**

- **Demo Setup:** Step-by-step migration from VMware to Azure using Copilot migration agent
- **Business Case and ROI Analysis:** Evaluating the value and risks of migration with built-in AI planning tools
- **Discovery and Assessment:** Using migration agent to automatically analyze applications and dependencies
- **Modernization Demo:** Real-time migration and modernization to Azure, with application readiness checks
- **AI Readiness:** Discussion about transitioning to AI-enabled apps post-migration
- **Platform Features:** Overview of Azure Accelerate program, platform landing zone configuration, and future customization options
- **Q&A:** Answers on environment prerequisites, workload boundaries, and new automatic application fingerprinting features
