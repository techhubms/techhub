# Top 10 Community Projects Built with GitHub Copilot SDK

GitHub challenged the Reddit community to build anything they could imagine using the GitHub Copilot SDK in just one weekend. The results were impressive: from an autonomous agent that plays Stardew Valley to a fully functional AI-powered web browser. This video features the ten most creative and technically sophisticated submissions, pushing the boundaries of what developers can achieve with Copilot technology.

## Highlights from the Challenge

- **Autonomous Agent for Stardew Valley:** Leveraging Copilot SDK to build a bot that interacts with and plays the game autonomously, demonstrating both game automation and AI-driven gameplay.
- **AI-Powered Web Browser:** A submission that integrates Copilot SDK to allow AI-powered browsing, showcasing creative interaction between web technologies and AI.
- **Other Standouts:** The remaining projects encapsulate a variety of real-world uses and novel applications, all built within the time constraint of one weekend.
