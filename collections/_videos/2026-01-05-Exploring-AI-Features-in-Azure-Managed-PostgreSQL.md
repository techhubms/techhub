# Exploring AI Features in Azure Managed PostgreSQL

**Author:** John Savill's Technical Training

This video takes a technical deep dive into the expanding role of AI in Azure Managed PostgreSQL, demonstrating how developers can harness Microsoft Azure's database services for enhanced intelligence and automation.

## Key Topics Covered

- **AI and PostgreSQL Overview**: Introduction to how Azure brings AI-powered features to its managed PostgreSQL service.
- **VSCode Extension**: Demonstrates connecting to PostgreSQL via Visual Studio Code for enhanced developer productivity.
- **AI-Driven Management & Dashboards**: Highlights server dashboards and AI-helped operations within Postgres.
- **Schema Visualization**: How AI and tooling can generate schema diagrams and database insights.
- **GitHub Copilot Interaction**: Outlines ways to leverage GitHub Copilot for easier database development and code generation tasks.
- **AI Extensions for PostgreSQL**: Examination of plugins and extensions that bring cloud AI features directly into the database environment.
- **Embeddings and Vector Search**: Steps to create embeddings, build vector indexes with DiskANN, perform semantic search, and reranking results using AI advanced logic.
- **Integration with Azure Foundry**: Connecting Azure PostgreSQL with Azure Foundry for scalable AI and data engineering workflows.
- **Practical AI Workflows**: Generating content, extracting information, and enhancing query logic using integrated AI capabilities.
