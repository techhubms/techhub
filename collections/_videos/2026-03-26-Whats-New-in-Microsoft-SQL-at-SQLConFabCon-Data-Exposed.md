# What's New in Microsoft SQL at SQLCon/FabCon | Data Exposed

This Data Exposed episode recaps announcements and updates shared at **SQLCon/FabCon 2026 (Atlanta, March 2026)** about **Microsoft SQL** and related data-platform work, featuring **Priya Sathy** (Microsoft SQL product leader) with insights from **Bob Ward** and **Anna Hoffman**.

## Video chapters

- **0:00** Introduction
- **3:10** What’s new in Microsoft SQL at SQLCon
- **8:28** What’s new in AI – SQL Developers Certification
- **10:50** Database Hub in Fabric

## Key topics mentioned

- Updates across **Microsoft SQL** (including **Azure SQL** and SQL Server as part of the broader SQL family)
- “What’s new in AI” as it relates to SQL, including a **SQL Developers Certification**
- **Database Hub in Microsoft Fabric**, positioning databases and Fabric together on a single data platform
