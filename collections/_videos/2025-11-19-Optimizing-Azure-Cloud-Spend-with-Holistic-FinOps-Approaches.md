# Optimizing Azure Cloud Spend with Holistic FinOps Approaches

**Speaker:** Alexander Gahrmann | Microsoft Ignite 2025

This session explores how Finout manages and optimizes cloud spend across Microsoft Azure and integrated services like Kubernetes, Datadog, and OpenAI. Key elements include:

## MegaBill Unified Cost View

- Holistic breakdown of Azure resources, subscriptions, and associated tags
- Enables detailed exploration of cloud expenses for granular insights

## Virtual Tags for Dynamic Cost Allocation

- Rules-based, flexible tagging system for allocating costs across teams and departments
- Supports dynamic assignment to reflect organizational structure and usage

## Shared Cost Distribution and Visualization

- Demonstrates techniques for distributing shared cloud costs among teams and stakeholders
- Tools for showbacks and chargebacks, providing clarity in cost ownership

## Dashboards and Analytics

- Custom dashboards to visualize cloud expenditure, projections, and historical trends
- Real-time analytics for proactive cost management

## Anomaly Detection & Automated Notifications

- Systems to identify abnormal spending patterns
- Automated alerts and notifications to relevant stakeholders

## Integration of Telemetry and Unit Economics

- Ingesting telemetry data to provide actionable insights on resource efficiency and unit economics within Azure

## FinOps Support and Best Practices

- Guidance on structuring FinOps within organizations
- Practical tips for improving cloud efficiency and ongoing cost management
