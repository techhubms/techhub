Microsoft Developer hosts Shawn Henry and Rong Lu as they walk through the Microsoft Agent Framework (MAF) 1.0 GA release, covering stable APIs, multi-agent handoff orchestration, Python and .NET support, plus the Foundry Toolkit for VS Code with debugging, testing-style evaluation, and GitHub Copilot integration.<!--excerpt_end-->

## Overview
Microsoft Agent Framework (MAF) has reached version 1.0 (GA), with the goal of making production-grade agent development feel closer to “normal” software development.

In this episode, Shawn Henry and Rong Lu cover what’s new in the v1.0 GA release and demonstrate an end-to-end developer workflow from local development to deployment using Foundry.

## What’s new in Microsoft Agent Framework 1.0 (GA)
- GA release with **stable APIs**.
- **Multi-agent orchestration** with **handoff patterns**.
- Support for **Python** and **.NET**.

## Demo highlights
### Getting started with MAF in VS Code
- Demo uses a **VS Code extension** to get started.

### Building agents with Foundry Toolkit for VS Code (GA)
Foundry Toolkit for VS Code (formerly **AI Toolkit**) is now GA and aims to provide a unified IDE experience:
- A single **“Create Agent”** entry point.
- **Agent Inspector** for **F5 debugging**.
- **Evaluation-as-tests** using **pytest**.
- **Deep GitHub Copilot integration**.

### Multi-agent workflows
- Demo shows creating and iterating on a **multi-agent workflow**.
- Includes debugging multi-agent behavior using **Agent Inspector**.
