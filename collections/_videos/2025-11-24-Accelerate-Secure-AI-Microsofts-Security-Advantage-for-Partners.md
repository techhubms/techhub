# Accelerate Secure AI: Microsoft’s Security Advantage for Partners

**Session Speakers**: Steve Dispensa, Ann Choi  
**Event**: Microsoft Ignite 2025  
**Session ID**: PBRK272

## Overview

This breakout session explores how Microsoft’s integrated security portfolio and AI-driven solutions continuously create value for partners serving customers of all sizes. Keynotes address the transformative role of AI in security, practical strategies for building resilient, data-safe systems, and specific partner opportunities.

## Key Points

- **Security Trends and Challenges**: Insights into modern security threats and how Microsoft addresses them at global scale.
- **AI’s Impact on Security**: AI adoption is accelerating, raising concerns about trust and regulation. Microsoft emphasizes foundational AI safety, protecting data, and building resilient infrastructures.
- **Unified Security Approach**: Microsoft’s platform integrates security solutions for enterprise, corporate, and SMB customers, streamlining protection against modern threats and AI infrastructure vulnerabilities.
- **Partner Opportunities**: Guidance on leveraging Microsoft’s resources, commercial strategy for FY26, and expanding security-AI synergies through collaboration.
- **Major Announcements**: Introduction of Agent 365, Security Copilot, and Sentinel Graph, demonstrating advancements in Microsoft’s security ecosystem.
