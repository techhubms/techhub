# Unleashing SAP Databricks on Azure: Modernize, Analyze, and Innovate

**Speakers:** Anavi Nahar, Shanku Niyogi

Microsoft Ignite 2025 session BRK136 delivers insights into modernizing your data platform by integrating SAP Databricks natively on Azure. Key highlights include:

## Introduction

- Azure Databricks as a Microsoft first-party service
- Joint goals in the Microsoft–SAP partnership
- Leveraging SAP and external data for enterprise analytics

## Technical Deep Dive

- **Delta Sharing:** Enables seamless data access between SAP Business Data Cloud and Azure Databricks without duplication.
- **Live Demo:** Workflow on integrating SAP data, performing exploratory analysis, and analyzing model predictions (feature importance, residual diagnostics).
- **Agents:** Tools that facilitate joining unfamiliar SAP data sets, improving understanding and collaboration.
- **Automation:** Automated forecasting and exploratory analysis use cases demonstrated.

## Power BI Integration

- Direct demonstration of using Power BI with Azure Databricks for business intelligence and dashboarding.
- Discussion on building intelligent apps and generating actionable insights.

## Use Cases and Benefits

- Unified data analytics and machine learning leveraging trusted SAP and third-party business data
- Real-world BI applications, data collaboration, and data product creation on Azure

SAP Databricks on Azure delivers a robust platform for enterprise-scale analytics, advanced AI/ML, and business intelligence, powered by seamless integration and native cloud capabilities.
