# Azure Cost Estimation: Navigate Database Pricing

Join host Thomas Maurer and guest Maria Jose Fernandez from Microsoft Azure on this episode of the Azure Essentials Show as they explore database cost estimation in Azure.

## What this episode covers

### Three-layer framework for database cost estimation
A practical way to estimate Azure database costs by working through three layers:

1. **Architecture**: understand the target architecture and requirements.
2. **Computing services**: choose the database service and compute approach.
3. **Usage patterns**: factor in how the workload behaves and is consumed.

### Configuration decisions that drive monthly cost
The episode calls out five configuration choices that have a big impact on database spend:

- **Purchasing model**
- **Compute tier and size**
- **Storage**
- **High availability**
- **Savings options**

### Hands-on demo: Azure Pricing Calculator
Maria Jose demonstrates how to use the **Azure Pricing Calculator** to:

- Model different database scenarios
- Compare cost options and trade-offs
- Make decisions before committing to cloud database spending

## Recommended next steps

- Open the Azure Pricing Calculator and start modeling your own database workloads.
- Try different configurations for:
  - **Azure SQL Database**
  - **Azure Database for PostgreSQL Flexible Server**
- Export estimates to share with finance and engineering teams.
