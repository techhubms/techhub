# One View to Rule Them All: OneLake Catalog Explorer | Data Exposed

## Introduction

Navigating multiple workspaces and searching for the right datasets can be challenging in large organizations. The OneLake Catalog Explorer, part of Microsoft Fabric, solves this by providing a single, unified view for all your Fabric data items.

## Key Features Demonstrated

- **Unified Data Estate**: Browse and manage all Microsoft Fabric items from a central place, alleviating the need to switch workspaces.
- **Personalized Filters**: Apply targeted filters for quick access to relevant datasets, tables, or resources.
- **Trust Signals for Data Quality**: Quickly evaluate the reliability and quality of data using built-in trust indicators, improving governance and compliance.
- **Immediate Actions**: Take direct actions such as refreshing datasets and managing permissions without navigating away.
- **Consistent Experience**: Enhanced UI prevents users from losing their place when working across multiple resources.

## Demo Walkthrough (Summary)

- Step-by-step navigation of the OneLake Catalog Explorer interface
- Applying personalized filters to locate resources
- Exploring the trust signals system for identifying high-quality datasets
- Quick actions: triggering dataset refreshes and adjusting permissions inline
