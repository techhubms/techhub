# Introducing Agent Skills in VS Code

Agent Skills in Visual Studio Code enhance the developer experience by enabling the creation and customization of AI-powered coding features within the editor. This video covers the following topics:

## What are Agent Skills?

Agent Skills allow developers to extend VS Code using AI-powered capabilities. These can automate tasks, offer contextual code completions, or provide intelligent suggestions.

## Why Are Agent Skills Game-Changing?

- Developers can tailor the coding environment to specific needs
- Integration with VS Code streamlines workflows
- Simple creation process: get started with your first skill in minutes

## Demo: Creating Your First Agent Skill

The demo illustrates step-by-step how to:

- Access Agent Skills in VS Code
- Create a new skill from scratch
- Test your skill in the coding environment
