# The Download: Agentic Workflows, New AI Models, and GitHub Updates

**Hosted by Andrea (GitHub)**

In this episode of The Download, Andrea explores several significant updates for developers and the open-source community:

## AI Model Releases

- **GPT-5.3-Codex:** Now generally available for GitHub Copilot, bringing advanced code generation and AI capabilities to developers ([GitHub changelog](https://github.blog/changelog/2026-02-09-gpt-5-3-codex-is-now-generally-available-for-github-copilot/)).
- **Claude Opus 4.6 and Claude Sonnet 4.6:** These newer models are also now available for GitHub Copilot, enhancing AI pairing and coding assistance ([GitHub changelog](https://github.blog/changelog/2026-02-05-claude-opus-4-6-is-now-generally-available-for-github-copilot/)).

## GitHub Agentic Workflows

- **Agentic Workflows:** Introduced in technical preview, these workflows leverage AI (Copilot, Claude, and Codex) to automate common developer tasks — fixing CI failures, responding to code issues, and more. ([Agentic workflows changelog](https://github.blog/changelog/2026-02-13-github-agentic-workflows-are-now-in-technical-preview/))
- **Agent HQ:** Lets teams choose and manage their preferred AI agents, including new /fleet commands for streamlined automation ([Agent HQ blog post](https://github.blog/2026-02-04-pick-your-agent-use-claude-and-codex-on-agent-hq/)).

## Repository and Security Controls

- **Repo-Level Pull Request Controls:** New settings allow maintainers to configure pull request access more securely, helping to reduce spam and maintain project health. ([Changelog](https://github.blog/changelog/2026-02-13-new-repository-settings-for-configuring-pull-request-access/))

## Community and Industry News

- **OpenClaw Creator Joins OpenAI:** The developer known for OpenClaw heads to OpenAI, signaling further collaboration between projects ([OpenClaw at OpenAI](https://steipete.me/posts/2026/openclaw)).
- **CIA World Factbook Sunsetting:** Noted as an important transition for open data and the infosec community ([CIA announcement](https://www.cia.gov/stories/story/spotlighting-the-world-factbook-as-we-bid-a-fond-farewell/)).
- **25 Years of the Agile Manifesto:** Reflecting on the impact of agile practices in software development ([Agile Manifesto](https://agilemanifesto.org/)).
