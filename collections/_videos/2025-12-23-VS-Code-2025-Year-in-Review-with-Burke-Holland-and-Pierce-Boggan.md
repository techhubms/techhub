# VS Code 2025: Year in Review with Burke Holland & Pierce Boggan

## Introduction

Burke Holland and Pierce Boggan host a detailed walkthrough of Visual Studio Code’s 2025 milestones, spotlighting key features and the integration of AI-powered tools for developers.

## Major Themes

- **Copilot Universe and Agent Mode:**
  - Launch of Copilot Free & Universe, marking a pivotal point for AI-assisted development.
  - Introduction of Agent Mode, enabling autonomous code tools and workflow loops.
- **Extensibility and Model Choice (MCP + BYOK):**
  - March releases focused on model flexibility and extensibility.
- **Prompt Engineering Tools:**
  - April brought prompt files and multi-window chat, empowering interactive command workflows.
- **Cloud Agent & PR Automation:**
  - May’s features included parallel work via Cloud Agent and automating PR workflows for productivity gains.
- **Open Source Copilot Chat & Security:**
  - Release of open source Copilot Chat and improvements in terminal safety/sandboxing environments.
- **Workflow Advancements:**
  - Introduction of worktrees, checkpoints, Auto Mode for smarter routing, and background agents.
- **Community and Open Source:**
  - Growth in the MCP Gallery, Codex integration, and increased community engagement.

## Developer Impact

- Faster code generation with AI-powered Copilot and VS Code integrations
- Enhanced extensibility and configurability for personalized workflows
- Improved safety in terminal and agent operations
- Greater support for parallelization and smart automation
- Community-driven improvements and open personalization
