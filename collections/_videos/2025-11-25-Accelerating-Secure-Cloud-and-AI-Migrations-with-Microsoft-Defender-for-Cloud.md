# Accelerating Secure Cloud and AI Migrations with Microsoft Defender for Cloud

**Speakers:** Gabriela Perez Arredondo, Jonathan Stock  
**Event:** Microsoft Ignite 2025  
**Session:** PBRK271 (Intermediate)

## Introduction and Session Overview

This session opens with the importance of protecting critical business assets: the cloud, AI platforms, and essential applications. Security and innovation are prioritized through unified and intelligent solutions, specifically Microsoft Defender for Cloud.

## Key Topics Covered

- **Azure Evolution & Partner Business Models**: Overview of new Azure changes relevant to partners and strategic opportunities for evolving business models.
- **Real-world Partner Examples**: Case studies show partners utilizing AI-driven migration to create new service offerings and business models.
- **Unified Security Across Multi-Cloud and Hybrid Environments**: How Microsoft Defender for Cloud streamlines security for infrastructure spanning multiple clouds and on-premises.
- **Security Assessments and Building Trust**: Detailed process for using security assessments to identify customer needs, build trust, and unlock potential new engagements.
- **Integration of Security and Infrastructure Teams**: Best practices for addressing common organizational obstacles when merging security and infrastructure functions, including collaborative workflows.
- **Microsoft Engagement Methodology & Orchestration**: Methodologies for partner and customer engagement, with focus on organizational orchestration and driving security outcomes.
- **Azure Accelerate Plus Proof of Value (POV)**: Leveraging Azure migration and security incentives, technical workshops, and proof-of-value pilots to speed up secure adoption.
- **Cloud Security Recommendations & Workshops**: Tools and Microsoft resources for delivering actionable cloud security recommendations to customers, including the Cloud Security Ambitioning Workshop.

## Technologies Highlighted

- **Microsoft Defender for Cloud**: Central to safeguarding workloads, enabling unified management, threat detection, and mitigation across Azure, multi-cloud, and hybrid scenarios.
- **Microsoft Purview**: Information protection and compliance capabilities integrated into the security review process.
- **Azure Security Services**: Practical guides on orchestrating migration and delivery of secure cloud workloads.
