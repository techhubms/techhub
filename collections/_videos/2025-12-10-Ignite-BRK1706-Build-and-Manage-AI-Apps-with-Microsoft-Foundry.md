# Ignite BRK1706: Build and Manage AI Apps with Microsoft Foundry

Join Microsoft engineering leaders as they introduce the next generation of modular AI with Microsoft Foundry. Discover how Foundry enables developers to:

- **Build intelligent agents**: Quickly create and deploy AI-powered agents tailored for enterprise use cases.
- **Orchestrate workflows**: Utilize new workflow capabilities to sequence, monitor, and manage interactions between agents, data, and services.
- **Unify models and tools**: Integrate diverse models, data sources, and developer tools within a single, secure platform.
- **Ensure enterprise-grade security and observability**: Leverage comprehensive security controls and observability features for governance at scale.

## Key Features Highlighted

- Modular AI architecture using Microsoft Foundry
- Workflow orchestration and management
- Centralized security and compliance
- End-to-end observability for debugging and monitoring
