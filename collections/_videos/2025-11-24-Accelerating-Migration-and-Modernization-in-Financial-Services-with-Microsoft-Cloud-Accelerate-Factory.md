# Accelerating Migration and Modernization in Financial Services with Microsoft Cloud Accelerate Factory

## Overview

This breakout session from Microsoft Ignite 2025, delivered by Pradeep Raman and Jesse Sheldon, focuses on how financial services organizations are leveraging Microsoft’s Cloud Accelerate Factory (CAF) to speed up their cloud transformation efforts.

## Key Topics Covered

- **Industry Focus & Poll**: Introduction to challenges and opportunities in financial services.
- **AI Adoption**: Examination of AI as a common solution for modernizing and overcoming industry hurdles, including agentic AI's impact on economic transformation and speed of modernization.
- **Microsoft’s Partnership Approach**: Overview of Microsoft's dual focus on innovation and implementation support for financial enterprises.
- **Cloud Accelerate Factory Benefits**: Emphasis on hands-on support, zero-cost scalable delivery, and tools like Azure Arc for managing hybrid and multi-cloud environments.
- **Migration & Modernization**: Streamlining migrations, re-architecting legacy applications, and deploying new scalable solutions with AI-powered monitoring.
- **Embracing Automation**: Advocacy for reassessing legacy modernization assumptions and embracing agentic automation to improve agility and resilience.

## Solution Architecture Highlights

- **CAF Framework**: Structured to accelerate cloud migration and modernization.
- **AI-powered Monitoring**: Introduction and integration of intelligent monitoring across migrated workloads.
- **Azure Arc**: Extending Azure management and services to diverse environments.
