# Levi’s Global IT Transformation: Migration and Modernization with Azure

**Speakers:** Maria Bledsoe, Shawn Harris, Michael Womack

Microsoft Ignite 2025 session BRK156

## Overview

This session details Levi’s extensive IT transformation, focusing on migrating and modernizing their global infrastructure by leveraging Azure services. The journey includes moving major enterprise workloads—Windows Server, SQL Server, SAP, Oracle, and more—to Azure, utilizing both data and application modernization strategies and infrastructure re-platforming for improved agility, scalability, and cost efficiency.

## Migration and Modernization Approach

- Migrated workloads from legacy data centers to Azure (Windows Server, SQL Server, SAP, Oracle)
- Used **Azure Kubernetes Service (AKS)** and **Azure SQL** for scalable modernization
- Combined data and application modernization with infrastructure re-platforming
- Centralized cloud cost management program (FinOps)
- Focused on agility, scalability, and measurable cost savings

## Technical Insights

- Defined clear migration keys to success
- Built internal cloud skills within Levi’s teams
- Carefully grouped applications for phased migration
- Integrated AI to optimize technical workflows

## Lessons Learned

- Importance of upskilling existing teams on Azure
- Strategic grouping of apps enhances migration outcomes
- Centralized cost management (FinOps) is critical
- Starting small and scaling up helps with risk management
- Early integration of AI into workflows yields higher operational efficiency
