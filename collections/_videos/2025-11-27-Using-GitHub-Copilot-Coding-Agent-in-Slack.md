# Using GitHub Copilot Coding Agent in Slack

The GitHub app for Slack now allows direct collaboration with GitHub Copilot. With this integration, developers can use the Copilot coding agent right inside their Slack conversations.

## Key Features

- **Copilot Coding Agent**: Translates natural language conversations in Slack into code and GitHub pull requests.
- **Collaboration in Chat**: Streamlines code collaboration without leaving Slack.
- **Initial Release**: Focuses on beginning to integrate Copilot's capabilities into Slack, setting the foundation for future enhancements.

## How It Works

- Add the GitHub app to your Slack workspace.
- Interact with Copilot via Slack messages to generate code based on your discussions.
- Directly create pull requests from Slack, increasing workflow efficiency for distributed teams.

## Benefits

- Enhance team productivity by bringing code generation and collaboration into the tools you're already using.
- Reduce context-switching by integrating GitHub Copilot's AI-driven coding assistance inside Slack.
