# Automatic Passkey Rollout Update for Microsoft Entra ID

**Author:** John Savill's Technical Training

## Overview

This video provides an update on the new automatic passkey rollout for Microsoft Entra ID (formerly Azure Active Directory), focusing on recent changes that impact identity security and user authentication.

### Key Topics Covered

- **Benefits of Passkeys:**
  - Enhanced security over traditional passwords
  - Support for phishing-resistant authentication
- **Types of Passkeys:**
  - Synced passkeys (work across devices)
  - Device-bound passkeys (restricted to a single device)
- **Authorization Layer:**
  - Explains how the authorization process integrates with passkeys
- **What Is Changing:**
  - Overview of updates to the registration campaign
  - How the rollout affects organizations and end users
- **Registration Campaign Changes:**
  - Adjustments in user registration flow
  - Implications for IT admins and organizational security posture
