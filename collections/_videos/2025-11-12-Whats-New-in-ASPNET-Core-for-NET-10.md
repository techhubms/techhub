# What's New in ASP.NET Core for .NET 10

Explore the latest enhancements coming to ASP.NET Core in .NET 10, a session presented by dotnet that spotlights major updates for building modern web applications and APIs.

## Key Topics

- **Performance Improvements**: Demonstrations of optimizations in ASP.NET Core, helping developers build faster applications.
- **Enhanced OpenAPI Support**: New capabilities make it easier to document and consume APIs using industry-standard OpenAPI tools.
- **Authentication with Passkey Support**: Added security features through native passkey support alongside traditional authentication methods.
- **Improved Validation**: Minimal APIs now feature enhanced validation options, reducing boilerplate code and improving reliability.
- **Blazor Updates**: Further advancements in Blazor empower full-stack web app development within the .NET ecosystem.

## Benefits for Developers

- Write more maintainable and secure applications.
- Take advantage of the latest C# and ASP.NET Core features.
- Build and orchestrate cloud-native web apps efficiently.
- Benefit from improved productivity with evolving tooling like .NET Aspire and Visual Studio.
