# VS Code 1.109 Release Highlights

**Presented by Reynald Adolphe and the Visual Studio Code team**

## Overview

This release video covers new and improved features in Visual Studio Code version 1.109, designed to enhance the developer editing experience.

## Key Highlights

- **Ask Questions:**
  - New feature enabling developers to interactively ask questions within the editor for better assistance and documentation lookup.

- **Support for MCP Apps:**
  - Enhanced compatibility and tooling for MCP (Microsoft Cloud Platform) applications, streamlining cloud-focused development workflows.

- **Agent Skills Now Generally Available:**
  - Expanded and stabilized agent skills integration, supporting intelligent automation in editor workflows.

- **Subagents in Parallel:**
  - Subagents can now function in parallel, enabling advanced coding agent capabilities during the development process.

- **Editor Selection Improvements:**
  - Double-clicking to select bracket and string content is now supported, increasing efficiency for structured code editing.

- **Integrated Browser (Preview):**
  - Preview of an integrated browser, allowing web testing and live previews directly within the editor for a streamlined developer experience.

- **Third-Party Coding Agents (Preview):**
