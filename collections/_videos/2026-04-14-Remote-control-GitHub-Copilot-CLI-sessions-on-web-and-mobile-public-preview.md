# Remote control GitHub Copilot CLI sessions on web and mobile (public preview)

Need to step away from your desk but keep working in the same terminal session? GitHub Copilot CLI has a new **remote control** feature (in **public preview**) that lets you access your **active terminal session** from **any device with a web browser**.

## What’s new

- **Remote access to an active Copilot CLI terminal session**
- Works from **web** and **mobile** (via a browser)
- Enables you to **remote control your session** so you can continue managing work when away from your primary machine
