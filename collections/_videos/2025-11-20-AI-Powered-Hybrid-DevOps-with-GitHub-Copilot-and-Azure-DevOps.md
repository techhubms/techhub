# AI-Powered Hybrid DevOps with GitHub Copilot and Azure DevOps

**Presented by:** Dave Burnison, Dan Hellem  
**Event:** Microsoft Ignite 2025, BRK106

## Overview

This advanced session details how to modernize DevOps strategies using Agentic DevOps. It demonstrates hybrid migration from Azure Repos to GitHub and shows how to continue leveraging Azure Boards and Pipelines, with AI and agent-based workflows through GitHub Copilot and MCP Servers.

### Key Topics

- **Hybrid Mode:** Defining integration patterns between Azure Boards, Azure Pipelines, and GitHub repositories for work item traceability.
- **Copilot Integration:** Demos of GitHub Copilot supporting branch, commit, and pull request tracing, and YAML pipeline development.
- **Agent-Based Workflows:** How Copilot agents and MCP Servers enhance DevOps automation, streamline merge commits, cleanup, and template suggestions.
- **Automation Patterns:** Using Copilot for code reviews, duplication detection, and reusable pipeline templates.

### Session Chapters

- **Defining Hybrid Mode and Traceability:** Technical explanation of connecting Azure Boards with GitHub and Azure Pipelines.
- **Demo:** Creating and linking branches, commits, and pull requests for traceability.
- **Merge and Cleanup Automation:** Using agents for automatic merging and branch management.
- **Copilot in Pipelines:** Integrating Copilot coding agents for YAML pipeline generation and review.
- **Template Recommendations:** Copilot identifies duplicate pipeline code and recommends reusable templates.
- **Azure DevOps Core:** Deeper technical overview and best practices when using Azure DevOps tools.
- **Work Items/AI Context:** Automating user story creation from features informed by AI.
