# Unlock agentic workflows for your apps using MCP on Windows

This video session, presented by Microsoft Developer, explores how Windows supports agentic workflows through the Microsoft Copilot Platform (MCP). The discussion centers on connecting native applications, developer tools, and AI agents via MCP.

## Key Topics Covered

- **MCP Server Development**: Learn approaches for packaging and securing MCP servers to enable robust interactions between agents and Windows apps.
- **MCP Registry Integration**: Understand how to register MCP servers—both built-in and custom—so agents can seamlessly automate tasks.
- **Automation Scenarios**: Discover how agents leverage MCP capabilities to orchestrate and automate a variety of tasks across Windows environments, bringing AI-powered experiences into enterprise and developer contexts.
- **Security, Scalability, and Extensibility**: Explore how MCP ensures secure operation, scales across user and enterprise needs, and provides extensibility for future scenarios.
