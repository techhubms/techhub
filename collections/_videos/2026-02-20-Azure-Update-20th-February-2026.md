# Azure Update – 20th February 2026

**Author:** John Savill's Technical Training

## Introduction

Stay up to date with Microsoft's rapidly evolving Azure cloud platform! This update covers key new features and improvements that matter to cloud and DevOps professionals working with Azure.

## Featured Topics

### 1. Prem SSDv2 & Ultra Instance Snapshot Access

- Details on new snapshot access for Premium SSDv2 and Ultra disk types, helping accelerate backup and restore operations for performance-sensitive workloads.

### 2. AKS (Azure Kubernetes Service) Enhancements

- **Auto-provision Encryption-at-Host**: Improved security for containers, ensuring encryption at the virtualization host level.
- **LocalDNS for AKS**: Reduces DNS latency and enhances reliability in Kubernetes clusters.
- **Node Autoprovision (Government & Private Cloud)**: Enables rapid cluster scaling and improved support for highly regulated environments.
- **MCP Server for AKS**: Management/Control Plane improvements.
- **Agentic CLI for AKS**: Streamlined management and automation via the new CLI tools.

### 3. Azure Functions Updates

- **Flex Consumption Tools**: Newly available tools for more flexible serverless consumption.
- **Azure Functions .NET 10 Support**: First-class developer support for the latest .NET runtime in serverless scenarios.

### 4. Claude Sonnet 4.6 Availability

- Announcing new AI model availability for integration and custom scenarios (details in video).

### 5. Agent Monitoring Dashboard

- Updated dashboard for agent monitoring across Azure, improving visibility and operations.
