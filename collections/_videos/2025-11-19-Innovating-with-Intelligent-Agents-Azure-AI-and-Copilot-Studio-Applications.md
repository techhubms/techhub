# Innovating with Intelligent Agents: Azure AI and Copilot Studio Applications

**Speakers:** Renu Nair, Bhargav Parnandi
**Event:** Microsoft Ignite 2025 (Session ID: ODSP1401)

## Overview

This session dives into the future of application innovation utilizing Azure AI and autonomous agents. Learn how intelligent systems can sense, decide, and act, unlocking adaptability and operational efficiency.

### Key Chapters

- **Role of Microsoft Technologies in Agentic AI:** Foundations for developing intelligent agents on Microsoft platforms.
- **Copilot Studio Integration:** Approaches for agent prototyping and workflow design.
- **Demo:** Operational intelligence system leveraging AI-driven modules for task summarization and recommendations.
- **Case 360 Architecture:** A deep dive into building business impact solutions on Microsoft Power Platform and Azure AI.
- **Supporting Modules:** User validation, admin configuration, and modules for robust AI-enabled operations.
- **Integration for Business Impact:** How Azure AI agents enhance workflows for scalable, actionable intelligence.

## Real-World Strategies

- Designing scalable AI solutions using Azure and Power Platform.
- Enhancing agent efficiency through summarization, recommendations, and operational AI modules.
- Architectural best practices for integrating autonomous agents with Microsoft cloud services.
