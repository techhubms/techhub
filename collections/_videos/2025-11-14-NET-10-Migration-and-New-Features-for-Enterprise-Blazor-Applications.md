# .NET 10 Migration and New Features for Enterprise Blazor Applications

In this session, dotnet showcases the successful migration of a large-scale enterprise Blazor application to .NET 10. The session covers:

- **Upgrade Strategies:** Common approaches for upgrading existing applications to .NET 10, with practical advice for large-scale enterprises.
- **Performance and Capabilities:** Areas where .NET 10 provides measurable improvements over earlier versions.
- **ASP.NET Core Identity Passkeys:** Introduction to passkey authentication for enhanced security.
- **Integration of Modern Features:**
  - [.NET Aspire 13](https://aka.ms/aspire13/announcement): Simplified orchestration for cloud-native apps.
  - [ASP.NET Core 10](https://aka.ms/asp/dotnet/2025): Modern, secure, and full-stack web development.
  - [.NET MAUI 10](https://aka.ms/dotnet/maui/2025): Cross-platform mobile/desktop development improvements.
  - [C# 14](https://aka.ms/csharp14/announcement/2025): More expressive language capabilities for developers.
  - [F# 10](https://aka.ms/fsharp10/2025/announcement): Enhanced safety and functionality for functional programming.
  - [Visual Studio 2026](https://aka.ms/visualstudio2026/announcement): Smarter, faster IDE innovations targeting modern .NET workflows.
- **Resources & Community:**
  - Official [.NET blog](https://aka.ms/dotnet/blog), [documentation](https://learn.microsoft.com/dotnet), [forums](https://aka.ms/dotnet/forums), and links for further learning.

The session reinforces how .NET 10 represents a significant evolution in the .NET ecosystem, supporting builders of secure, high-performance, and cloud-native applications. Viewers are encouraged to explore the latest features using the linked resources.
