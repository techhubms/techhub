# Innovation in Microsoft Databases: AI-Ready Data and Modern Apps

## Overview

Presented at Microsoft Ignite 2025, this session led by Shireesh Thota dives into the latest innovations across Microsoft’s database services, showing how new features support the development of intelligent applications and enable robust data-driven experiences for organizations.

## Key Topics

- **SQL Server 2025:**
  - Introduction of AI-powered capabilities including semantic search and vector data types.
  - Enhanced security features for enterprise applications.
- **SQL Con Announcement:**
  - Launch of a dedicated SQL data conference in 2026 supporting collaboration and learning.
- **PostgreSQL Discussions:**
  - Microsoft’s community engagement and open-source contributions for PostgreSQL in the cloud.
- **Horizon DB Demo:**
  - Provisioning directly from VS Code.
  - Demonstrates AI integration: vector indexing, embeddings, search.
- **GPT-5 for Data Extraction:**
  - Use of AI (GPT-5) to automate and enrich SQL data extraction tasks.
- **DocumentDB Multi-Cloud Demo:**
  - Features global data consistency and failover across clouds.
- **Customer Spotlight – Adobe VP Platform AI Engineering:**
  - Real-world insights and implementation of AI within Microsoft data platforms.
- **Fabric Integration and Next-Gen Apps:**
  - Overview of Microsoft Fabric’s role in unifying data analytics and operational databases.

## Resources

- Detailed session recordings and related resources are available at the [Microsoft Ignite site](https://ignite.microsoft.com) and [Azure SQL AI development resources](https://aka.ms/ignite25-plans-AzureSQLAIDevelopment).

## Highlights

- Modern improvements in vector and semantic search for databases to support advanced AI and machine learning workloads.
- Community-enablement for open-source database technologies led by Microsoft.
- Live demos showcasing the integration of new features into developer workflows via VS Code.
- Adoption of multi-cloud strategies and robust consistency features.
