# What's New in Azure App Service for .NET Developers

Byron Tardif shares updates and demonstrations aimed at helping .NET developers leverage the full capabilities of Azure App Service for web APIs, websites, and microservices.

## Key Topics Covered

- **New Features for .NET Developers**: Overview of recent enhancements tailored for .NET 10, including improved deployment workflows and integration capabilities.
- **Deployment Workflows**: Updates on seamless deployment processes leveraging Azure DevOps and GitHub Actions for CI/CD.
- **Diagnostics & Monitoring**: Enhanced tools to troubleshoot applications and monitor performance for cloud-native solutions.
- **Performance Optimization**: Practical tips for scaling applications efficiently and optimizing response times in the cloud.
- **Security Best Practices**: Recommendations for securing web apps on Azure, with built-in features and configuration strategies.
- **App Service Integration**: Deep dive into new integration points with Azure DevOps and GitHub Actions, simplifying developer productivity.
- **Supported Technologies**: Insights on support for .NET 10, ASP.NET Core 10, .NET MAUI 10, C# 14, F# 10, and new Visual Studio versions.

## Actionable Demonstrations

- Step-by-step guides for deploying .NET applications using App Service
- Real-world examples of diagnostics and monitoring configurations
- Performance tuning approaches for scalable cloud applications
- Security configurations including authentication and authorization options for web apps
