# Building Production-Ready AI Agents with Microsoft Foundry

## Session Overview

This intermediate-level (200) session from Microsoft Ignite 2025 provides strategies and demonstrations for developers building agentic applications using Microsoft Foundry, Microsoft's unified AI-native platform.

### Key Topics Covered

- **Challenges in AI Agent Deployment & Governance**
  - Overview of complexities involved in deploying agentic applications at scale, including real-time business context integration and governance.
- **Microsoft Foundry Platform Capabilities**
  - Introduction and announcement of new features for centralized management of agents, policy enforcement, visibility, and custom tooling.
  - Turnkey solutions for building and deploying AI-powered agents.
- **Technical Demonstrations**
  - Live demonstration of code-based agent creation:
    - Example: Multi-Agent Coupon Generation Workflow using Python.
    - Integration of workflows to automate business processes.
  - AI Gateway integration for central policy, security, and visibility management.
- **Enterprise Integration & Governance**
  - Detailed walkthrough on connecting agentic workflows to real business process automation.
  - Discussion on policy-driven governance and visibility for enterprise systems.
- **Customer Success: KPMG Case Study**
  - Insights into accelerating AI innovation and achieving business impact using Microsoft Foundry in real-world enterprise scenarios.
