# Scale Smarter: Azure Infrastructure for the Agentic Era

**Presented by Microsoft Events at Ignite, with speakers Jeremy Winter, Nidhi Chappell, and Annie Pearl**

## Session Overview

Learn how Azure’s intelligent infrastructure and agentic cloud operations are redefining cloud reliability, security, and agility for every workload. This session highlights:

- Real-time data transformation (BMW example)
- Advancements in distributed AI training
- Azure Resiliency and Copilot Resiliency Agent for reliability
- Launch of Azure Copilot and agentic operations for smarter cloud management
- Immersive and enhanced experiences in Azure Portal
- Pillars of modern governance, compliance, and policy integration
- SQL Server 2025 release and extended Red Hat partnership
- Simplified operations with Azure Kubernetes Service Automatic

## Key Innovations

### 1. Intelligent Infrastructure and AI Operations

- Azure is advancing distributed AI training, supporting modern agentic workloads and facilitating scalable, secure, reliable cloud solutions.

### 2. Agentic Cloud Management

- Azure Copilot and Copilot Resiliency Agent introduce reliability automation and agentic management patterns for operational excellence.

### 3. Enhanced Azure Portal Experiences

- New immersive and streamlined experience pillars in the Azure Portal, focusing on ease of management and proactive control.

### 4. Governance, Compliance, and Policy

- Integrated governance tools and compliance frameworks help enforce enterprise requirements, automate policy, and improve audit reliability.

### 5. Platform Updates

- Announcing SQL Server 2025 and celebrating a 10-year milestone partnership with Red Hat.
- Azure Kubernetes Service Automatic rolls out for streamlined, automated operations—reducing manual labor, improving scalability.

## Technical Highlights

- Distributed AI infrastructure for agentic workloads
- Copilot-driven reliability and operational agents
- Automated governance enforcement and compliance in Azure
- Next-gen data and AI platform upgrades
