# Secure-by-Design Transformation: PwC and Microsoft Purview Enhancing Data Security

**Session Overview**

This intermediate-level Microsoft Ignite 2025 session gathers insights from PwC and Microsoft, with guest Scott Sikorski, to discuss secure-by-design strategies using Microsoft Purview.

**Key Topics Covered**

- **Deploying Microsoft Purview Catalog:**
  - PwC helps clients deploy Purview to accelerate innovation and improve data asset management.

- **Addressing Security Gaps:**
  - The session details how Purview can be used to identify and address security gaps in enterprise environments.

- **Improving Data Asset Visibility:**
  - Techniques for leveraging Microsoft Purview to centralize and enhance data asset visibility across organizations.

- **Efficiency Challenges in Enterprise:**
  - Discussion of 'doing more with less,' focusing on how automation and smart cataloging can overcome resource constraints.

- **Automating Alert Management:**
  - Integration of intelligent agents for automating data security alerts, enabling proactive risk management and faster response times.

- **Building a Strong Data Foundation:**
  - Strategies to establish a robust data foundation that supports secure innovation and compliance.

**Session Highlights**

- Collaboration between PwC and Microsoft in advancing secure data governance
- Use of Microsoft Purview to streamline compliance initiatives and improve operational efficiency
- Real-world enterprise examples and practical takeaways for deploying intelligent automation in data security workflows
