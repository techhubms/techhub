# AI-First Mobile Development: Live Coding with Copilot and .NET MAUI

**Presented by:** Jakub Florkowski, Shane Neuville, and Gerald Versluis

This session features the .NET MAUI team taking on the challenge of building a real cross-platform mobile app entirely with the help of AI—specifically GitHub Copilot and related AI agents—showcasing:

- **Project Scaffolding**: Setting up a .NET MAUI project with minimal manual input, relying on AI for initialization.
- **UI Generation**: Allowing AI to design and implement user interfaces according to modern standards.
- **Automated Bug Fixing**: Using AI to identify, explain, and correct issues as they arise in real time.
- **Code Review & Validation**: AI-powered code evaluation and suggestions to optimize quality and maintainability.
- **Deployment & Shipping**: Following AI-driven instructions to validate and deploy the app to multiple platforms.

The session examines both the effective parts and the stumbling blocks of relying on AI, giving viewers a realistic look at current capabilities and limitations of AI-powered tools like GitHub Copilot in .NET MAUI/mobile app development workflows. All critical steps from initial project creation to shipping the app are powered by AI, with the team highlighting decision-making, troubleshooting, and maximizing productivity with these new technologies.
