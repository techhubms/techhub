# Teaching Modern Web Development with .NET, Razor Pages, and HTMX

Learn how the Textbooker.pl app exemplifies a simplified educational approach to web development. This session features Tomasz Osmanowski, Michał Siegień, and Tymoteusz Bielski, hosted by Cam Soper and Scott Addie.

## Key Topics Covered

- **Unified .NET Platform:** How the .NET ecosystem integrates backend, frontend, and deployment, lowering barriers for new developers and students.
- **Razor Pages:** Modern web development leveraging .NET Razor Pages for fast, responsive apps with clean separation of concerns.
- **HTMX:** Reduce reliance on JavaScript with HTMX, enabling dynamic, interactive user experiences directly from the server.
- **Educational Benefits:** Focus on programming fundamentals without overwhelming learners with complex frontend frameworks.
- **Real-World Case Study:** Insights from building and teaching with the Textbooker.pl application.

## Highlights

- Demonstrates how combining Razor Pages and HTMX leads to highly interactive applications while minimizing frontend code complexity.
- Discussion on why these technologies are especially suited for education: easier setup, faster feedback loops, and comprehensible full-stack patterns.
- Tips for educators looking to modernize their web development curriculum using .NET technologies.
