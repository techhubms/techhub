# Automation in Copilot Studio: Agent Flows and Computer Use

Presented by Jose Luis Fernandez and Heather Orta-Olmo at Microsoft Ignite 2025.

## Overview

This session demonstrates how Copilot Studio empowers users to build sophisticated automation solutions using agent flows and computer use agents (CUA). Attendees will learn to:

- Build solutions automating tasks across apps, sites, and desktops
- Integrate with secured workflows via Microsoft security and governance frameworks
- Confidently scale automation initiatives from pilot to enterprise-wide use

## Key Topics

- **Customer Success Stories**: How Uber and Virgin Money leveraged AI automations for business efficiency
- **Security and Credentials**: Best practices in managing credentials and configuring machine access for automation
- **Cloud PC Pool Integration**: Seamless automation with Windows 365 environments, announced new integration
- **Built-In Credential Store**: Streamlining secure credential management in Copilot Studio
- **Agent Flows**: Introduction to agent flows, intelligent approvals, and information request actions
- **Flow Interface Updates**: Latest enhancements in automation interface
- **Enterprise Automation Demo**: Nationwide insurance claim automation walkthrough
