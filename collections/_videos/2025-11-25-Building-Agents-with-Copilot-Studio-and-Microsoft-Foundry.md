# Building Agents with Copilot Studio and Microsoft Foundry

## Session Overview

This advanced session demonstrates how to build and deploy enterprise-grade AI agents using Copilot Studio, Microsoft Foundry, and the Microsoft Agent Framework. Presenters discuss integrating connected agents, deploying custom models and knowledge, and establishing observability using unified tracing. Live demos illustrate practical applications across policy and complex claims processing, and the discussion includes future directions for these technologies.

## Key Topics Covered

- **Unifying Copilot Studio and Microsoft Foundry**: Challenges faced by enterprises using fragmented AI tools and disconnected data silos, motivating integrated agent solutions.
- **Integrating Users and Developers**: Collaborating power users and developers to solve organizational tasks such as policy management and claims processing.
- **Microsoft Agent Framework and Unified SDK**: Introduction of unified SDK supporting both Copilot Studio and Foundry, enabling cross-platform agent orchestration.
- **Live Demonstrations**: Examples include NHL query processing and live tracing in Copilot Studio, as well as system capabilities like task management and queuing.
- **Deploying to Microsoft 365 Copilot**: Demonstration of agent solutions targeted for deployment into Microsoft 365 environments.
- **Observability and Unified Tracing**: Best practices for monitoring agent performance and activity across platforms.
- **Future Guidance**: Recommendations for engineering seamless solutions with Copilot Studio and Foundry based on current trends and preview announcements.
