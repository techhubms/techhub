# Accelerate Data and AI Transformation with Azure Databricks (BRK128)

This session from Microsoft Ignite 2025 demonstrates Azure Databricks as a central data and AI platform for organizations pursuing scalable, collaborative, and data-driven solutions.

## Key Topics

- **Customer Showcase**: EyeFi implementation for advanced analytics
- **AIBI and Genie**: Conversational Analytics, governed and secure with Unity Catalog
- **Microsoft Foundry Integration**: Announcement of Genie for AI agents and real-time solution launches
- **Azure Databricks Connector**: June release supporting real-time interactions across Microsoft platforms
- **Monitoring and Evaluation Tools**: Genie capabilities, built-in evaluation without extra setup
- **Knowledge Assistant**: Handling unstructured data sources for analytics
- **Multi-Agent Supervisor**: Routing queries between Genie and Knowledge Assistant for advanced decision support

## Microsoft Ecosystem Integration

- **Power BI & Power Apps**: Seamless analytics and visualization, driving actionable data-driven decisions
- **Microsoft Foundry**: Enables AI tools and agent development within enterprise workflows

## Real-Time Decision Making

- **Built-in Monitoring**: Immediate insights and evaluation tools to support ongoing analytics
- **Governance**: Unity Catalog for secure management of analytics assets
