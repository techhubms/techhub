# GitHub Copilot in JetBrains: Demo of MCP and Agent Mode

This video demonstrates the latest capabilities of GitHub Copilot within JetBrains IDEs, especially focusing on the integration with the Model Context Protocol (MCP).

## Key Topics Covered

- **Agent Mode:** Explore how Copilot's agent mode autonomously manages issues, assisting developers throughout the coding process.
- **MCP Enhancements:** Includes MCP sampling for customized model configurations and the use of MCP servers for broader context awareness.
- **MCP Prompts:** Leverage intelligent prompts to enhance workflow and streamline development tasks.
- **MCP Elicitation:** Dive into structured dialogue flows to improve requirement gathering and task clarification.
