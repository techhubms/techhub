# Copilot Studio Innovations and Roadmap: Building Low-Code AI Agents (BRK313)

**Speakers:** Omar Aftab, Erez Altus, Zac Cadi  
**Event:** Microsoft Ignite 2025  
**Session:** BRK313 | Foundational (100)

## Overview

This session immerses viewers in the evolving capabilities of Copilot Studio, focusing on the creation and management of intelligent, low-code agents. Attendees learn about the two main Copilot Studio editions: the embedded builder for seamless integration and the enterprise-grade version for advanced customization and governance.

## Key Topics

- **Knowledge Capabilities Enhancements:** Discover how Copilot Studio leverages improved knowledge bases to build smarter agents.
- **Generative Orchestration:** Learn about new orchestration features supporting complex workflows and conversational logic.
- **Testing and Debugging:** Explore new platform tools to streamline agent evaluation, including question set generation and configurable scoring methods.
- **Agent Governance and Management:** Find out how governance and cost controls are being improved for enterprise scenarios.
- **Agent Identity Demonstration:** See a live demo highlighting agent identity and operational transparency.

## Upcoming Features

- Enhanced integration with Microsoft 365 Copilot
- Advanced enterprise-agent creation suite
- Scoring strictness and custom evaluation methods
- Improved platform governance and management controls
- Expanded resource management tools

## Real-World Impact

The session includes customer success stories demonstrating:

- Efficiency gains with automated, AI-powered agents
- Elevated user experience through Copilot Studio innovations
- Seamless low-code automation for business workflows
