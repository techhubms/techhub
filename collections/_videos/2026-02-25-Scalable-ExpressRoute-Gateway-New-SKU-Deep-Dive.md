# Scalable ExpressRoute Gateway: New SKU Deep Dive

**Author:** John Savill's Technical Training

## Introduction

This video provides a technical overview of Microsoft's new scalable ExpressRoute Gateway SKU for Azure. The session introduces gateway basics, examines available SKUs, explains fixed throughput and scaling mechanisms, and covers practical aspects such as pricing and migration.

## Chapters Covered

- Gateway fundamentals
- ExpressRoute Gateway SKUs
- Gateway functions and fixed throughput
- New scalable gateway SKU
- Scale units (min/max)
- Fixed and auto scaling capabilities
- Pricing and cost considerations
- Migration and transition guidance
- Future roadmap for ExpressRoute Gateways

## Key Takeaways

- **ExpressRoute Gateway Basics:** Understand the function of Azure's ExpressRoute Gateways as part of hybrid or enterprise networking, allowing private connectivity between on-premises infrastructure and Azure.
- **Gateway SKUs:** Comparison between standard vs. scalable SKUs, their throughput and performance limitations.
- **Scalable Gateway Mechanism:** New SKU introduces support for scaling throughput dynamically via scale units—clarified with minimum and maximum scale unit information.
- **Auto Scaling:** How gateways can auto-scale based on demand, supporting more connections and higher bandwidth for large deployments.
- **Pricing:** Details on new pricing models related to scalable gateways, practical budgeting advice.
- **Migration:** Guidance for transitioning from non-scalable to scalable gateways.
- **Future Outlook:** Speculation and guidance on the continued evolution of ExpressRoute networking within Azure.
