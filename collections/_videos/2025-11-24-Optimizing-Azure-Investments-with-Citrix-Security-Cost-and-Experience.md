# Optimizing Azure Investments with Citrix: Security, Cost, and Experience

**Speakers:** Shawn Bass, Quentin DRION, Hermann Pais  
**Event:** Microsoft Ignite 2025 (Session BRKSP466)

## Overview

This session addresses how organizations leveraging **Azure** and **Windows 365** can maximize their investments using the Citrix platform as a multiplier. Attendees learn practical strategies for cloud cost optimization, advanced security features, and the delivery of strong user experiences for hybrid and remote workforces.

## Key Topics Covered

- **Citrix as the #1 Secure Application Access Provider:** Introduction to Citrix’s leadership in secure remote access.
- **Modern Work Challenges:** Solutions for hybrid environments and managing application sprawl across diverse platforms.
- **Unified Communications Compatibility:** Ensuring seamless interoperability with major unified communication and contact center standards.
- **AI-Powered Video Upscaling:** Leveraging client-side GPU to boost video quality and reduce bandwidth use.
- **Zero Trust Security with Citrix Secure Private Access:** Replacing VPNs, implementing zero trust, and deploying protections like session watermarking and screen capture prevention.
- **IT Troubleshooting Tools:** Introduction to User Session Dashboard for efficient monitoring and troubleshooting.
- **Migration and Upgrade Support:** Using Citrix tools (e.g., Adrian) to plan and execute upgrades and migrations from on-premises to Citrix Cloud.
- **Real-World Transition Example:** Case study of MSC’s migration from a traditional on-premises setup to a Citrix Cloud environment.

## Actionable Insights

- Assess your Azure and Windows 365 deployments to identify areas where Citrix can reduce total cost of ownership.
- Enhance security posture using modern access controls, moving towards zero trust principles.
- Use Citrix’s integrated dashboards and planning tools to streamline IT support and migration efforts.
