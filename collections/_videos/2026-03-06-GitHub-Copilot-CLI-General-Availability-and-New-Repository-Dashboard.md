# GitHub Copilot CLI General Availability and New Repository Dashboard

**Presented by GitHub**

## Episode Highlights

- **GitHub Copilot CLI GA:**
  - The GitHub Copilot CLI is now generally available for all Copilot users.
  - It includes new agentic features and supports multiple AI models.
  - The CLI aims to increase productivity by enabling developers to leverage Copilot's AI directly from the command line.
- **Repository Dashboard Launch:**
  - GitHub has released a new dashboard to help users quickly find and manage their projects.
  - The dashboard is generally available for all users, streamlining repo navigation and project oversight.
- **GitHub Copilot Dev Days:**
  - An event where developers can learn more about Copilot, its integrations, and practical use cases.
  - For more information, visit [GitHub Copilot Dev Days](https://gh.io/ghcdevdays).
- **npmx Open Source Project:**
  - The story behind npmx.dev, a newly launched open source project for managing npm packages, is featured.
  - Visit [npmx.dev](https://npmx.dev/) to learn more.
