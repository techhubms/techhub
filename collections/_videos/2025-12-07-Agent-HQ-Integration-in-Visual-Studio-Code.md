# Agent HQ Integration in Visual Studio Code

Agent HQ provides developers with a single view to initiate, monitor, and review agent sessions directly within VS Code. Whether the agents are local or remote, you can seamlessly incorporate them into your development workflow.

## Key Features Covered

- **Agent Sessions View:** Manage all of your agent processes in one place, supporting both local and remote agents.
- **OpenAI Codex Integration:** Easily add AI-powered code features to your workflow via Codex.
- **Plan Agent:** Use the plan agent functionality to automate and organize session activities, helping you break down tasks and plan your coding work.
- **Delegate to Agent:** Delegate tasks to specific agents, harnessing the flexibility of workflow automation within VS Code.
- **Copilot CLI Integration:** Enhance your workflow using Copilot CLI for command-line-driven agent interactions.
- **Custom Agents:** Build and incorporate custom agents tailored to your specific needs, including handoff configurations and workflows.
- **Download Insiders:** Access the latest VS Code features by downloading the Insiders build.
