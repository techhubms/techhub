# What’s New in the GitHub Copilot CLI?

GitHub presents the most recent improvements to the GitHub Copilot CLI, focusing on features that help developers be more productive and automate their workflows more efficiently.

## Key Updates

- **Custom Agents Built-In**: Developers can now utilize built-in custom agents within the Copilot CLI to tailor automation to their specific needs.
- **Improved Installation Options**: The CLI is easier to install with new support for [Homebrew](https://brew.sh/) and [Winget](https://learn.microsoft.com/en-us/windows/package-manager/winget/), simplifying setup on macOS and Windows.
- **Copilot SDK Technical Preview**: The newly introduced Copilot SDK enables developers to automate workflows and build more robust developer tools.

## Developer Benefits

- Automate common and repetitive development workflows directly from the command line.
- Integrate with powerful AI-driven features through the SDK, streamlining tool building.
- Enhanced flexibility in how Copilot CLI is installed and updated across development environments.

## Getting Started

1. **Installation**: Use Homebrew or Winget to install the Copilot CLI quickly on your preferred operating system.
2. **Custom Agents**: Explore the built-in agent functionality to automate personalized workflows.
3. **Copilot SDK**: Experiment with the SDK technical preview to create deeper integrations and custom solutions.
