# Microsoft Agent Framework WorkFlows Explained

Discover how to build complex, intelligent automation solutions with Microsoft Agent Framework Workflows. This session features deep dives and advanced demos led by Evan Mattson and Victor Dibia, showcasing core concepts and the Workflow Web UI:

## Key Topics Covered

- **Agent Framework Core Concepts**: Introduction to AI agents and their orchestration in business processes.
- **Type-Safe Architecture**: How type safety simplifies workflow development and reduces errors.
- **Workflow Web UI Demo**: Live demonstration of creating and managing workflows using the web interface.
- **Developer Feedback**: Insights from developers using Agent Framework in real-world scenarios.
- **Scalable Automation**: Strategies for orchestrating advanced, multi-agent systems on Azure.

## Demo Highlights

- Walkthrough of setting up agent workflows
- Step-by-step configuration of business logic
- Integration tips for cloud-based AI automation
