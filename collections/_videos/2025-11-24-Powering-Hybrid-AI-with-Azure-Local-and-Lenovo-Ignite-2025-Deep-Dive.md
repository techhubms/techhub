# Powering Hybrid AI with Azure Local and Lenovo

## Overview

This Microsoft Ignite 2025 breakout session explores the collaboration between Microsoft and Lenovo in hybrid cloud and AI, focusing on Azure Local running on Lenovo's latest server platforms. The session covers practical use cases including:

- Real-time video analytics
- Predictive maintenance
- Medical diagnostics
- Low-latency inferencing

## Speakers

- Stuart Mcrae
- Jayanth Yajaman

## Session Highlights

### Partnership and Industry Impact

- Details on the Microsoft-Lenovo partnership
- Industry recognition for transforming hybrid cloud and AI use cases

### Customer Journey: Kavli's Azure Local Implementation

- Migrating from legacy IT infrastructure to Azure Local
- Regulations and modernization challenges
- Deployment on Lenovo ThinkAgile MX hardware

### AI Flexibility and Innovation

- Support for NVIDIA GPUs and custom AI models through Azure AI Foundry
- Enabling seamless migration and edge AI innovation
- Invisible infrastructure benefits for end customers

### Audience Q&A

- Data sovereignty solutions, especially for small businesses via local service providers
- Multi-tenancy in Azure Local: Clustered workload boundaries and scalable compute islands
- AKS (Azure Kubernetes Service) load balancing availability—GA targeted for early next year

## Technical Takeaways

- Azure Local leverages hardware flexibility and partnership integrations to deliver real-time AI applications at the edge
- Customers can modernize their estates and unlock new business potential
- Data security, sovereignty, and scalability are emphasized for enterprise and SMB customers
