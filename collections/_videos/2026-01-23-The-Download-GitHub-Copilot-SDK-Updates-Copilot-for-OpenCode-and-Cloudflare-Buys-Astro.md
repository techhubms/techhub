# The Download: GitHub Copilot SDK Updates, Copilot for OpenCode, and Cloudflare Buys Astro

Hosted by Christina Warren, this episode of The Download delivers the latest in developer news:

## Highlights

- **Game Off 2025 Winners**: Recognition of standout entries from the latest Game Off competition. [Details here](https://gh.io/gameoff25).

- **GitHub Copilot Now in OpenCode**: GitHub Copilot subscribers can now use their subscription directly with the OpenCode platform, expanding Copilot’s usability for more developers. [Announcement](https://github.blog/changelog/2026-01-16-github-copilot-now-supports-opencode/)

- **Copilot SDK (Technical Preview)**: Introduction to the new GitHub Copilot SDK—enabling developers to build agent capabilities right into their applications. This includes highlights of features, use-cases, and the technical preview release. [Read more](https://github.blog/news-insights/company-news/build-an-agent-into-any-app-with-the-github-copilot-sdk/?utm_source=the-download)

- **Copilot CLI Updates**: Major updates to the Copilot command-line interface with enhanced agents, improved context management, and installation options. [Changelog](https://github.blog/changelog/2026-01-14-github-copilot-cli-enhanced-agents-context-management-and-new-ways-to-install/)

- **Cloudflare Acquires Astro Technology Company**: Cloudflare’s acquisition of Astro signals future innovation in web technologies. [Astro’s statement](https://astro.build/blog/joining-cloudflare/) and [Cloudflare’s announcement](https://blog.cloudflare.com/astro-joins-cloudflare/)

- **Project Spotlight: Handy**: Spotlight on Handy, a project that helps developers improve productivity. Find more at [handy.computer](https://handy.computer/) and the [GitHub repo](https://github.com/cjpais/Handy).

- **Pick of the Week**: Christina’s personal pick—Harry Styles’ new single 'Aperture'.
