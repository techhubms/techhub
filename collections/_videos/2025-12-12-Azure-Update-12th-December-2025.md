# Azure Update - 12th December 2025

**Author:** John Savill's Technical Training

## Introduction

A brief update highlighting recent changes and new features in Microsoft Azure. The channel continues to grow and, due to the volume of support requests, the author no longer answers questions directly—viewers are guided to external communities for specific queries.

## Key Announcements & Product Changes

- **Azure Batch Default Outbound Retirement**
  - Azure Batch default outbound access is being retired. Users should review existing workloads and update outbound connectivity configurations.
- **App Gateway v2 FIPS Compliance**
  - Azure Application Gateway version 2 now supports FIPS compliance, offering enhanced security for regulated workloads.
- **Azure Files Premium LRS Zonal Improvements**
  - Azure Files Premium now has Locally Redundant Storage (LRS) zonal support, improving resilience within single Azure regions.
- **Azure Databricks Dashboard Teams Support**
  - New support for Teams integrations within Azure Databricks Dashboards, streamlining collaboration in data projects.
- **VibeVoice Availability**
  - VibeVoice update—availability announced, but no technical details provided.
- **Azure Sphere OS 25.12 Release**
  - Latest version (25.12) of Azure Sphere OS released; users should consider testing and upgrading devices.
- **GPT-5.2 Release Mention**
  - Reference to GPT-5.2, but no in-depth coverage or application details.
