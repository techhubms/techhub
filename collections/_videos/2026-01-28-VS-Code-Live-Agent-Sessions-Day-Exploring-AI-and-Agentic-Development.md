# VS Code Live: Agent Sessions Day – Exploring AI and Agentic Development

**Event Date:** February 19th  
**Hosted by:** Visual Studio Code Team, featuring Olivia Guzzardo McVicker and community guests

## Overview

VS Code Live: Agent Sessions Day is a global virtual event highlighting how Visual Studio Code continues to evolve as a premier code editor, now enhanced with AI and agentic workflows. The event includes:

- Live demos showing real-world applications of agentic development in VS Code
- Behind-the-scenes discussions with the product team and community builders
- Features on customizable workflows, open integrations, and extensibility
- Insights on empowering developers with AI-assisted coding

## Key Topics

- **Agentic Development:** Discover how agents in VS Code help automate tasks and enhance productivity.
- **AI Integration:** Practical demonstrations of how AI is redefining coding workflows and developer experiences within the editor.
- **Custom Workflows:** Learn to tailor VS Code to your team's needs using open, community-driven solutions.
- **Open Source and Community:** Emphasis on extensibility and staying community-focused.

## Who Should Attend

- Developers interested in AI-assisted coding
- Teams exploring customizable development environments
- Anyone seeking inspiration for smarter, more efficient workflows using VS Code

## Speakers

- Olivia Guzzardo McVicker (@OliviaGuzzardo) and members of the Visual Studio Code Team
- Special guests from the developer community

## What You'll Learn

Attendees will leave with:

- Practical ideas for integrating agents and AI in their workflow
- Knowledge of open integrations and extensibility features in VS Code
- Inspiration for shaping personalized, productive development environments
