# Visual Studio Code Deployment and AI Governance for IT Pros

## Session Overview

This session, delivered at Microsoft Ignite 2025, is designed for IT professionals responsible for scaling and governing Visual Studio Code (VS Code) in enterprise settings. It focuses on maintaining security, compliance, and productivity while meeting developer needs and introducing AI features responsibly.

## Key Topics

- **VS Code at Scale:** Approaches for deploying VS Code across organizations, handling updates effectively, and maintaining consistency for developers.
- **Extension and Telemetry Policies:** Enforce extension use through private marketplaces, control telemetry to meet compliance, and authenticate using GitHub or Entra ID.
- **Security and Policy Enforcement:** Demonstrations of restricting developer environments, applying policies for compliance, and keeping systems secure without impacting productivity.
- **Scaling AI Responsibly:** Practical strategies—including the use of Model Context Protocol (MCP), prompt templates, and extension governance—to boost GitHub Copilot effectiveness, foster AI alignment, and drive collaboration between IT and development teams.
- **Organizational Adoption:** Tips for establishing internal champions and promoting effective collaboration between IT pros and developers to scale adoption of AI and productivity tools organization-wide.
