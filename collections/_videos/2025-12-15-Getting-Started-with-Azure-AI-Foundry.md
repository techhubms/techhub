# Getting Started with Azure AI Foundry

Hosted by Seth Juarez, this video guides developers through the process of beginning with Azure AI Foundry, Microsoft’s comprehensive platform for AI development on Azure.

## Key Topics Covered

- **Model Selection:** How to choose your first AI model within Azure AI Foundry
- **Building Conversational Agents:** Steps to create agents you can interact with, including practical examples
- **Code Structuring:** Insights on how Azure AI Foundry might change the way you organize and structure your code for AI solutions
- **Developer-Friendly Approach:** Breaking down intimidating aspects for a quick start

## Getting Started Steps

1. **Access Azure AI Foundry:** Visit [Get Started with Azure AI Foundry](https://msft.it/6053tm871)
2. **Pick Your First Model:** Guidance on selecting a model appropriate for your solution
3. **Build Agents:** Tutorial and demonstration for building agents capable of conversation
