# Meet the Performance-Enhanced Next Gen Azure SQL Managed Instance

**Speakers:** Anitha Adusumilli, Aditya Badramraju, Troy Brovold, Craig Henninger  
**Microsoft Ignite 2025 – Session BRK125**

## Overview

This advanced session unveils the new General Purpose tier for Azure SQL Managed Instance, delivering double the performance and five times more database capacity for fully managed SQL workloads.

- **Key Value:** Fully managed SQL with unprecedented scale and performance.
- **AI-Assisted Tooling:** New migration tools leverage AI (including Copilot) to help you migrate large, complex SQL estates more efficiently—completing end-to-end migrations in days.
- **Customer Case Study:** Hear from Hexure, an insurance services organization, as they share their journey rebuilding their digital platform on Azure SQL Managed Instance and Azure App Service, leading to improved customer experience and efficiency.

## Session Highlights

- **AI as Central Theme:** Ignite 2025 focuses on the role of AI within modern database management and digital transformation.
- **Consistency Across Editions:** SQL Server 2025 is introduced as the AI-ready database, emphasizing consistency and advanced capabilities for enterprise scenarios.
- **Next-Gen Performance:** Learn about the technical underpinnings and real-world benchmarks achieving faster workload execution and expanded capacity.
- **Migration Journey with Copilot Assistance:** Experience a step-by-step walkthrough of Azure Portal's migration workflow, validation processes, query performance optimization, and go-live (cutover) execution, all enhanced with Copilot.
- **Customer Success Story:** Hexure details their transition, working closely with Microsoft teams to modernize their estate with Managed Instance and Azure App Service.
