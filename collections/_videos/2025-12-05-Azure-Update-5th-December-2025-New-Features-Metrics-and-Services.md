# Azure Update - 5th December 2025: 10% More Fancy Edition

## Introduction

This video delivers an overview of the latest developments in Microsoft Azure for early December 2025. It is designed for technical professionals working with Azure services.

## Highlights

- **ALB Metric - New Dimension**: Updates to Application Load Balancer with improved metrics for monitoring and optimization.
- **Blob SFTP Resumable Uploads**: Azure Blob Storage now supports resumable uploads via SFTP, improving reliability for large transfers.
- **PostgreSQL Flex Upgrades**:
  - Available in new regions for expanded geographical redundancy.
  - pg_squeeze extension updated to version 1.9.1 for better table management.
  - Additional extensions: ip4r for IP range types and credcheck for credential validation.
- **Azure Databricks Serverless Workspaces**: Introduction of serverless compute options in Azure Databricks, streamlining analytics workloads without infrastructure management.
- **Regional Expansion**: Perth Azure Extended Zone launches, increasing resiliency options for workloads in Australia.
- **Azure ML SDK Retirement**: Azure Machine Learning SDK v1 announced for retirement—users should plan to upgrade to newer versions.
- **Azure MCP Server - Confidential Ledger Support**: Enhanced support for confidential ledgers in managed compute platforms, improving security for sensitive data.
- **AI Enhancements**:
  - Mistral Large 3 integration available in Foundry, expanding options for AI application development.

## Developer Actions

- Review and update scripts and pipelines to use new ALB metrics dimensions.
- Test and implement resumable SFTP uploads with Blob Storage for improved reliability.
- Upgrade PostgreSQL extensions and validate environments in new regions.
- Prepare for migration if currently using Azure ML SDK v1, per official retirement timeline.
- Explore Databricks serverless workspaces for scaling analytics jobs with minimal management overhead.
- Consider confidential ledger for scenarios requiring increased data integrity and auditing.
- For AI workloads, evaluate capabilities of Mistral Large 3 in Foundry for next-generation applications.
