# Advanced Capabilities and Innovation of Azure Storage Solutions

*Presented at Microsoft Ignite 2025 by Yuemin Lu, Aung Oo, and Lena Ye*

## Overview

Explore how Azure Storage addresses diverse cloud storage challenges through advanced features and continuous innovation. This session provides:

- Practical guidance for modernizing legacy data environments
- Strategies to optimize business-critical Tier-0 workloads
- Approaches for accelerating new AI and cloud-native applications

## Key Topics and Demonstrations

### 1. New Capabilities in Blob Storage

- **Smart Tiering**: Automates data placement for cost and performance optimization
- **Priority Replication**: Quickly replicate critical data sets
- **FS Spec Compatibility**: Enhanced integration with different storage protocols

### 2. AI and High-Performance Workloads

- **NVIDIA DGX Cloud Collaboration**: Building high-throughput AI infrastructure via Azure Managed Lustre
- **Kaito Framework & Azure NVMe**: Optimizing inference workloads for massive parallel access

### 3. Practical Demos by Lena Ye

- **Azure Storage Discovery**: Enhanced visibility into storage deployments
- **Copilot for Data Management**: Simplifies management workflows for administrators
- **Instant Snapshots and Access**: Provides rapid restore and quick data availability

### 4. Migration and Data Modernization

- **Cloud-native NAS Migration**: Updated Azure Files management model for legacy NAS environments
- **Identity-only Authentication for VDI**: Secure user profile management with Azure Files
- **Azure NetApp Files Migration Assistant**: Streamlines transfer and migration for enterprise datasets

## Best Practices

- Ensuring business continuity with faster restores and instant access
- Applying new management models for efficient cloud migrations
- Leveraging cloud-native tools for data discovery and security
