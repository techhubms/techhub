# Microsoft Entra: What's New in Secure Access on the AI Frontier

## Session Overview

This Microsoft Ignite 2025 breakout session highlights the latest secure access innovations in Microsoft Entra, focusing on the expanding role of AI.

### Key Topics Covered

- **AI-driven Identity and Access Expansion:** Extend traditional Zero Trust with new capabilities designed for agent and AI integration.
- **Agent Governance with Entra Agent ID:**
  - Introduction of Agent ID for managing non-human identities.
  - Distinct identity, authentication, and permissions for agents.
- **Demo: Managing Agents via Entra Agent ID Interface:**
  - Step-by-step demonstration of agent management workflows.
- **Prompt Injection Risk Management:**
  - Tools and controls to detect and mitigate prompt injection attacks in AI scenarios.
- **Unified App Visibility and Control:**
  - Introduction of the App Lifecycle Management Agent.
  - Enhanced visibility and unified management for applications.
- **Conditional Access Policies:**
  - Per-app policies provide granular controls over access requirements.
  - Deep analysis capabilities to detect excessive or missing policy exclusions.
- **Account Recovery Setup:**
  - Improved admin workflows for secure recovery within Microsoft Entra.

## Chapters & Demos

- **Reflection on AI Innovations and Rapid Adoption:** Review of the current AI landscape and impact on security requirements.
- **Deep Dive into Agent Governance:** Technical walkthrough of agent management features, focusing on permission granularity and compliance.
- **App Lifecycle Management and Conditional Access Demonstrations:** Concrete examples of visibility, control, and access policy definition.
- **Risk Management Innovations:** Practical advice on mitigating emerging AI risks, including prompt injection protection mechanisms.
