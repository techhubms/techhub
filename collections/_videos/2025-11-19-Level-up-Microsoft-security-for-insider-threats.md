# Level up Microsoft security for insider threats

**Session Overview**  
Maximize your organization's security posture by leveraging Microsoft Defender and Microsoft Entra ID in combination with OpenText™ Core Threat Detection and Response. This Microsoft Ignite 2025 session highlights:

- How behavior-based analytics unmask insider, novel, and advanced persistent threats
- Complementing existing tools for streamlined investigations and noise reduction—with no manual rule-writing required
- Making the most of trusted telemetry

**Key Chapters & Insights**

- **Malicious insiders:** Understanding risks, motives, and financial impact
- **Human behavior:** Central to detecting advanced threats
- **Detection challenges:** Credential theft, lateral movement, and how insiders exploit legitimate access
- **Beyond detection:** Importance of context in investigation and risk mitigation
- **Operational improvements:** Reduced investigation time and increased team confidence

**Technologies and Solutions Covered**

- Microsoft Defender: Threat detection and incident response
- Microsoft Entra ID (formerly Azure AD): Identity protection and access control
- OpenText™ Core Threat Detection: Augmenting Microsoft tools with behavior analytics
