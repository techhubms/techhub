# Build AI‑Powered .NET Apps with Telerik

**Featuring:** Ed Charbeneau (Progress Telerik), Daniel Roth

Join the ASP.NET Community Standup to learn how to build AI-powered Blazor apps using the latest tools and techniques from Telerik. Ed Charbeneau demonstrates practical applications of AI coding assistants and intelligent UI components integrated with .NET technologies.

## Topics Covered

- Integrating Telerik’s AI coding tools with ASP.NET Core and Blazor
- Using AI coding assistants to accelerate .NET app development
- Leveraging MCP (Microsoft Cloud Platform) servers for AI capabilities
- Building intelligent user experiences with smart UI components
- Generating clean, maintainable code assisted by AI technology
- Best practices for combining .NET and AI-driven components

### Resources

## Key Takeaways

- AI tools can help automate code generation and improve productivity in .NET development
- Telerik’s UI components support intelligent user interactions within Blazor apps
- MCP integration facilitates more advanced cloud-based AI scenarios
- Combining AI and established UI frameworks leads to richer user experiences

This session offers practical examples and expert discussion for .NET developers seeking to harness AI within their modern applications.
