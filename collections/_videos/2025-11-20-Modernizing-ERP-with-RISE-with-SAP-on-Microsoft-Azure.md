# Modernizing ERP with RISE with SAP on Microsoft Azure

## Session Overview

Presented by Wael Elkabbany and Hiren Shah at Microsoft Ignite 2025, this session showcases ways for SAP customers to modernize business processes using the Microsoft Cloud, especially through the RISE with SAP program. The session covers Azure innovations, advanced security, and AI-driven monitoring tools tailored for SAP workloads.

## Key Topics Covered

### 1. Microsoft–SAP Partnership

- Strategic partnership overview and benefits for enterprises

### 2. Azure as the Platform for SAP

- Why Azure is positioned as the leading cloud for SAP workloads
- Infrastructure advances, including high SLA (Service-Level Agreement) for RISE deployments

### 3. Go-To-Market Strategy: ‘Innovate Rise’

- New strategic initiatives to accelerate SAP adoption and modernization

### 4. Infrastructure Innovation

- Introduction of CXL (Compute Express Link) technology to support SAP HANA cost and performance optimization

### 5. AI-Driven Monitoring and Automation

- Use of proactive AI agents in Azure to detect configuration drift in SAP environments
- Automated remediation support

### 6. Integration and Analytics

- How organizations deploy RISE with SAP on Azure
- Integrating SAP BTP (Business Technology Platform) and BDC (Business Data Connectors)
- Accelerating analytics setup through business process solutions

### 7. Security and Compliance

- Securing SAP workloads with Microsoft Defender and Sentinel
- Addressing data sovereignty and compliance requirements for regulated industries

### 8. Demonstrations

- Demo setup leveraging Copilot and Joule in a real-world sales scenario
- Showcasing AI-powered data retrieval and integrated SAP workflows
