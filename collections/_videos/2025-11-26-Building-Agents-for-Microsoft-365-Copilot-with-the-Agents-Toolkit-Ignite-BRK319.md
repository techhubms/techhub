# Building Agents for Microsoft 365 Copilot with the Agents Toolkit (BRK319)

**Speakers:** Robert Howard, Matthew Barbour, Sébastien Levert  
**Event:** Microsoft Ignite 2025  
**Session Level:** Advanced (300)

## Session Overview

Learn how to build and bring your own agent into Microsoft 365 Copilot using the Microsoft 365 Agents Toolkit. The session explores:

- Declarative Agents: Configuration-based agents with showcased examples from Persistent and Fujitsu.
- Custom Engine Agents: Deeper integration and customization options.
- The Microsoft 365 Agents SDK: Tools and resources to build, test, and deploy agents.
- Extending with Copilot APIs: Ways to securely ground answers in Microsoft 365 data.
- Real-world business automation scenarios, such as procurement and contractor management and insurance claim assessment using photo analysis models (e.g., Minstrel Model).

## Key Topics and Demonstrations

- **Agent Architecture:** Architectural walkthrough and review of agent source code for custom implementations.
- **Policy Compliance Validation:** Use of Copilot Retrieval API to validate compliance through SharePoint data integration.
- **Showcase Demos:** Live demonstrations including procurement management and custom insurance agent solutions.
