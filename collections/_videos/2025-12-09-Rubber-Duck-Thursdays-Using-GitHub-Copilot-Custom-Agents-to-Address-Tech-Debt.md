# Rubber Duck Thursdays: Using GitHub Copilot Custom Agents to Address Tech Debt

Join GitHub's lighthearted Rubber Duck Thursdays livestream as developers tackle technical debt through live coding sessions.

## What You'll Learn

- How to use GitHub Copilot custom agents to assist in refactoring code
- Live exploration and development of GitHub Action workflows
- Practical strategies for addressing and reducing technical debt with AI tools
- Real-time demonstration of AI-powered workflow automation

## Stream Highlights

- Review of previous session: Authoring workflow files with Copilot custom agents
- Applying custom agents to identify and manage areas of tech debt
- Open discussion and informal tips from hosts as they code

## Who Is This For?

- Developers interested in practical uses of GitHub Copilot and its custom agents
- Teams looking to streamline actions and CI/CD automation
- Anyone curious about live AI-assisted coding or refactoring

## About the Stream

Rubber Duck Thursdays is an informal coding session meant for learning, sharing, and exploring developer tools with the broader community. The hosts use GitHub Copilot, especially its custom agent features, to show how AI can enhance everyday workflows and help manage software maintenance challenges.
