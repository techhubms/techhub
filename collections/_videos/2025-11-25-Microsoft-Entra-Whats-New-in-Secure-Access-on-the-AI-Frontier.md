# Microsoft Entra: What's New in Secure Access on the AI Frontier

Presented at Microsoft Ignite 2025, this session dives into the most recent security advancements in Microsoft Entra, tailored for AI-driven identity and access environments.

## Key Highlights

- **AI Innovation Reflection**: Explores the rapid adoption of AI and its security implications.
- **Agent Governance**: Introduction of Microsoft Entra Agent ID, allowing for distinct identities, authentication, and permissions for agents in AI workflows.
- **Hands-On Demo**: Shows how to manage agents with the Agent ID interface.
- **Zero Trust Expansion**: Advises on expanding foundational Zero Trust strategies to handle an increasing number of AI agents and applications.
- **AI Risk Management**: Covers new protections against AI-specific threats like prompt injection.
- **Lifecycle Management**: Demonstrates unified app visibility and control using the App Lifecycle Management Agent in Entra.
- **Conditional Access Enhancements**: Details new per-app conditional access policies for granular security enforcement.
- **Deep Policy Analysis**: Discusses the agent’s ability to detect excessive or missing policy exclusions.
- **Account Recovery Setup**: Administration setup for account recovery workflows within Microsoft Entra.
