- Announcement on the retirement of the AKS Windows Server Annual Channel, with recommendations for migration.

### App Gateway WAF Insights Updates

- Insights into new diagnostics and configuration features for Application Gateway Web Application Firewall (WAF).

### Premium SSDv2 New Availability

- Updates on the regional expansion of Premium SSDv2 and its benefits for cloud storage.

### Blob User Delegated SAS to Entra ID

- How Blob Storage’s user-delegated Shared Access Signatures (SAS) now integrate with Microsoft Entra ID for enhanced security and access management.

### Azure AI Search Sensitivity Label Support

- Azure AI Search now supports sensitivity labels, improving data governance and security compliance for organizations running sensitive workloads.

### PostgreSQL Premium SSDv2 Geo Backup

- New geo-backup options for PostgreSQL Premium SSDv2 deployments, enabling resilient disaster recovery setups.

### SQL Server VS Code Extension Updates

- Overview of significant improvements to the SQL Server extension for Visual Studio Code, helping developers boost productivity.

### New OpenAI Models in Foundry

- Presentation of the latest OpenAI models available in Foundry and their applicability for enterprise AI use cases.

### Azure Monitor Pipeline Updates

- Enhancements in Azure Monitor pipeline capabilities for better telemetry and observability.

### GitHub Copilot CLI

- Coverage of new features and improvements in the GitHub Copilot CLI tool to help automate coding workflows and command completion.
