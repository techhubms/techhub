# Building the Most Intelligent Agents with the Latest Knowledge Sources

**Session:** BRK318 (Microsoft Ignite 2025)  
**Speakers:** Anjli Chaudhry, Eli Coon, Hardik Modi, Nick Taylor  
**Session Level:** Intermediate (200)

## Overview

This session explores how to power your AI agents with the most comprehensive suite of knowledge sources using Copilot Studio. It highlights native knowledge integrations, the ability to use connectors, APIs, and MCP server authentication to design intelligent agents capable of coordinating and executing workflows.

## Featured Highlights

- **Genie Agent (Gilead Sciences Preview):** Real-world guest appearance shows practical AI agent deployment focusing on enterprise knowledge integration.
- **Knowledge Sources:** Use of document filtering and relevant information extraction for agent intelligence.
- **Structured Data Analysis:** Demonstration of tools and scripts tailored for management and extraction of insights from structured data sources.
- **Personalization:** Techniques for leveraging end-user context to deliver personalized response experiences.
- **Visualization Example:** Example scenario where an agent creates visualizations from Excel data, displaying outputs and insights for business users.
- **Quality and Evaluation:** Approaches to ensure output quality, handle complex and unstructured data, and agent evaluation strategies.

## Architecture & Integration

- **Copilot Studio:** Core development platform for authoring, integrating, and deploying intelligent AI agents with low-code tools.
- **Integration Examples:** Deep dives into practical integrations with Microsoft Graph, Azure AI Search, and Active Directory for robust access control and dynamic data querying.
- **Workflow Coordination:** Authenticating and managing agent workflows across multiple knowledge bases and systems.
