# What's New in SQL Server Management Studio 22 GA

Presented by Microsoft Developer, this Data Exposed episode gives an in-depth overview of the general availability highlights for SQL Server Management Studio (SSMS) version 22. The segment includes:

## Introduction and Recap

- Review of SSMS development over the past year, from SSMS 21 preview to SSMS 22 GA release.

## Key Features in SSMS 22

- Walkthrough of what's new in SSMS 22, including usability enhancements and new capabilities.
- Demonstration segment showing updates to the SSMS interface and toolset.
- Instructions for installing SSMS versions side-by-side to compare features or transition workloads.
- Discussion on adding new workloads and extensibility options in SSMS 22.
