# Blueprint for Intelligent AI Agents with PostgreSQL and Azure Integration

**Speakers:** Sharon Chen, Maxim Lukiyanov, Abe Omorogbe
**Event:** Microsoft Ignite 2025 | BRK130 | Advanced (300)

## Overview

This session dives into how modern AI agents are reshaping business operations. Learn how to design and deploy intelligent agents that deeply understand enterprise data using Azure Database for PostgreSQL. The talk covers:

- Techniques for building agents incorporating knowledge graphs and vector search
- Leveraging latest AI model integrations available directly in Azure Database for PostgreSQL
- Architectural blueprint: separation of compute and storage layers for performance and availability
- AI Model Management: streamlined provisioning, lifecycle control, and integrated billing
- Automatically provisioned AI models such as GPT-5, text embeddings, and semantic ranking

## Key Features & Demonstrations

- **Generate Operator:** Summarize product reviews directly inside SQL
- **Extract Operator:** Identify entities and construct knowledge graphs within PostgreSQL
- **Microsoft Foundry Integration:** Integrating AI and database capabilities for advanced scenarios
- **AZD Up Command:** Step-by-step guide for connecting to Postgres
- **Live Demo:**
  - Metrics Intelligence tooling
  - Using GitHub Copilot for identifying and fixing PostgreSQL database performance issues
  - Automation for protocol amendment management and intelligent revision tracking

## Best Practices & Customer Insights

- Real-world blueprints and patterns shared by Azure customers
- Guidance on augmenting existing applications and custom copilots with these new AI and ML innovations
- Tips for orchestrating AI model lifecycles and enabling advanced analytic capabilities
