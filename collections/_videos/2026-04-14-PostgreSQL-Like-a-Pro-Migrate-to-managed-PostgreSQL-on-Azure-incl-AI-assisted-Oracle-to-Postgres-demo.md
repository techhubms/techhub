# PostgreSQL Like a Pro: Migrate to managed PostgreSQL on Azure

Meet **Jonathon Frost** from the PostgreSQL team at **Microsoft Azure**. This session focuses on modernizing databases by migrating to **managed PostgreSQL on Azure**, with an overview of migration approaches and a short demo.

## What this video covers

- Why migrating to **managed PostgreSQL on Azure** can be simpler than expected
- **Two migration types**
  - **Homogeneous migration** (PostgreSQL → PostgreSQL)
  - **Heterogeneous migration** (e.g., Oracle → PostgreSQL)
- Benefits and platform notes
  - General benefits of moving to managed PostgreSQL on Azure
  - Performance-related benefits mentioned with **AMD**
  - **Built-in migration support**
- Tooling and demo
  - **AI-assisted migration tooling** in **VS Code**
  - Using **GitHub Copilot** as part of the AI-assisted workflow
  - A demo of a PostgreSQL migration
