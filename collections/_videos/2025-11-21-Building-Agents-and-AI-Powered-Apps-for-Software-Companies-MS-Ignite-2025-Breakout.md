# Building Agents and AI-Powered Apps for Software Companies (MS Ignite 2025 Breakout)

## Overview

This session explains how agents and AI-powered applications are transforming software development for growth and innovation. It covers collaboration opportunities, technical integrations, and the use of advanced Microsoft technologies.

## Key Topics

- **Partnerships in Graph Analytics and AI:** Insights on how engaging with Microsoft's ecosystem and partner network enhances graph analytics and AI capabilities.
- **Microsoft’s AI Infrastructure for Developers:** Microsoft’s role in advancing AI infrastructure through strategic partnerships with software companies.
- **Open Source and Security Integration:** Commitment to open source technologies and integrating Microsoft Defender with GitHub for improved development security practices.
- **Microsoft Startups and the AI Challenge:** Exploration of the Microsoft Startups program and challenges developers face in implementing AI.
- **Rapid Development with Azure Logic Apps:** Connecting over 1400 services using Azure Logic Apps to streamline app development and system integration.
- **Demo: Building Agents and Local AI Pair Programming:** Live demonstration of building agentic solutions and local development workflows using AI-assisted pair programming.
- **Scaling AI—Experimentation to Implementation:** Strategies for transitioning from experimental AI projects to large-scale production deployments.
- **Industrial AI Stack and New AI Offerings:** Insights into building a robust industrial AI stack and new AI kit announcements available via the Microsoft Marketplace.
