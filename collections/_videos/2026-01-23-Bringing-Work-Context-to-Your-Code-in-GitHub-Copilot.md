# Bringing Work Context to Your Code in GitHub Copilot

In this video, Microsoft Developer introduces a preview of a new integration between GitHub Copilot and Microsoft 365 data, enabled by the Work IQ MCP server. This feature empowers developers to access their organization's knowledge directly while coding, streamlining the process of bringing relevant work context and insights into development workflows.

## Key Highlights

- **Work IQ MCP Server Preview**: A new server that unlocks business knowledge from Microsoft 365 and makes it accessible within GitHub Copilot.
- **Enhanced Coding Experience**: Integrate essential documents, discussions, and enterprise knowledge into the coding process for improved productivity and smarter code suggestions.
- **Developer Empowerment**: Developers can now leverage contextual information from their own work environment to write better, more relevant code.
