# Defending Cloud Platforms: Unified Security with Microsoft Defender

## Overview

This session, presented at Microsoft Ignite 2025 by Drew Nicholas, Greg Schellenberg, and Oz Wilder, explores the challenges and strategies for securing cloud environments as AI adoption expands attack surfaces.

## Key Topics

- **AI-Driven Threats & Cloud Expansion:** Discussion on how the rise of AI increases both the complexity and opportunities for attackers, expanding organizational attack surfaces.
- **Defender for Cloud Platform:** Introduction to unified security strategies using Microsoft Defender for Cloud, integrating security from code development to live runtime.
- **Security Posture & Risk Management:** Insights into posture assessments, risk prioritization, and workload protection for hybrid/multi-cloud environments.
- **Real-World Azure Attack Case:** Detailed review of adversary actions and data theft from an Azure Kubernetes cluster, including detection and prevention strategies.
- **Exposure Management:** Overview of proactive defense mechanisms and exposure management to reduce vulnerability across environments.

## Practical Takeaways

- How Microsoft Defender for Cloud integrates with hybrid and multi-cloud deployments, offering a unified view of security risks.
- Approaches to attack detection and transitions toward prevention using the Defender platform.
- Techniques for security posture improvement and exposure management against emerging AI-driven threats.
