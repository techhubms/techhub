# Microsoft Foundry Workflows - Pt. 1: Creating a Sequential Multi-Agent Workflow

**Presented by April — Microsoft Developer**

This video is the first installment of the Microsoft Foundry Workflows series, focusing on building sequential multi-agent workflows using Foundry's latest templates and orchestration tools.

## Tutorial Overview

- **Portal Access & Project Setup**  
  Walks through accessing the Microsoft Foundry portal and creating a new workflow project.

- **Model Deployment**  
  Demonstrates deploying an AI model directly within the Foundry environment, leveraging Azure AI infrastructure.

- **Sequential Workflow Creation**  
  Guides users through configuring a workflow using the provided Foundry template, connecting and sequencing three distinct AI agents for process automation.

- **Testing the Workflow**  
  Shows how to validate and run the workflow, emphasizing troubleshooting and iteration techniques.
