# Agentic AI for Creatives with Microsoft Foundry and Data | BRK200

**Speakers**: Marco Casalaina, James Codella, Eva Jennings, Rohani Jotshi, Jari Koister, Kristijan Risteski, Bill Wittress

## Session Overview

This Microsoft Ignite breakout session focuses on how creative industries leverage agentic AI, powered by Microsoft Foundry and Azure databases, to empower artists, designers, and storytellers.

### Key Topics

- **Foundry for AI**: Tools designed for creative tasks including new IQ capabilities and control plane improvements
- **Azure Integrations**: Announcement of Azure SQL and Azure Cosmos DB support in Microsoft Fabric, enabling AI and semantic search for creative scenarios
- **Industry Use Cases**:
  - **Pantone**: Demo of a Palette Generator that creates color-blind friendly palettes using AI
  - **Kantar**: CTO demonstrates Automated Ad Analysis, Scene Detection, and Feedback Systems leveraging agent agents and technical architecture with Microsoft Foundry
  - **TomTom**: Showcases EV driver experience enhancements using agentic AI for Charging Finder and route optimization

### Technical Highlights

- Integration patterns for bringing AI into creative pipelines
- Technical architecture of Microsoft Foundry and intelligent agent systems
- Feedback mechanisms and recommendation application
- Live demonstration of creative AI tools in real scenarios
