# SQL to Insights in Minutes with Copilot for Data Factory

**Presented by:** Microsoft Developer

In this episode of Data Exposed, you'll learn how Copilot and AI capabilities within Microsoft Fabric Data Factory can accelerate going from raw Azure SQL data to actionable insights.

## Key Topics Covered

- **Modern Data Integration:** See how Copilot simplifies transforming and moving SQL data using natural language instructions.
- **Automation & Orchestration:** Automate data workflows, integration, and notifications without writing complex SQL scripts.
- **User Experience:** Guided demos illustrate loading data to various destinations, orchestrating entire data pipelines via intuitive interfaces.
- **AI-Powered Features:** Fabric Data Factory leverages Copilot to streamline operations formerly requiring deep SQL or data engineering knowledge.

## Demo Highlights

- Walkthrough of Data Factory in Microsoft Fabric
- Step-by-step AI-driven demo of loading and transforming Azure SQL tables
- Live orchestration and notification automation in minutes

## How This Helps Practitioners

- Reduces manual coding for common data engineering tasks
- Empowers SQL users to build data pipelines faster
- Accelerates time to insight with automated AI-driven workflows
