# Build AI Agents with Zero Code Using Azure Logic Apps

**Host:** Priyanka Vergadia (Principal Cloud Advocate, Microsoft)  
**Guest:** Rohitha Hewawasam (Engineering Manager, Azure Logic Apps)

## Overview

Explore how developers can create powerful, enterprise-grade AI agents using Azure Logic Apps with a no-code/low-code approach. The discussion includes leveraging over 1400 enterprise connectors, integrating with tools like Microsoft Teams and ServiceNow, and using GPT-4 and prompt engineering. The session features practical demos, insights on compliance and authentication, workflow debugging tips, and strategies for scaling to multi-agent orchestration—all with minimal or no coding required.

## What You'll Learn

- Building AI agents with Azure Logic Apps, without writing code
- The importance of Logic Apps’ 1400+ connectors for enterprise integration
- Addressing authentication, compliance, and network boundaries
- Integrating GPT-4 and using prompt engineering for dynamic agent logic
- Embedding live chat clients and connecting with Microsoft Teams and ServiceNow
- Monitoring, debugging, and ensuring transparency in AI workflows
- Scaling from single agents to multi-agent orchestration
- Extending from no-code to pro-code scenarios

## Demo Highlights

- **Chatbot for Fort Coffee:** Building a production workflow integrating multiple services
- **ServiceNow & Teams Integration:** Real-world scenarios showing off cross-tool orchestration
- **Monitoring Tools:** Techniques for live agent conversations and tool usage visibility
