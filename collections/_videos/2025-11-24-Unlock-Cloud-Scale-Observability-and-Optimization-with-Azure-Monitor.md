# Unlock Cloud-Scale Observability and Optimization with Azure

**Speakers:** Rahul Bagaria, Ritesh Kini, Farzana Rahman  
**Session:** Microsoft Ignite 2025 – BRK149

## Overview

This session dives deeply into **Azure Monitor's** capabilities for delivering end-to-end observability across both cloud and hybrid infrastructures. Key areas include proactive issue detection, reduced mean time to recovery, and unlocking operational optimizations using cutting-edge AI functionality.

## Key Topics

### 1. AI-Driven Operations and Observability

- Leveraging Azure Monitor for real-time, comprehensive monitoring
- Detecting and addressing issues earlier using AI-enhanced tools
- Demo: Agentic AI troubleshooting within the Azure Portal

### 2. Copilot in Azure Agents

- Introduction of Copilot integration in Azure Monitor agents
- Extends visibility to cost and carbon efficiency metrics
- Helps identify optimization opportunities with actionable insights
- Validates recommendations and automates performance improvements

### 3. Advanced Visualization & Integration

- Using **Azure Monitor** with **Grafana** and **Prompt QL** for enhanced alerts and visualization
- AI-powered troubleshooting simplifies root cause and performance investigation

### 4. OpenTelemetry & Logging

- Announcement of **OpenTelemetry** as a first-class citizen in Azure Monitor
- Easier integration and data collection for distributed applications
- Multi-tier logging improves audit log management
- New simplified OpenTelemetry setup via **AKS Application Monitoring**

### 5. Optimization and Automation

- AI-guided agent identifies high-ROI cost savings and usability improvements
- Copilot functionality to auto-generate CLI scripts for optimization tasks, such as VM resizing to reduce costs and environmental impact

## Session Chapters

- Opportunities in AI-driven operations (Intro)
- Live demonstration of agentic AI troubleshooting
- Using visualization tools: Grafana & Prompt QL
- AI-powered troubleshooting workflows
- OpenTelemetry integration and logging improvements
- Simplified AKS monitoring deployments
- Identifying and prioritizing optimization actions
- Automating cost savings and VM management with Copilot CLI generation
