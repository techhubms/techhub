# GitHub for Beginners Season 3: Getting Started with GitHub Issues, Actions, and Pages

**Presented by GitHub**

Welcome to Season 3 of GitHub for Beginners! Designed for new developers and anyone starting out with GitHub, this season focuses on practical skills to help you:

- Create and manage GitHub Issues for project tracking
- Build and customize your developer portfolio with GitHub Pages
- Automate development workflows using GitHub Actions
- Collaborate on code within Visual Studio Code

## Topics Covered

- **Introduction to GitHub Issues**: Learn how to use Issues for bug tracking, feature requests, and team collaboration.
- **Getting Started with GitHub Actions**: Set up CI/CD workflows to automate builds, tests, and deployments.
- **Building Your Portfolio**: Step-by-step guidance on using GitHub Pages to showcase your work and projects.
- **Collaboration in Visual Studio Code**: Tips on working with others directly from VS Code using GitHub integrations.
