# Integrating GraphQL APIs with Fabric SQL Databases

**Presented by:** Microsoft Developer, featuring MVP Nicky van Vroenhoven

This session explains how to leverage GraphQL for simplified API access to Microsoft Fabric SQL Databases. Key topics include:

## 1. Introduction

- Overview of the integration scenario
- Relevance for modern, data-driven applications

## 2. Setup & Schema Generation

- Steps to expose a Fabric SQL Database via GraphQL
- Schema generation techniques using Microsoft Fabric
- Authentication configuration for secure access

## 3. Query Execution

- Demonstration of GraphQL queries against Fabric SQL databases
- Discusses response structures and common query patterns

## 4. Integration Options

- Explains how to connect GraphQL-enabled databases to various app platforms
- Details for integrating with Power BI and other Microsoft tools

## 5. Resources & Learning Paths

- [GraphQL API Overview](https://learn.microsoft.com/en-us/fabric/data-engineering/api-graphql-overview)
- [GraphQL API Limits](https://learn.microsoft.com/en-us/fabric/data-engineering/api-graphql-limits)
- [GraphQL Source Control & Deployment](https://learn.microsoft.com/en-us/fabric/data-engineering/graphql-source-control-and-deployment)
- [Microsoft Fabric GraphQL Module](https://learn.microsoft.com/en-us/training/modules/get-started-with-graphql-microsoft-fabric/)

## 6. Presenter Background

Nicky van Vroenhoven is a Microsoft Data Platform MVP specializing in BI solutions with Power BI, Power Platform, and T-SQL. With experience in SQL Server and BI consultancy, Nicky shares insights on using GraphQL with the Microsoft stack.
