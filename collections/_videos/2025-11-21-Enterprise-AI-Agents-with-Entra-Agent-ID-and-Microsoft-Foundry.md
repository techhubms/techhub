# Enterprise AI Agents with Entra Agent ID and Microsoft Foundry

## Overview

This Microsoft Ignite 2025 session features Akshaya Annavajhala, Gerald Ertl, and Amanda Foster, and demonstrates how Commerzbank and other organizations drive agentic AI at enterprise scale. The focus is on implementing AI agents that meet stringent standards for security, identity, connectivity, extensibility, and lifecycle governance using Microsoft Foundry and Entra Agent ID.

## Key Topics Covered

- **Introduction to Enterprise AI Agents**
  - Discussion of an AI agent (Ava) handling customer requests.
  - Introduction to Microsoft Foundry's role in deploying and governing agents.

- **Enterprise-Grade Constraints**
  - Addressing practical challenges like networking, identity, and compliance.
  - Assigning individual identities and organizational roles to agents via Entra Agent ID.
  - Provisioning and managing agent catalogs at the tenant level.

- **Agent Lifecycle and Operations**
  - Full lifecycle support for agents: deployment, monitoring, and retirement.
  - Custom tool integration and unified inventory through the Entra Admin Center.
  - Foundry Control Plane dashboard for real-time operational metrics.

- **Framework Benefits**
  - Agent-based approach achieves efficiency, consistency, and scalable governance.
  - Taxonomy of agent development methods and supported capabilities.

## Practical Takeaways

- How to manage secure agent authentication and authorization using Entra Agent ID.
- Best practices for networking configuration and compliance in enterprise AI deployments.
- Life cycle management and monitoring with control planes and dashboards.
- Techniques for integrating custom tools and managing agent catalogs.
