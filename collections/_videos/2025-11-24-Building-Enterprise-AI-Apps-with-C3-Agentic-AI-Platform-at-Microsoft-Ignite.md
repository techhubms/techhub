# Building Enterprise AI Apps with C3 Agentic AI Platform

**Speakers:** Nikhil Krishnan, Joel Morgenstern  
**Event:** Microsoft Ignite 2025, Session BRKSP488

## Session Overview

Learn how to turn business ideas into production-ready enterprise apps with C3 AI Studio. The platform lets you:

- Build apps instantly using a low-code Application Canvas or full-code VSCode tools
- Define goals in natural language and generate applications rapidly
- Refine generated apps with code for custom functionality
- Deploy securely with built-in governance, version control, and observability
- Create workflows, AI-powered agents, and dashboards in minutes
- Connect seamlessly to existing data sources and APIs
