# Multi-agent workflows in VS Code

Visual Studio Code demonstrates **multi-agent support in VS Code**, where you can run multiple agent sessions in parallel so you can work on several tasks at the same time instead of waiting for one task to finish before starting another.

## What the video covers

- Running **multiple agent sessions** side by side in VS Code
- Using agents in parallel to tackle different workstreams, including:
  - Implementing **color themes**
  - Adding **storage functionality**
  - Creating **documentation**
