# AI-assisted Migration: The Path to Powerful Performance on PostgreSQL

**Speakers:** Guy Bowerman, Jonathon Frost, Luigi Nardi, Ashokkan Somuveerappan  
**Event:** Microsoft Ignite 2025 | BRK123 | Advanced (300)

## Session Overview

This session covers how Apollo Hospitals achieved faster performance and innovation by migrating legacy databases, such as Oracle, to Azure Database for PostgreSQL. The presentation guides you through:

- Why organizations choose PostgreSQL and Azure together
- The benefits of managed PostgreSQL on Azure for enterprise workloads
- Steps to migrate existing Oracle workloads using AI-assisted migration tools
- Context and technical considerations for Oracle-to-Postgres migration
- Live demonstrations using Visual Studio Code and validation processes
- Capturing and verifying schema conversions between Oracle and PostgreSQL
- Continuous performance tuning leveraging advanced, automated tuning tools within Azure
- Key scenarios demonstrating the impact of intelligent tuning for demanding workloads

## Chapters

- **0:00** — Why Postgres and why on Azure
- **03:05** — Benefits of managed Postgres and how Azure meets enterprise needs
- **17:49** — Live demo: Creating and executing a migration project, including validation in VS Code
- **20:54** — Oracle-to-Postgres migration context and discussion
- **22:28** — Techniques for capturing coding notes and schema verification
- **35:01** — Performance tuning strategies and demonstration of the DB Tune automated workflow

## Key Takeaways

- **Migration Made Seamless:** Learn practical steps for migrating Oracle databases to Azure using AI-assisted guidance and automated tooling.
- **Database Resilience:** Discover new Azure Database for PostgreSQL features that support extensibility, reliability, and intelligent application growth.
- **Advanced Tuning:** Benefit from continuous tuning powered by machine learning to support complex and performance-intensive database scenarios.
