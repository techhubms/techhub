# GitHub Copilot Arm Cloud Migration Agent Deep Dive

In this GitHub Code & Community live stream, Eric Sondhi (PM, Arm) and Alexis Abril (Sr. Partner Engineer, GitHub) join Andrea Griffiths to explore a custom agent built for GitHub Copilot. This agent assists developers in migrating workloads from traditional x86 environments to Arm-based cloud infrastructure.

- **Practical Guidance**: Learn common pitfalls and best practices for multi-architecture cloud deployments
