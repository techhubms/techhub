# Bolster Your Data Security in the AI Era with Microsoft and Netskope

**Session Type:** Microsoft Ignite 2025 (ODSP1476)  
**Speaker:** Ankur Chadda  
**Format:** English (US), Pre-recorded, Foundational (100)

## Overview

This session helps organizations achieve consistent data security and regulatory compliance by unifying visibility and policy enforcement over all endpoints, SaaS, IaaS, and network traffic. Ankur Chadda demonstrates how Microsoft Purview's expanded discovery and classification features—integrated with Netskope's unified security platform—offer a seamless and centralized security experience for administrators and end users.

## Agenda

- **Balancing AI and Cloud Security:** Introduction to challenges and opportunities related to AI advancement and cloud expansion, and their impact on security and compliance.
- **Unified Security Approach:** Discussion of holistic solutions for enterprise security, focusing on Microsoft and Netskope integration.
- **Netskope One Coverage:** Overview of the architectural coverage provided by Netskope One and supported use cases.
- **Microsoft and Netskope Integrations:** Over 20 different touchpoints highlighting how these ecosystem integrations strengthen enterprise security across Microsoft solutions.
- **Improved Security & Compliance Outcomes:** Deep dive into how these integrations improve data protection, compliance reporting, and operational efficiency within the Microsoft cloud ecosystem.
- **Integrated Security Service Edge (SSE):** Exploration of administrating security policies in a unified manner for both data at rest and in motion.

## Technologies and Features

- **Microsoft Purview:** Data discovery, classification, and compliance management for Microsoft 365, cloud apps, and endpoints.
- **Netskope:** Security Service Edge (SSE) framework for real-time policy enforcement, data, and threat protection.
- **Expanded Integrations:** API-based and native integrations between Microsoft cloud and Netskope platforms to extend visibility and controls.

## Key Takeaways

- Strategies for maintaining visibility and compliance across complex cloud and endpoint environments.
- How to orchestrate policy and discovery spanning multiple platforms using Microsoft and third-party security controls.
- Practical administrator tips for leveraging unified security architectures.
