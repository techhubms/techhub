# VS Code Live: Agent Sessions Day – AI and Agentic Development in Visual Studio Code

Join GitHub and the Visual Studio Code product team for a global virtual event highlighting how VS Code is evolving as a developer editor redefined by AI. The agenda features demos and talks from leading team members, focusing on agents, customizable workflows, integration with AI models, and maintaining open-source community values.

## Agenda Highlights

- **Welcome** by Olivia Guzzardo McVicker, introducing Agent HQ Vision
- **Keynote Demo** with Kyle Daigle
- **How VS Code Builds with AI** by Harald Kirschner
- **Unified Agent Experience** with Peng Lyu and Pierce Boggan
- **Extending Agents with MCP** — Josh Spicer
- **The Browser in Your Editor** by Connor Peet
- **Bring Your Own Model in VS Code**: Kyle Cutler, Justin Chen, Joaquin Ruales
- **How We Ship Models**: Sandeep Somavarapu
- **Customizing Your Agents** with Julia Kasper
- **Copilot CLI in VS Code**: Courtney Webster
- **Best Practices for Agentic Coding** with Alex Weininger, Burke Holland, and Pierce Boggan
- **Closing Message** from Kai Metzel

## What You'll Learn

- How to leverage AI-powered agents and workflows in Visual Studio Code
- Customizing and extending VS Code with open integrations and model support
- Real-world demos of agentic development and Copilot CLI integration
- Best practices for building smarter, more efficient coding solutions with VS Code AI tools
