# If .NET Brewed Beer: IoT Brewing Automation with Raspberry Pi

Presented by dotnet

## Overview

This session shares the speaker's experience automating the beer brewing process, using a combination of Raspberry Pi and Meadow boards as the computing foundation. It emphasizes fast, efficient operation on low-powered devices, demonstrating how common software development skills can target embedded and IoT platforms with .NET.

## What You'll Learn

- Approaches to building control systems on low-powered hardware
- Practical, hands-on usage of .NET to interface with machine automation
- Hardware integration with Raspberry Pi and Meadow Boards
- Considerations for developing software for IoT and embedded environments

## Key Highlights

- Automating beer brewing through device programming
- Use of .NET for device management and automation
- Hands-on developer perspectives, not just high-level theory

## Who Should Watch

Anyone interested in learning about:

- Embedded software development
- Automating real-world processes using .NET
- IoT prototyping and project ideas
