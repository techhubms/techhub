# Build AI Apps Fast with GitHub and Microsoft Foundry in Action

**Speakers**: Kate Catlin, Facundo Santiago  
**Event**: Microsoft Ignite 2025 | Session BRK110  
**Level**: Intermediate (200)

## Overview

This session helps IT professionals accelerate AI application development with Microsoft’s Foundry platform and GitHub. It highlights the end-to-end process, from initial experimentation to large-scale deployment of intelligent applications.

## Key Topics Covered

- **AI App Lifecycle:** Full workflow from experimentation to production deployment of AI-driven apps
- **Integration of Tools:** Live demonstrations of the AI Toolkit, integrating GitHub, Microsoft Foundry, and Agent Framework
- **Orchestration & Security:** Focus on robust workflows, orchestration of multi-agent systems, and ensuring security and governance throughout development
- **Demo Use Case:** Classifying bug reports using GitHub Action and an AI model

## Chapters & Highlights

1. **Retrieval-Augmented Generation (RAG) to Agentic AI**
   - Introduction to leveraging RAG and proactive agentic approaches
2. **Demo: AI Toolkit**
   - Integration of GitHub, Microsoft Foundry, and the Agent Framework
3. **Classifying Bug Reports**
   - Running experiments with GitHub Actions and AI modeling
4. **Modular Agent Design**
   - Methods for scalable and maintainable agent architectures
5. **Tool Selection Challenges**
   - Insights on making choices in complex multi-agent ecosystems
6. **Multi-Agent Workflow Construction**
   - Avoiding monolithic designs through modular, conditional workflows
7. **Efficient Packaging & Deployment**
   - Using Azure Cloud Deploy (ACD) for agent packaging and rollout
8. **Conclusion**
   - Encouragement to build multi-agent systems with Azure Foundry

## Learnings & Best Practices

- **Workflow Orchestration:** Utilizing Foundry’s orchestration to efficiently move AI projects from concept to deployment
- **Security and Governance:** Leveraging built-in features for managing app security throughout the lifecycle
- **Scalable Architectures:** Designing modular agents for scalability and future-proof applications
- **DevOps Integration:** Using GitHub Actions and advanced DevOps practices to automate testing and deployment
