# ASP.NET Community Standup: What's Next for Orleans?

## Overview

In this episode, Daniel Roth, Mike Kistler, and Reuben Bond discuss the roadmap and upcoming features for the Orleans framework within the .NET ecosystem. Orleans is designed to simplify building scalable, distributed applications using the virtual actor model.

## Key Topics Discussed

- **Upcoming features in Orleans**: Details on future enhancements planned for scalability, reliability, and integration.
- **Orleans architecture improvements**: Insights into architectural evolutions to better support cloud-scale workloads.
- **Integration with ASP.NET Core**: How Orleans complements ASP.NET Core for modern, distributed web applications.
- **Real-world use cases**: Examples of how developers and organizations are leveraging Orleans for production workloads.

## Why Use Orleans?

Orleans enables .NET developers to write distributed systems with a simplified programming model, offering advantages for cloud native and scalable architectures.

Learn more about actor-based programming, new developer tools for distributed applications, and upcoming changes in the Orleans project directly from the team and .NET community leaders.
