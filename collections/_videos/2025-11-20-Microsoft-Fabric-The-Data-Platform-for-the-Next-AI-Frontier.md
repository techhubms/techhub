# Microsoft Fabric: The Data Platform for the Next AI Frontier

Presented by **Amir Netz, Technical Fellow and CTO of Microsoft Fabric** at Microsoft Ignite 2025.

## Session Overview

As AI adoption accelerates, organizations must transform their data platforms to provide a unified foundation for AI success. Amir Netz shares major updates and developments for Microsoft Fabric, focusing on building enterprise-ready solutions for data and analytics.

## Key Highlights

### 1. New Data Services

- **SQL and NoSQL SaaS Databases:** Announced for Microsoft Fabric, enabling cloud-based data management tailored for enterprise scale and flexibility.
- **Cosmos DB Integration:** Alert functions now integrate into development environments with Cosmos DB, supporting operational and event-driven use cases.

### 2. Data Warehousing and Migration

- **Synapse vs. Fabric Data Warehouses:** Comparison of migration strategies and feature advancements to help organizations transition and maximize performance.

### 3. Advanced Data Modeling

- **Fabric Graph Modeling:** Introduced to enable robust knowledge and relationship modeling, streamlining complex data scenarios.
- **Fabric Graph for Microsoft Sentinel:** Powers advanced threat detection and monitoring, connecting data platform insights with security operations.

### 4. Platform Operations and Management

- **Capacity Overage Billing and Surge Protection:** Workspace-level controls to manage billing and usage spikes.
- **Extended OneLake Access:** Open APIs and formats allow integration with multiple Microsoft and third-party tools, enhancing flexibility in data access and analytics.
- **Fabric IQ and Semantic Modeling:** Launch of Fabric IQ introduces semantic models into operational data models and supports ontology-driven data analysis.
