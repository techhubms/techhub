# Partners: Choosing the Best Azure Database for Customer Scenarios

**Speakers:** Shireesh Thota, Jesse Sullivan, Mike Thomsen  
**Session Type:** Microsoft Ignite 2025 (Intermediate)

## Overview

Choosing the right database solution is essential when supporting customers migrating to the cloud, modernizing enterprise applications, or building new AI-driven solutions. This session delivers actionable knowledge for partners, highlighting managed database options within Azure and key use cases for each.

## Key Topics Covered

- **Azure Database Options:**
  - Overview of Microsoft’s managed database portfolio
  - When to use Azure SQL Database, Cosmos DB, and others
  - Advantages of co-locating databases with apps, analytics, and AI workloads on Azure

- **Partner Scenarios:**
  - Migration of on-premises applications to the cloud
  - Application modernization strategies
  - Building AI-powered and analytics solutions with Azure databases

- **Pitching Value to Customers:**
  - Differentiation of Azure offerings for various workload needs
  - How partners can position Azure as the platform that supports broad business goals
