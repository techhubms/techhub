# Transform Manufacturing with Digital Twins and Real-Time Simulation

**Presented by Microsoft Events | Microsoft Ignite 2025**

## Overview

This foundational session delves into how digital twins and real-time simulation are revolutionizing manufacturing and mobility. Learn how companies are using Microsoft Azure, NVIDIA AI, and Ansys solutions—integrating Omniverse and OpenUSD technologies—to optimize processes and achieve faster insights.

### Speakers

- Donald Ferguson
- Ganesh Sivaraman

## Session Highlights

- **Introduction & Greetings** (Seth Juarez)
- **Real-Time Traffic Data Processing with AI and Azure Cloud**
- **Optimizing Signal Timing and Designing Future-Proof Roadways**
- **Visualizing Live City Traffic on Interactive 3D Maps**
- **Improving Emergency Response with Incident Detection**
- **Collaboration Between Microsoft and NVIDIA**
- **Rapid AI Model Deployment Using Azure and NVIDIA DeepStream**
- **Sharing Lessons Learned from Traffic AI Projects**
- **Closing Remarks & Gratitude**

### Technologies Featured

- Microsoft Azure cloud platform
- NVIDIA DeepStream and accelerated computing
- Omniverse™ libraries
- Digital twin and real-time simulation architectures
- OpenUSD (Universal Scene Description)
- AI model deployment and integration
- Traffic data processing and visualization

### Key Takeaways

- How to leverage cloud and AI for smarter cities and manufacturing
- Strategies for rapid deployment and scaling of AI models
- Enhancing emergency response and traffic management through real-time analytics
- Importance of collaboration between technology providers for innovative manufacturing solutions
