# Accelerating Agentic Journeys in Financial Services with Microsoft AI

**Speakers:** Jared Lambert, Hemanth Sundararaj, Adam Wheat  
**Session:** PBRK393, Microsoft Ignite 2025

## Overview

This session explores how frontier partners in the financial services industry are leveraging Microsoft AI solutions—especially Copilot Studio—to build intelligent agents that address banking, capital markets, and insurance challenges.  Case studies such as Morningstar illustrate how data, integration, and AI tools drive business value and digital transformation.

## Agenda & Chapters

- **AI Use Cases in Financial Services:**  
  Introducing intelligent agent applications across banking, capital markets, and insurance.
- **Three Phases of AI Adoption:**  
  1. Assisted work  
  2. Agent collaboration  
  3. Autonomous operations
- **Project Tips and Recommendations:**  
  Focus on agent intent, constructing robust responses, and designing effective conversational interfaces.
- **Extending Copilot:**  
  Using connectors to power client-specific agents and enable advanced automation.
- **Morningstar Case Study:**
  - Overview of Morningstar’s mission
  - The importance of data as a core asset
  - Integrating client holdings using the MCP Server
  - Deploying agents for client-centric analytics
  - Real-world demo analyzing Microsoft stock exposure
