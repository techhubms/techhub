# Migrating to Microsoft Fabric: Unifying Your Data Journey

**Session Speakers:** Jenny Jiang, Mark Kromer, Priyanka Langade, Pushpendra Nagtode

This breakout session from Microsoft Ignite 2025 covers the modernization and migration journey from Azure Synapse Gen 2 and Azure Data Factory to Microsoft Fabric. Attendees will discover:

## Key Topics Covered

- **Migration Scenarios**: Real-world paths from Synapse and Data Factory to Fabric
- **Best Practices**: Strategies and tips for seamless platform migration
- **Unified Data Platform**: How Microsoft Fabric serves as a single destination for integration, transformation, and analytics
- **Fabric Warehouse Roadmap**: Advanced features and future plans for data warehousing within Fabric
- **Transition Details**: Deep dives into migration mapping between Synapse workloads and Fabric
- **Spark Engineering**: How Fabric leverages Spark for modern analytics
- **Copy Job Feature**: Simplified data movement with new Fabric functionality
- **Cloud Accelerate Program**: Support options and resources for large-scale migration
