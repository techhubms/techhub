# Add an AI Agent to Your App with GitHub Copilot SDK

The GitHub Copilot SDK is now available in technical preview, offering developers a straightforward way to embed AI-powered agents into any application. Whether you utilize your existing GitHub Copilot subscription or provide your own model key, the SDK manages complex integration steps automatically.

## Key Features

- **Easy Integration**: Add AI functionality to apps with just a few lines of code.
- **Flexible Authentication**: Use an existing Copilot subscription or bring your own model key.
- **Outcome Driven**: Describe the agent's outcome and capabilities within the SDK.
- **Rapid Deployment**: Quickly ship AI-enabled agents into your projects.

[Learn more about GitHub Copilot SDK](https://github.com/github/copilot-sdk?utm_source=youtube-sizzle-marketing-video-copilot-sdk&utm_medium=social&utm_campaign=cli-sdk-jan-2026)

## Getting Started Steps

1. **Install the GitHub Copilot SDK** for your application language.
2. **Configure your agent** by describing its desired capabilities and required outcomes.
3. **Authenticate** using your Copilot subscription or supply your own model key.
4. **Deploy or test your application** to begin interacting with your new AI agent.

> “You can now add an AI agent to any application with just a few lines of code... the SDK handles the complex steps for you.”
