# Building Autonomous Enterprise Agents with Reasoning in Microsoft Foundry

**Speakers:** Ashish Agrawal, Alexander Hughes, Jennifer Marsman, Naomi Moneypenny, Zachary Ziegler

## Overview

This session from Microsoft Ignite 2025 delves into constructing autonomous agents using advanced reasoning models. These agents are capable of planning, executing, and adapting across diverse enterprise tasks, including:

- Lead scoring
- Content generation
- Customer support

Attendees will learn how to leverage Microsoft Foundry for agent development, ensuring alignment with enterprise goals and robust operational reliability.

## Key Topics Covered

- **Showcase of Partner and NVIDIA Nematron Reasoning Models**: Introduction and demonstrations highlighting new capabilities.
- **Model Router**: Jennifer Marsman explains model routing and how agents select the most appropriate reasoning engine.
- **Multi-Agent Collaboration**: Introduction of GPT-5.1 and collaboration between agents such as researcher, writer, editor, verifier, and reviewer.
- **Free-Form Tool Calling**: Integration of tools like weather and time for broader functionality using flexible workflows.
- **Benefits of Parallel and Free-Form Execution**: Faster and more cost-efficient applications through simultaneous and flexible operations.
- **Life-Critical Use Cases**: Ensuring uptime and accuracy for physician queries using the Deep Consult system.

## Demonstrations

- Step-by-step demos detail how to create agents within Microsoft Foundry and connect multiple enterprise systems.
- Examples showcase both the technical and business aspects of agentic solutions for advanced enterprise scenarios.
