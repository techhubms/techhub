# Customize GitHub Copilot Agents in Visual Studio Code

Discover how to turn GitHub Copilot in VS Code into a specialized AI teammate, tailored to your team's workflows. This session covers:

- **Building Custom Agents**: Learn how to create agents with specific roles to assist in coding tasks within VS Code.
- **Agent Skills**: Understand how to equip your agents with specialized skills and connect them with GitHub Copilot for enhanced performance.
- **MCP Integrations**: See how to leverage MCP (Microsoft Copilot Platform) integrations to strengthen your AI companion's capabilities.
- **Workflow Handoffs**: Set up seamless transitions and handoffs between agents to guide development workflows for specific processes.
- **Practical Team Patterns**: Explore real-world patterns and best practices for making Copilot adapt to the way your team collaborates and codes.
