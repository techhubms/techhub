# Accelerating Oracle to SQL Server Migrations with AI and Copilot in SSMA

**Presented by Microsoft Developer**

This episode explores how Copilot, powered by AI, is now a part of the SQL Server Migration Assistant (SSMA) for Oracle. Key highlights include:

## Key Topics

- **Overview of Copilot in SSMA for Oracle**  
  Discover how Copilot leverages AI to:
  - Improve the accuracy of code conversion from Oracle to SQL Server
  - Reduce the need for manual migrations and corrections
  - Assist users in troubleshooting and resolving complex migration blockers

- **Demo Walkthrough**  
  The episode presents a demonstration showing:
  - The step-by-step process of migrating from Oracle to SQL Server using SSMA with Copilot assistance
  - How Copilot identifies and addresses conversion challenges

- **Authentication Methods**  
  A breakdown of the authentication methods supported by SSMA for Oracle, enhancing security and flexibility during migration.

- **Roadmap and Future Investments**  
  Preview of ongoing and future enhancements planned for Copilot in SSMA, aimed at further automating and simplifying the migration journey.
