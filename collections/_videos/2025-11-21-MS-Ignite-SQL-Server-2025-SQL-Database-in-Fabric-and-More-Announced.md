# MS Ignite: SQL Server 2025 GA, SQL DB in Fabric GA & More!

Discover everything new in the Microsoft SQL technology space as announced at Microsoft Ignite 2025.

## Key Announcements

- **SQL Server 2025 General Availability (GA)**: The latest version, focusing on improved scalability, cloud-native integrations, and new features for enterprise data management.
- **SQL Database in Microsoft Fabric GA**: SQL Database now integrates with Microsoft Fabric, opening up analytics and data engineering capabilities in Fabric's unified platform.
- **Azure SQL Updates**: Announcements include service enhancements and developer resources for Azure SQL.
