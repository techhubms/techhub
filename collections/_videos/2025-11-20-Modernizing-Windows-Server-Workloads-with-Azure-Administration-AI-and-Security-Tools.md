# Modernizing Windows Server Workloads with Azure Administration, AI, and Security Tools

**Speakers:** Dan Cuomo, Jeff Woolsey  
**Microsoft Ignite 2025 — Session BRK142**

## Optimizing Windows Server Workloads in Azure

This session demonstrates how to migrate, configure, and administer Windows Server workloads using modern Azure infrastructure and management tools:

- **Azure Administration Tools:**
  - **Windows Admin Center (WAC)**: Streamline Windows Server and workload management within Azure.
  - **Azure Copilot and AI Shell:** Leverage AI-powered features for automation, rapid troubleshooting, and operational insights.
  - **Azure Arc Integration:** Centralize virtual machine management within the Azure portal.

## Key Migration and Management Areas

- **Azure Migrate Capabilities**: Assess and support migration for various workloads.
- **Monitoring and Performance:**
  - Use **Azure Monitor** for dependency mapping and dashboarding.
  - Share dashboards across teams to help break down silos.
- **Workload Flexibility:** Manage both cloud-connected and on-premises workloads seamlessly.
- **Virtualization Setup**: Walkthrough of prerequisites and setup steps for running workloads in virtualization mode.
- **Disaster Recovery:**
  - Utilize **Hyper-V Replica Management** for simplified BCDR (Business Continuity and Disaster Recovery).
- **Security and Compliance:**
  - Configure AI-enhanced monitoring and compliance tools.
  - Manage updates and implement best practices for workload security.
- **Identity Management:**
  - Simplify Windows Server identity services using **Entra Domain Services** within Azure.

## Roadmap and Future Capabilities

- Plans for expanded virtualization management features and further integration of AI and Copilot experiences in Windows Server administration.
- Upcoming enhancements to deployment workflows and dashboard customization.
