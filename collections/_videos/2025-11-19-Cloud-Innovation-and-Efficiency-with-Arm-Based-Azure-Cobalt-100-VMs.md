# Cloud Innovation and Efficiency with Arm-Based Azure Cobalt 100 VMs

Session from Microsoft Ignite 2025 | Speaker: Sameer Nori

## Overview

Learn how enterprises unlock increased performance, efficiency, and scalability using Azure Cobalt 100 Arm-based virtual machines (VMs). The session highlights customer adoption examples, technical deep dives, and guidance for migrating and modernizing workloads on Arm-powered Azure infrastructure.

## Why Developers Choose Arm

- **Performance, maturity, and efficiency** are cited as top reasons for adopting Arm-based VMs on Azure.

## Software Collaboration

- Integration with **GitHub runners** for CI/CD
- Windows performance optimizations for development and testing on Arm
- Methods to optimize software for Arm platforms

## Workload Performance

Detailed benchmarks and technical results for workloads including:

- **NGINX**
- **Redis**
- **Postgres**
- **SPECjbb**

Performance gains are highlighted for typical cloud applications deployed on Arm infrastructure.

## Partner Highlights

- **One Trust** deployment and use cases with Cobalt 100 VMs in Azure
- Insights on adoption, outcomes, and optimization strategies

## AI and Data Workload Integration

- **Databricks** integrates Cobalt 100 VMs for scalable AI and data workloads
- Demonstrates how Arm-powered infrastructure accelerates machine learning and analytics tasks

- Over **150 interactive learning paths and tutorials** provided for migrating, deploying, and optimizing workloads on Arm-based Azure VMs
