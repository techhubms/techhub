# What’s New in Copilot Studio and Roadmap

**Speakers:** Omar Aftab, Erez Altus, Zac Cadi  
**Event:** Microsoft Ignite 2025

Explore the newest innovations and upcoming features in Copilot Studio, Microsoft's platform for building AI agents with low-code capabilities. This session provides a deep dive into both embedded builder and enterprise-grade versions of Copilot Studio, along with hands-on demonstrations and use cases.

## Key Highlights

- **Enhanced Knowledge Capabilities:** Improvements enabling more robust and contextual AI agent knowledge management.
- **Generative Orchestration:** New advancements in orchestrating generative AI workflows and processes within agent solutions.
- **Testing and Debugging Tools:** Introduction of updated toolsets for easier debugging and validation of AI agents before deployment.
- **Microsoft 365 Copilot Integration:** New features assisting information workers through tighter integration with M365 Copilot, including the Copilot Agent Builder for streamlined enterprise agent creation.
- **Governance and Cost Management:** Expansions in governance controls, allowing organizations to better manage platform usage and agent identities, as well as track and control costs associated with agent operations.
- **Real-World Impact:** Customer success stories demonstrating increased efficiency, improved experiences, and effective automation through Copilot Studio.

## Session Chapters

1. **Overview of Copilot Studio Versions** (Embedded builder vs. enterprise-grade suite)
2. **Features for Information Workers** (Including M365 Copilot integration)
3. **Building and Managing Agents** (Tools and practices for rapid agent development)
4. **Evaluation and Governance** (Creating evaluation question sets, scoring, and platform management)
5. **Demo: Agent Identity in Action**
