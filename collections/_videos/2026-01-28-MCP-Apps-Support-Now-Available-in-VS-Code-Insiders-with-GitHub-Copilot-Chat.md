# MCP Apps Support in VS Code Insiders

Visual Studio Code has announced that MCP Apps are now supported in VS Code Insiders.

Developers can leverage this integration to interact with MCP Apps through GitHub Copilot Chat, allowing responses to be rendered as GUI outputs within the agent environment.

**Key Highlights:**

- **MCP Apps Integration**: Available within VS Code Insiders for hands-on experimentation.
- **GUI Responses**: MCP agent responses can now produce GUI-based outputs, improving the user experience.
- **GitHub Copilot Chat**: Seamlessly utilize Copilot Chat to query and manage MCP Apps workflows.
- **Try it Now**: Download the latest VS Code Insiders build to start using this feature.
