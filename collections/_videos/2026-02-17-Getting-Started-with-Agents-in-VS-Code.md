# Getting Started with Agents in VS Code

Join the Cozy AI Kitchen for an episode featuring Elijah King, Product Designer on the VS Code team. Elijah demonstrates the power of VS Code’s new multi-agent background workflow that lets anyone—even non-developers—build a fully functional iOS chatbot app using only simple prompts and natural language.

## Key Topics Covered

- **Building No-Code Apps**: Learn how non-developers and designers can transition to app creators using VS Code's agent-powered workflows, reducing the reliance on manual coding.
- **AI-Powered Workflows**: See VS Code’s multi-agent design in action, where background agents automate complex logic while users focus on high-level ideas and prompts.
- **Model Switching**: Discover how seamlessly switching between different AI models enhances flexibility and creativity in app development.
- **Rapid Prototyping**: Watch Elijah build a no-code iOS chatbot from scratch using prompt-based interactions, accelerating the transformation from idea to app.
- **Design to Engineering**: Insights into how designers can leverage AI to cross into software engineering roles, expanding the accessibility of development tools.

## Episode Highlights

- Live demo of iOS chatbot app creation
- Explanation of agent workflows in VS Code
- Tips for leveraging natural language prompts for app prototyping
- Discussion on the evolving landscape of AI-powered developer tools
