# Build Secure Agentic AI Apps with SQL Server 2025

## Introduction and Session Overview

Welcome to Microsoft Ignite 2025. In this STUDIO33 session, Anna Hoffman and Sirjad Parakkat discuss how Ivanti is deploying next-generation service management solutions powered by SQL Server 2025’s advanced AI features.

## Key Themes

- **Security, Performance, Availability:** SQL Server 2025 focuses on strengthening core enterprise requirements while introducing new AI-driven capabilities.
- **Agentic AI:** The integration of agentic AI simplifies IT operations, drives productivity, and supports ongoing innovation and employee satisfaction.
- **Vectorization and Embeddings:** SQL Server 2025 introduces support for vectorization and embeddings, enabling more sophisticated AI workloads directly within the database ecosystem.
- **Alternative Vector Database Challenges:** The session compares SQL Server 2025 to other platforms like Pinecone and ChromaDB, highlighting improved integration and operational comfort.

## One Consistent SQL for the Era of AI

Anna Hoffman details the overarching vision for SQL Server as a unified platform for AI-powered applications, emphasizing a consistent developer experience and enterprise reliability.

## Developer Enablement

- **SQL AI Developer Certificate:** An upcoming certification helps validate skills in building AI-enabled solutions with SQL Server 2025.

## AI Adoption and Enterprise Guardrails

Speakers describe fostering trust in AI through robust enterprise-grade guardrails, making the adoption of AI approachable and secure for organizations.
