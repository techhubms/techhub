# Security Copilot: Empowering Security Teams with AI

*Presented by Dorothy Li, Fabrizio Dell'Oglio, Dilip Radhakrishnan at Microsoft Ignite 2025 (Session BRK244)*

## Session Overview

Security teams are under increasing pressure, facing high complexity and resource constraints. This session introduces Security Copilot, Microsoft's generative AI-powered platform that transforms security and IT workflows. Participants discover how AI-driven automation, unified data platforms, and deep integrations help modernize SecOps.

## Key Topics & Demonstrations

- **Overview of Microsoft Fabric**: Learn how Microsoft Fabric provides a unified data platform to support modern security operations.
- **Esri Partnership and ArcGIS Maps Integration**: See how ArcGIS Analytics Engine integrates with Fabric to unlock spatial insights for security data.
- **Development Environments for Fabric Functions**: Guidance on establishing dev, test, and production environments that automate security and data workflows.
- **Automation with Fabric Agents**: Connect Fabric agents to custom data functions, enabling IT automation and rapid response.
- **Agent Intelligence for Custom Logic**: Explore AI-driven voice and data agent capabilities to streamline workflows, including refund logic integration.
- **OneLake Shortcuts for Data Unification**: Introduce features that unify data across SharePoint, OneDrive, Google BigQuery, Oracle, and Snowflake for better security visibility and real-time analytics.
- **Snowflake Integration Demo**: Live demonstration of real-time queries and analytics using OneLake and Snowflake.

## Speaker Insights

Microsoft product leaders share practical strategies for building resilient security architectures, leveraging AI for threat detection, and automating repetitive tasks. The session is tailored to security professionals, data engineers, and IT practitioners seeking to maximize the speed and scale of their operations through Microsoft platforms and partner integrations.
