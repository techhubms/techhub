# What’s New and Next for Azure Virtual Desktop: Ignite 2025 Recap

**Speakers:** Roop Kiran Chevuri, Mark Douglas, Eric Orman  
**Event:** Microsoft Ignite 2025, Session BRK339

## Overview

Azure Virtual Desktop (AVD) continues to improve, offering enhanced management efficiency, better scalability, and new connectivity features. This Ignite 2025 session provides IT professionals with actionable updates and demonstrations relevant for AVD planning and operations.

## Key Innovations

- **App Attach Enhancements:** Expanded third-party support enables smoother application management and delivery.
- **B2B Access (General Availability):** New secure B2B login with improved identity and security controls, including Multi-Factor Authentication (MFA).
- **User Licensing and Pricing:** Introduction of per-user access pricing and external user licensing, making AVD more flexible for different organizational scenarios and partners.
- **SLB Demo:** Live demonstration of secure B2B login using AVD and MFA.
- **Dynamic Auto Scale:** Configuration options now support maintaining 100% minimum active hosts, enhancing reliability and scaling.
- **User Experience Improvements:** Enhanced session reliability and client connectivity.
- **RDP Multipath:** Major breakthrough allowing multipath capability, demonstrated with live failover (no user disconnection).
- **Migration API:** Streamlined migration approach for moving VDI desktops to Windows 365, simplifying cloud adoption.

## Industry Applications

- **Healthcare Focus:** Azure Virtual Desktop is highlighted as especially attractive for healthcare organizations requiring strict compliance, remote access, and robust security controls.
