# How to Modernize Oracle Workloads on Azure | BRK162

## Overview

This Microsoft Ignite 2025 session, delivered by Kalyan Kaki and Tushar Pandit, provides a roadmap for organizations seeking to migrate and modernize their Oracle applications and databases by leveraging Microsoft Azure’s cloud services.

### Key Agenda Points

- **Oracle Database at Azure and AI Integration**: Understanding how Oracle workloads can move seamlessly to Azure, benefiting from the secure and high-performance Azure network.
- **Demonstration: Agent Usage Across Sales Teams**: Showcasing practical adoption within business processes.
- **Oracle Database Cloud Services and Adoption**: Exploring Azure’s portfolio for Oracle database workloads and enterprise adoption strategies.
- **Portfolio Overview**: Details on available Oracle Database services within Azure, including deployment models and configurations.
- **Oracle MAA Configurations**: Recap of Maximum Availability Architecture (MAA) for high resilience.
- **Azure Marketplace Procurement**: Streamlining financial transactions and subscriptions for Oracle on Azure.
- **Security Integrations**: How Azure Key Vault and Sentinel enhance the security posture of Oracle workloads.
- **Frontier AI and Data Unification**: Using Microsoft Fabric to combine Oracle data with other critical sources and harness the potential of Azure AI for application modernization.

## Detailed Session Topics

### 1. Migrating Oracle to Azure

- Outline of steps and considerations for moving on-premises Oracle applications and databases to Azure’s cloud infrastructure.
- Benefits such as operational streamlining, increased agility, and cost flexibility.

### 2. Data Unification with Microsoft Fabric

- Techniques for combining Oracle data with other enterprise data sources.
- Leveraging Microsoft Fabric for business intelligence, analytics, and unified data experiences.

### 3. Accelerating Modernization with Azure AI Services

- Introduction to how Azure AI services can modernize legacy Oracle workloads.
- Integration scenarios for embedding AI-driven insights and automation into Oracle-based solutions.

### 4. Security and Compliance

- Using Azure Key Vault to securely store connection secrets and credentials for Oracle databases.
- Integrating with Azure Sentinel for proactive threat detection and incident response for Oracle workloads.
- Discussion of enterprise security standards and cloud governance.

### 5. Enterprise Adoption and Operations

- Overview of Oracle Database cloud services offered natively on Azure.
- Enterprise procurement and financial integration via Azure Marketplace.
