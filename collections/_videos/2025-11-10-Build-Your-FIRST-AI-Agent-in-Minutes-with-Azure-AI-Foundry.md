## Getting Started

1. **Set up your Azure AI Foundry account:**
   - Go to [Azure AI Foundry](https://ai.azure.com/) and create an account.
2. **Create an AI Agent Project:**
   - Use the Foundry interface to launch a new agent. Select basic parameters.
3. **Configure Agent Instructions and Personality:**
   - Define how your agent responds and its core behavior traits.
4. **Add Knowledge Using RAG (PDF Upload):**
   - Upload private documents for agent use. Enables Q&A with your data (RAG-style retrieval).
5. **Test the Agent Live:**
   - Try prompts; verify conversational performance.
6. **Monitor Performance:**
   - Use Foundry dashboard to track usage and response quality.
7. **Clean Up Resources:**
   - Delete unused projects to avoid unexpected Azure billing.
