# Azure Update - 19th December 2025: Service Improvements, AI, and Security News

**Author:** John Savill's Technical Training

Welcome to the final Azure weekly update of 2025, bringing a fast-paced summary of recently released features, security advancements, and AI-related improvements, plus a festive Azure Christmas song segment.

## Highlights This Week

- **Service Bus Trigger for Functions**: Overview of recent improvements for Azure Functions utilizing the Service Bus trigger, focusing on enhanced reliability and performance.
- **ANF CZRR & Advanced Ransomware Protection**: Updates on Azure NetApp Files (ANF), with new ransomware protection features designed to strengthen enterprise storage security.
- **ASM Blob to Blob Feature**: New advancements for ASM storage accounts enabling blob-to-blob operations.
- **Azure SQL Database Serverless Auto-Resume**: Details about automatic resume features that improve the efficiency of serverless Azure SQL Database deployments.
- **Cosmos DB SRE Enhancements**: Emphasis on Site Reliability Engineering practices introduced to Cosmos DB, targeting improved cloud database availability.
- **AI and GPT Model Updates**: Highlights include the rollout of GPT-image-1.5 in Foundry and updated voice models now available in the Azure AI platform.
