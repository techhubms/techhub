# .NET Data Community Standup: How dotConnect and Entity Developer Streamline .NET Workflows

**Presenters:** Yurii Hryhoriev, Jiri Cincura, Shay Rojansky

## Overview

This video demonstrates the use of dotConnect and Entity Developer to enhance the .NET data development experience. The speakers share practical tips and walk through live demos to show how these tools:

- Simplify connection to various databases in .NET projects
- Integrate with Entity Framework Core for streamlined ORM (Object-Relational Mapping)
- Automate code and model generation
- Improve data modeling and management via visual tools

## Key Topics Covered

- Setting up dotConnect in a .NET solution
- Using Entity Developer to create and maintain data models visually
- Integrating data access layers with Entity Framework Core and ADO.NET
- Productivity gains and workflow improvements for database-oriented .NET applications
- Real-world scenarios where these tools save time and reduce errors

## Who Should Watch

- .NET developers working with databases
- Teams building data-centric enterprise applications
- Anyone interested in improving workflow efficiency with Microsoft-centric tools
