# Build a Planning App with the GitHub Copilot SDK | demo

What if a personal planning app could “think with you”? This video shows how to use the **GitHub Copilot SDK** (via the **GitHub Copilot CLI SDK**) to upgrade a basic **Node.js** application into a dynamic **AI-powered assistant**.

## What the demo builds

- **Weekly meal plans** generated dynamically
- Ability to **regenerate** options as preferences change
- A **weekend activity schedule** feature
- A brief **code walkthrough** so you can build something similar
