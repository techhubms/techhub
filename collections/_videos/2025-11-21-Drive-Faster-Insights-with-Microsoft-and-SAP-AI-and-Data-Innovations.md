# Drive Faster Insights with Microsoft and SAP AI and Data Innovations

**Speakers:** Robert Hernandez, Nathan Weaver  
**Event:** Microsoft Ignite 2025 | BRK163 | Advanced (300)

## Session Overview

This session explores how Microsoft and SAP are combining efforts to unlock enterprise data potential with artificial intelligence and advanced data architectures. Attendees will see:

- Microsoft and SAP joint innovations connecting data silos for unified analytics
- How SAP Business AI and Microsoft Copilot streamline workflows and drive automation
- Strategies to merge SAP and non-SAP data in Azure for actionable business insights

## Key Topics and Demos

- **Three Pillars:** Integration across productivity, data, and security
- **SAP Business Data Cloud:** Deployment and partnership with Azure data centers
- **Enterprise Data Merging:** Microsoft’s approach to transforming SAP and non-SAP data sources by business function
- **SAP and Microsoft 365 Connectivity:** Introducing Jewel Connector, simplifying unified access
- **MCP (Microsoft Copilot for SAP):** Connecting Large Language Models (LLMs) with enterprise SAP systems for task automation
- **Live Demo:** Using Copilot to aggregate business insights and execute transactional actions within SAP systems
- **Researcher/Analyst Agents:** Enhancing reasoning and analytics using integrated AI assistants

## Benefits and Outcomes

- Greater efficiency in analytics and reporting by bridging SAP and Microsoft platforms
- Automation of frontline and back-office processes via AI-driven insights
- Simplified enterprise architecture that leverages Azure for global data unification
