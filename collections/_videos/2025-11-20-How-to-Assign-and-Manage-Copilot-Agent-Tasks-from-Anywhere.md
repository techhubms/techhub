# How to Assign and Manage Copilot Agent Tasks from Anywhere

Explore the latest features of GitHub Copilot coding agents with insights from the GitHub team. This video covers how developers can efficiently assign, monitor, and manage coding agent tasks from a variety of platforms, improving workflow flexibility and productivity.

## Key Topics Covered

- **Assigning Tasks from Visual Studio Code**: Learn how to directly assign Copilot agent tasks within VS Code, streamlining integration into your development workflow.
- **Managing Autonomous Agents Locally with Copilot CLI**: Discover how Copilot CLI allows you to run and steer coding agents on your local environment for increased control.
- **Assigning Tasks from Slack**: Understand how you can delegate coding tasks to Copilot agents via Slack, enabling seamless task management right from your communication platform.
- **Centralized Copilot Coding Agents Page**: Get familiar with the new agent page that provides a single location to monitor and steer all Copilot coding agent tasks, offering unified visibility and oversight.
- **Custom Agents for Specific Tasks**: Find out how to create customized Copilot agents tailored for particular coding or operational functions, allowing you to automate unique developer workflows.
