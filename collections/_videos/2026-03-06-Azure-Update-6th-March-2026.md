## Key Azure Platform Updates

### Confidential VMs (DCe and ECe v6)

- Announced enhancements to confidential compute with new VM types for sensitive workloads, offering improved isolation and security for data in use.

### Azure Firewall: Draft and Deploy

- Introduction of a workflow allowing administrators to draft and test firewall configurations before deploying to production, minimizing risk of access errors.

### Databricks Updates

- **Workspace network configuration**: Improved controls for networking in Azure Databricks workspaces.
- **Lakebase**: New capability for managing and analyzing large datasets efficiently in Databricks environments.

### AI and Model Integration

- **Grok 4.0** and **Qwen3.5 Medium**: Release of new AI model series, further expanding Azure's AI ecosystem for advanced reasoning and text generation tasks.
- **OpenAI GPT-5.3 & GPT-5.4**: Integration of the latest GPT models within Azure AI Foundry and GitHub Copilot-related services, enabling developers to access cutting-edge language capabilities.
- **Model use and integration**: Focus on practical application and selection of the right model for specific scenarios, highlighting integration strategies with Azure tools.
- **Phi-4-Reasoning-Vision**: New model focused on combining visual and language reasoning, providing broader AI use cases.

### Azure Arc Gateway

- Introduction of gateway functionality to extend Azure management and security to on-premises and hybrid environments via Azure Arc.

### Azure Policy Enhancements

- Speed improvements for policy enforcement, reducing lag between configuration and compliance monitoring.
