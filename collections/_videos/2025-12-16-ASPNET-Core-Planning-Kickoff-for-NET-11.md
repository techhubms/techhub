# ASP.NET Core Planning Kickoff for .NET 11

**Featured Guests:** Daniel Roth, Mike Kistler, Jose Perez Rodriguez, Larry Ewing

Join the .NET team in this ASP.NET Community Standup focused on the planning kickoff for ASP.NET Core in .NET 11. This session explores:

- How the ASP.NET team approaches planning for major releases
- Opportunities for developer and community involvement in the planning process
- Early priorities and directions for ASP.NET Core features in .NET 11
- Ways to give feedback and join the conversation via community links and platforms

## Key Points

- **Planning Approach:** Learn about the methods and frameworks used by Microsoft’s ASP.NET Core team when planning new features and improvements.
- **Community Participation:** Find out how you can get involved, share ideas, and help shape the future of ASP.NET Core.
- **Feature Roadmap:** Early insights into what’s being considered for .NET 11 in terms of web development frameworks and capabilities.
