# Agents at Work: Windows, Copilot Studio, and AI-Powered Cloud Innovation

Presented by Phil Gerity, Stefan Kinnestrand, and Navjot Virk at Microsoft Ignite 2025, this session dives into the transformative role of Windows as the canvas for intelligent productivity.

## Session Highlights

- **Intro & Agenda**: Stefan Kinnestrand outlines how 'Agents at Work' leverages Windows for a new era of work.
- **Windows as AI Canvas**:
  - Three innovation pillars driving AI evolution in Windows.
  - Copilot Plus PCs introduced with enhanced AI performance and exclusive features.
- **Demonstrations**:
  - **Semantic Search**: Improved search and knowledge retrieval.
  - **Click to Do** and **Agent Mode (Excel on Copilot Plus PC)**: Streamlined task automation.
  - **Fluid Dictation**: Accessibility enhancement for users.
- **Enterprise Innovation**:
  - Transition to enterprise-grade cloud integration with Windows.
  - Copilot Studio showcased with over 3 million agents built, supporting developer and maker workflows.
- **Partner Innovations**:
  - Windows 365 for Agents as a platform for scalable agent solutions (e.g., Manus AI).
- **Summary**:
  - Empowering developers, IT professionals, and end users through AI, security, and cloud-powered tools.

## Technical Takeaways

- Windows is evolving to support intelligent agents driven by Copilot Studio and Copilot Plus PCs
- Semantic Search and Fluid Dictation enhance both productivity and accessibility
- Windows 365 for Agents enables enterprise scalability for developer-built agents
- Copilot Studio is central to agent creation for automation and decision support
- AI and cloud are now deeply integrated into Windows environments for improved adaptability and security
