# Building Microsoft Agent Framework Solutions for Microsoft 365 with AI Integration

**Presenters:** Robert Bruckner, James Oleinik  
Session from Microsoft Ignite 2025 | BRK305

## Overview

Learn to build A365-ready agents designed for enterprise use with the Microsoft Agent Framework and Agent 365 SDK. Agents can be developed on any AI stack and integrated seamlessly across Microsoft 365, focusing on unified discovery, governance, security, and observability.

### Key Topics Covered

- **Telstra Case Study:** Demonstrates scaling of AI agents responsibly with Agent 365
- **Security & Compliance:** Monitoring and governance utilizing Microsoft Defender integration
- **Agent Creation & Testing:** Steps for creating and deploying agents in a tenant environment
- **Partner Showcases:** Workflow agent solutions from Cognition (Devon and Nana)
- **Agent Notification & Response:** How agents asynchronously process document mentions
- **Collaborative AI Agent Demo:** Automated onboarding tasks and Microsoft tool integrations
- **Observability:** Evaluation harness, agent registration, and leaderboards
- **Interoperability:** MCP server integration and interoperability best practices

## Technical Highlights

- Building agents using the Microsoft Agent Framework or Agent 365 SDK
- Security and compliance best practices for agent-based workflows
- Real-time monitoring and observability of deployed agent instances
- Integrating with Defender for security event detection and response
- Live demonstrations on agent registration and collaborative onboarding workflows
- Techniques for agent evaluation and tracking performance via leaderboards
- Tools for agent administration in enterprise tenant environments
