# Azure Update – 14th November 2025

**Presented by John Savill’s Technical Training**

This update compiles the latest Azure announcements and feature improvements with relevance for architects, administrators, developers, and DevOps professionals:

## Highlights

### Azure Kubernetes Service (AKS) and Container Platform

- **AKS Azure Linux OS Guard:** Security and compliance enhancements for managed Linux nodes.
- **AKS Flatcar Container Linux support:** New node OS option for hardened security requirements.
- **AKS LocalDNS:** Simplified DNS management for AKS workloads, improving resolution speed.
- **AKS agentic CLI:** CLI updates for easier AKS cluster interaction and management.
- **AKS schedule profile configuration:** Advanced workload scheduling and resource control.

### Application Security

- **WAF for App GW for Containers:** Enhanced web application firewalling for container-hosted apps.
- **Azure Firewall Packet Capture & DNS Flow Trace Logs:** Improved diagnostics and analysis for network and security teams.
- **AFD WAF JavaScript challenge:** Updated web protection mechanisms for Azure Front Door.

### Network and Management

- **Azure Virtual Network Manager (AVNM) updates:** Expanded capabilities for managing diverse network environments.
- **Object Priority Replication:** Fine-grained control over Azure Storage object replication.
- **GRS and GZRS Priority Replication:** More flexible disaster recovery with granular geo-replication.

### Data Storage/Platform Developments

- **ADLS Gen2 Vaulted Backup:** New backup options for Data Lake scenarios.
- **PostgreSQL Flexible 3 AZ Japan West:** High availability in additional Azure regions.
- **openCypher Support & KQL Graph Semantics:** Expanded graph data capability for data-intensive workloads.
- **MySQL Flexible Lower Case Table Names:** Case-handling updates for better compatibility.
- **MySQL Azure Function Trigger:** Event-driven integration for cloud-native automations.

### Security and Monitoring

- **Azure Migrate Security Insights:** Security posture checks as part of cloud migration planning.

### Development Platform

- **.NET 10 Release:** New features and improvements for Microsoft’s core development platform.
- **GPT-5.1 Models:** Preview of next-generation AI capabilities available through Azure services.
- **Atlanta AI Superfactory:** Expansion of AI infrastructure in North America.
