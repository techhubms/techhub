## Introduction

- Covers all key Azure and related innovations released or previewed during Ignite 2025.
- Ideal for consultants, developers, and admins tracking Microsoft's cloud roadmap.

## Compute & Containers

- **AKS (Azure Kubernetes Service)**: Multiple upgrade cycles, automatic updates, network improvements, and KAITO MCP support.
- **App Service & Logic Apps**: Service updates including managed instances, Arm64 trusted launch, custom error pages, and new language/runtime support.
- **Azure Functions**: Node.js 24, Java 25 support, integration improvements.
- **OpenShift Virtualization**: Enhanced capabilities for running VM workloads on AKS.
- **App Gateway (App GW)**: New WAF v2 HTTP DDoS rules, container support, slow start, managed instance support.

## Networking & Security

- **VM scheduled actions** and new Ebsv6/NCv6 SKUs.
- **Azure Boost**: Confidential device improvements.
- **ACR (Azure Container Registry)**: Granular ABAC controls.
- **ExpressRoute**: Scalable gateways, advanced NAT, and AVNM updates.
- **Azure DNS**: Threat intelligence feed, new filtering options.

## Storage & Backup

- **Azure Files**: Entra-only auth, smart tiering, single file restore.
- **Managed Lustre, Blob Storage** (archival expansion, smart tiering).
- **Azure VM backup**: Threat detection.

## Databases & Data

- **Cosmos DB & DocumentDB**: Major feature and performance updates.
- **Microsoft Fabric databases**: Integration advances and analytics improvements.
- **PostgreSQL & HorizonDB**: Expanded support and performance enhancements.
- **Azure SQL/SQL MI**: Next-gen GP, more SQL DB updates, SQL Server 2025 features.

## AI & Microsoft Fabric

- **Microsoft Foundry**: Major new naming conventions and platform controls for deploying, fine-tuning, and managing AI models.
- **Foundry IQ, LLM Speech, Agent Service**: Durable AI agents, model routing, content understanding.
- **Durable AI agents**: Work IQ, Fabric IQ tools.
- **Azure Copilot**: New agents, SRE support (AI integration with operational tooling).
- **Microsoft Fabric DB, advanced connectors**: Improved analytics and data integrations.

## Identity & Governance

- **Entra ID (Azure AD)**: Agent ID registry, passkey/account recovery, new/updated agents.
- **APIM MCP Governance**: Policy and management updates.
- **Foundry Control Plane**: Expanded governance and security for AI workloads.

## DevOps & Monitoring

- **Application Gateway, APIM v2**: Integration and governance enhancements.
- **Azure Monitor**: OpenTelemetry SDK support, query-based metrics, dynamic thresholds, and log improvements.
- **Marketplace, APIC Standard**: Platform integrations for deploying new cloud resources.

## Other Notable Updates

- **Learning & Certification**: Links shared for Azure Master Class, DevOps training, PowerShell content, and certification resources.
