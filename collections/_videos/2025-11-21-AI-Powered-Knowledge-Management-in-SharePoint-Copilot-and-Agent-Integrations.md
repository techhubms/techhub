# AI-Powered Knowledge Management in SharePoint: Copilot and Agent Integrations

This Microsoft Ignite 2025 video session, delivered by Adam Harmetz and John Mighell, explores how SharePoint leverages the latest AI innovations to improve knowledge management in the Microsoft 365 ecosystem.

## Key Topics Covered

- **Work IQ**: Introduction to Microsoft's intelligence layer for Copilot and agents within SharePoint.
- **AI-Ready Content**: Emphasis on the need for well-organized, structured content to enhance AI responses ("garbage in, garbage amplified").
- **Smart Floating Action Button**: Overview of how AI-driven user interface elements are integrated into SharePoint for streamlined interaction.
- **Copilot Integration**: Demonstrations of Copilot working with SharePoint, such as extracting information from images in PowerPoint slides and using metadata for responding to queries (e.g., expense reports).
- **Content Gap Analysis**: Automatic discovery and recommendation of missing or outdated content.
- **Build and Automate with Agents**: Introduction to no-code SharePoint agents for automating specialized tasks within Microsoft 365 workflows.
- **Integration with Microsoft 365**: How SharePoint AI and agents seamlessly plug into the broader Microsoft 365 ecosystem for more intelligent knowledge experiences.

## Practical Demonstrations

- Hands-on examples of Copilot querying SharePoint content and using metadata.
- Use cases where AI recognizes and fills information gaps in organizational knowledge bases.
- How no-code agents can be built and applied by SharePoint makers to automate business processes.
