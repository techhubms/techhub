# Microsoft Agent 365: Managing AI Agents with Enterprise Controls

This session from Microsoft Ignite 2025 focuses on introducing Microsoft Agent 365, a dedicated control plane for managing AI-powered digital labor within enterprise environments.

## Key Topics

- **AI Agents in the Enterprise:** How Agent 365 enables the hiring, onboarding, and training of autonomous AI agents that scale productivity and adapt to business needs.
- **Management and Oversight:** Every agent in Pro Agent 365 has clear ownership to support accountability and governance.
- **Integration Ecosystem:** Manus agents are integrated into the Microsoft ecosystem, with the admin center view allowing seamless linking across organizational agents.
- **Developer & IT Collaboration:** Agent 365 includes features that facilitate collaboration between IT, developers, and human agents to foster efficiency.
- **Microsoft 365 Tooling:** The platform empowers AI agents to operate and collaborate using tools from the Microsoft 365 suite.
- **Compliance & Security:** Built-in controls ensure compliance with organizational policies, authorization requirements, and secure data access. Emphasis is placed on agent governance and enterprise trust.
- **EY Real-World Journey:** A guest segment highlights EY’s practical experiences using Agent 365 in real enterprise scenarios.
