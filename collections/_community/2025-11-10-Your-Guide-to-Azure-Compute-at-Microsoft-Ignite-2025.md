# Your Guide to Azure Compute at Microsoft Ignite 2025

Microsoft Ignite 2025 is approaching fast, and this guide is your one-stop resource to all Azure Compute events, announcements, and opportunities at the conference. Whether you're joining in person or virtually, the curated sessions below will help you deepen your Azure knowledge and cloud skills.

## Featured Sessions

### Tuesday

- **BRK171: [What's new and what's next in Azure IaaS](https://ignite.microsoft.com/en-US/sessions/4ad4a2b1-09f6-4942-94d0-ae47734b7059) (Intermediate 200)**  
  Insights into the latest capabilities in compute, storage, and networking for Azure IaaS, with a focus on boosting performance, resiliency, cost efficiency, and supporting AI workloads.  
  *Tuesday, November 18 | 2:30 PM - 3:15 PM PST*

### Wednesday

- **BRK430: [Inside Azure Innovations with Mark Russinovich](https://ignite.microsoft.com/en-US/sessions/BRK430?source=sessions) (Advanced 300)**  
  Mark Russinovich covers the newest advances in Azure architecture, including Azure Boost and support for intelligent, scalable, and innovative applications.  
  *Wednesday, November 19, 2:45 PM PST*

## Other Related IaaS Sessions

### Thursday

- **BRK176: [Driving efficiency and cost optimization for Azure IaaS deployments](https://ignite.microsoft.com/en-US/sessions/BRK176?source=sessions) (Intermediate 200)**  
  Focus on controlling cloud spend while maintaining performance. Covers Azure tools and strategies for aligning infrastructure choices with workloads and reducing TCO.  
  *Thursday, November 20, 9:45 AM PST*
- **BRK217: [Resilience by design: Secure, scalable, AI-ready cloud with Azure](https://ignite.microsoft.com/en-US/sessions/BRK217?source=sessions) (Advanced 300)**  
  Learn about secure, scalable architectures, simulating failures, and orchestrating recovery using Azure's built-in resiliency features.  
  *Thursday, November 20, 1:00 PM PST*
- **BRK178: [Architecting for resiliency on Azure Infrastructure](https://ignite.microsoft.com/en-US/sessions/BRK178?source=sessions) (Intermediate 200)**  
  Practical approaches for high availability and fault tolerance using Azure availability zones and multi-region deployments.  
  *Thursday, November 20, 1:00 PM PST*
- **BRK148: [Architect resilient apps with Azure backup and reliability features](https://ignite.microsoft.com/en-US/sessions/BRK148?source=sessions) (Advanced 300)**  
  Deep dive into resilience techniques across VMs, databases, and containers, featuring enhanced data and cyber resiliency with threat detection against ransomware.  
  *Thursday, November 20, 3:30 PM PST*

### Friday

- **BRK146: [Resiliency and recovery with Azure Backup and Site Recovery](https://ignite.microsoft.com/en-US/sessions/BRK146?source=sessions) (Advanced 300)**  
  Demonstrates backup and disaster recovery across Azure environments using modern techniques and compliance-focused methods.  
  *Friday, November 21, 9:00 AM PST*
- **BRK149: [Unlock cloud-scale observability and optimization with Azure](https://ignite.microsoft.com/en-US/sessions/BRK149?source=sessions) (Advanced 300)**  
  Showcases Azure Monitor for end-to-end cloud observability and new Copilot in Azure agents for actionable insights on cost and resource optimization.  
  *Friday, November 21, 10:15 AM PST*
- **BRK173: [Azure IaaS best practices to enhance performance and scale](https://ignite.microsoft.com/en-US/sessions/BRK173?source=sessions) (Advanced 300)**  
  Explores how to optimize compute, storage, and networking in Azure for high performance and scalable solutions.  
  *Friday, November 21, 10:15 AM PST*
- **BRK172: [Powering modern cloud workloads with Azure Compute](https://ignite.microsoft.com/en-US/sessions/BRK172?source=sessions) (Advanced 300)**  
  Latest VM innovation announcements and best practices for unlocking new customer value.  
  *Friday, November 21, 11:30 AM PST*
- **BRK168: [Azure IaaS platform security deep dive](https://ignite.microsoft.com/en-US/sessions/BRK168?source=sessions) (Advanced 300)**  
  Comprehensive overview of Azure’s security architecture with best practices for foundational, compute, network, and storage security pillars.  
  *Friday, November 21, 11:30 AM PST*

## Upskill Yourself with Hands-On Labs

### Tuesday

- **LAB500: [Attain unified observability and optimization in Azure](https://ignite.microsoft.com/en-US/sessions/LAB500?source=sessions) (Intermediate 200)**  
  Use AI-powered analytics to monitor Azure workloads, drive optimization, and apply FinOps and sustainability strategies.  
  *Tuesday, November 18, 2:45 PM PST*
- **LAB520: [Start, Get and Stay Resilient with Azure](https://ignite.microsoft.com/en-US/sessions/LAB520?source=sessions) (Intermediate 200)**  
  Practical lab to assess, validate, and improve resiliency across Azure applications with management and configuration tools.  
  *Tuesday, November 18, 4:30 PM PST*
