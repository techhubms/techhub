# Build Your Microsoft Ignite Schedule: Top Sessions for Solution Architects

Microsoft Ignite 2025 is a major event for IT professionals and architects looking to deepen their knowledge, connect with experts, and stay ahead on cloud and AI advancements. This guide helps solution architects make the most of Ignite by recommending sessions that emphasize Azure architecture, agentic AI, security, data, and infrastructure.

## Top Recommended Sessions for Solution Architects

**1. [From start to scale: Realize agentic AI value](https://ignite.microsoft.com/en-US/sessions/BRKSP464?source=sessions)**  *(BRKSP464)*  
Learn strategies for designing and scaling intelligent systems using Microsoft's Agentic AI framework, balancing performance and responsible governance.

**2. [Unleashing SAP Databricks on Azure: Modernize, analyze, and innovate](https://ignite.microsoft.com/en-US/sessions/BRK136?source=sessions)**  *(BRK136)*  
Discover how Azure and Databricks integrate with SAP to build modern, high-performance data architectures.

**3. [Migrate and Modernize Windows and SQL Server Workloads to Azure](https://ignite.microsoft.com/en-US/sessions/LAB506-R1?source=sessions)**  *(LAB506-R1)*  
Explore Azure's modernization approach for securing, scaling, and optimizing enterprise systems built on Windows Server and SQL Server.

**4. [Innovation Session: Security in the Agentic Era – the Core Primitive](https://ignite.microsoft.com/en-US/sessions/BRK1712?source=sessions)**  *(BRK1712)*  
Examine the evolving security landscape as driven by agentic AI, with real-world IT and enterprise impacts.

**5. [Microsoft Purview Compliance Manager: AI Compliance & Resilience](https://ignite.microsoft.com/en-US/sessions/THR750?source=sessions)**  *(THR750)*  
Understand how Microsoft Purview and AI simplify compliance for hybrid and multi-cloud deployments, automate risk, and enhance resilience.

## Keynotes and Infrastructure Sessions

Before developer deep-dives, consider these sessions to set your context:

- **[Opening Keynote](https://ignite.microsoft.com/en-US/sessions/KEY01?source=sessions)** *(KEY01)* — Hear Microsoft leadership unveil the latest AI, cloud, and developer ecosystem updates.
- **[Scale Smarter: Infrastructure for the Agentic Era](https://ignite.microsoft.com/en-US/sessions/BRK1704?source=sessions)** *(BRK1704)* — Learn what's new in Azure infrastructure including compute, networking, and hybrid management.
- **[Microsoft Fabric and Azure Databases - the data estate for AI](https://ignite.microsoft.com/en-US/sessions/BRK1702?source=sessions)** *(BRK1702)* — Explore how Azure's data services enable unified management, analytics, and intelligent app creation at scale.
