## Product Group News

### [New Azure API Management Service Limits](https://techcommunity.microsoft.com/blog/integrationsonazureblog/new-azure-api-management-service-limits/4497574)

Detailed breakdown of updated limits across API Management service tiers (Classic, v2). Key resource limits, rollout schedule, and guidance for existing services plus instructions for requesting increases.

### [Accessing Shared OneDrive Folders in Azure Logic Apps](https://techcommunity.microsoft.com/blog/integrationsonazureblog/how-to-access-a-shared-onedrive-folder-in-azure-logic-apps/4484962)

Explains supported methods for accessing shared OneDrive folders from Logic Apps, including using Microsoft Graph and specific API configurations.

### [Stop Writing Plumbing! Use the Logic Apps MCP Server Wizard](https://techcommunity.microsoft.com/blog/integrationsonazureblog/stop-writing-plumbing-use-the-new-logic-apps-mcp-server-wizard/4496702)

Introducing a guided setup experience for MCP servers in Logic Apps Standard, centralizing API authentication, exposure, and tool generation for building agent-enabled workflows (compatible with VS Code, Copilot Studio, Foundry).

### [Logic Apps Agentic Workflows with SAP – AI Agents](https://techcommunity.microsoft.com/blog/integrationsonazureblog/logic-apps-agentic-workflows-with-sap---part-2-ai-agents/4492362)

Describes an AI-driven validation agent retrieving business rules and outputting structured data for SAP integrations. Details deterministic design patterns, multi-model workflows, and orchestration techniques with Logic Apps and SharePoint.

### [Logic Apps Agentic Workflows with SAP – Infrastructure](https://techcommunity.microsoft.com/blog/integrationsonazureblog/logic-apps-agentic-workflows-with-sap---part-1-infrastructure/4491906)

Covers contract setup, workflow orchestration, data shaping, and connectivity for SAP/Logic Apps integration, emphasizing robust error handling and notification flows.

### [Unified AI Gateway Pattern with Azure API Management](https://techcommunity.microsoft.com/blog/integrationsonazureblog/azure-api-management---unified-ai-gateway-design-pattern/4495436)

A customer success story using APIM as a centralized AI gateway to standardize requests, enforce authentication, and streamline multi-provider model access (with policy examples and modular implementation guidance).

### [BizTalk Migration Tool: Orchestrations to Logic Apps](https://techcommunity.microsoft.com/blog/integrationsonazureblog/a-biztalk-migration-tool-from-orchestrations-to-logic-apps-workflows/4494876)

Open-source toolkit converting BizTalk artifacts (.btm, .odx) into Logic Apps workflows with mapping and automation features; includes demo resources and caveats.

### [Azure Arc Jumpstart Template: Hybrid Logic Apps Deployment](https://techcommunity.microsoft.com/blog/integrationsonazureblog/azure-arc-jumpstart-template-for-hybrid-logic-apps-deployment/4493996)

Walkthrough for provisioning hybrid Logic Apps Standard on Arc-enabled AKS via deployment scripts, integrating Azure SQL, Azure Storage, and Kubernetes resources. Includes test and troubleshooting steps.
