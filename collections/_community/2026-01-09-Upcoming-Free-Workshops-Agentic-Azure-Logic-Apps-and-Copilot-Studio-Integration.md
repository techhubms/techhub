# Upcoming Agentic Azure Logic Apps Workshops

KentWeareMSFT is hosting two free online workshops covering enterprise integration and automation using Azure Logic Apps and Copilot Studio MCP Servers. These sessions are designed for technical audiences interested in building agent-based business processes with Microsoft platforms.

## 1. Connecting the Enterprise with MCP Servers from Copilot Studio

- **Date & Time:** Tuesday, January 13th at 2PM CET / 6AM Mountain
- **Description:** Learn how to connect enterprise systems by leveraging MCP Servers from Copilot Studio in conjunction with Logic Apps. This session covers integration scenarios and technical architecture.
- [Registration Link](https://www.linkedin.com/events/7406108337387233280/)

## 2. Building Agentic Business Processes in a Day using Azure Logic Apps Agent Loop

- **Date & Time:** Friday, January 16th at 2PM CET / 6AM Mountain
- **Description:** Discover how to construct agentic business processes using Azure Logic Apps Agent Loop in a practical, day-long format. This session walks through hands-on steps for automating complex workflows.
- [Registration Link](https://www.linkedin.com/events/7406997279758471170/)
