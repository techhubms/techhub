# Azure Virtual Desktop Expands to USGov Texas in Azure Government

Azure Virtual Desktop (AVD) is now available in the USGov Texas region, providing government and regulated customers with a new option for deploying secure, flexible virtual desktop environments tailored to diverse mission needs.

## Key Benefits

- **Regional Choice:** Deploy host pool objects directly in the USGov Texas region, supporting geographic distribution and specific mission requirements.
- **Performance Gains:** Experience improved connection performance and reduced latency for users in or near the Texas region.
- **Scalability and Flexibility:** Take advantage of multiple Azure Government regions for better alignment with compliance, security, and operational priorities.
- **Security and Compliance:** Expands options while maintaining Azure Government’s robust compliance and security standards.

## Creating Host Pools in USGov Texas

Customers can now set up host pools in the USGov Texas region, which can help:

- Lower latency for end-users
- Improve responsiveness of virtual desktops
- Enable faster access and interaction with critical applications

## Getting Started

To leverage these benefits, you can begin deploying your Azure Virtual Desktop environments in USGov Texas today. This regional expansion supports missions with strict compliance or geographic requirements and helps optimize the user experience for US government workloads.

For full documentation and step-by-step deployment instructions, see the [Azure Government documentation](https://learn.microsoft.com/en-us/azure/azure-government/documentation-government-welcome).
