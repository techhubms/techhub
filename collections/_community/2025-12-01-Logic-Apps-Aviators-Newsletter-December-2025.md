# Logic Apps Aviators Newsletter - December 2025

**Compiled by WSilveira**

## Ace Aviator of the Month: Daniel Jonathan

Azure Integration Architect @ Cnext shares how he helps organizations modernize integration architectures using Azure services. Daniel uses Logic Apps, Azure Functions, Service Bus, API Management, and has added AI intelligence via Semantic Kernel and LangChain. Key advice includes learning reliable messaging, retries, and error handling before advancing into complex toolchains.
