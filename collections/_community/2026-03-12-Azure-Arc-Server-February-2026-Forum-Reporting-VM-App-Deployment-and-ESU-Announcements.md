# Azure Arc Server February 2026 Forum Recap

**Author:** Aurnov Chattopadhyay  
**Date Published:** March 12, 2026

The February 2026 edition of the Azure Arc Server Forum featured several important updates and discussions focused on Azure Arc-enabled server management and related services. Below is a recap of the main topics:

## Key Agenda Items

### 1. Arc Server Reporting & Dashboard  

*Speaker: Jeff Pigot, Sr. Solution Engineer*

- Demonstrated a visual reporting dashboard that integrates management services and experiences across Azure Arc-enabled servers.
- The Arc Software Assurance Benefits Dashboard is available on [GitHub](https://github.com/wjpigott/ArcBenefitsDashboard/tree/main/workbook) for community reference and use.

### 2. VM Applications: Third-Party App Deployment & Patching (Private Preview)  

*Speaker: Yunis Hussein, Product Manager*

- Shared insights into the private preview for third-party application deployment and patching on Azure Arc-enabled servers.
- Interested participants can sign up for the private preview by filling out [this form](https://forms.office.com/r/qhQ4H2JrFU).

### 3. Windows Server 2016 ESUs Enabled by Azure Arc: Portal Experience Feedback  

*Speaker: George Enninful*

- Discussed how Azure Arc enables Extended Security Updates (ESUs) for Windows Server 2016.
- Feedback on the portal experience is requested via [this form](https://forms.office.com/Pages/DesignPageV2.aspx?prevorigin=shell&origin=NeoPortalPage&subpage=design&id=v4j5cvGGr0GRqy180BHbRyemPojJK7tEkoALdt6d-B9UOUpIVkNKMUZPVlpaV0dKQUI1UUxKUURUUy4u&topview=Preview).
