# Azure NCv6 Public Preview: Unified Platform for Converged AI & Visual Computing

Azure introduces the NCv6 GPU Virtual Machine series, aiming to bridge high-end visualization and enterprise AI workloads in the cloud. Powered by NVIDIA RTX PRO 6000 Blackwell Server Edition GPUs and Intel Xeon Granite Rapids processors, NCv6 delivers a flexible platform for industrial digitalization, LLM inference, and agentic workflows.

## Key Features

- **NVIDIA RTX PRO 6000 Blackwell GPU (96 GB GDDR7):** Supports multimodal AI models, 3D rendering, and complex physics simulations.
- **Intel Xeon Granite Rapids CPUs (up to 4.2 GHz):** Ensures no bottleneck during rendering or AI processing.
- **Flexible Sizing:** Choose from General Purpose (up to 320 vCPUs), Compute Optimized (higher vCPU for simulations), or Memory Optimized (up to 1,280 GB RAM). Fractional GPU options available for cost-effective VDI and inference.
- **Azure Accelerated Networking (up to 200 Gbps)** and premium storage options (up to 2TB local, SSD v2, Ultra Disk support).

## Supported Workloads

- **Digital Twins & Omniverse Simulation:** Real-time simulation of virtual replicas for engineering and industrial use.
- **LLM Inference & RAG:** Cost-effective deployment of self-hosted language models and retrieval-augmented generation pipelines.
- **Agentic AI Applications:** Build and deploy autonomous AI-driven solutions.
- **Scientific Visualization & HPC:** Run physics-driven simulations and large-scale scientific workloads.
- **Media & Entertainment:** Accelerate rendering, VFX, and AI-powered content generation.
- **Virtual Desktop Infrastructure (VDI):** Modernize remote workstations with high-fidelity GPU acceleration.

## Technical Specifications

| Specification        | Details                                                  |
|---------------------|----------------------------------------------------------|
| GPU                 | NVIDIA RTX PRO 6000 Blackwell (96GB GDDR7)               |
| Processors          | Intel Xeon Granite Rapids (up to 4.2 GHz Turbo)           |
| vCPUs               | 16-320 (scalable)                                        |
| Memory              | 64-1,280 GB DDR5                                         |
| Network             | Up to 200,000 Mbps (200 Gbps) Accelerated Networking      |
| Storage             | Up to 2TB local; Premium SSD v2 & Ultra Disk supported    |

For more details, see the [official product documentation](https://learn.microsoft.com/en-us/azure/virtual-machines/sizes/gpu-accelerated/nc-rtxpro6000-bse-v6-series?tabs=sizebasicgp%2Csizebasicco%2Csizebasicmo).

## Example Applications

- **AI Research:** Efficiently host and serve LLMs, ranking models, and RAG pipelines for enterprise solutions.
- **Industry & Manufacturing:** Simulate environments for autonomous vehicles, digital twins, and sensor validation.
- **Remote Work and VDI:** Fractional GPU allows multiple high-fidelity desktop sessions per VM, modernizing distributed team workflows.
- **Entertainment Production:** Run large render farms for VFX and generate AI-driven content with burst performance.

## Conclusion

The Azure NCv6 series advances cloud infrastructure for innovation, uniting AI, simulation, and visualization at scale on a single platform. Enterprises and developers can leverage NCv6 to build the industrial metaverse, deploy agentic applications, or accelerate scientific and creative workloads while ensuring scalability and cost-efficiency.
