## Product Group Updates

### [Microsoft BizTalk Server Product Lifecycle Update](https://techcommunity.microsoft.com/blog/integrationsonazureblog/microsoft-biztalk-server-product-lifecycle-update/4478559)

- **BizTalk Server 2020** is the final version, with support through 2030
- Microsoft recommends a gradual migration to Azure Logic Apps
- Migration tooling, hybrid deployment, and artifact reuse supported for a smooth transition

### [Data Mapper Test Executor for Logic Apps Standard](https://techcommunity.microsoft.com/blog/integrationsonazureblog/data-mapper-test-executor-a-new-addition-to-logic-apps-standard-test-framework/4472440)

- Native XSLT and Data Mapper testing in Logic Apps Standard test framework
- Streamlines transformation validation with automated map testing and feedback cycles
- Integrates with latest SDK for reliable CI/CD pipeline testing

### [General Availability: AI & RAG Connectors in Logic Apps (Standard)](https://techcommunity.microsoft.com/blog/integrationsonazureblog/%F0%9F%8E%89-announcing-general-availability-of-ai--rag-connectors-in-logic-apps-standard/4474337)

- Logic Apps Standard adds **AI and RAG connectors** for document processing, semantic search, embeddings, and agentic workflows
- Enables intelligent, context-aware automation powered by organizational data

### [Logic Apps Labs – Agentic Workflows Learning Path](https://azure.github.io/logicapps-labs/docs/category/build-agentic-solutions-with-azure-logic-apps)

- Guided labs for conversational and autonomous agents with Logic Apps
- Expand skills in MCP tool integration and advanced orchestration patterns
