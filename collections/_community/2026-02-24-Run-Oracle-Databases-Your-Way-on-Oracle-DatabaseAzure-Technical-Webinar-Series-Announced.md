# Run Oracle Databases Your Way on Oracle Database@Azure – Technical Webinar Series

Oracle and Microsoft are joining forces to launch a technical webinar series crafted for database and cloud professionals. The 'Run Oracle Your Way—On Oracle Database@Azure' series, led by Oracle and Microsoft product managers, delves into real-world topics including architecture, networking, performance, security, and day-one operations.

### Why Oracle Database@Azure?

- Joint engineering by Oracle and Microsoft
- Enables seamless migration of existing Oracle databases to Azure with no refactoring required
- Maintains or improves on performance, resiliency, and latency
- Supports Bring Your Own License (BYOL) and license-included models
- Lets organizations modernize applications on Azure while retaining familiar Oracle tools and practices

### About the Webinar Series

- 30-minute technical deep dives hosted by Oracle and Microsoft experts
- Each session centers on a key practical topic (e.g., architecture, networking, security, performance)
- Tailored for practitioners planning or optimizing Oracle Database@Azure deployments
- Real-world demonstrations and actionable guidance
