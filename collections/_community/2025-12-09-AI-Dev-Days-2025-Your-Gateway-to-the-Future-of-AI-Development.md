# AI Dev Days 2025: Your Gateway to the Future of AI Development

**Event Dates:** 10–11 December 2025  
**Format:** Virtual, global access

## Overview

AI Dev Days 2025 is a comprehensive, hands-on event designed for AI developers, engineers, and creators. Participants will explore the latest advances across Microsoft platforms including Azure, Microsoft Foundry, GitHub, Visual Studio Code, and more.

## Event Highlights

### Day 1: Building AI Applications with Azure, GitHub, and Foundry

- **Agentic DevOps:** Insights into automating software delivery with intelligent agents
- **Azure SRE Agent:** Learn about site reliability engineering on Azure using AI-powered agents
- **Microsoft Foundry:** Explore Microsoft's platform for building and scaling AI
- **MCP (Microsoft Cloud Platform):** Updates and integration strategies
- **Models for AI Innovation:** Latest advancements in AI models on Microsoft platforms

[Join Day 1 Sessions](https://developer.microsoft.com/en-us/reactor/events/26536/)  
[Video Link](https://www.youtube.com/live/GCLYC5d7kpI?si=1SZmVVO0qFAzsSOW)

### Day 2: Using AI to Boost Developer Productivity

- **Agent HQ:** Discover new tools for managing and deploying AI agents
- **VS Code & Visual Studio 2026:** See what’s new in Microsoft’s flagship development tools for AI work
- **GitHub Copilot Coding Agent:** Hands-on demos of Copilot’s latest coding and productivity features
- **App Modernisation Strategies:** Techniques and best practices for updating legacy apps using AI and Microsoft tools

[Join Day 2 Sessions](https://developer.microsoft.com/en-us/reactor/events/26537/)  
[Video Link](https://www.youtube.com/live/GCLYC5d7kpI?si=1SZmVVO0qFAzsSOW)

## Why Attend?

- **Hands-on Labs:** Experience new product features in real time
- **Key Takeaways from Microsoft Ignite and GitHub Universe 2025:** Stay updated on industry trends
- **Global Workshops:** Sessions in multiple languages, including LATAM and EMEA
- **Networking:** Connect with leading industry figures and peers
