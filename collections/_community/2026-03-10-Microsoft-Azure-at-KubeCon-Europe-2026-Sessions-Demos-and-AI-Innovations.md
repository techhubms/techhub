# Microsoft Azure at KubeCon Europe 2026: Sessions, Demos, and AI Innovations

**Author:** Coryskimming (Microsoft)
**Event Dates:** March 23–26, 2026  
**Location:** Amsterdam, Netherlands

Microsoft returns to Amsterdam as a Diamond Sponsor for KubeCon + CloudNativeCon Europe 2026 with a week of live technical sessions, demos, and innovative announcements—centered on AKS, Kubernetes, cloud-native AI/ML, DevOps, security, and open-source collaboration. Here's what attendees can expect:

## Azure Day with Kubernetes (March 23)

- **Free full-day technical event** focused on Azure Kubernetes Service (AKS) at Hotel Casa Amsterdam
- **Morning:** Latest in AKS, building/running AI apps on Kubernetes
- **Afternoon Tracks:**
  - **Hands-on AKS Labs:** Practical, instructor-led deep-dives
  - **Expert Roundtables:** Interactive sessions on security, autoscaling, AI workloads, and performance with AKS engineers
- **Networking:** Evening social and direct Q&A with Microsoft teams

## KubeCon + CloudNativeCon (March 24–26)

- **Microsoft Azure Booth (#200):**
  - Demos on AKS, scaling Kubernetes, cloud-native AI/ML, observability, and security
  - Interact with engineers behind open-source projects (Inspektor Gadget, Headlamp, Drasi, Radius, Notary Project, ORAS, Ratify, Flatcar, Istio, and more)

### Featured Keynotes and Sessions

- **Keynote:** _Scaling Platform Ops with AI Agents: Troubleshooting to Remediation_ ([Video](https://sched.co/2HgFq))
  - Presenters demo HolmesGPT, an open-source project connecting LLMs to operational data for automated diagnostics and remediation in Kubernetes
  - Covers incident reduction, RBAC compliance, auditability, and challenges in building reliable AI agents
- **Customer Keynote:** _Rules of the Road for Shared GPUs: AI Inference Scheduling at Wayve_ ([Video](https://sched.co/2HgE0))
  - Shows real-world approaches to scheduling multi-tenant AI inference on GPU clusters using Kubernetes and Kueue
- **Theatre Demo:** _Building Cross-Cloud AI Inference on Kubernetes with OSS_ ([Video](https://sched.co/2IEHs))
  - Walk-through of a Kubernetes-native, multi-cloud approach to elastic AI inference with Karpenter and GPU flex nodes for cross-provider scheduling

### Microsoft-Led Breakout Sessions

- **Radius:** Custom resource/Headlamp integration for real-world workloads
- **Networking:** Deep dives into the Kubernetes network stack, Istio production best practices, and multi-cluster solutions
- **Security:** Data protection as infrastructure for AI, supply chain security, Compliance-as-Code, confidential containers
- **Operational Excellence:** Fleet Manager, codeless OpenTelemetry, autoscaling, modern Ingress, root cause analysis automation
- **AI/ML:** Running large-scale inference workloads, GPU utilization, and cost optimization for AI spend
- **Cloud-Native Modernization:** Terraform automation, GitHub Copilot-enabled app modernization, developer portals

### Booth Theatre Highlights

- Zero-Migration AI, Model Context Protocol for AI observability, AKS cluster optimization with Terraform
- Cost management for AI apps, secure deployments using confidential containers, network security, multi-cluster workload strategies
- “Modern Apps, Faster” sessions demonstrating GitHub Copilot integration for AKS development workflows

### Bonus: Other Microsoft-Attended Events

- **Cloud Native Rejekts** (March 21): Additional community talks
- **Maintainer Summit** (March 22): Meet Azure maintainers and CNCF contributors
