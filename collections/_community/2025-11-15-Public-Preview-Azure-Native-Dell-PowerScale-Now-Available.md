# Public Preview: Azure Native Dell PowerScale Now Available

**Author: Aung_Oo**

## Overview

Dell and Microsoft have extended their partnership by launching Dell PowerScale as a fully managed, Azure Native ISV service, now available in public preview. Built on the OneFS platform, this service delivers enterprise-grade storage capabilities natively within Azure, offering a seamless experience for managing unstructured data at scale.

## Key Features

- **Deployment Options:**
  - *Customer Managed*: Deploy and manage OneFS clusters within your own Azure tenant, handling all compute, networking, and storage resources.
  - *Dell Managed*: Dell PowerScale for Microsoft Azure is fully managed by Dell, simplifying deployment and operational overhead. Customers consume it just like any other Azure Native service without managing underlying infrastructure.

- **Enterprise Benefits:**
  - Up to 8.4PB in a single namespace
  - Multi-protocol support
  - Performance, efficiency, and resiliency for AI, HPC, analytics, and enterprise workloads
  - Data replication between on-premises and Azure deployments
  - Data reduction via compression, deduplication, and tiering with Cloud Pools to Azure Blob Storage
  - Integration with Azure’s ecosystem (AI/Analytics, DR, application bursting)

- **Azure Native Integrations:**
  - Deployed and managed through Azure Portal
  - Payment facilitated via Azure Marketplace using customer’s existing Azure commitments/contracts
  - Management tools: Azure Portal, Azure CLI, PowerShell

## Use Cases

- AI and ML workloads
- High-performance computing (HPC)
- Analytics and big data storage
- Hybrid cloud and multi-region data replication
- Cloud-based disaster recovery
- Application bursting scenarios

This public preview of Dell PowerScale for Microsoft Azure provides customers with flexibility and simplicity for managing petabyte-scale, unstructured data in the cloud. By leveraging a fully managed ISV service native to Azure, organizations can optimize hybrid, cloud, and data-intensive workloads with enterprise storage features at cloud scale.
