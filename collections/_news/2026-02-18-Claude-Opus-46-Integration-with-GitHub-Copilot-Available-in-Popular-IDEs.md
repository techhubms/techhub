# Claude Opus 4.6 Integration with GitHub Copilot Now Available in Popular IDEs

Claude Opus 4.6 is now generally available to users of Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+ tiers. This release expands the accessibility of Claude Opus 4.6 within GitHub Copilot Chat across a range of integrated development environments (IDEs).

## Where You Can Access Claude Opus 4.6

- **GitHub Copilot Chat** on github.com
- **GitHub Mobile**
- **Visual Studio Code**
- **Visual Studio**
- **JetBrains IDEs**
- **Xcode**
- **Eclipse**

You can select Claude Opus 4.6 via the chat model picker in agent, ask, and edit modes. If the option is not immediately available, users are encouraged to check back later as rollout completes.

## Enabling Access for Organizations

Administrators of Copilot Business and Copilot Enterprise accounts must opt in by enabling the Claude Opus 4.6 policy within Copilot settings. Once enabled, organization users will gain access to the model in supported development environments.
