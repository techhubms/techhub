# GPT-5.1-Codex-Max Now Available in GitHub Copilot

GitHub Copilot users now have access to the GPT-5.1-Codex-Max model, which is generally available for Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+ plans.

## Supported Platforms

- **GitHub Copilot Chat on github.com**
- **GitHub Mobile**
- **Visual Studio Code**
- **Visual Studio**
- **JetBrains IDEs**
- **Xcode**
- **Eclipse**

Access to GPT-5.1-Codex-Max can be managed through the chat model picker in agent, ask, and edit modes. Availability may vary based on rollout status.

## Enabling Access

### For Copilot Business and Copilot Enterprise

- Administrators must enable the GPT-5.1-Codex-Max policy in Copilot settings for organizational access.
- Once enabled, eligible users will see GPT-5.1-Codex-Max as an option in the model picker across all supported IDEs.

### For Individual Plans

- Users can select GPT-5.1-Codex-Max in the model picker within any supported IDE.
- Confirm access via a one-time prompt when enabling for the first time.
