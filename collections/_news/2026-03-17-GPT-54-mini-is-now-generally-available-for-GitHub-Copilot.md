# GPT-5.4 mini is now generally available for GitHub Copilot

OpenAI’s **GPT-5.4 mini** (a fast, agentic coding model) is now rolling out in **GitHub Copilot**. In early tests, GPT-5.4 mini is described as OpenAI’s highest-performing “mini” model so far, with:

- Faster **time to first token**
- Stronger **codebase exploration**
- Better results when using **grep-style tools**

Pricing note: the model is launching with a **0.33x premium request multiplier**, but pricing is **tentative and subject to change**.

## Availability in GitHub Copilot

GPT-5.4 mini will be available to **Copilot Pro, Pro+, Business, and Enterprise** users:

- [GitHub Copilot plans](https://github.com/features/copilot/plans?utm_source=changelog-codex53-cta&utm_medium=changelog&utm_campaign=newmodels)

You can select the model from the model picker in:

- **Visual Studio Code** (and later) in all modes: chat, ask, edit, agent
- **Visual Studio** (version and later) in all modes: agent, ask
- **JetBrains** (versions and later) in all modes: ask, edit, agent
- **Xcode** (versions and later) in all modes: ask, agent
- **Eclipse** (versions and later) in all modes: ask, agent
- **github.com**
- **GitHub Mobile** (iOS and Android)
- **GitHub CLI**

Prompting and model parameters are expected to work best in the latest versions of each product, so upgrading is recommended.

## Enabling access

- **Enterprise & Business plans**: administrators must enable the **GPT-5.4 mini policy** in Copilot settings. After it’s enabled, users in the org will see GPT-5.4 mini in the model picker (for example, in Visual Studio Code).
- **Bring Your Own Key (BYOK)**: use **Manage Models** in the picker, choose **GPT-5.4 mini**, and enter your API key when prompted.
