# AI Dev Days: Level-Up Your AI Skills with Microsoft Reactor

**Author:** Jenn Cockrell  
**Event:** [AI Dev Days](https://developer.microsoft.com/reactor/series/s-1612/)  
**Dates:** December 10-11, 2025

> _Tis’ the holiday season, and some would say there’s no greater gift than the gift of learning._

If you’re interested in deepening your expertise in building intelligent applications, join the free, hands-on **AI Dev Days** virtual series. Designed for developers, these sessions cover the latest in AI, agentic workflows, app modernization, and coding productivity with Microsoft platforms.

## What to Expect

**Day 1 – December 10, 2025:**

- Deep dives into **Azure**, **GitHub**, and building AI applications
- Topics: **Agentic DevOps**, **Azure SRE Agent**, **Microsoft Foundry**, **MCP (Microsoft Cloud Platform)**, and working with latest AI **Models**
- Highlights from **Microsoft Ignite** and **GitHub Universe 2025**
- Hands-on lab for new product features

**Day 2 – December 11, 2025:**

- Focus on how AI can improve your coding workflows
- Topics: **Agent HQ**, **VS Code**, **Visual Studio 2026**, **GitHub Copilot Coding Agent**, **app modernization**
- More real-world insights from major Microsoft events
- Additional practical lab for skills growth with AI developer tools

Whether you’re experimenting or building production-ready solutions, these sessions help accelerate innovation and bring your ideas to life.

👉 [Register for AI Dev Days](https://developer.microsoft.com/reactor/series/s-1612/)

### Featured Technologies and Topics

- Azure and Microsoft Foundry
- GitHub and GitHub Copilot
- Agentic DevOps, Azure SRE Agent, Agent HQ
- VS Code and Visual Studio 2026
- App modernization strategies
- Hands-on AI model integration
- Expert insights from Microsoft Ignite and GitHub Universe
