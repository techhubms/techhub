# Gemini 3 Pro deprecated

On **March 26, 2026**, **Gemini 3 Pro** was **deprecated across all GitHub Copilot experiences**, including:

- Copilot Chat
- Inline edits
- Ask mode
- Agent mode
- Code completions

## Deprecation details

| Model | Deprecation date | Suggested alternative |
| --- | --- | --- |
| Gemini 3 Pro | 2026-03-26 | Gemini 3.1 Pro |

## What you should do

- **Update workflows and integrations** to use **supported models** (suggested: **Gemini 3.1 Pro**).
- **Copilot Enterprise administrators** may need to enable access to alternative models via **model policies** in Copilot settings.
  - Verify availability by checking **your individual Copilot settings** and confirming the policy is enabled for the specific model.
  - After enabling, the model should appear in the **Copilot Chat model selector** in:
    - **VS Code**
    - **github.com**
- **No action is required to remove** deprecated models.
