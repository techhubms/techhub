# Dependabot Now Supports Automatic Updates for pre-commit Hooks

GitHub Dependabot has expanded its native capabilities to include automatic updates for [pre-commit](https://pre-commit.com/) hooks. By specifying `pre-commit` as a package ecosystem in your `dependabot.yml` configuration, Dependabot will parse your `.pre-commit-config.yaml`, check each hook’s repository for new tags or releases, and automatically open pull requests that update the `rev` field to the latest version.

## Key Features

- **Tag and SHA-based revisions**: Supports hooks pinned to specific Git tags (e.g., `v4.5.0`) and commit SHAs to ensure precise dependency management.
- **Grouped Updates**: You can use Dependabot’s [grouped updates configuration](https://docs.github.com/code-security/dependabot/dependabot-version-updates/configuration-options-for-the-dependabot.yml-file#groups) to combine multiple hook updates into a single pull request, simplifying PR management.
- **Changelog and Release Notes**: Pull requests generated by Dependabot will include release notes and changelogs from each updated hook repository, allowing teams to review changes before they merge.
- **YAML Structure Preservation**: Updates are made in a way that preserves your original `.pre-commit-config.yaml` formatting, including inline comments (e.g., `# frozen:`).
- **Local and Meta Repo Awareness**: Dependabot intelligently skips hooks defined as `local` or `meta` repositories, focusing only on dependencies that require version management.
- **Multi-host Support**: Works seamlessly with hooks hosted not only on GitHub, but also on GitLab, Bitbucket, and other common Git hosting providers.

## Getting Started

To enable this feature, edit your `dependabot.yml` file to include `pre-commit` as a package ecosystem under version updates. More details and guidance can be found in the [official documentation](https://docs.github.com/code-security/dependabot/dependabot-version-updates/configuring-dependabot-version-updates).

Participate in the ongoing discussion or provide feedback in [dependabot-core issue #1524](https://github.com/dependabot/dependabot-core/issues/1524).

## Why This Matters

Automating pre-commit hook dependency updates helps teams improve supply chain security and ensures their codebases use the latest, most secure hooks with minimal manual overhead.
