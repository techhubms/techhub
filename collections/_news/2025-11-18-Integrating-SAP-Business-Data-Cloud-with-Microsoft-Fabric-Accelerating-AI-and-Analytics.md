# Integrating SAP Business Data Cloud with Microsoft Fabric: Accelerating AI and Analytics

**Authors:** Microsoft Fabric Blog, with contributions from Irfan Khan (SAP) and Arun Ulag (Microsoft)

At Microsoft Ignite in San Francisco, SAP and Microsoft announced the launch of SAP Business Data Cloud Connect for Microsoft Fabric, marking a significant step in their partnership. This integration enables secure, bi-directional, zero-copy sharing between SAP Business Data Cloud (BDC) and Microsoft Fabric, allowing organizations to access and unify semantically rich enterprise data for advanced analytics and AI applications.

## Key Features

- **Unified Data Foundation:** Harmonize SAP and non-SAP data using Microsoft Fabric's data engineering and warehousing capabilities.
- **Zero-Copy Data Sharing:** Access SAP data instantly from Microsoft Fabric and vice versa without data replication delays.
- **OneLake Integration:** SAP data products are integrated into Microsoft OneLake, providing an AI-ready data lake and extending data access within familiar tools such as Excel and Teams.
- **Advanced Analytics and AI:** Use Power BI, Copilot in Power BI, AI Foundry, and Copilot Studio to build intelligent applications and analyze data with natural language interfaces.
- **Multi-Agent Collaboration:** Enable collaboration across platforms using M365 Copilot and SAP Joule for unified business productivity and insights.

## Use Cases

- Building semantically rich, harmonized data foundations for analytics and AI solutions.
- Deploying intelligent agents and applications grounded in enterprise business data.
- Integrating advanced analytics with natural language interaction via Copilot and Fabric data agents.

## Availability

SAP Business Data Cloud Connect for Microsoft Fabric is projected to reach general availability in Q3 2026.
