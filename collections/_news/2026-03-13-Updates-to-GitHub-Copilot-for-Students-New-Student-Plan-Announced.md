# Updates to GitHub Copilot for Students: New Student Plan Announced

**Starting today, students with GitHub Education benefits are being transitioned to the new GitHub Copilot Student plan.** As part of this change, GitHub has updated the range of AI models available to student users.

## What’s New?

- **GitHub Copilot Student Plan:** A Copilot experience specifically designed for students receiving GitHub Education benefits.
- **Model Lineup Update:** The set of Copilot AI models accessible to students has been revised, though specific models are not detailed in the announcement.
- **Focus on AI-Native Learning:** The new plan is aimed at supporting students with tools and features optimized for AI-driven learning, reflecting GitHub’s ongoing investment in education-focused technologies.

## Further Information

- For a full breakdown of the changes and discussion from the GitHub community, visit the [official announcement on GitHub Blog](https://github.blog/changelog/2026-03-13-updates-to-github-copilot-for-students) or join the [Community Discussion](https://gh.io/copilot4edu).

## Key Points

- Transition to a student-specific Copilot plan for GitHub Education members
- Updated available Copilot model lineup
- Focus on sustainable and long-term AI tool investment for students

Students and educators should review the community discussion for additional implementation details or questions about eligibility.
