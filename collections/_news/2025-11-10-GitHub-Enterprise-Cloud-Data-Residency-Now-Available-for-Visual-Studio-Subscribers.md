# GitHub Enterprise Cloud Data Residency Now Available for Visual Studio Subscribers

Visual Studio subscriptions now offer support for data residency when used with GitHub Enterprise Cloud. This addition is available to both subscription-based and usage-based Visual Studio subscribers, allowing organizations and developers to utilize GitHub Enterprise features while complying with regional data storage requirements.

## What Changed?

- Visual Studio subscriptions and GitHub Enterprise Cloud now integrate data residency features
- Users can maintain compliance with regional data regulations without changing their workflows

## Key Benefits

- **Regulatory Compliance**: Aligns with organizational and regional data storage policies
- **Enterprise Alignment**: Facilitates the use of Visual Studio subscriptions in regulated industries
- **Frictionless Integration**: Subscribers get seamless access to GitHub Enterprise Cloud without compromising compliance
