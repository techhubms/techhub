# Copilot coding agent works faster with semantic code search

[Copilot coding agent](https://docs.github.com/copilot/concepts/agents/coding-agent/about-coding-agent) now has access to a semantic code search tool. This lets it find relevant code based on meaning, instead of relying only on exact text matches with tools like `grep`.

## What changed

- The agent can use **semantic search** to locate relevant code even when it doesn’t know the exact names or patterns to search for.
- This is intended to help the agent find the right code **faster** in situations where text-based search is too brittle.

## Reported impact

- GitHub reports that in testing, this tool allowed Copilot to complete tasks in **2% less time** with **no change in quality**.

## Configuration

- **No configuration is required.**
- The agent automatically uses semantic search when appropriate.
