# Visual Studio Code 1.114 (Insiders)

*Last updated: March 25, 2026*

Welcome to the 1.114 release of Visual Studio Code.

To try new features as soon as possible, download the nightly Insiders build: https://code.visualstudio.com/insiders

## March 23, 2026

- **Pinned chat sessions indicator**: Pinned chat sessions show a **pin icon** in the sessions list, making it easier to distinguish pinned from unpinned sessions. (Issue: https://github.com/microsoft/vscode/issues/304046)
- **New task variable `${taskVar:name}`**: A new task variable lets you capture dynamic values from a task's **problem matcher output** and use them in **launch configurations**.

  Example scenario:
  - Capture a server URL from task output
  - Reference it in a launch configuration as `${taskVar:serverUrl}`

  (Issue: https://github.com/microsoft/vscode/issues/303361)
