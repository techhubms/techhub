# Claude Opus 4.5 Preview Available in GitHub Copilot Chat and IDEs

GitHub Copilot has expanded its AI model offerings by adding Claude Opus 4.5 in public preview for Copilot Enterprise, Business, Pro, and Pro+ users. This new model enhances coding assistance and productivity across several integrated development environments (IDEs).

## Where Claude Opus 4.5 Is Available

You can access Claude Opus 4.5 in GitHub Copilot Chat on:

- GitHub.com
- GitHub Mobile
- Visual Studio Code
- Visual Studio
- JetBrains IDEs (IntelliJ IDEA, PyCharm, etc.)
- Xcode
- Eclipse

The model is selectable via the chat model picker, supporting agent, ask, and edit modes for an interactive coding experience.

## How to Enable Access

For Copilot Business and Enterprise users, organization administrators need to enable Claude Opus 4.5 in Copilot settings by updating the policy configuration. Once enabled, all users in the org can choose Claude Opus 4.5 as the active model in Copilot Chat.

- [Supported Copilot Models](https://docs.github.com/copilot/reference/ai-models/supported-models)
