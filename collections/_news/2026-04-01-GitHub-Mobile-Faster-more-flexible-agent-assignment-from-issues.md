# GitHub Mobile: Faster, more flexible agent assignment from issues

GitHub Mobile now includes a faster way to delegate work to an agent from issues.

## What’s new

- A new **Assign an Agent** option is available in the issue overflow menu.
- When assigning an agent, you can:
  - Add **custom instructions**.
  - Choose a **different repository** to control where the work is delegated.
- The same agent assignment options are also available when **creating a new issue**.

## Why it matters

These changes streamline delegating work to agents within GitHub Mobile, reducing context switching during issue creation and issue management.

## Availability

Available in the latest production build of GitHub Mobile:

- iOS: https://apps.apple.com/us/app/github/id1477376905
- Android: https://play.google.com/store/apps/details?id=com.github.android
