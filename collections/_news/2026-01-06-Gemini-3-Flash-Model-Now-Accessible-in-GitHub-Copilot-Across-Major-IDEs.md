# Gemini 3 Flash is Now Available in Visual Studio, JetBrains IDEs, Xcode, and Eclipse

Allison announces the public preview of the Gemini 3 Flash model, which can now be accessed by GitHub Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+ users.

## Key Details

- **Availability:**
  - Accessible in GitHub Copilot Chat on github.com, GitHub Mobile, Visual Studio Code, Visual Studio, JetBrains IDEs, Xcode, and Eclipse using the chat model picker (agent, ask, and edit modes).
  - For those who do not see the option yet, access may be rolled out over time.
- **Supported IDEs and Versions:**
  - Visual Studio: Versions 17.14.12 and later, 18.1.0 and later, all chat modes (chat, ask, edit, agent).
  - GitHub Copilot for JetBrains: Version 1.5.62 and later in all modes (ask, edit, agent).
  - GitHub Copilot for Xcode: Version 0.46.0 and later in ask and agent modes.
  - GitHub Copilot for Eclipse: Version 0.14.0 and later in ask and agent modes.

## Enabling Gemini 3 Flash

- For Copilot Business and Enterprise accounts, administrators need to enable the Gemini 3 Flash policy in Copilot settings.
- After enabling, users in the organization will have Gemini 3 Flash available in the model picker.
