# Claude Opus 4.7 is generally available in GitHub Copilot

**Claude Opus 4.7**, Anthropic’s latest Opus model, is now rolling out on **GitHub Copilot**.

In early testing, Opus 4.7 is described as delivering:

- Stronger **multi-step task** performance
- More reliable **agentic execution**
- Improvements in **long-horizon reasoning**
- Better handling of **complex, tool-dependent workflows**

## Model lineup changes

To improve service reliability, GitHub is **streamlining model offerings**. Over the coming weeks:

- **Opus 4.7** will replace **Opus 4.5** and **Opus 4.6** in the model picker for **Copilot Pro+**.

## Pricing / request multiplier

This model is launching with a **7.5× premium request multiplier** under promotional pricing **until April 30**.

## Availability in GitHub Copilot

Claude Opus 4.7 will be available to:

- **Copilot Pro+**
- **Copilot Business**
- **Copilot Enterprise**

You’ll be able to select the model in the model picker in:

- Visual Studio Code
- Visual Studio
- Copilot CLI
- GitHub Copilot Coding Agent
- github.com
- GitHub Mobile (iOS and Android)
- JetBrains
- Xcode
- Eclipse

Rollout is **gradual**, so it may not appear immediately for all users.

## Enabling access (Business / Enterprise)

For **Copilot Enterprise** and **Copilot Business**, plan administrators must enable the **Claude Opus 4.7 policy** in Copilot settings.
