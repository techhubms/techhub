# GPT-5.1 and GPT-5.1-Codex Now Available in GitHub Copilot

**Author: Allison**

GitHub Copilot now offers the GPT-5.1 and GPT-5.1-Codex models to subscribers of Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+.

## Where You Can Use the New Models

- **GitHub Copilot Chat** on [github.com](https://github.com), GitHub Mobile
- Visual Studio Code, Visual Studio
- JetBrains IDEs, Xcode, Eclipse

These models can be selected via the chat model picker in agent, ask, and edit modes within supported IDEs.

## How to Enable GPT-5.1 and GPT-5.1-Codex

### For Organizations (Business & Enterprise)

- **Administrators** must enable the new GPT-5.1 and GPT-5.1-Codex policies in Copilot settings.
- Once enabled, all users in the organization see the models in the picker across supported IDEs.

### For Individual Plans

- Enable a model by choosing it from the picker in any supported IDE and confirming the prompt.

## Additional Resources

- For a full list of supported models in Copilot, refer to [Copilot AI Models Documentation](https://docs.github.com/copilot/reference/ai-models/supported-models).
- Share your experiences or ask questions in the [GitHub Copilot Community Discussions](https://github.com/orgs/community/discussions/categories/copilot-conversations).

## Feedback

Feedback is encouraged through the provided community forums and documentation links. If you do not yet see the new models, check again later as availability may be rolling out.
