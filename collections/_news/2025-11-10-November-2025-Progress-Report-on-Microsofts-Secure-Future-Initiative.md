# November 2025 Progress Report on Microsoft’s Secure Future Initiative

Microsoft’s Secure Future Initiative (SFI) aims to accelerate innovation, strengthen resilience, and lead the digital industry toward a safer future. This progress report outlines technical achievements and ongoing security strategies across Microsoft’s products and services.

## Key Highlights

- **Improved Security Engineering:**
  - Enhanced security for Microsoft Azure, Microsoft 365, Windows, Surface, and Microsoft Security products.
  - Azure enforced secure defaults, expanded hardware-based trust, and updated security benchmarks for cloud protection.
  - Microsoft 365 added an AI Administrator role and improved governance for agent lifecycle and data transparency.
  - Windows and Surface introduced expanded passkey support, auto-recovery, and memory-safe firmware.
  - Microsoft Security rolled out new data security posture management for AI and evolved Sentinel into an AI-first SIEM platform.

- **Fostering Security-First Mindset:**
  - 95% of employees completed top-rated training on AI-powered cyberattacks.
  - Developed internal security awareness resources now also available to customers.

- **Governance and Global Collaboration:**
  - The Cybersecurity Governance Council broadened its function, especially in response to European regulations.
  - Launched the Microsoft European Security Program and collaborative initiatives like Advancing Regional Cybersecurity in the global south.

- **Engineering Benchmarks:**
  - 99.6% of employees and devices now use phishing-resistant MFA.
  - High-risk users migrated to locked-down Azure Virtual Desktop environments.
  - 99.5% live secrets in code detected and remediated.
  - Over 50 new detections across infrastructure, with updates added to Microsoft Defender.
  - $17 million awarded to encourage responsible vulnerability disclosure.

- **Actionable Guidance for Customers:**
  - Microsoft shares 10 SFI patterns and practices for customer risk reduction.
  - Zero Trust workshops integrating SFI-based assessment support organizations in evaluating and strengthening their security postures.

- **Security Foundation:**
  - With 35,000 engineers working full-time on security efforts, Microsoft reports that SFI remains the largest cybersecurity initiative in digital history.
  - Emphasis on accelerating AI-driven security innovations, anomaly detection, and automated remediation.
