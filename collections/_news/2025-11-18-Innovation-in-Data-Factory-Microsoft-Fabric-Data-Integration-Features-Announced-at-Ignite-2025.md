# Innovation in Data Factory: Microsoft Fabric Data Integration Features Announced at Ignite 2025

Microsoft Fabric continues to advance enterprise data integration, highlighted by major announcements at Ignite 2025. Central to these innovations is Microsoft OneLake, which enables unified, zero-copy, and zero-ETL strategies for data estates spanning multi-cloud and on-premises environments.

## Key Data Factory Announcements

### Next-generation Data Integration

- **Fabric Data Factory** now combines Azure Data Factory capabilities with Power Query, offering 170+ connectors, ETL/ELT support, and deep integration with AI and analytics services.

### dbt Integration

- **dbt jobs** can now be orchestrated natively in Fabric, allowing data engineering teams to embed version-controlled transformations with governance, CI/CD, and monitoring. dbt Fusion is coming soon for advanced performance.

### SAP Mirroring & Multi-Cloud Integration

- **SAP Business Data Cloud Connect**: Enables seamless, zero-copy sharing between SAP products and Fabric, simplifying analytics and AI on enterprise SAP data.
- **Mirroring for SAP Datasphere**: Maintains data fidelity for real-time replication into OneLake.
- **Mirroring support** extended to PostgreSQL, CosmosDB, SQL Server (2016-2025), Snowflake (Iceberg Table), and more; Qlik joins Open Mirroring ecosystem.

### AI Function Transforms

- **AI Function Transforms in Dataflow Gen2** infuse AI tasks like entity extraction, sentiment, and language detection directly into dataflows, sparking advanced analytics and machine learning without custom code.
- **Copilot for pipeline automation**: Generate pipeline expressions and error insights summaries using natural language.

### Developer Productivity and Automation

- **Power BI Desktop integration** for direct export to OneLake and SQL.
- **Power Query Online IntelliSense** for enriched authoring experiences.
- **Synapse pipeline migration** now available via open source PowerShell tools.
- **Apache Airflow public APIs** allow job orchestration with files and external requirements.

### Security and Governance

- **Snowflake key pair authentication** and workspace private link provide secure, automated data integration scenarios.
- **Workspace outbound access protection** enhances compliance by governing external connections.

## Summary

Microsoft Fabric Data Factory delivers secure, automated, and extensible data integration and orchestration—empowering organizations to unify analytics, ML, and operational data across clouds. Engaged collaboration with partners like dbt Labs and Qlik, and AI-powered features with Copilot, underline Microsoft’s leadership in modern data engineering.
