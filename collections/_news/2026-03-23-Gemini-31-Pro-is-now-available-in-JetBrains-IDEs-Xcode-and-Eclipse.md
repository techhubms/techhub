# Gemini 3.1 Pro is now available in JetBrains IDEs, Xcode, and Eclipse

Gemini 3.1 Pro is available in **public preview** to **Copilot Enterprise**, **Copilot Business**, **Copilot Pro**, and **Copilot Pro+**.

You can access the model in **GitHub Copilot Chat** through the **chat model picker** (in **agent**, **ask**, and **edit** modes) in the following clients:

- github.com
- GitHub Mobile
- Visual Studio Code
- Visual Studio
- JetBrains IDEs
- Xcode
- Eclipse

If you do not yet have access, you may need to check again later.

## Enabling access

For **Copilot Business** and **Copilot Enterprise**:

- Organization administrators must **opt in** by enabling the **Gemini 3.1 Pro policy** in **Copilot settings**.
- Once enabled, users in that organization will see **Gemini 3.1 Pro** in the model picker.

For details on available models in Copilot, see: https://docs.github.com/copilot/reference/ai-models/supported-models?utm_source=model-release-announcement&utm_medium=changelog&utm_campaign=newmodels
