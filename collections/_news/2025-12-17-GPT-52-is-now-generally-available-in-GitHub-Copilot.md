# GPT-5.2 is Now Generally Available in GitHub Copilot

**Author:** Allison

## Overview

GPT-5.2 is now available in GitHub Copilot for Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+ users. This release enables access to GPT-5.2 through GitHub Copilot Chat in various development environments.

## Supported Environments

- **Web:** github.com
- **Mobile:** GitHub Mobile
- **IDEs:** Visual Studio Code, Visual Studio, JetBrains IDEs, Xcode, Eclipse

All users can choose GPT-5.2 using the chat model picker, covering agent, ask, and edit modes. Some users may need to wait for access if it hasn't reached their account yet.

## Enabling Access

- **Enterprise and Business Plans:**
    - Administrators must opt in by enabling the GPT-5.2 policy in Copilot settings.
    - Once enabled, the model appears for all supported IDEs within the organization.
- **Individual Plans:**
    - Enable GPT-5.2 by selecting it in any supported IDE and confirming the one-time prompt.

This release expands AI model availability within the GitHub Copilot ecosystem, allowing developers and teams to utilize GPT-5.2 for improved coding assistance across a wide range of supported environments. Feedback and questions can be shared through GitHub Community forums.
