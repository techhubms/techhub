# JavaScript AI Build-a-thon Season 2

**Author:** Julia Muiruri

Are you a JavaScript or TypeScript developer looking to break into AI without switching to Python? The JavaScript AI Build-a-thon Season 2 brings you a free, immersive four-week experience designed to bridge that gap using familiar tools and the latest AI technologies on the Microsoft stack.

## Program Overview

- **Duration:** March 2 – March 31, 2026
- **Format:** Online, hands-on, self-paced with live expert sessions and a community hackathon
- **Audience:** JavaScript/TypeScript developers seeking practical AI skills

### What You'll Learn

1. **Local AI Development:** Run AI models 100% on-device
2. **Multi-Service, Multi-Agentic Systems:** Learn to design and build complex agent-based applications
3. **RAG (Retrieval-Augmented Generation) Pipelines:** Implement core AI architectures using RAG for intelligent querying and response
4. **End-to-End Agent Design:** From back-end API design to building and scaling AI agents
5. **MCP Tools:** Backend tool development for specialized AI agents

## Program Structure

### Phase I: Learn & Skill Up (March 2–13)

- **Self-Paced Quests:** Foundational AI concepts, practical code samples, and architecture patterns for JavaScript/TypeScript AI solutions
- **Expert-led Sessions:** Live streams on Microsoft Reactor and interactive Q&A on Discord
- **Guided Projects:**
  1. **Local Serverless AI Chat with RAG**
     - RAG architecture and ingestion pipeline
     - Query/retrieval and LLM-based response generation
  2. **AI Burger Ordering Agent**
     - Designing AI agents and back-end APIs using MCP Tools

### Phase II: Global Hackathon (March 13–31)

Put your skills to the test in a community hackathon, building AI-powered projects that solve real-world problems. Includes:

- **Product Demos:** Showcases of the latest features and tools
- **Community Support:** Collaborate solo or in teams up to four
- **Awards:** Six categories recognizing excellence and ingenuity
- **Live Office Hours:** Ongoing support and guidance from Microsoft & GitHub experts
