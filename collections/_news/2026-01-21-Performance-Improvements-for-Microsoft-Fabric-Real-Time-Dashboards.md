# Performance Improvements for Microsoft Fabric Real-Time Dashboards

Performance is crucial when making decisions with live data. The Microsoft Fabric team has announced a set of enhancements to Real-Time Dashboards, aimed at delivering faster, smoother, and more responsive experiences—driven by customer feedback.

## Key Improvements

- **Initial dashboard loads are up to 6× faster** in certain scenarios
- **Large dataset visualizations display more quickly**, minimizing delays and improving workflow
- **Charts render with enhanced efficiency**, including up to 10× faster pie charts
- **User interface is now smoother and more responsive**, with freezes and visual jumps eliminated

Combined, these changes result in a noticeably improved experience, especially for those working with high-volume, real-time analytics.

## User Benefits

- **Faster insights** with reduced dashboard loading times
- **Smoother data exploration**, less interruption in interaction or tile refreshes
- **Greater trust and dependability** in real-time analytics environments

## Try the New Dashboards

To experience these improvements, try the updated [Real-Time Dashboards quick start](https://learn.microsoft.com/fabric/real-time-intelligence/dashboard-real-time-create?tabs=create-manual%2Ckql-database). Set up your first dashboard in minutes and experience faster live data exploration.

The Microsoft Fabric team encourages ongoing user feedback to keep improving the platform.
