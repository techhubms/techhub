# GitHub Mobile: Research and code with Copilot cloud agent anywhere

Copilot cloud agent now supports more than pull request workflows on **GitHub Mobile**, so you can continue development work when you’re away from your desk.

## What’s new

From GitHub Mobile, you can ask Copilot to:

- **Research your codebase**
- **Generate an implementation plan** before writing code
- **Make code changes on a branch** without immediately opening a pull request

You can then:

- **Review the diff**
- **Iterate** until you’re ready
- **Create a pull request** when the time is right — all from your mobile device

## Optional: Automatically open a pull request

If you already know you want to open a pull request, specify that in your prompt and Copilot will **create a pull request automatically** when the session completes.
