# Codespaces with data residency now available in Japan

GitHub Codespaces with data residency now supports **Japan** as an available region, in addition to the existing **EU** and **Australia** regions.

## What changed

In January, GitHub announced that **Codespaces is in public preview for GitHub Enterprise with data residency**, and Japan was listed as “coming soon” during the preview period.

Japan is now included in the **public preview** as a supported region.

## Who this affects

If your **enterprise is hosted in the Japan region**, you can now:

- Create codespaces with data residency
- Manage codespaces with data residency
