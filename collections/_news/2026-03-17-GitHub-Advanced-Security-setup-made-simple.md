# GitHub Advanced Security setup made simple

GitHub Advanced Security (GHAS) is now easier to manage at the organization level. GitHub added a new guided experience to help set up and configure Advanced Security, with faster workflows for editing configurations and repository targeting.

## What changed

- A new guided experience for setting up and configuring GitHub Advanced Security in organizations
- Faster editing of:
  - Configurations
  - Repository targeting

## How to view detailed settings

To view detailed settings:

- Select **Custom configuration**, or
- Open a configuration by name

## Availability

- Available now on **GitHub Enterprise Cloud**
- Planned release for **GitHub Enterprise Server 3.22**
