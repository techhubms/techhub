# AI Dev Days 2025: Microsoft & GitHub Virtual Event for Developers

**Dates:** December 10-11, 2025  
**Format:** Live-streamed on YouTube, free to attend

## Event Overview

AI Dev Days brings the latest innovations from Microsoft Ignite, GitHub Universe, and .NET Conf into one virtual conference. It's designed for developers, technical decision makers, and DevOps professionals who want to master AI integration in their workflows and applications. Attendees will gain insights into AI agents, developer tools, cloud solutions, and hands-on experience with practical workshops.

## Key Highlights

- **Live keynotes and technical sessions** on AI development, agents, automation, and productivity tools
- **Hands-on workshops**: Build AI agents with Azure AI Foundry & MCP; Customize GitHub Copilot for your workflow
- **Expert speakers**: Microsoft and GitHub engineering leaders, including Jay Parikh, Amanda Silver, David Fowler, Martin Woodward, and more
- **Source code and demos**: Access all materials on GitHub
- **Regional editions**: Follow-up events for China, LATAM, EMEA, APAC, GCR

## Sessions and Workshops

### Day 1: Building AI Applications (December 10)

- **Orchestration with Microsoft Foundry**: Build, manage, and deploy agentic AI using Foundry's modular workflows; unify models, data, tools with security and observability
- **Multi-agent automation (Foundry)**: Use Semantic Kernel + AutoGen and the Agent Framework for fleet-level orchestration
- **Agentic workflows on Windows via MCP**: Develop secure, scalable MCP servers, automate tasks across Windows platforms
- **Aspire for AI Application Development**: Streamline frontends, APIs, containers, and databases using Aspire’s new features
- **End-to-end DevOps with AI-powered workflows**: Combine GitHub Copilot and Azure to accelerate app planning, coding, quality assurance, and deployment
- **Technical deep dive: MCP Protocol**: Secure, scalable agent-tool communication across platforms
- **Build & deploy AI Agents (VS Code + GitHub Copilot)**: Instrument, trace, and deploy agents from local prototype to Azure cloud
- **Integrate OpenAI and Anthropic models via Microsoft’s AI ecosystem**: Enterprise-ready AI apps using Foundry Models
- **Workshop**: Build a pizza-ordering agent with Azure AI Foundry & MCP

### Day 2: Enhancing Developer Productivity (December 11)

- **VS Code Release 1.07**: Latest features & developer enhancements
- **GitHub Copilot CLI**: AI-powered coding in your terminal; advanced scripting and automation
- **Unified agent management in VS Code**: Manage cloud/local agents to streamline developer tasks
- **Leveraging MCP in daily development**: Integrate external tools/services through unified interfaces in VS Code
- **Multi-agent workflows for design & coding**: Automate and track tasks with agents and plan tools
- **Tech debt management using Copilot Cloud Agent**: Modernize legacy codebases with AI
- **Visual Studio 2026**: Advanced AI-driven development, debugging, and profiling
- **Spec-driven development with Spec Kit & GitHub Copilot**: Predictable outcomes, scenario-focused coding
- **Maximizing productivity with GitHub Copilot**: Lessons on using Copilot across GitHub, VS Code, and Visual Studio for shipping products
- **Workshop**: Customize GitHub Copilot for your workflow

## Who Should Attend?

- Developers integrating AI into applications
- DevOps professionals adopting agentic workflows
- Technical leaders exploring Microsoft and GitHub solutions
- Anyone interested in the latest cloud and AI tooling
