Allison announces GitHub Copilot’s new long-term support (LTS) commitment for enterprise customers, starting with GPT-5.3-Codex, including availability dates, base-model rollout timing, and how model approvals and pricing multipliers work for Copilot Business and Copilot Enterprise.<!--excerpt_end-->

## Overview

GitHub is introducing a **long-term support (LTS)** commitment for **Copilot Business** and **Copilot Enterprise**, aimed at giving enterprises more stability for **internal security and safety reviews**. Any model designated as LTS will be available for **12 months from the model’s launch date**.

The first LTS model is **GPT-5.3-Codex** (in partnership with OpenAI).

## GPT-5.3-Codex LTS availability

- **Launch date:** February 5, 2026
- **Available through:** February 4, 2027
- **Applies to:** Copilot Business and Copilot Enterprise

GitHub Copilot data indicates GPT-5.3-Codex has a **high code survival rate** among enterprise customers.

## GPT-5.3-Codex as the newest base model

GPT-5.3-Codex will also become the **newest base model** for Copilot, **replacing GPT-4.1**.

- The **base model** is used when an organization **has not approved other models** through its internal review process.
- GitHub will **automatically enable GPT-5.3-Codex** for all Copilot Business and Copilot Enterprise organizations **within 60 days** of the announcement.

### Premium request unit multipliers

- **GPT-5.3-Codex:** **1x premium request unit multiplier**
- **GPT-4.1:** remains **force-enabled at a 0x multiplier** for now

GitHub encourages organizations to begin using the latest models.

## Key dates

- **March 18, 2026:** LTS and base model changes announced
- **May 17, 2026:** GPT-5.3-Codex becomes the base model for all Copilot Business and Copilot Enterprise organizations
- **February 4, 2027:** End of the LTS availability window for GPT-5.3-Codex

## Who this applies to

These changes apply **only** to:

- Copilot Business
- Copilot Enterprise

They do **not** apply to:

- Copilot Pro
- Copilot Pro+
- Copilot Free

For individual plans, model deprecation follows the standard timeline.
