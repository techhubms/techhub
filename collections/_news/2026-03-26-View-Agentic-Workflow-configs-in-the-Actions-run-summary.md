# View Agentic Workflow configs in the Actions run summary

GitHub Actions now shows your **Agentic Workflow markdown configuration** directly in the **Actions run summary** for any agentic workflow.

## What changed

When you open a workflow run, the run summary can display the **exact markdown config** that was used when that run executed.

## Why this matters

This update is intended to make reviewing runs easier by:

- Reducing the need to switch between pages.
- Showing the exact configuration used when the workflow ran.
