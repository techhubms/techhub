# Fabric Influencers Spotlight: Microsoft Fabric Community Highlights (February 2026)

The February 2026 edition of the Fabric Influencers Spotlight brings together a range of technical articles, video tutorials, and presentations curated from Microsoft MVPs and Fabric Super Users, illustrating best practices and actionable guidance for Microsoft Fabric adopters. Here are key highlights from this month's featured content:

## Power BI Development

- **Valerie Junk** explains “Fit to Content” vs “Grow to Fit” options for Power BI table and matrix visuals, clarifying how to select and optimize these formatting settings in reports.
- **Cengizhan Arslan** compares DAX filtering approaches—Fact vs. Dimension, FILTER vs. KEEPFILTERS—and discusses their impact on query performance and report behavior, especially in large Fabric semantic models.

## Data Engineering

- **Matthias Falland** demonstrates automating Spark workloads in Microsoft Fabric using Spark Job Definitions instead of notebooks for production pipelines, focusing on robust scheduling, parameterization, and monitoring.
- **Pallavi Routaray** discusses Direct Lake in Microsoft Fabric, outlining high-performance architecture, storage strategy, and modeling patterns distinct from Import and DirectQuery models.

## Data Science

- **Sahir Maharaj** presents advanced time intelligence techniques for data science, using pandas, datetime, and the holidays library within Microsoft Fabric’s Notebooks to extract relevant date and seasonality features, supporting predictive analytics.

## Fabric Platform, Administration, and Governance

- **Shubham Rai** provides an end-to-end walkthrough of the Fabric Data Agent, integrating language-based queries with a sales lakehouse and demonstrating the process from setup through testing, including current limitations.
- **Kevin Chant** covers configuration-based deployment patterns integrating Microsoft Fabric and Azure DevOps, leveraging the fabric-cicd toolset to streamline operationalization.

## Real-Time Intelligence

- **Amit Chandak** builds a real-time dashboard by combining YouTube Live API polling, Fabric Eventstream HTTP connectors, KQL Database/EventHouse, and live dashboard tiles in Microsoft Fabric.

## Databases

- **Inturi Suparna Babu** delivers a hands-on guide to Open Mirroring in Microsoft Fabric, with practical steps and insights into incremental updates and schema synchronization.

## Certifications & Learning

- **Kevin Chant** provides an updated checklist for the DP-600 Microsoft Fabric exam, covering what candidates need for Fabric Analytics Engineer Associate certification.

The spotlight recognizes the expertise of MVPs and Super Users, capturing technical contributions that advance the Fabric ecosystem. For broader details and full learning resources, consult the provided reference links.
