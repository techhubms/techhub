# GitHub Copilot Code Review Preview Features Now Available in Enterprise Cloud with Data Residency

GitHub has announced that Copilot code review preview features are now supported in GitHub Enterprise Cloud with data residency. This update enables organizations to leverage the latest agentic code review capabilities directly within their pull requests while ensuring compliance with data residency requirements.

## Key Features

- **Agentic Tool Calling:** Access tools that gather project-wide context, including code, directory structures, and symbol references, to provide more comprehensive code reviews.
- **Integration with Deterministic Tools:** Configure and run static analysis tools such as CodeQL and ESLint alongside Copilot code reviews for advanced issue detection.
- **Seamless Handoffs:** Easily hand issues found by Copilot code review to a Copilot coding agent within the review process, streamlining conflict resolution and code updates.
- **Mission Control Visibility:** Monitor Copilot code review sessions using Copilot agent’s mission control interface, providing detailed oversight into the review process.

## How to Get Started

- Explore official documentation for [Copilot code review tool setup and usage](https://docs.github.com/copilot/concepts/agents/code-review#new-tools-public-preview-in-github-copilot-code-review).
- Learn about the public preview [announcement from GitHub Universe](https://github.blog/changelog/2025-10-28-new-public-preview-features-in-copilot-code-review-ai-reviews-that-see-the-full-picture/).
- Join conversations in the [GitHub Community announcements](https://github.com/orgs/community/discussions/categories/announcements).

Organizations interested in adopting modern, AI-powered code review workflows can now trial these features in GitHub Enterprise Cloud environments, benefiting from improved review depth, automated tool integrations, and compliance with regional data regulations.
