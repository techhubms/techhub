# Ask Copilot in security assessments now available

Organization admins and security managers can now jump into a GitHub Copilot experience directly from **secret risk assessment** or **Code Security risk assessment** results to get:

- Contextual explanations of findings
- Guided next steps for remediation

## Learn more

- Code Security risk assessment docs: https://docs.github.com/code-security/concepts/code-scanning/code-security-risk-assessment
- Secret risk assessment docs: https://docs.github.com/en/code-security/how-tos/secure-at-scale/configure-organization-security/configure-specific-tools/assess-your-secret-risk
