# GPT-5.4 mini is now available in Copilot Student auto model selection

GPT-5.4 mini is now generally available to the **Copilot Student** plan via **Copilot auto model selection**.

- Details on auto selection: [Copilot auto model selection](https://docs.github.com/copilot/concepts/auto-model-selection)
- GPT-5.4 mini is included as part of **Auto** in **GitHub Copilot Chat** across the following IDEs:
  - Visual Studio Code
  - Visual Studio
  - JetBrains IDEs
  - Xcode
  - Eclipse

To review which models are available in Copilot, see the documentation:
- [Supported models in Copilot](https://docs.github.com/copilot/reference/ai-models/supported-models?utm_source=model-release-announcement&utm_medium=changelog&utm_campaign=newmodels)
