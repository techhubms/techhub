# Gemini 3 Pro Model Launches for GitHub Copilot in Popular IDEs

**Author: Allison**

Gemini 3 Pro, a new AI model, is now available in public preview for GitHub Copilot subscribers with Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+. This release enables users to access Gemini 3 Pro within GitHub Copilot Chat across multiple platforms, including github.com, GitHub Mobile, Visual Studio Code, Visual Studio, JetBrains IDEs, Xcode, and Eclipse using the chat model picker in agent, ask, and edit modes.

## Where You Can Use Gemini 3 Pro

- **Visual Studio**: Versions 17.14.12 and above, 18.1.0 and above (all modes: chat, ask, edit, agent)
- **JetBrains IDEs**: GitHub Copilot plugin version 1.5.62 and above (all modes: ask, edit, agent)
- **Xcode**: GitHub Copilot version 0.46.0 and above (modes: ask, agent)
- **Eclipse**: GitHub Copilot version 0.14.0 and above (modes: ask, agent)

## How to Enable Gemini 3 Pro

- Copilot Business and Enterprise administrators must opt in by enabling the Gemini 3 Pro policy in Copilot settings.
- Once this policy is set, users in the organization will be able to access Gemini 3 Pro through the model picker in Copilot.

Refer to [official Copilot documentation on model support](https://docs.github.com/copilot/reference/ai-models/supported-models?utm_source=model-release-announcement&utm_medium=changelog&utm_campaign=newmodelshttps://docs.github.com/copilot/reference/ai-models/supported-models) for technical details and version requirements.
