# Introducing Anthropic’s Claude Opus 4.5 in Microsoft Foundry

Anthropic’s Claude Opus 4.5 is now in public preview on Microsoft Foundry, GitHub Copilot paid plans, and Copilot Studio, marking a major milestone for AI development in the Microsoft ecosystem. This model sets new standards for coding, agentic workflows, and enterprise productivity.

## Key Highlights

- **Public Preview in Microsoft Foundry, GitHub Copilot Paid Plans, and Copilot Studio**
- **Enterprise-scale Access**: Azure customers benefit from immediate integration, deployment, and scalability for advanced AI agents.
- **Expanded Model Selection**: Developers can pick from leading frontier AI models within Microsoft’s secure platform.

## Technical Benchmarks

Claude Opus 4.5 achieves state-of-the-art results:

| Capability                | Claude Opus 4.5 | Claude Sonnet 4.5 | Claude Opus 4.1 | Gemini 3 Pro |
|---------------------------|------------------|-------------------|-----------------|--------------|
| Agentic Coding (SWE-bench)| 80.90%           | 77.20%            | 74.50%          | 76.20%       |
| Tool Use (Retail)         | 88.90%           | 86.20%            | 86.80%          | 85.30%       |
| Visual Reasoning (MMMU)   | 80.70%           | 77.80%            | 77.10%          | -            |
| Graduate Reasoning (GPQA) | 87.00%           | 83.40%            | 81.00%          | 91.90%       |

*These benchmarks show Opus 4.5's strong multi-domain performance for engineering, reasoning, and workflow execution.*

## Developer Features in Foundry

- **Programmatic Tool Calling**: Direct tool invocation (e.g., Python) for efficient workflows.
- **Tool Search**: Dynamic tool discovery without context window overload.
- **Effort Parameter (Beta)**: Fine-tune computational effort for balance between speed and cost using new SDK helpers.
- **Compaction Control**: Manage long-running tasks over extended agent interactions.

## Use Cases

- **Software Engineering**: Automate multi-system development tasks, code generation, and architectural analysis.
- **Cybersecurity**: Threat detection and incident response leveraging agentic tool use.
- **Financial Modeling**: Advanced predictive modeling, compliance monitoring, and structured analysis.
- **Enterprise Operations**: Agent-driven orchestration across multiple tools and data sources.

## Security and Safety Improvements

- **Reduced Prompt-Injection Vulnerability**
- **Greater Robustness Across Complex Tasks**
- **Enterprise-Grade Compliance**: Supporting centralized governance and observability.
