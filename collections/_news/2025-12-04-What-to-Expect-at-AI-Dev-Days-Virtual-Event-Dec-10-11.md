# What to Expect at AI Dev Days Virtual Event Dec. 10-11

AI Dev Days is a major two-day virtual event hosted by Microsoft and GitHub, designed to bring together developers, builders, and technical leaders to learn about the latest advancements in AI development, tools, and workflows. The event will be held on December 10-11, 2025, streamed live on YouTube and entirely free to attend.

## Event Overview

- **Dates:** December 10-11, 2025
- **Start Time:** Day 1 at 9:00 AM PT, Day 2 at 8:00 AM PT
- **Format:** Virtual, live sessions and interactive workshops

## What You'll Learn

AI Dev Days synthesizes content from top industry events (Microsoft Ignite, GitHub Universe, .NET Conf) into sessions and hands-on labs dedicated to:

- Developing AI applications using Microsoft Foundry
- Building and orchestrating AI agents
- Leveraging GitHub Copilot and Copilot CLI in development workflows
- Enhancing developer productivity with advanced tooling in Visual Studio Code and Visual Studio
- Integrating multi-agent workflows and new protocols (like MCP)
- Streamlining DevOps with AI-powered processes

## Day 1 Highlights: Building AI Applications

- **Microsoft Foundry**: Learn to build, orchestrate, and manage intelligent agents with modular, enterprise-oriented architecture (sessions featuring Semantic Kernel, AutoGen, and agent orchestration).
- **Agentic Workflows**: Dive into multi-agent orchestration, the Model Context Protocol (MCP), and building containerized, observable agents.
- **AI Integration**: See how to extend apps with MCP on Windows and modernize workflow automation.
- **GitHub Copilot & Azure**: Explore end-to-end development, deployment, and diagnostics with Copilot and SRE/testing agents.
- **Workshop**: Build a Pizza Ordering Agent hands-on using Azure AI Foundry and MCP.

## Day 2 Highlights: Developer Productivity with AI

- **GitHub Copilot CLI**: Transform your terminal into an AI-powered development environment; automate coding and scripting.
- **VS Code & Visual Studio**: Discover new features in VS Code 1.07 and Visual Studio 2026, focusing on AI-driven development and agent integration.
- **Plan & Specification Tools**: Implement spec-driven development using open source toolkits with Copilot.
- **Real-World Insights**: Hear how engineers ship more code and modernize legacy software using GitHub Copilot and agentic workflows.
- **Workshop**: Customize GitHub Copilot for your personal developer workflow.

## Who Should Attend?

- Developers integrating AI into their apps
- AI developers wanting to deepen their tool expertise
- Technical leaders and DevOps professionals focused on AI/ML workflows
- Anyone keen on Microsoft’s AI, GitHub, and Azure ecosystem advancements

## Additional Notes

- Sessions will be available on-demand after the event.
- Global regional editions (China, LATAM, EMEA, APAC) provide localized workshops in Jan 2026.
