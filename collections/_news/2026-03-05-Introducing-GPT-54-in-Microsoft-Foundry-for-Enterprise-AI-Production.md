# Introducing GPT-5.4 in Microsoft Foundry for Enterprise AI Production

**Author:** Naomi Moneypenny  
**Published:** March 5, 2026

## Overview

OpenAI’s GPT‑5.4 and GPT‑5.4 Pro are now (or soon to be) generally available in Microsoft Foundry. These models are designed for organizations requiring highly reliable AI systems in production, with a focus on advanced reasoning, robust execution, and enterprise-grade operational control.

## Key Features

- **Stronger Reasoning:** Improved consistency and context retention across long, multi-step workflows
- **Agentic Workflows at Scale:** Enables automation across tools, files, and complex tasks
- **Enhanced Computer Use Capabilities:** Structured orchestration of file access, tool invocations, and code execution
- **Reduced Oversight Needs:** Lower reliance on prompt tuning and manual intervention
- **Predictable Output:** Higher stability and quality for artifacts such as documents, presentations, and spreadsheets

## Production-Ready Design

GPT-5.4 is tailored to support production workloads, moving beyond planning into reliable task completion. Key improvements include:

- **Consistent Multi-turn Reasoning:** Maintains intent over long, complex interactions
- **Instruction Alignment:** More accurate following of user or process instructions
- **Low-Latency Performance:** Responsive outputs suited for real-time enterprise needs
- **Improved Tool Invocation:** More dependable integration with workflow tools

## GPT-5.4 Pro: For Deep Analytical Workflows

The premium ‘Pro’ model enables:

- **Multi-path Reasoning:** Explores alternative solutions before selecting a final response
- **Higher Analytical Depth:** For research or scenarios with multiple trade-offs
- **Stable Extended Reasoning:** Useful in sustained analytical or research tasks
- **Enhanced Decision Support:** Designed when rigor is prioritized over speed

## Enterprise Deployment via Microsoft Foundry

- **Operational Controls:** Policy enforcement, monitoring, auditability, and version management
- **Secure and Compliant:** Built to align with enterprise security and compliance needs
- **Lifecycle Management:** Full support from deployment to monitoring and upgrades

## Scenarios and Use Cases

- **Agent-driven Automation:** Customer support, research, business process workflows
- **Enterprise Knowledge Work:** Data analysis, drafting documents, presentation generation
- **Developer Workflows:** Code generation, refactoring, debugging, UI scaffolding
- **Complex Analytical Tasks:** Long-chain logic and reasoning for high-stakes decisions

## Getting Started and Pricing

- **Standard Model:** $2.50 per million input tokens, {{CONTENT}}.25 per million cached input tokens, $15.00 per million output tokens
- **Pro Model:** $30.00 per million input tokens, $180.00 per million output tokens
- **Deployment Zones:** Available in Standard Global and Standard Data Zone (US); additional regions forthcoming

## Next Steps

Organizations can immediately begin integrating GPT‑5.4 into their workflows via Microsoft Foundry, leveraging enterprise controls and advanced agentic features.

For pricing and deployment details, visit the [Microsoft Foundry catalog](https://ai.azure.com/catalog).
