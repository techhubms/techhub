# GPT-5.4 is Generally Available in GitHub Copilot

**Author:** Kate Catlin

OpenAI’s latest agentic coding model, GPT-5.4, is now being rolled out to users of GitHub Copilot. Early testing highlights its improved real-world coding, logical reasoning, and multi-step task execution, especially for agentic and tool-dependent developer workflows.

## Availability and Supported Platforms

- **GitHub Copilot Editions:** Pro, Pro+, Business, and Enterprise
- **Where You Can Use GPT-5.4:**
  - Visual Studio Code (v1.104.1+) — chat, ask, edit, agent modes
  - Visual Studio (v17.14.19+) — agent, ask modes
  - JetBrains (v1.5.66+) — ask, edit, agent modes
  - Xcode (v0.48.0+) — ask, agent modes
  - Eclipse (v0.15.1+) — ask, agent modes
  - github.com web interface
  - GitHub Mobile (iOS/Android)
  - GitHub CLI
  - GitHub Copilot Coding Agent

**Tip:** Prompting and model features work best in the latest versions of each supported product.

## Enabling Access

- **For Organizations:** Copilot Enterprise and Copilot Business plan admins must enable the GPT-5.4 model policy in their Copilot settings.
