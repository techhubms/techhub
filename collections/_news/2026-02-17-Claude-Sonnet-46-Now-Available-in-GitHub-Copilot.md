# Claude Sonnet 4.6 Now Available in GitHub Copilot

**Author:** Kate Catlin

Claude Sonnet 4.6, the newest agentic coding model from Anthropic, is being rolled out for GitHub Copilot users. Early testing indicates strong performance in agentic coding tasks and enhanced results in search operations.

## Where You Can Use Claude Sonnet 4.6

- **GitHub Copilot Pro, Pro+, Business, and Enterprise** plans
- **Visual Studio Code**: All modes (chat, ask, edit, agent)
- **Visual Studio**: All modes (agent, ask)
- **github.com**
- **GitHub Mobile (iOS & Android)**
- **GitHub Copilot CLI**
- **GitHub Copilot Coding Agent**

The rollout is gradual, so some users may not see the model immediately.

## Enabling Claude Sonnet 4.6

Administrators for Copilot Enterprise and Business plans need to enable Claude Sonnet 4.6 in the Copilot settings.

## Pricing

The model launches with a 1x premium request multiplier, and pricing may change.
