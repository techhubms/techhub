# Get started with GitHub Copilot CLI: A free, hands-on course

*By Dan Wahlin*

GitHub Copilot now extends beyond your code editor and offers powerful functionality right from your terminal with GitHub Copilot CLI. This post introduces a free, open-source, 8-chapter hands-on course designed to teach developers how to use Copilot CLI effectively, even without prior AI expertise.

## What is GitHub Copilot CLI?

GitHub Copilot CLI is a conversational AI assistant that runs in your terminal and leverages AI to:

- Review code for quality and suggest improvements
- Generate tests based on source files
- Debug code by analyzing files for issues
- Explain confusing logic or unfamiliar code
- Generate commit messages and refactor functions
- Write new features, ranging from front-end to APIs

It maintains conversational context, allowing you to build upon previous queries.

## About the Course

The **GitHub Copilot CLI for Beginners** course is structured into 8 chapters, progressing from basic installation to advanced workflow integration:

1. **Quick Start**: Covers installation and authentication
2. **First Steps**: Explains the three interaction modes – interactive, plan, and one-shot (programmatic)
3. **Context and Conversations**: Shows how to point Copilot at files/directories and manage sessions
4. **Development Workflows**: Focuses on code review, refactoring, debugging, test generation, and Git integration
5. **Custom Agents**: Guides on creating AI-powered assistants for specialized tasks
6. **Skills**: Demonstrates task-specific instructions that auto-trigger based on prompts
7. **MCP Servers**: Explains connecting Copilot to external resources and APIs through the Model Context Protocol
8. **Putting It All Together**: Combines agents, skills, and MCP servers for full development workflows

Every step is practical and hands-on, centered on a book collection management app that evolves through each exercise.

## Who Should Use This Course?

- **Developers comfortable with terminal-based workflows**
- **Teams wanting to adopt AI-assisted development practices**
- **Students or early-career developers** (AI concepts are introduced with clear explanations and relatable analogies)

If you can use the command line, you can leverage this course—there’s no prerequisite AI or machine learning background.

## Teaching Approach

Each chapter uses real-world analogies—for example, comparing interaction modes to ordering food—to help you build a conceptual understanding before diving into technical details. You’ll also find:

- Clear, copyable CLI commands
- Explanations of when to use which Copilot CLI features (agents, skills, MCP servers)
- Realistic integration patterns
- Assignments with well-defined criteria
