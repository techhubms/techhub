# Fabric Influencers Spotlight January 2026

Welcome to the latest edition of the Fabric Influencers Spotlight. This recurring post brings together exceptional technical content from Microsoft MVPs and Fabric Super Users, covering multiple facets of the Microsoft Fabric platform. Below are highlights from this month's featured community contributions:

## Data Warehouse

**Amit Chandak** demonstrates how to utilize IDENTITY columns in Microsoft Fabric Data Warehouse for the automatic generation of surrogate keys. The video covers key differences from SQL Server and offers a practical, step-by-step implementation guide. [Watch on YouTube](https://youtu.be/kgFiIRukiJM)

## Power BI

- **Jennifer Rattan** explores the Power BI Narrative Visual, showing how it elevates the data prep process with AI-driven documentation and modeling that aids data engineering and reporting tasks. [Read the post](https://community.fabric.microsoft.com/t5/Power-BI-Community-Blog/Rethinking-Data-Prep-Power-BI-Narrative-Visual-as-a-Clever-Ally/ba-p/4919996)

- **Ilgar Zarbaliyev** compares DAX UDFs and Fabric UDFs within Power BI environments, offering detailed insights on creation, use cases, and how these functions strengthen data modeling and integration. [Read the article](https://www.linkedin.com/pulse/understanding-differences-between-dax-udf-fabric-power-ofivf)

## Data Engineering

- **Vojtech Sima** presents a tutorial on implementing write-back from Power BI reports to Azure SQL Database using Fabric User Data Functions, bypassing Power Apps for cost efficiency and direct reporting. Detailed architectural considerations and technical implementation guidance are provided. [Read the blog](https://www.vojtechsima.com/post/power-bi-write-back-to-azure-sql-fabric-udf)

## Data Science

- **Sahir Maharaj** walks through the concept of regularization for data science in Microsoft Fabric—explaining L1, L2, and ElasticNet penalties, their practical distinctions, and use cases for combating overfitting in machine learning workflows. [Read the article](https://community.fabric.microsoft.com/t5/Data-Science-Community-Blog/Mastering-Regularization-for-Data-Science-in-Microsoft-Fabric/ba-p/4818743)

## Fabric Platform, Administration & Governance

- **Inturi Suparna Babu** details configuring OneLake security, demonstrating row-level security (RLS) for fine-grained access control, including real-world implementation steps. [Read the blog](https://community.fabric.microsoft.com/t5/Fabric-platform-Community-Blog/Fine-grained-ReadWrite-access-to-data-with-OneLake-security-Row/ba-p/4914092)

- **Aleksi Partanen** provides an overview of Fabric data stores and databases—Lakehouse, Warehouse, Eventhouse, SQL Database, and Cosmos DB—and explains their interaction within OneLake. [Watch on YouTube](https://youtu.be/QLD6M_GRu_c)

## Real-Time Intelligence

- **Greg Low** introduces "Fabric RTI 101," a series delving into the capabilities and technical implementations of Real-Time Intelligence within Microsoft Fabric.

## Learning & Certification

- **Kevin Chant** provides updated advice for Microsoft certifications in the age of Fabric and AI. [Read the blog](https://chantifiedlens.com/2026/01/01/updated-advice-if-microsoft-certifications-are-your-new-years-resolution/)

- **Frank Geisler** hosts "The F-Files Podcast"—a discussion blending technical deep-dives on Microsoft Data, Fabric, RTI, and AI with practitioner stories and practical feedback. [Listen on YouTube](https://www.youtube.com/watch?v=97AYiuvexe0&list=PLZ4JMqbYd0BYbdjyHDFLZFF1M3PkYBqm_)
