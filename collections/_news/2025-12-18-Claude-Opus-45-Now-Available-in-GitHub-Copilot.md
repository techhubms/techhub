# Claude Opus 4.5 Now Available in GitHub Copilot

Claude Opus 4.5 is now generally available for Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+ users. This enhancement allows developers to use Claude Opus 4.5 within GitHub Copilot Chat on multiple platforms, including github.com, GitHub Mobile, Visual Studio Code, Visual Studio, JetBrains IDEs, Xcode, and Eclipse. Users can select the model using the chat model picker, which supports agent, ask, and edit modes.

## Enabling Access

- **For Organizations:**
  - Copilot Business and Copilot Enterprise administrators need to enable the Claude Opus 4.5 policy in Copilot settings.
  - Once enabled, users in the organization will have access to the model picker across all supported IDEs.
- **For Individuals:**
  - Enable Claude Opus 4.5 by selecting it in the model picker in any supported IDE.
  - Confirm access via a one-time prompt.

If access is not immediately available, users are advised to check back later as rollout may take time.
