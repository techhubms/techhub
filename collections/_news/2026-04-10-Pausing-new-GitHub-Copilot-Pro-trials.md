# Pausing new GitHub Copilot Pro trials

As GitHub Copilot continues to grow, GitHub has seen a significant rise in abuse of the free trial system. To protect the experience and integrity of the platform for legitimate developers, GitHub is **pausing new GitHub Copilot Pro trials** while it investigates and strengthens protection mechanisms.

## What this means

- **New users cannot start a free Copilot Pro trial** at this time.
- **Existing trials are not affected.**
- **Copilot Free** and **paid Copilot Pro subscriptions** remain available.

## What’s next

This is described as a **temporary pause**. GitHub is working on improved safeguards to prevent misuse of the trial system and will reopen trials once those protections are in place.

GitHub says it will share an update when new trials are available again.

## How to get access now

If you need Copilot immediately, GitHub recommends subscribing to **Copilot Free** or choosing one of the **paid Copilot subscriptions** directly.
