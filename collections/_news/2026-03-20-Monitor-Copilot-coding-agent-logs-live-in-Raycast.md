Allison explains how to use the GitHub Copilot extension for Raycast to monitor Copilot coding agent tasks and view live agent logs directly from Raycast, including what Copilot plans support the feature and when an admin must enable it.<!--excerpt_end-->

## Overview

[Raycast](https://raycast.com/) is a free launcher for macOS and Windows that can launch apps, search files, control your system, and run community-built extensions.

With the [GitHub Copilot extension for Raycast](https://www.raycast.com/github/github-copilot), you can hand tasks to the [Copilot coding agent](https://docs.github.com/copilot/concepts/agents/coding-agent/about-coding-agent) (a cloud-based background agent) and track progress from Raycast.

## What’s new

You can now **watch Copilot’s logs live from Raycast** (without switching to GitHub), making it easier to monitor what the coding agent is doing and track progress with keyboard-driven workflow.

## How to view live Copilot coding agent logs in Raycast

1. Install the GitHub Copilot extension for Raycast.
2. Open Raycast.
3. Select the **View Tasks** command.
4. Pick your session from the list.

![Screen recording of live agent logs in Raycast](https://github.com/user-attachments/assets/6eaedf15-11c7-4392-8edc-8fee0b4c06ac)

## Availability and prerequisites

Copilot coding agent is available to:

- Copilot Pro
- Copilot Pro+
- Copilot Business
- Copilot Enterprise

If you’re on **Copilot Business** or **Copilot Enterprise**, an administrator must enable Copilot coding agent in GitHub:

- [Enable Copilot coding agent (Policies page)](https://docs.github.com/enterprise-cloud@latest/copilot/how-tos/agents/copilot-coding-agent/enabling-copilot-coding-agent)
