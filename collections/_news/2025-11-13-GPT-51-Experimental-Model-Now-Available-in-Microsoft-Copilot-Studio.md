# GPT-5.1 Experimental Model Now Available in Microsoft Copilot Studio

Microsoft has released the GPT-5.1 model as an experimental feature in [Microsoft Copilot Studio](https://www.microsoft.com/microsoft-copilot/microsoft-copilot-studio/), following OpenAI’s latest advancements. This model is accessible to U.S. customers in early release cycle Power Platform environments.

## Key Highlights

- **Experimental Model**: GPT-5.1 enables users to trial improved adaptability in both chat and reasoning tasks.
- **Availability**: Limited to U.S. customers and early release Power Platform environments for initial feedback and evaluation.
- **Usage Guidance**: Microsoft recommends using the experimental model for non-production scenarios, allowing customers to test capabilities and assess performance relative to current models.
- **Deployment Readiness**: This early access helps organizations prepare for future production use, while Microsoft gathers feedback to ensure the model meets quality standards.

![Screenshot showing agent's model choices, highlighting GPT-5.1 Chat and Reasoning](https://www.microsoft.com/en-us/microsoft-copilot/blog/wp-content/uploads/2025/11/5.1-final-1-scaled.png)

For more information and step-by-step guides on using the latest models in Copilot Studio, see the official [Microsoft documentation](https://learn.microsoft.com/microsoft-copilot-studio/authoring-select-agent-model).

## Summary

Experimental model releases like GPT-5.1 in Copilot Studio give organizations early exposure to new AI features, supporting evaluation and feedback before general availability.
