# Join or Host a GitHub Copilot Dev Days Event Near You

The way we build software is changing rapidly as AI becomes a core part of development practice. GitHub Copilot Dev Days is a global series of hands-on, in-person, community-led events focused on helping developers of all experience levels explore real-world AI-assisted coding.

## About GitHub Copilot Dev Days

GitHub Copilot Dev Days are designed to provide practical sessions with:

- Live demonstrations of GitHub Copilot features
- Hands-on workshops and interactive exercises
- Professional training on using tools like GitHub Copilot CLI, Cloud Agent, Copilot in VS Code and Visual Studio, plus support for other editors
- Content tailored for both newcomers to AI-assisted development and experienced users looking to learn new techniques

Each event is run by local community leaders such as GitHub Stars, Microsoft MVPs, GitHub Campus Experts, Microsoft Student Ambassadors, and employees from both GitHub and Microsoft. Attendees can expect quality technical sessions, opportunities for networking, snacks, and community swag.

**Sample Agenda:**

- Introductory session (30-45 min) on GitHub Copilot
- Local community session (30-45 min) on relevant topics
- Workshop (1 hour) with hands-on coding exercises

## Who Should Attend?

Anyone interested in improving their software development workflow with AI. Sessions are suitable for all skill levels, from students to experienced professionals. Whether you're new to AI development tools or already use Copilot, you'll find content to help you get the most out of AI-assisted coding.

## How to Join

Events are happening worldwide starting in March, but space is limited. To find an event or to register, visit [the official GitHub Copilot Dev Days page](https://aka.ms/githubcopilotdevdays).

Want to bring an event to your local developer group? Organizers can complete [this form](https://aka.ms/githubcopilotdevdays/form) for more information.

## Learn, Connect, and Code

GitHub Copilot Dev Days combines practical learning, community engagement, and the latest in AI for software development. Join your peers, expand your skills, and experience AI-assisted coding firsthand.

For more details and registration, check out the [GitHub Copilot Dev Days page](https://aka.ms/githubcopilotdevdays).
