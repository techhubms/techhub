# Azure DevOps Server Now Generally Available for Self-Hosted Enterprises

Azure DevOps Server has officially moved to general availability (GA), offering robust DevOps capabilities for enterprises that require self-hosted solutions. This transition from the Release Candidate (RC) phase to full production marks a significant milestone, making the platform ready for organization-wide adoption.

## Upgrade Path and Compatibility

- Direct upgrades supported from Azure DevOps Server RC or any Team Foundation Server (TFS) version from 2015 onwards.
- **Important:** TFS 2015 will reach the end of Extended Support on October 14, 2025. Organizations are strongly advised to upgrade to Azure DevOps Server for continued security and compliance.
