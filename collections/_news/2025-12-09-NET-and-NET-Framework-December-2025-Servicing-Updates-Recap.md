# .NET and .NET Framework December 2025 Servicing Updates Recap

The .NET Team and Tara Overfield provide a rundown of the latest servicing updates for December 2025. This update refreshes .NET and .NET Framework, offering non-security fixes and valuable links for developers.

## What’s New in December 2025

- **.NET 10.0.1** has received non-security fixes this month.
- The update covers core areas: Runtime, ASP.NET Core, SDK, WinForms, and Entity Framework Core.
- No new security or additional non-security updates are available for .NET Framework.

## .NET Framework December 2025 Update

There are no new updates available for .NET Framework this month. Developers can reference prior changes or issues using the [release notes for .NET Framework](https://learn.microsoft.com/dotnet/framework/release-notes/release-notes).

## Summary

- Update your .NET 10.0 installations to the latest service release for ongoing improvements.
- Keep up to date with servicing changes via provided resource and changelog links.
- Look forward to the next monthly update.
