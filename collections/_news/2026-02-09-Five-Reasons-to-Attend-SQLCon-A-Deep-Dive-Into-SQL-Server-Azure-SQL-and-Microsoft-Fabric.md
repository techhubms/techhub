# Five Reasons to Attend SQLCon: A Deep Dive Into SQL Server, Azure SQL, and Microsoft Fabric

**Author:** Anna Hoffman  
**Event:** SQLCon + FabCon, Atlanta, March 16-20

The SQL community is gathering in Atlanta for the first-ever [SQLCon](https://sqlcon.us/), held alongside **FabCon**, the Microsoft Fabric Community Conference. Attendees get access to deep SQL expertise and broad exposure to the latest in Microsoft Fabric, Power BI, data engineering, real-time intelligence, and AI developments.

## 1. One Pass, Two Conferences—Double the Value

- One registration provides admission to both SQLCon and FabCon
- Mix and match sessions: attend a deep SQL session in the morning and a Microsoft Fabric or AI session in the afternoon
- Shared expo and community lounge foster interdisciplinary learning

## 2. Dive Deep with Interactive Sessions and Hands-On Workshops

- 50+ SQL-focused sessions available ([full track details](https://sqlcon.us/program/tracks))
- In-depth content covers:
  - SQL Server
  - Azure SQL
  - SQL database in Microsoft Fabric
  - Performance tuning, security, governance
  - Migration and modernization strategies
  - Building AI-powered experiences with SQL
- Monday and Tuesday feature hands-on workshops—participants can bring their laptops for practical learning, scripts, and patterns to take home
- Wednesday through Friday: stacked conference sessions to inform annual planning

## 3. Experience Atlanta: The Perfect Setting for SQLCon

- Hosted at the Georgia World Congress Center in downtown Atlanta
- Proximity to key attractions: Centennial Olympic Park, State Farm Arena
- Special attendee party at the Georgia Aquarium for networking with peers and product teams
- City promotes easy access to dining, entertainment, and conference venues

## 4. Announcements, Roadmap, and Engineering Insights

- Direct updates from Microsoft SQL engineering team (30+ team members attending)
- First-look at upcoming features and live demos:
  - SQL Server, Azure SQL, SQL database in Microsoft Fabric
  - SQL tooling, driver updates, SSMS/VS Code extension enhancements
  - Copilot integrations for SQL
  - Fabric SQL experiences
- Attendees gain clarity on Microsoft data roadmap and investment priorities

## 5. Revitalized and Engaged SQL Community

- SQLCon is designed for connection: meetups, community lounge discussions, Ask-the-Experts sessions with Microsoft engineers, MVPs, and product teams
- Shared keynotes, open Q&A, and informal networking extend the experience beyond structured sessions

## Bonus: Make the Budget Work

- Early bird pricing, team discounts, and buy-one-get-one deals available on the registration page
- Use promo code **SQLCMTY200** for $200 off

## Wrap-Up

SQLCon + FabCon delivers an intensive, high-impact week of technical learning, community building, and hands-on engagement with Microsoft’s data and AI platforms. If SQL is integral to your technology stack, this event promises to inform, inspire, and connect you with the latest strategies, tools, and engineers shaping the future of Microsoft data solutions.

**See you in Atlanta!**
