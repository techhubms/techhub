# Japan’s ARUM turns craftsmanship into scalable AI for precision manufacturing

*Read the story in Japanese: https://news.microsoft.com/source/asia/features/apans-arum-turns-craftsmanship-into-scalable-ai-for-precision-manufacturing/?lang=ja*

## Overview

In Kanazawa, Japan, ARUM Inc. is prototyping an automated CNC (computer-numerical control) machining center called **TTMC Origin**. The system is designed to help junior workers operate precision machining equipment by turning expert know-how into step-by-step guidance delivered in natural language.

## KAYA: conversational interface for CNC operations

A worker interacts with **KAYA**, a conversational AI interface that:

- Presents an on-screen avatar and prompts the operator (in Japanese) to select the file for the precision part to be manufactured.
- Analyzes the selected file.
- Offers to set up the machining process.
- Guides the operator through intricate manual steps, such as:
  - Changing a drill bit
  - Repositioning the item being cut

## Microsoft technologies used

KAYA is described as being:

- Built using **Azure AI Speech**
- Built using **Azure OpenAI** in **Microsoft Foundry** (Azure AI Foundry)
- Running on **GPT-5** (OpenAI’s large language model)

## Why this approach

ARUM’s goal is to make a CNC machining center that can be operated by **junior workers**, not only highly skilled experts—particularly relevant in Japan where experienced specialists are described as a “dwindling breed.”

According to **Tetsuya Kitanishi**, lead software developer for TTMC Origin, **generative AI** is central to voicing and guiding these intricate manual steps.
