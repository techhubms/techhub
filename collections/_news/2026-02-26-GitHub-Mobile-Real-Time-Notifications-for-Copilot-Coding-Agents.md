# GitHub Mobile: Real-Time Notifications for Copilot Coding Agents

GitHub has launched Live Coding Agent Notifications on GitHub Mobile, making it easier for developers to stay updated on their Copilot coding agent and supported third-party agent sessions while on the move.

## Key Features

- **Live Activities Integration**: Receive real-time updates about agent sessions started from GitHub.com, GitHub Mobile, or the CLI. Notifications include pull request titles and current status, with direct navigation to the pull request within the app.

- **Supported Session States**:
  - In progress
  - Completed
  - Failed
  - Cancelled

- **Supported Agents and Platforms**:
  - Copilot coding agent
  - Third-party agents, e.g., Codex and Claude
  - Available for cloud-run sessions only
  - iOS 17.2+ (Live Activities required)
  - Android 16+ for Live Update Notifications (older versions receive basic notifications)

## Customization

- Users can disable Live Activities in the app settings.
- First notification includes a 'Disable Live Updates' quick opt-out option.

## Availability

- Feature is available now on the latest production build of [GitHub Mobile for iOS](https://apps.apple.com/us/app/github/id1477376905) and [Android](https://play.google.com/store/apps/details?id=com.github.android).
