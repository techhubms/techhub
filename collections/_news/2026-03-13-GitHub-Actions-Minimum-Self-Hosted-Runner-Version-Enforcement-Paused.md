# GitHub Actions: Minimum Self-Hosted Runner Version Enforcement Paused

**Author:** Allison  
**Type:** News

GitHub has communicated that enforcement of the minimum self-hosted runner version requirement for GitHub Actions (v2.329.0) will be temporarily paused. The enforcement, which was scheduled to take effect on March 16, 2026, is postponed to allow users additional time for a smooth transition.

During this period, you can continue to register and configure self-hosted runners running versions below v2.329.0. However, GitHub strongly encourages all administrators and teams to upgrade their self-hosted runners to v2.329.0 or later as soon as possible. Detailed upgrade instructions can be found in the [upgrade documentation](https://docs.github.com/enterprise-cloud@latest/actions/reference/runners/self-hosted-runners#runner-software-updates-on-self-hosted-runners).

This pause does not represent a change in GitHub's long-term direction. The plan to eventually block older runner versions from registration and configuration remains unchanged. The temporary hold is intended to ensure users have sufficient time to adapt and prevent operational disruption.

#### Key Points

- Minimum version enforcement (v2.329.0) is postponed
- Older runners can still be registered/configured temporarily
- Upgrading soon is strongly encouraged
- Timeline and enforcement updates will follow
