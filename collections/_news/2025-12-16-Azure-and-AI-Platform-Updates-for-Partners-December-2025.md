# Azure and AI Platform Updates for Partners: December 2025

By Cyril Belikoff

Microsoft Ignite 2025 introduced a new era—Frontier transformation—where AI becomes foundational within organizations. This monthly Azure update aggregates announcements and insights relevant to Microsoft partners seeking to deliver next-generation, AI-powered solutions, manage unified data, and optimize cloud governance.

## Key Announcements and Updates

### Azure Copilot (Private Preview)

- Introduces specialized AI agents in the Azure portal, PowerShell, and CLI.
- Streamlines migration, modernization, and governance tasks across customer environments.
- Integrates with Azure Resource Manager and leverages agentic workflows alongside GitHub Copilot.
- [Read more](https://techcommunity.microsoft.com/blog/azureinfrastructureblog/ushering-in-the-era-of-agentic-cloud-operations-with-azure-copilot/4469664?previewMessage=trueus/products/copilot)

### Foundry Control Plane (Public Preview)

- Unified visibility, security, and policy management for AI agents.
- Centralizes lifecycle governance and observability across Microsoft Cloud.
- Supports enterprise-grade management for safe, scalable AI agent deployments.
- [Learn about Foundry Control Plane](https://learn.microsoft.com/azure/ai-foundry/control-plane/overview?view=foundry)

### Foundry IQ (Public Preview)

- Offers a unified endpoint for agent knowledge integrated with Azure AI Search.
- Enables secure, contextually-aware RAG solutions by connecting agents to enterprise data sources.
- [Foundry IQ details](https://techcommunity.microsoft.com/blog/azure-ai-foundry-blog/foundry-iq-unlocking-ubiquitous-knowledge-for-agents/4470812)

### Fabric IQ (Public Preview)

- Delivers unified views of enterprise data and AI agents using OneLake.
- Supports real-time analytics across hybrid and multicloud.
- Improves performance and governance for intelligent business solutions.
- [Fabric IQ intro](https://blog.fabric.microsoft.com/blog/from-data-platform-to-intelligence-platform-introducing-microsoft-fabric-iq)

### Agent Factory (Now Available)

- Streamlines procurement and provisioning of Microsoft AI agent services.
- Features the Microsoft Agent Pre-Purchase Plan (P3) for simplified access and spend predictability.
- Includes support and training for AI solution deployment.
- [Read more](https://techcommunity.microsoft.com/blog/azure-ai-foundry-blog/introducing-microsoft-agent-factory/4470732)

### Microsoft Foundry: Anthropic Claude Model Availability

- Anthropic Claude models now available in Microsoft Foundry.
- Expands advanced AI reasoning and workflow capabilities for research and coding scenarios.
- Maintains unified governance across Azure, Foundry, and Microsoft 365 Copilot environments.
- [Details](https://azure.microsoft.com/blog/introducing-anthropics-claude-models-in-microsoft-foundry-bringing-frontier-intelligence-to-azure/?msockid=162c0e87ae0865133121180fafe264d0)

### Resale Enabled Offers (Marketplace)

- Expands partners' sales opportunities via Microsoft Marketplace.
- Simplifies listing management and expands customer reach for ISVs and channel partners.
- [Marketplace offers guide](https://learn.microsoft.com/partner-center/marketplace-offers/resale-enabled-offers-overview?wt.mc_id=pp2e7gt31s)

### Azure HorizonDB for PostgreSQL (Private Preview)

- A PostgreSQL cloud database architected for mission-critical and AI workloads.
- Features auto-scaling, advanced vector indexing, and deep analytics/AI integration.
- Supports modernization and high-performance applications in secure, scalable environments.
- [HorizonDB details](https://azure.microsoft.com/products/horizondb)

### Microsoft Agent 365: Unified Control for AI Agents

- Brings consistent agent governance across Microsoft 365, Azure, and Foundry.
- Integrates with Defender, Entra, and Purview for security and compliance.
- Streamlines agent deployment and visibility for IT administrators and partners.
- [Learn more](https://learn.microsoft.com/microsoft-agent-365/overview)
