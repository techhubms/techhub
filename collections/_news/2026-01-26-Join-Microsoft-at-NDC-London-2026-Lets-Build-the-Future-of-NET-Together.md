# Join Microsoft at NDC London 2026 – Let’s Build the Future of .NET Together

NDC London is back, bringing together .NET and Azure developers for a packed schedule of workshops and main conference sessions from January 26–30, 2026. If you're eager to grow your expertise in modern Microsoft technologies, this event is a top opportunity.

## Meet Microsoft Engineers and Experts

You'll have a chance to attend deep technical talks delivered by top Microsoft speakers and MVPs. Topics include:

- Advanced .NET 10 development
- AI-powered development workflows (including GitHub Copilot)
- Cloud-native patterns and architecture
- Application modernization strategies
- Performance tuning and best practices

Speakers like Chris Ayers, Damian Brady, Gerald Versluis, and Steve Sanderson will share practical guidance and real-world experiences, helping you apply modern patterns in your projects.

## Hands-on Opportunities

Pre-conference workshops run January 26–27, while the main conference is January 28–30. During the event, visit the Microsoft Booth (#11) to:

- See live demos of new Microsoft developer technologies
- Get your technical questions answered by engineers building .NET and Azure
- Discuss topics like AI, modernization, and architecture 1:1
- Access resources, swag, and guidance on hot products like Aspire, Azure SRE Agent, Managed Instance on Azure App Service, and more

## Core Focus Areas

This year, expect a special emphasis on:

- AI-assisted development using GitHub Copilot and related tools
- Scaling legacy applications with .NET modernization best practices
- Hands-on demonstrations and practical use cases with Azure

## Connect with the Community

Whether you're modernizing your stack, learning about the latest in .NET and Azure, or wanting to try out new AI tools, NDC London is an opportunity to collaborate with Microsoft experts and network with other professionals shaping the software industry.

Find the Microsoft team at Booth #11 from January 28–30. Engage directly with product teams, share your challenges, and explore what's possible with the newest Microsoft technologies.
