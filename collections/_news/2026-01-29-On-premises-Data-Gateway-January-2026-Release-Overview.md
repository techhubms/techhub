# On-premises Data Gateway January 2026 Release

## Version

- 3000.302

## Improvements

- **Performance optimization for CSV reading**: Enhanced performance when reading CSV format in Copy jobs and Pipeline activities.
- **Adaptive performance tuning**: Improved read and write operations in Pipeline activities through adaptive tuning.

## Power BI Desktop Compatibility

- The gateway is updated to be fully compatible with the January 2026 version of Power BI Desktop.
- Ensures reports published to Power BI Service and refreshed via the gateway use consistent query execution logic with the latest Power BI Desktop.
