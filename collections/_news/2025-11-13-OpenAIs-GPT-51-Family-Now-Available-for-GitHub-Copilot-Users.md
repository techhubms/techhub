# OpenAI's GPT-5.1 Family Now Available for GitHub Copilot Users

OpenAI's GPT-5.1, GPT-5.1-Codex, and GPT-5.1-Codex-Mini models are now rolling out in public preview within GitHub Copilot.

## Model Availability in GitHub Copilot

The new models are available in various environments and subscription tiers:

**Supported Plans:**

- Copilot Pro
- Copilot Pro+
- Copilot Business
- Copilot Enterprise

**Supported IDEs and Platforms:**

- Visual Studio Code (chat, ask, edit, agent)
- JetBrains (ask, edit, agent)
- Xcode (ask, agent)
- Eclipse (ask, agent)
- GitHub.com
- GitHub Mobile (iOS and Android)
- Copilot CLI

**Model-Specific Requirements:**

- *GPT-5.1* is supported across all major IDEs and modes listed above.
- *GPT-5.1-Codex* and *GPT-5.1-Codex-Mini* require specific IDE versions:
  - Visual Studio Code v1.104.1+
  - JetBrains v1.5.61+
  - Xcode v0.45.0+
  - Eclipse v0.13.0+

Rollout is gradual. If you don’t have access yet, check again soon.

## How to Enable the New Models

- **Enterprise and Business:**
  - Admins must enable policies for the new models in Copilot settings.
  - Users will then select the model in the Copilot Chat model picker.
- **Pro and Pro+:**
  - Users can manually select and confirm model activation in the picker.
- **Bring Your Own Key:**
  - Choose 'Manage Models' and enter your OpenAI API key for custom access.

## Summary

The release of GPT-5.1, GPT-5.1-Codex, and GPT-5.1-Codex-Mini expands the AI capabilities of GitHub Copilot across major development environments. Users and administrators should review their Copilot plan settings and IDE requirements to enable access to these advanced models.
