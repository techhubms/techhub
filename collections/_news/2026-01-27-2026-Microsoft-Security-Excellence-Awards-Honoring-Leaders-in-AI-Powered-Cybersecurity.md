# 2026 Microsoft Security Excellence Awards: Honoring Leaders in AI-Powered Cybersecurity

**Author:** Maria Thomson

## Introduction

Security in today's digital landscape relies on collaboration. Microsoft recognized outstanding teams and individuals at the 2026 Security Excellence Awards, celebrating achievements in advancing cybersecurity with cutting-edge Microsoft tools and AI innovation.

## Event Overview

On January 26, 2026, Microsoft gathered partners, members of the [Microsoft Intelligent Security Association (MISA)](https://aka.ms/MISA), finalists, and security leaders in Redmond, Washington. The event honored those pushing boundaries with new approaches to threat intelligence, secure access, data protection, and more.

## Award Categories and Winners

### Security Trailblazer

Partners delivering innovative AI-powered solutions leveraging Microsoft Security products.

- **Winner:** Avertium
- Finalists: Avanade, Bulletproof, ExtraHop, Ontinue

### Data Security and Compliance Trailblazer

Recognizing leaders using [Microsoft Purview](https://www.microsoft.com/en-us/security/business/microsoft-purview) for data protection, compliance, and AI-driven outcomes.

- **Winner:** BlueVoyant
- Finalists: Invoke LLC, Netrix Global, Quorum Cyber, water IT Security GmbH

### Secure Access Trailblazer

Partners driving innovation with [Microsoft Entra](https://www.microsoft.com/en-us/security/business/identity-access/microsoft-entra-id) and [Microsoft Intune](https://www.microsoft.com/en-us/security/business/microsoft-Intune), advancing secure identity and endpoint management.

- **Winner:** Tata Consultancy Services
- Finalists: Cayosoft, Devicie, IBM Consulting, Inspark

### Security Changemaker

Honoring individual contributors making a significant impact within their organizations or the wider community.

- **Winner:** Anna Bordioug, Protiviti
- Finalists: Jon Kessler (Epiq), Justine Wolters (Cloud Life), Mario Espinoza (Illumio), Nithin RameGowda (Skysecure Technologies Pvt Ltd)

### Security Software Development Company of the Year

Awarded to software companies building standout AI-secured solutions deeply integrated with [Microsoft Security products](https://www.microsoft.com/en-us/security/business/solutions/generative-ai-cybersecurity).

- **Winner:** Illumio
- Finalists: ContraForce, Darktrace, inforcer, Tanium

### Security Services Partner of the Year

Partners excelling in integrating Microsoft security products with service offerings, AI-driven innovation, and customer support.

- **Winner:** Invoke LLC
- Finalists: BlueVoyant, Cloud4C, Shanghai Flyingnets, Quorum Cyber

## Recognized Themes

- **AI and Microsoft Security:** Many winners implemented AI-powered security as a central theme, leveraging the latest available features across Microsoft's security suite.
- **Zero Trust Strategies:** Recognition for advancing Zero Trust approaches to strengthen identity and endpoint security.
- **Community Collaboration:** Emphasis on teamwork, partnership, and knowledge sharing in the cybersecurity field.
