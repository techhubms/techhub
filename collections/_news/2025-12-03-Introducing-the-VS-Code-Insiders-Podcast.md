# Introducing the VS Code Insiders Podcast

The VS Code Insiders Podcast is the official podcast from the Visual Studio Code team, offering listeners a unique, behind-the-scenes view of the most popular code editor in the world.

## About the Podcast

Hosted by James Montemagno and other members of the VS Code team, this podcast features conversations with the people shaping VS Code's future—including core developers, product managers, and open source contributors. Listeners can expect discussions about:

- The design decisions influencing new features
- The reasoning behind experimental tooling and extensions
- Accessibility engineering for diverse developer needs
- Open source community building and governance
- The integration of AI and agents into core developer workflows

## Recent Episode Highlights

- **Designing for Everyone: Accessibility in VS Code** (with Megan Rogge) – Insights on implementing accessibility features such as screen reader support and high-contrast themes.
- **Behind the Scenes of VS Code's Planning Agent** (with Bhavya U) – Explores the Planning Agent feature and how AI helps plan and orchestrate complex coding tasks inside VS Code.
- **Building & Scaling Open Source Communities** (with Steve Francia) – Guidance on governance, contributor experience, and nurturing sustainable open source communities.
- **Beyond the Keynote: VS Code at GitHub Universe** (with Pierce Boggan) – Analyzes how VS Code operates as an AI-native tool, and key takeaways from GitHub Universe 2025.

## Why Listen?

Whether you are an experienced developer, a VS Code power user, or new to coding, the show provides valuable insights into the ongoing evolution of developer tooling. Topics extend from accessible code environments to integrating next-gen AI features, making it relevant for anyone working in software engineering today.

## How to Subscribe

- Visit the [VS Code Insiders Podcast homepage](https://www.vscodepodcast.com) to listen to past episodes and subscribe.
- Download [VS Code Insiders](https://code.visualstudio.com/insiders) for early access to upcoming features.
- Stay updated with the [Insiders release notes](https://code.visualstudio.com/updates) for the latest improvements.
