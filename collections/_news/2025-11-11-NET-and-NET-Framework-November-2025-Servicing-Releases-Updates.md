# .NET and .NET Framework November 2025 Servicing Releases Updates

**Authors**: Tara Overfield, Victor Israel-Bolarinwa

## Overview

This update summarizes the latest servicing releases for both .NET and .NET Framework in November 2025. The primary focus is on maintenance and non-security fixes across versions .NET 8.0 and .NET 9.0.

## Highlights

- **Security Improvements**: While the November update references security improvements in general, this month specifically delivers non-security fixes.
- **Release Timeline**: Updates were released as of November 11, 2025.
- **Applies to**: .NET 8.0 (version 8.0.22) and .NET 9.0 (version 9.0.11).

## .NET Framework November 2025 Updates

- No new security or non-security updates are available for .NET Framework this month.
- Recent updates and details can be found in the [release notes for .NET Framework](https://learn.microsoft.com/dotnet/framework/release-notes/release-notes).

## Action Items for Developers

- Review the linked changelogs to stay informed of fixes or changes that might impact your workloads.
- Download installers or refresh images/SDKs as needed to keep your environments current.
- Monitor known issues for possible current limitations.
