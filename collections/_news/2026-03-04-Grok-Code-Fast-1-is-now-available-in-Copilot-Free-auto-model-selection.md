# Grok Code Fast 1 is now available in Copilot Free auto model selection

Grok Code Fast 1 is now generally available to users with Copilot Free plans through Copilot's auto model selection feature.

This update means that Grok Code Fast 1 is one of the possible models Copilot can automatically use when making coding suggestions. The change affects several integrated development environments, including Visual Studio Code, Visual Studio, JetBrains IDEs, Xcode, and Eclipse.

If you'd like to learn more about Copilot's supported models, visit the [GitHub Copilot models documentation](https://docs.github.com/copilot/reference/ai-models/supported-models?utm_source=model-release-announcement&utm_medium=changelog&utm_campaign=newmodels).

Developers are invited to share their feedback and discuss Copilot features in the [GitHub Community Copilot Conversations](https://github.com/orgs/community/discussions/categories/copilot-conversations?utm_source=model-release-announcement&utm_medium=changelog&utm_campaign=newmodels).
