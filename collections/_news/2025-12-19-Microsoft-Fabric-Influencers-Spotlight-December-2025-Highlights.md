# Microsoft Fabric Influencers Spotlight: December 2025 Highlights

The December 2025 edition of the Fabric Influencers Spotlight brings together high-impact contributions from Microsoft MVPs and Fabric Super Users. This collection covers a broad range of practical, technical topics around Microsoft Fabric, including real-world solutions for data engineering, analytics, DevOps, administration, and more.

## Featured Areas

- **Data Warehousing**: Practical demos on Microsoft Fabric Warehouse, exploring SQL capabilities, Delta tables in OneLake, ingestion, cross-database queries, time travel, and zero copy cloning.
  - [Aleksi Partanen: Understanding Microsoft Fabric Warehouse (YouTube)](https://youtu.be/DKRfvGz0v54)
  - [Kevin Chant: The F-Files Podcast – Historical Data Tracking (YouTube)](https://www.youtube.com/watch?v=C6tA5nyU4FU)

- **Power BI & Context Engineering**: Innovations in combining Power BI MCP server with large language models and practical blog posts on advanced DAX/SQL for engagement analysis.
  - [Rishi Sapra: Power BI MCP Server & LLM Agent Mode (Blog)](https://msfabrictraining.com/power-bi-mcp-server-how-context-engineering-changes-everything/)
  - [Parul Sagar: Sequence Logic for Engagement Metrics (Blog)](https://community.fabric.microsoft.com/t5/Power-BI-Community-Blog/When-Semi-Additive-Metrics-Fall-Short-Why-You-Need-Sequence/ba-p/4894989)

- **Data Engineering**: Techniques for modeling data with Fabric Graph in one-lake, tracking permissions, and downstream data flows, plus deep dives on semantic models and entity architecture in Fabric IQ.
  - [Jake Duddy: Fabric Graphs (Blog)](https://evaluationcontext.github.io/posts/FabricGraph/)
  - [Matthias Falland: Fabric IQ Ontology Workshop (YouTube)](https://youtu.be/VgJvDastcb0)

- **Data Science & AI**: Tutorials on leveraging new features for Microsoft Fabric Data Agents and detailed guides on autoencoder-based dimensionality reduction for wide datasets.
  - [Gaston Cruz: Fabric Data Agents Creator Features (YouTube)](https://youtu.be/asAG2u1xh3A)
  - [Sahir Maharaj: Autoencoders in Microsoft Fabric (Blog)](https://community.fabric.microsoft.com/t5/Data-Science-Community-Blog/Autoencoder-Dimensionality-Reduction-for-Data-Science-in/ba-p/4899083)

- **Administration & DevOps**: Hands-on workflows for automating Lakehouse and Warehouse deployments using Azure DevOps pipelines, plus CI/CD strategies for Fabric SQL databases.
  - [Srisakthi Kannan: Lakehouse/Warehouse Deployment with Azure DevOps (Blog)](https://community.fabric.microsoft.com/t5/Fabric-platform-Community-Blog/Lakehouse-and-Warehouse-Deployment-with-Azure-DevOps-A-Practical/ba-p/4876100)
  - [Kevin Chant: CI/CD for SQL Databases in Fabric (YouTube)](https://www.youtube.com/watch?v=LJzNNR8nFlI)

- **Real-Time Analytics**: Step-by-step guidance to building real-time dashboards using Eventstream, Eventhouse, and visualization in Microsoft Fabric.
  - [Inturi Suparna Babu: Real-Time Weather Dashboard (Blog)](https://community.fabric.microsoft.com/t5/Real-Time-Intelligence-Community/Building-a-Real-Time-Weather-Dashboard-in-Microsoft-Fabric-Step/ba-p/4897230)

- **Learning & Events**: Updates from Microsoft Ignite 2025, featuring insights on Power Platform, Copilot Studio, Azure, and the evolution of business applications through AI.
  - [Gaston Cruz & Alex Rostan: Microsoft Ignite 2025 Updates (YouTube)](https://youtu.be/vERYvSJeIVs?si=ykKdzX7V5MKd1u1J)
