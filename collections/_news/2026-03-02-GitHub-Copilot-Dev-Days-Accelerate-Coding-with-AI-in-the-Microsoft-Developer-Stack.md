# GitHub Copilot Dev Days: Accelerate Coding with AI in the Microsoft Developer Stack

Modern software development is evolving rapidly, with AI now being a core part of how Microsoft developers build and deliver applications. To help developers maximize productivity without compromising quality, **GitHub Copilot Dev Days** offer a series of global, in-person, hands-on events.

## About GitHub Copilot Dev Days

GitHub Copilot Dev Days events are designed for developers at all levels who use Microsoft technologies. They focus on practical, applied learning, enabling attendees to:

- Experience live demonstrations of GitHub Copilot working in **Visual Studio** and **VS Code**
- Participate in hands-on workshops using Copilot alongside **.NET, Java, Python, TypeScript**, and more
- Explore real-world development scenarios, from writing code to deploying cloud-native applications on **Azure**
- Learn about the **GitHub Copilot CLI**, cloud agent, and integration in a variety of editor environments

## Who Should Attend?

These events are ideal for:

- Developers using Visual Studio and VS Code across multiple languages
- Those working with the command-line interface (CLI) and looking to integrate AI assistance into their workflows
- Professionals, students, and community members aiming to modernize their approach with AI-assisted development tools

## What Will You Learn?

Attendees will benefit from:

- **Live Demos:** See GitHub Copilot in action within Visual Studio and VS Code, accelerating development tasks
- **Workshops:** Engage in guided coding exercises using Copilot for hands-on experience
- **Real-World Scenarios:** Practical examples and advanced use cases, including integration into enterprise codebases and team-based workflows
- **AI Pair Programming:** Learn how to apply AI-powered coding patterns for increased productivity

Sample event agenda includes:

- **Introductory Session:** Overview of GitHub Copilot’s role in the Microsoft developer toolchain
- **Community Session:** Local experts share their Copilot experiences
- **Hands-on Workshop:** Guided practical exercises with AI-assisted development

## Community and Networking

Events are hosted by a range of community leaders, including Microsoft MVPs, GitHub Stars, Microsoft Student Ambassadors, and others. Attendees will connect with local peers, share experiences, and take away actionable knowledge. Enjoy networking opportunities—including snacks and swag.
