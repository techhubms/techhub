# Fabric Runtime 2.0 (Preview)

Fabric Runtime provides integration within the Microsoft Fabric ecosystem, providing an environment for data engineering and data science projects built on Apache Spark.

Fabric Runtime 2.0 is described as a next-generation runtime for large-scale data computations in Microsoft Fabric, aimed at enabling scalable analytics and advanced workloads.

## Switching to Runtime 2.0 (Preview)

You can switch to Runtime 2.0 Preview in two places:

- **Workspace settings level**
- **Environment level**

## Components and versions in Fabric Runtime 2.0

Fabric Runtime 2.0 includes the following components and upgrades:

- **Apache Spark**: 4.0
- **Operating System**: Azure Linux 3.0 (Mariner 3.0)
- **Java**: 21
- **Scala**: 2.13
- **Python**: 3.12
- **Delta Lake**: 4.0

## What the preview enables

The preview stage provides:

- Access to new features and APIs from **Spark 4.0** and **Delta Lake 4.0**
- Early usage of the latest Spark and Delta-based enhancements
- A transition path for changes related to newer **Java**, **Scala**, and **Python** versions

## Upcoming upgrades mentioned

Based on customer feedback, additional component upgrades are stated as targeted for release in the next few weeks:

- **Apache Spark**: 4.1
- **Delta Lake**: 4.1
- **Python**: 3.13
