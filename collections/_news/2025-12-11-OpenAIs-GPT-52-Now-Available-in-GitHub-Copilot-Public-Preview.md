# OpenAI’s GPT-5.2 Now Available in GitHub Copilot Public Preview

**Author:** Allison

OpenAI’s latest model, GPT-5.2, focused on long context understanding and front-end UI generation, is now in public preview for users of GitHub Copilot Pro, Pro+, Business, and Enterprise plans.

## Availability in GitHub Copilot

You can now choose GPT-5.2 within GitHub Copilot across several platforms:

- **Visual Studio Code** (chat, ask, edit, agent modes)
- **Copilot Chat on GitHub.com**
- **GitHub Mobile** (iOS and Android)
- **Copilot CLI**

The rollout is gradual. If you don't see GPT-5.2 yet, check again soon.

## Enabling Access

- **Enterprise and Business Plans:**
  - Admins must enable GPT-5.2 in Copilot settings.
  - Once enabled, all organization users can select GPT-5.2 via the model picker.
- **Pro and Pro+ Plans:**
  - Users can select GPT-5.2 directly in the model picker after confirming a one-time prompt.
- **Bring Your Own Key (BYOK):**
  - In Visual Studio Code, go to **Manage Models** in the model picker.
  - Choose **OpenAI GPT-5.2** and follow prompts to enter your OpenAI API key.

## Learn More
