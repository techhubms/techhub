## A smoother navigation experience in GitHub Mobile for Android

GitHub has revamped navigation in **GitHub Mobile for Android** to create a more consistent and intuitive experience across the app.

## What changed

- Moving between key areas like **Home** and **Inbox** should feel smoother.
- The **bottom navigation** is now more consistently available where it matters.
- Each tab does a better job of **preserving your place** as you explore.

## Why it matters

- Easier to switch contexts and pick up where you left off.
- Faster navigation back to important destinations.
- A more connected, predictable, and efficient overall app experience.

## Availability

This update is available in the latest production build on Google Play:

- GitHub Mobile on Android: https://play.google.com/store/apps/details?id=com.github.android
