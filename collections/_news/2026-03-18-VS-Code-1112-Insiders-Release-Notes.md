# VS Code 1.112 Insiders Release Notes

*Last updated: March 11, 2026*

Explore the latest features, enhancements, and fixes introduced in the Insiders build of Visual Studio Code 1.112. These updates are primarily targeted at developers seeking to leverage Copilot integrations, improved agent management, and productivity enhancements within the VS Code environment.

## Key Highlights

### Copilot CLI and Agent Enhancements

- Add default location for hook configurations in `~/.copilot/hooks` for easier management (Mar 11).
- Support added for steering, queuing messages, and permission levels for Copilot CLI agents, enabling improved command and control (Mar 10).

### AI and Chat Features

- VS Code chat now supports direct analysis of stored images, broadening its AI-driven capabilities.
- Visual distinction for reserved context in chat, improving usability and information tracking.
- Claude agent mode is now generally available (no longer in preview).

### Editor and Workflow Improvements

- Minimap context menu now offers a Side option for positioning.
- Integrated browser gets right-click menu with standard actions and ability to add elements to chat.
- Editor dialog can be resized and moved within the workbench.
- New setting `editor.find.closeOnResult` optionally closes the Find dialog upon a successful search.

### Agent and Plugin Controls

- Agent plugins and MCP servers can now be toggled at both workspace and global levels, giving finer control over developer environments.

### Additional Improvements

- Update notification is now accessible from more UI areas, even if the Status Bar is hidden.
- Continued focus on incremental usability through the commit log and closed issues.

For the complete changelog and direct downloads, visit the [official release notes](https://code.visualstudio.com/updates/v1_112).
