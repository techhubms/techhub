# Fabric Influencers Spotlight: November 2025

Welcome to the November edition of Microsoft Fabric Influencers Spotlight, where the Fabric Community team highlights insightful work by MVPs and Super Users working with all aspects of Microsoft Fabric. This month’s feature includes technical blog posts, instructional videos, and presentations on power topics like platform architecture, data engineering, data science, warehousing, and administration.

## Power BI Developments

- **Wyn Hopkins (MVP)**: [Simpler DAX with New Calendar Functionality – Power BI Preview](https://youtu.be/BDL6nLPKFI4?si=crU2iS2rUI-mjI63)
  - YouTube guide helps implement new Calendar Options in Power BI with examples and tips for developers working with DAX and date dimensions.

- **Parul Sagar (Super User)**: [Dynamic Change Tracking for LMS Enrolments Using Power Query M](https://community.fabric.microsoft.com/t5/Power-BI-Community-Blog/Dynamic-Change-Tracking-for-LMS-Enrolments-Using-Power-Query-M/ba-p/4854637)
  - Tutorial demonstrates building a snapshot change tracking solution using Power Query (M language), capturing added and deleted records for analytics.

## Data Engineering Insights

- **Thomas 'Tom' Martens (MVP)**: [Entra ID Group Expansion – The Complete Guide](https://minceddata.net/Getting+social/Blog/Monitoring+Fabric/Entra+ID+Group+Expansion+-+the+complete+guide)
  - Explains techniques to expand nested Entra ID groups to individual users, with practical strategies for Row Level Security and answering permission queries. Includes handling of Graph API request limitations and automation tips.

- **Aleksi Partanen (MVP)**: [Understanding Delta Tables and Lakehouses in Microsoft Fabric](https://youtu.be/eZD89I6xLXg)
  - Beginner-friendly video describing Lakehouse and Delta Table foundations in Fabric. Covers partitioning, performance optimization, and virtualization of external sources including ADLS, S3, and Dataverse.

## Data Science & Advanced Pipelines

- **Sahir Maharaj (MVP)**: [Advanced Pipelines & Transformers for Data Science in Microsoft Fabric](https://community.fabric.microsoft.com/t5/Data-Science-Community-Blog/Advanced-Pipelines-amp-Transformers-for-Data-Science-in/ba-p/4812307)
  - Explores how advanced pipelines enable consistent project management and introduces step-by-step custom transformer development in Microsoft Fabric, bringing domain-specific logic to data workflows.

## Fabric Platform, Administration, and Governance

- **Andy Cutler (MVP)**: [Fabric Architecture: Azure Tenants](https://www.serverlesssql.com/fabric-architecture-azure-tenants/)
  - A technical look at the role of Azure Tenants in Fabric Capacity planning, workspace strategy, multi-tenancy, lakehouse and warehouse creation, and domain topology.

- **Rajendra Ongole (MVP)**: [GraphQL Explained: Transforming Data Access in Microsoft Fabric](https://community.fabric.microsoft.com/t5/Fabric-platform-Community-Blog/GraphQL-Explained-Transforming-Data-Access-in-Microsoft-Fabric/bc-p/4869106#M99)
  - Discusses using GraphQL as a modern, flexible API layer inside Fabric to make data integration and efficient data access easier for analytics developers and architects.

## Database Integration

- **Inturi Suparna Babu (Super User)**: [Google BigQuery Mirroring in Microsoft Fabric: A Step-by-Step Guide](https://community.fabric.microsoft.com/t5/Databases-Community-Blog/Google-BigQuery-Mirroring-in-Microsoft-Fabric-A-Step-by-Step/ba-p/4871175)
  - Walks through mirroring Google BigQuery tables into Microsoft Fabric using preview features that enable cross-cloud data operations without building complex ETL pipelines.

## Community & Learning Programs

- **Fabric Data Days**: Ongoing event offering Power BI upskilling sessions, community challenges, exam discounts, and collaboration opportunities. Learn more: [Fabric Data Days](https://aka.ms/fdd/schedule)
