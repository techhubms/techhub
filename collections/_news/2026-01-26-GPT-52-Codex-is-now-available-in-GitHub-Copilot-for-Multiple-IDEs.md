# GPT-5.2-Codex is Now Available in GitHub Copilot for Multiple IDEs

The GPT-5.2-Codex model is now generally available for Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+ subscribers.

## Accessing GPT-5.2-Codex in GitHub Copilot

You can now interact with the GPT-5.2-Codex model in GitHub Copilot Chat on the following platforms:

- **github.com**
- **GitHub Mobile**
- **Visual Studio Code**
- **Visual Studio**
- **JetBrains IDEs**
- **Xcode**
- **Eclipse**

The model can be selected through the chat model picker, supporting agent, ask, and edit modes. If the model isn't visible to you yet, access may be rolling out—check back periodically.

### Supported IDE Versions

- **Visual Studio:** 17.14.19 and later; 18.0.0 and later (all modes: ask, agent)
- **GitHub Copilot for JetBrains:** 1.5.61 and later (all modes: ask, edit, agent)
- **GitHub Copilot for Xcode:** 0.45.0 and later (all modes: ask, agent)
- **GitHub Copilot for Eclipse:** 0.13.0 and later (all modes: ask, agent)

## How to Enable GPT-5.2-Codex for Organizations

Administrators for Copilot Business and Copilot Enterprise must enable GPT-5.2-Codex by updating Copilot settings. Once opted in, users will see this model in their available options.
