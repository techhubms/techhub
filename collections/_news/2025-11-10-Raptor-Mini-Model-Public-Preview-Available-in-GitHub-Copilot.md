# Raptor Mini Model Public Preview for GitHub Copilot

**Author:** Allison

Raptor mini, a new experimental AI model, is now gradually rolling out to GitHub Copilot users on Pro, Pro+, and Free plans within Visual Studio Code.

## Key Points

- **Model Availability:**
  - Raptor mini can be selected using the Copilot Chat model picker in Visual Studio Code.
  - Accessible in chat, ask, edit, and agent modes.
  - The rollout is gradual; if not visible yet, users should check back soon.

- **How to Access:**
  1. Open Visual Studio Code with GitHub Copilot enabled.
  2. Use the model picker in Copilot Chat to select Raptor mini.
  3. Use the model in various Copilot interaction modes (chat, ask, edit, agent).

- **Further Information:**
  - Review the [official Copilot documentation](https://docs.github.com/copilot/reference/ai-models/supported-models) to explore all available AI models.
  - Users are encouraged to share feedback and experiences via the [community discussion forum](https://github.com/orgs/community/discussions/categories/copilot-conversations).

## Release Notes

- **Gradual Rollout**: Some users may receive access before others depending on gradual rollout progression.
- **Plans Supported**: Raptor mini is available to Pro, Pro+, and Free plan holders.
- **Supported Features**: Works with Copilot Chat across different operational modes within VS Code.

## Additional Resources

- [Raptor mini documentation](https://docs.github.com/copilot/reference/ai-models/supported-models)
- [GitHub Copilot community discussions](https://github.com/orgs/community/discussions/categories/copilot-conversations)

## Summary

Raptor mini is an experimental addition to GitHub Copilot's suite of AI models, providing users with more options for code completion and AI-assisted development directly within Visual Studio Code.
