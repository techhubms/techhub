# Explore a Repository Using GitHub Copilot on the Web

GitHub Copilot on the web now features an enhanced file exploration experience for developers using chat.

## What's Changed

- **File Tree Browsing:** You can now ask Copilot to show a specific file, browse the entire file tree, and view how files relate within the broader codebase.
- **Contextual Navigation:** Seamlessly jump between files and maintain context in Copilot chat, making it easier to ask codebase-specific questions.

## Adding References to Chat

- **Temporary References:** Selecting files in the explorer automatically adds them as temporary references in your current Copilot chat session.
- **Permanent References:** You can make a reference permanent by clicking the reference token or double-clicking the file preview tab, enriching the chat's context further.

## How to Use

- This feature is currently available in public preview for all GitHub Copilot users on the web.
- Try it now to streamline your workflow and make Copilot chat more code-aware.
