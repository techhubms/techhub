# GPT-5.2-Codex Now Available in GitHub Copilot

**Author:** Allison

GitHub has announced the general availability of the OpenAI GPT-5.2-Codex model in Copilot Enterprise, Copilot Business, Copilot Pro, and Copilot Pro+ plans. This update enables users to select GPT-5.2-Codex as their preferred model in various environments and platforms.

## Supported Platforms

- **Visual Studio Code:** All modes (chat, ask, edit, agent)
- **Copilot Chat on GitHub.com**
- **GitHub Mobile:** iOS and Android apps
- **Copilot Coding Agent:** Available to Copilot Pro and Pro+ subscribers
- **Copilot CLI**

## Rollout Details

The rollout of GPT-5.2-Codex in GitHub Copilot will be gradual. Users who don't see the new model immediately are encouraged to check back as it becomes available in their environment.

## Enabling Access

### For Enterprise and Business Plans

- Administrators must enable the GPT-5.2-Codex model in the Copilot organizational settings.
- Once enabled, users within the organization can select GPT-5.2-Codex from the Copilot model picker.

### For Pro and Pro+ Plans

- Individual users can enable GPT-5.2-Codex by selecting it in the model picker in their Copilot interface.
- Users must confirm a one-time prompt to finalize model selection.

### Bring Your Own Key

- In Visual Studio Code, select **Manage Models** from the model picker.
- Choose **OpenAI GPT-5.2-Codex** and enter your OpenAI API key as prompted.
