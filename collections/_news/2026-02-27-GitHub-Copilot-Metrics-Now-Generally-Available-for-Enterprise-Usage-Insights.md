# GitHub Copilot Metrics Now Generally Available for Enterprise Usage Insights

Allison announces the general availability of GitHub Copilot usage metrics, providing enterprises with a powerful suite of analytics and reporting tools. These new features enable organizations to monitor Copilot adoption, analyze usage patterns, and build custom reports for better decision-making.

## Key Features

### Dashboards

- **Copilot Usage Dashboard:** Offers clear insights on how teams utilize GitHub Copilot, including code completion activity, IDE usage, and model/language breakdowns.
- **Code Generation Dashboard:** Quantifies Copilot's code output by measuring lines suggested, added, or deleted across completions, chat, and agent features.
- **Enterprise & Organization-Level Granularity:** Allows leaders to track team-specific trends beyond simple enterprise-wide aggregates.

### API Access

- **Enterprise-Level APIs:** Aggregate data across the entire organization to analyze Copilot engagement, usage rates, pull request throughput, and more.
- **Organization-Level APIs:** Enable programmatic access for organizations to monitor how specific teams are adopting Copilot, spot usage trends, and identify enablement needs.
- **User-Level APIs:** Provide daily insights into individual Copilot usage, aiding in targeted training and support initiatives.

For details, see the [API documentation](https://docs.github.com/enterprise-cloud@latest/rest/copilot/copilot-usage-metrics?apiVersion=2022-11-28#get-copilot-organization-usage-metrics-for-a-specific-day).

## Access and Permissions

- Access Copilot metrics through your enterprise account under **AI Controls** → **Copilot** → **Metrics**.
- Find dashboards under **Insights** → **Copilot usage** or **Code generation**.
- Permissions required include Enterprise Administrator, Billing Manager, or custom roles with the `View enterprise Copilot metrics` permission.
- Includes **fine-grained access controls** and support for GitHub Enterprise Cloud with exacting data residency requirements.

## What's Next

This release lays a foundation for connecting Copilot usage data with engineering outcomes, helping organizations transition from adoption tracking to measuring real development impact. Future updates will further link Copilot patterns to tangible productivity metrics.

GitHub Copilot usage metrics empower organizations to understand and optimize their AI-driven development efforts, with dashboards and APIs giving leaders precise visibility into Copilot adoption at every organizational level.
