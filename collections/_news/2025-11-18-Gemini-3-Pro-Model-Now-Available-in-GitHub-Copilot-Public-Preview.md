# Gemini 3 Pro Is in Public Preview for GitHub Copilot

GitHub Copilot, the developer coding assistant, is gradually rolling out support for **Gemini 3 Pro**, Google’s latest AI frontier model. The model will be available for subscribers at multiple plan levels: Pro, Pro+ Business, and Enterprise.

## How to Access Gemini 3 Pro

Developers will be able to select Gemini 3 Pro using the model picker from several Copilot interfaces:

- Visual Studio Code (Agent, Plan, Ask, and Edit modes)
- GitHub.com
- GitHub Mobile (iOS and Android)
- GitHub CLI

## Enabling Gemini 3 Pro

**Enterprise and Business Plans:**

- Administrators must enable the Gemini 3 Pro policy in Copilot settings. Once activated, users in the organization can select Gemini 3 Pro in supported environments.

**Pro and Pro+ Plans:**

- Individuals can enable Gemini 3 Pro by choosing the model in the Visual Studio Code picker and confirming the one-time prompt.

**Bring Your Own Key:**

- Users can select 'Manage Models' from the picker, choose Gemini 3 Pro, and enter their API key when prompted.
