# On-premises Data Gateway November 2025 Release (Version 3000.294)

The November 2025 update for the on-premises data gateway brings it in line with the latest version of Power BI Desktop. This ensures reports you publish and refresh via the gateway in the Power BI Service use the same query execution logic as the November Power BI Desktop release.

## Key Highlights

- **Version:** 3000.294
- **Compatibility:** Fully compatible with the November 2025 Power BI Desktop edition
- **Use Case:** Ensures consistency and reliability for report publishing and data refreshes
