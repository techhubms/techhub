# Developer-Focused Azure AI Foundry Sessions at Microsoft Ignite 2025

Microsoft Ignite 2025 offers an exceptional lineup of sessions and labs for developers eager to advance their expertise in Azure AI Foundry. This guide curates the key sessions and hands-on opportunities, ensuring you make the most of your time, whether attending in person or virtually.

## Must-See Innovation Session

- [**Your AI Apps and Agent Factory**](https://ignite.microsoft.com/en-US/sessions/BRK1706?source=sessions) (BRK1706)  
  Dive into agentic AI and the Azure AI Foundry platform for building and managing intelligent agents. Gain insights on workflow tools, orchestration, integration with VS Code and GitHub, and approaches to security and observability. Presented by product leaders including Yina Arenas, Sarah Bird, and Amanda Silver, this is a foundational session for learning about Azure AI Foundry's capabilities.

## Highlighted Breakout Sessions

- [**AI agents in Azure AI Foundry, ship fast, scale fearlessly**](https://ignite.microsoft.com/en-US/sessions/BRK189?source=sessions) (BRK189): Explore building with Azure AI Foundry's enterprise-ready Agent Factory.
- [**AI builder’s guide to agent development in Foundry Agent Service**](https://ignite.microsoft.com/en-US/sessions/BRK201) (BRK201): Practical developer guidance on workflows and best practices for AI agent development.
- [**AI-Powered Automation & Multi-Agent Orchestration in Azure AI Foundry**](https://ignite.microsoft.com/en-US/sessions/BRK197?source=sessions) (BRK197): Learn about orchestrating multiple agents for automation and complex workflows.
- [**Building Responsible AI Agents with Azure AI Foundry**](https://ignite.microsoft.com/en-US/sessions/BRK202?source=sessions) (BRK202): Focus on responsible AI frameworks and tools within the platform.
- [**AI tools for agents in Azure AI Foundry: Plug, play and reach**](https://ignite.microsoft.com/en-US/sessions/BRK192?source=sessions) (BRK192): Introduction of new production-ready tools for agentic AI apps, available via API or MCP servers.
- [**AI fine-tuning in Azure AI Foundry to make your agents unstoppable**](https://ignite.microsoft.com/en-US/sessions/BRK188?source=sessions) (BRK188): Techniques for customizing and fine-tuning agents using Azure OpenAI and open-source models.
- [**Building autonomous enterprise agents with reasoning**](https://ignite.microsoft.com/en-US/sessions/BRK203?source=sessions) (BRK203): Learn about deploying reasoning-powered autonomous agents for enterprise use cases.

## In-Person Activations and Labs

Attendees in San Francisco can interact with experts at Microsoft Showcase expo booths featuring real-world Azure AI Foundry innovations. Don't miss hands-on labs for practical skill building:

- [Prototyping multimodal agents with Azure AI Foundry and the AI Toolkit](https://ignite.microsoft.com/en-US/sessions/LAB512?source=sessions) (LAB512)
- [Build Agentic Knowledge Bases: Next-Level RAG with Azure AI Search](https://ignite.microsoft.com/en-US/sessions/LAB511?source=sessions) (LAB511)
- [Safeguard your agents with AI Red Teaming Agent in Azure AI Foundry](https://ignite.microsoft.com/en-US/sessions/LAB516?source=sessions) (LAB516)

Seats are limited, and an RSVP is required for onsite labs.
