# Hands-On Lab: The Power of GitHub Copilot in Visual Studio Code

Get practical with the advanced capabilities of GitHub Copilot integrated in Visual Studio Code. This lab goes beyond the basics, introducing agentic development through VS Code's agent mode, MCP servers, and custom agents, empowering you to build and integrate features rapidly into your applications.

## What You'll Learn

- **Agentic Development:** Explore how agent mode in VS Code, along with MCP servers and custom agents, automates coding tasks.
- **Live Feature Integration:** Build and integrate new app features in real time, observing the impact of changes immediately.
- **Customization:** Learn to fine-tune Copilot's AI-generated results to fit your team's coding standards and best practices.
- **Asynchronous Agents:** Discover how to run agents outside of VS Code, such as with Copilot CLI or Copilot Cloud Agent, supporting background and asynchronous workflows in the cloud.
