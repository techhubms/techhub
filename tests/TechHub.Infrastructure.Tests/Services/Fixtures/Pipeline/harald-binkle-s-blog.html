GitHub Copilot Dev Days | Trier

Posted on:March 19, 2026 |  at 07:50 PM

AI verändert die Softwareentwicklung und GitHub Copilot ist für viele Entwicklerinnen und Entwickler bereits ein wichtiges Tool im Alltag. Ich freue mich sehr, dass ich gemeinsam mit JAM Software GmbH die GitHub Copilot Dev Days in Trier am 14. April 2026 organisiere und hoste.

GitHub Copilot Dev Days in Trier

Dieses Community Event richtet sich an Entwicklerinnen und Entwickler, die AI Assistant Coding mit GitHub Copilot praxisnah erleben möchten. Egal ob du gerade erst startest oder Copilot bereits täglich nutzt, hier bekommst du konkrete Impulse für deinen Entwicklungsalltag.

Eventdetails

📅 Datum: Dienstag, 14. April 2026
⏰ Uhrzeit: 16:30 – 20:00 CET
📍 Ort: JAM Software GmbH, Am Wissenschaftspark 26, 54296 Trier

Agenda laut Luma Anmeldung

Hier sind die Sessiontitel in der Originalfassung von der Luma Registrierungsseite:

  • 16:30 – Check-in & Networking
  • 17:00 – Begrüßung/Eröffnung & Keynote (Harald Binkle, Xebia)
  • 17:30 – Session 1: Überblick über GitHub Copilot (Roland König, Freier Software Engineer)
  • 15 Min. Break
  • 18:30 – Session 2: “Out of Context” - Erfahrungen zu Coding mit AI (Marcel Dütscher, JAM-Software)
  • 19:15 – Session 3: GitHub Copilot effektiv nutzen (Nico Orschel, Xebia)
  • 20:00 – Abschluss & Ausklang

Das nimmst du mit

In diesem Event erwarten dich:

  • Ein klarer Überblick über Möglichkeiten und Grenzen von GitHub Copilot
  • Echte Erfahrungswerte aus dem Entwicklungsalltag mit AI Unterstützung
  • Konkrete Tipps für den produktiven Einsatz in Projekten
  • Austausch mit der Community vor Ort in Trier
  • Direkte Fragen und Antworten mit den Speakerinnen und Speakern

Veranstaltungsort

JAM Software hostet das Event in den eigenen Räumlichkeiten auf dem Petrisberg in Trier. Als Hersteller von TreeSize und weiteren etablierten Softwarelösungen steht JAM Software seit vielen Jahren für starke Produkte und technische Qualität.

Unterstützt von Xebia

Dieses Event wird von Xebia unterstützt. Nico Orschel und ich arbeiten bei Xebia und teilen in diesem Rahmen unsere Erfahrungen rund um moderne Entwicklungspraktiken mit GitHub Copilot.

Speaker

Harald Binkle (Keynote und Host)

Als Principal Consultant bei Xebia und Microsoft MVP unterstütze ich Teams bei der Einführung moderner Entwicklungspraktiken und Tools wie GitHub Copilot.

Roland König

Roland ist freier Software Engineer mit starker .NET Expertise und langjähriger Praxiserfahrung in moderner Softwareentwicklung.

Marcel Dütscher

Marcel ist Entwickler bei JAM Software und bringt direkte Projekterfahrung beim Einsatz von AI Coding Tools in realen Entwicklungsteams mit.

Nico Orschel

Nico ist Consultant bei Xebia mit Fokus auf Developer Productivity, Tooling und effektive Workflows im Alltag.
