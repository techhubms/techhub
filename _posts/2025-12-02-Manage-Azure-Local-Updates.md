# Manage Azure Local Updates

Keeping Azure Local instances up to date is vital to maintain security, performance, and leverage new Azure innovations. ## Key Topics Covered

### 1. Updating Azure Local Instances from the Azure Portal

- Initiate and monitor updates through the familiar Azure Portal interface.
- Apply the latest features, security patches, and performance improvements without leaving the portal.

### 2. Solution Builder Extension Updates

- Manage and monitor Solution Builder Extensions designed to extend Azure Local functionality.
- Understand how these updates bring new capabilities and ensure ongoing optimization and security.

### 3. Best Practices for Update Management

- Schedule regular updates to minimize downtime and disruption.
- Use built-in monitoring tools to track update status and health.
- Implement version control and review update logs for compliance and troubleshooting.

## Practical Guidance

- **Unified Management:** Leverage a single pane of glass for Azure Local and on-premises resources.
- **Automation:** Set up automated notifications or policies for update management.
- **Security & Compliance:** Ensuring updates are applied promptly protects against vulnerabilities and maintains compliance for regulated workloads.
